# Связь мозга и тела: EEG ↔ Рост/Вес

## Гипотеза

**Синхронизация мозга (S_brain) коррелирует с синхронизацией тела (S_body).**

---

## 1. Формальная модель

### 1.1. Синхронизация мозга

**EEG:**
```
S_brain = 1 - 0.05 · σ(Δ_brain)
```

**Где:**
```
Δ_brain = -ln(f_brain)
f_brain = средняя частота мозговых волн
```

### 1.2. Синхронизация тела

**Тело:**
```
S_body = 1 - 0.05 · σ(Δ_body)
```

**Где:**
```
Δ_body = -ln(f_body)
f_body = средняя частота биологических процессов
```

### 1.3. Гипотеза корреляции

**Связь:**
```
S_brain ≈ S_body
```

**Проверка:**
```
Если S_brain ≥ 0.7 → S_body ≥ 0.7
Если S_brain < 0.7 → S_body < 0.7
```

## 2. Биологические процессы как осцилляторы

### 2.1. Сердечный ритм

### 2.2. Дыхание

### 2.3. Метаболизм

## 3. Синхронизация тела

### 3.1. Среднее Δ

**Среднее Δ тела:**
```
⟨Δ_body⟩ = (Δ_heart + Δ_breath + Δ_metab + Δ_sleep) / 4
⟨Δ_body⟩ = (-0.285 + 1.322 + 1.204 + 11.37) / 4
⟨Δ_body⟩ = 13.611 / 4 ≈ 3.403
```

### 3.2. Рассеяние

**Рассеяние Δ:**
```
σ(Δ_body) = √[(1/4) ((-0.285-3.403)² + (1.322-3.403)² + (1.204-3.403)² + (11.37-3.403)²)]
σ(Δ_body) = √[(1/4) (13.61 + 4.33 + 4.84 + 63.47)]
σ(Δ_body) = √[21.56] ≈ 4.64
```

### 3.3. Синхронизация

**Синхронизация тела:**
```
S_body = 1 - 0.05 · 4.64 = 1 - 0.232 = 0.768
```

**S_body ≥ 0.7 → здоровье**

## 5. Связь с ростом и весом

### 5.1. Рост

**Рост как функция синхронизации:**
```
L ∝ S_body
```

**Проверка:**
```
Здоровый рост: S_body ≈ 0.77 → L ≈ 1.7 м
Задержка роста: S_body < 0.7 → L < 1.5 м
```

### 5.2. Вес

**Вес как суммарная энергия:**
```
M = Σ Q_i
```

**Где Q_i — энергия вибрации i-го органа.**

**Проверка:**
```
Здоровый вес: Σ Q_i ≈ 70 кг
Ожирение: Σ Q_i > 100 кг
Анорексия: Σ Q_i < 50 кг
```

### 5.3. BMI

**BMI как функция синхронизации:**
```
BMI = M / L²
BMI ∝ (Σ Q_i) / S_body²
```

**Проверка:**
```
Норма: BMI ≈ 20-25
Ожирение: BMI > 30
Дефицит: BMI < 18.5
```
