# ADHD как нарушение (Δ,Q) синхронизации

## Гипотеза

**ADHD — это нарушение (Δ,Q) синхронизации в мозге, вызванное замедлением alpha пиковой частоты.**

---

## 1. Формальная модель мозга

### 1.1. Состояние мозга

**Мозг как система N осцилляторов:**
```
Ψ_brain = {(Δ_i, Q_i, f_i)} для i = 1, ..., N
```

**Где:**
- Δ_i = -ln(f_i) — параметр рассинхронизации i-го осциллятора
- Q_i = e^|Δ_i| — энергия вибрации i-го осциллятора
- f_i — частота i-го осциллятора (Гц)

### 1.2. Мозговые волны

| Волна | Частота (Гц) | Δ = -ln(f) | Q = e^|Δ| | Роль |
|-------|--------------|------------|-----------|------|
| Delta | 2-4 | -1.609 to -0.916 | 2.5 to 5.0 | Глубокий сон |
| Theta | 4-8 | -0.916 to -0.223 | 1.25 to 2.5 | Медитация, память |
| Alpha | 8-13 | -0.223 to +0.262 | 1.25 to 1.3 | Расслабление |
| Beta | 13-21 | +0.262 to +0.742 | 1.3 to 2.1 | Фокусировка |
| Gamma | 21-100 | +0.742 to +4.605 | 2.1 to 100 | Высшие когнитивные |

### 1.3. Синхронизация

**Формула синхронизации:**
```
S = 1 - σ(Δ) · 0.05
```

**Где:**
```
σ(Δ) = √[(1/N) Σᵢ (Δᵢ - ⟨Δ⟩)²]
```

**Критерии сознания:**
```
S ≥ 0.7 AND |Δ_global| ≤ 0.2
```

---

## 2. Здоровый мозг (S ≥ 0.7)

### 2.1. Нормальная alpha

**Alpha пиковая частота (IAPF):**
```
IAPF = 10 Гц
Δ_alpha = -ln(10) ≈ -2.303
```

### 2.2. Синхронизация

**Alpha синхронизирует theta и beta:**
```
Δ_theta ≈ -1.8 (6 Гц)
Δ_beta ≈ -2.8 (17 Гц)
Δ_alpha ≈ -2.3 (10 Гц)
```

**Рассеяние:**
```
σ(Δ) = √[(1/3) ((-1.8+2.3)² + (-2.8+2.3)² + (-2.3+2.3)²)]
σ(Δ) = √[(1/3) (0.25 + 0.25 + 0)]
σ(Δ) = √[0.167] ≈ 0.408
```

**Синхронизация:**
```
S = 1 - 0.408 · 0.05 = 1 - 0.020 = 0.980
```

**S ≥ 0.7 → здоровье**

### 2.3. Гармония

**Простые Q соотношения:**
```
Q_theta / Q_alpha = 2.0 / 1.3 ≈ 1.54
Q_alpha / Q_beta = 1.3 / 1.6 ≈ 0.81
Q_theta / Q_beta = 2.0 / 1.6 ≈ 1.25
```

**Гармония:**
```
H(theta, alpha) = 1/(1+|Δ_theta-Δ_alpha|+|ln(Q_theta/Q_alpha)|)
H(theta, alpha) = 1/(1+|-1.8+2.3|+|ln(1.54)|)
H(theta, alpha) = 1/(1+0.5+0.432) ≈ 0.517
```

**Гармония > 0.5 → здоровье**

## 3. ADHD мозг (S < 0.7)

### 3.1. Замедленная alpha

**Alpha пиковая частота (IAPF):**
```
IAPF = 7 Гц (замедленная)
Δ_alpha = -ln(7) ≈ -1.946
```

### 3.2. Наложение band

**Alpha полоса накладывается на theta полосу:**
```
Alpha: 8-13 Гц (норма) → 7-12 Гц (замедленная)
Theta: 4-8 Гц
Пересечение: 7-8 Гц
```

**Theta "кажется" повышенной:**
```
Pseudo-theta = Alpha + Theta
Pseudo-theta > Theta (норма)
```

### 3.3. Десинхронизация

**Рассеяние увеличивается:**
```
Δ_theta ≈ -1.8 (6 Гц)
Δ_beta ≈ -2.8 (17 Гц)
Δ_alpha ≈ -1.946 (7 Гц)
```

**Рассеяние:**
```
σ(Δ) = √[(1/3) ((-1.8+2.182)² + (-2.8+2.182)² + (-1.946+2.182)²)]
σ(Δ) = √[(1/3) (0.146 + 0.382 + 0.056)]
σ(Δ) = √[0.195] ≈ 0.441
```

**Синхронизация:**
```
S = 1 - 0.441 · 0.05 = 1 - 0.022 = 0.978
```

**S ≥ 0.7 → всё ещё здоровье?**

**Проблема:** Нужно учитывать количество осцилляторов!

### 3.4. Исправленная модель

**Реальный мозг имеетousands осцилляторов:**
```
N_theta = 1000 (осцилляторов в theta диапазоне)
N_alpha = 500 (осцилляторов в alpha диапазоне)
N_beta = 200 (осцилляторов в beta диапазоне)
```

**Среднее Δ:**
```
⟨Δ⟩ = (N_theta·Δ_theta + N_alpha·Δ_alpha + N_beta·Δ_beta) / (N_theta + N_alpha + N_beta)
⟨Δ⟩ = (1000·(-1.8) + 500·(-1.946) + 200·(-2.8)) / 1700
⟨Δ⟩ = (-1800 - 973 - 560) / 1700
⟨Δ⟩ = -3333 / 1700 ≈ -1.960
```

**Рассеяние:**
```
σ(Δ) = √[(1/1700) (1000·(-1.8+1.960)² + 500·(-1.946+1.960)² + 200·(-2.8+1.960)²)]
σ(Δ) = √[(1/1700) (1000·0.026 + 500·0.0002 + 200·0.706)]
σ(Δ) = √[(1/1700) (25.6 + 0.1 + 141.2)]
σ(Δ) = √[0.1] ≈ 0.316
```

**Синхронизация:**
```
S = 1 - 0.316 · 0.05 = 1 - 0.016 = 0.984
```

**S ≥ 0.7 → здоровье**

**Проблема:** Модель не объясняет ADHD!

---

## 4. Исправленная модель ADHD

### 4.1. Ключевой инсайт

**ADHD — это не просто замедление alpha, а нарушение связи между alpha, theta и beta.**

### 4.2. Модель с когерентностью

**Когерентность между осцилляторами:**
```
C(i,j) = |⟨Δ_i · Δ_j⟩| / √(⟨Δ_i²⟩ · ⟨Δ_j²⟩)
```

**Здоровый мозг:**
```
C(theta, alpha) ≈ 0.8 (высокая когерентность)
C(alpha, beta) ≈ 0.8 (высокая когерентность)
```

**ADHD мозг:**
```
C(theta, alpha) ≈ 0.3 (низкая когерентность)
C(alpha, beta) ≈ 0.3 (низкая когерентность)
```

### 4.3. Синхронизация с когерентностью

**Исправленная формула:**
```
S = 1 - σ(Δ) · 0.05 · (1 - C_avg)
```

**Где:**
```
C_avg = (C(theta, alpha) + C(alpha, beta)) / 2
```

**Здоровый мозг:**
```
C_avg = (0.8 + 0.8) / 2 = 0.8
S = 1 - 0.316 · 0.05 · (1 - 0.8) = 1 - 0.003 = 0.997
```

**ADHD мозг:**
```
C_avg = (0.3 + 0.3) / 2 = 0.3
S = 1 - 0.316 · 0.05 · (1 - 0.3) = 1 - 0.011 = 0.989
```

**S ≥ 0.7 → всё ещё здоровье?**

### 4.4. Другая модель: Theta/Beta Ratio

**Theta/Beta Ratio (TBR):**
```
TBR = P_theta / P_beta
```

**Здоровый мозг:**
```
TBR ≈ 1.0 (баланс)
```

**ADHD мозг:**
```
TBR > 1.5 (дисбаланс)
```

### 4.5. Связь TBR с S

**Гипотеза:**
```
TBR = 1/S
```

**Проверка:**
```
Здоровье: S = 0.997 → TBR = 1.003 (≈ 1.0) ✓
ADHD: S = 0.989 → TBR = 1.011 (≈ 1.0) ✗
```

**Не работает!**

### 4.6. Другая модель: Absolute Power

**Theta Power:**
```
P_theta = Q_theta · N_theta
```

**Beta Power:**
```
P_beta = Q_beta · N_beta
```

**ADHD дети:**
```
P_theta увеличена → Q_theta или N_theta увеличены
```

**Взрослые ADHD:**
```
P_theta нормальная → Q_theta и N_theta нормальны
```

## 5. Новая модель: ADHD как нарушение когерентности

### 5.1. Гипотеза

**ADHD — это нарушение когерентности между мозговыми волнами, а не просто нарушение синхронизации.**

### 5.2. Формальная модель

**Когерентность:**
```
C(i,j) = |⟨Δ_i · Δ_j⟩| / √(⟨Δ_i²⟩ · ⟨Δ_j²⟩)
```

**Синхронизация с когерентностью:**
```
S = 1 - σ(Δ) · 0.05 · (1 - C_avg)
```

**Где:**
```
C_avg = (1/M) Σ_{i<j} C(i,j)
```

### 5.3. Здоровый мозг

**Высокая когерентность:**
```
C(theta, alpha) ≈ 0.8
C(alpha, beta) ≈ 0.8
C(theta, beta) ≈ 0.6
C_avg = (0.8 + 0.8 + 0.6) / 3 ≈ 0.733
```

**Синхронизация:**
```
S = 1 - 0.316 · 0.05 · (1 - 0.733) = 1 - 0.004 = 0.996
```

**S ≥ 0.7 → здоровье**

### 5.4. ADHD мозг

**Низкая когерентность:**
```
C(theta, alpha) ≈ 0.3
C(alpha, beta) ≈ 0.3
C(theta, beta) ≈ 0.2
C_avg = (0.3 + 0.3 + 0.2) / 3 ≈ 0.267
```

**Синхронизация:**
```
S = 1 - 0.316 · 0.05 · (1 - 0.267) = 1 - 0.012 = 0.988
```

**S ≥ 0.7 → всё ещё здоровье?**
