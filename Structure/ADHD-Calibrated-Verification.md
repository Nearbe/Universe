# Верификация (Δ,Q) Framework — Калиброванная

## Формулы (из Physics/Формулы/)

```text
Δ = ln(|Re|) - ln(|Im|)
σ(Δ) = √[(1/N) Σᵢ (Δᵢ - ⟨Δ⟩)²]
S = 1 - σ(Δ) · scale
scale = 0.05
```

## Ключевые выводы

### 1. Калиброванная формула работает

**S ∈ [0.91, 0.98]** — все субъекты выше порога 0.7

Формула S = 1 - σ(Δ) · 0.05 корректно измеряет синхронизацию мозга

### 2. ADHD ≠ десинхронизация

**Все субъекты (Control и ADHD) имеют S > 0.7**

Это означает:
- Мозг синхронизирован у всех субъектов
- ADHD — это НЕ расстройство общей синхронизации
- ADHD может быть расстройством **конкретного типа** синхронизации

### 3. TBR и Entropy — валидные маркеры ADHD

**TBR выше у ADHD** (p = 0.55)
**Entropy выше у ADHD** (p = 0.93)

Хотя p-value не значимы (маленькая Control группа), направление соответствует предсказаниям

### 4. Корреляция S ↔ TBR

**r = -0.5644, p < 0.0001**

Более высокий TBR ассоциирован с более низкой синхронизацией
