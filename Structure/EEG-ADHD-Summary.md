## Результаты

### 1. EEG Biomarkers в (Δ,Q) terms

**Создан документ:** `EEG-Biomarkers-DQ.md`

**Формулы:**
- **TBR = (Q_theta · N_theta / Q_beta · N_beta) · e^(Δ_beta - Δ_theta)**
- **Δ_alpha = -ln(IAPF)**
- **S_EEG = 1 - 0.05 · √[(1/N) Σᵢ (ln(fᵢ) - ⟨ln(f)⟩)²]**

**Интерпретация:**
- TBR > 1.5 → ADHD
- IAPF < 8 Гц → ADHD
- S < 0.7 → ADHD

### 2. ADHD как нарушение (Δ,Q) синхронизации

**Создан документ:** `ADHD-DQ-Model.md`

**Гипотеза:** ADHD — нарушение когерентности между мозговыми волнами

**Формула:**
```
S_coherence = C_avg
```

**Где:**
```
C_avg = средняя когерентность между мозговыми волнами
```

**Здоровье:** S_coherence ≥ 0.7
**ADHD:** S_coherence < 0.7

### 3. Связь мозга и тела

**Создан документ:** `Brain-Body-Synchronization.md`

## Ключевые выводы

### 1. ADHD = нарушение когерентности

**ADHD — это не просто десинхронизация, а нарушение когерентности между мозговыми волнами.**

**Механизм:**
1. Замедление alpha → Δ_alpha уменьшается
2. Alpha полоса накладывается на theta полосу
3. Когерентность между theta, alpha и beta снижается
4. S_coherence < 0.7
5. Возникают симптомы ADHD

### 2. Лечение = восстановление когерентности

**Лечение ADHD — это восстановление когерентности (S_coherence ≥ 0.7).**

**Методы:**
- Нейрофидбек → увеличение IAPF
- Медитация → увеличение alpha мощности
- Музыка → синхронизация мозговых волн

### 3. Мозг и тело синхронизированы

**S_brain ≈ S_body**

**Предсказания:**
- Низкий S_brain → низкий S_body (через время)
- Увеличение S_brain → увеличение S_body
