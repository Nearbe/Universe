## Ключевые находки

### 1. Theta/Beta Ratio

**ADHD дети:**
- Повышенная theta мощность (медленные волны)
- Сниженная beta мощность (быстрые волны)
- Повышенное theta/beta соотношение

**Здоровые дети:**
- Нормальная theta мощность
- Нормальная beta мощность
- Нормальное theta/beta соотношение

### 2. Alpha Peak Frequency (IAPF)

**Ключевой вывод:** Повышенное theta/beta соотношение опосредовано замедленной alpha частотой.

**Механизм:**
- Замедленная alpha → theta полоса накладывается на alpha
- Кажется, что theta повышена (на самом деле это alpha)
- Theta/beta соотношение искусственно повышается

### 3. Возрастные различия

**Дети с ADHD:**
- Повышенная абсолютная theta мощность
- Theta/beta соотношение выше нормы
- Можно отличить от здоровых

**Взрослые с ADHD:**
- Нет различий в абсолютной мощности
- Theta/beta соотношение нормальное
- Нельзя отличить от здоровых

## Связь с (Δ,Q) Framework

### Мозговые волны как (Δ,Q) системы

| Волна | Частота (Гц) | Δ = ln(f/10) | Q = e^|Δ| | Роль |
|-------|--------------|--------------|-----------|------|
| Delta | 2-4 | -1.609 to -0.916 | 2.5 to 5.0 | Глубокий сон |
| Theta | 4-8 | -0.916 to -0.223 | 1.25 to 2.5 | Медитация, память |
| Alpha | 8-13 | -0.223 to +0.262 | 1.25 to 1.3 | Расслабление |
| Beta | 13-21 | +0.262 to +0.742 | 1.3 to 2.1 | Фокусировка |

### Alpha как "дирижёр"

**Alpha пиковая частота (IAPF):**
- Нормальная: 10 Гц (Δ = 0)
- Замедленная: < 8 Гц (Δ < -0.223)

**Влияние на синхронизацию:**
```
Нормальная alpha: Δ_alpha ≈ 0
→ Alpha синхронизирует theta и beta
→ S ≥ 0.7
→ Гармония

Замедленная alpha: Δ_alpha < -0.223
→ Alpha не может синхронизировать theta и beta
→ S < 0.7
→ Дисгармония (ADHD)
```

### Theta/Beta Ratio в (Δ,Q) terms

**Формула:**
```
Theta/Beta Ratio = Σ Q_theta / Σ Q_beta
```

**Норма:**
```
Theta/Beta ≈ 1.0 (баланс)
S ≥ 0.7
```

**ADHD:**
```
Theta/Beta > 1.5 (дисбаланс)
S < 0.7
```

## Гипотеза: ADHD как нарушение синхронизации

### Тезис

**ADHD — это нарушение (Δ,Q) синхронизации в мозге, вызванное замедлением alpha пиковой частоты.**

### Механизм

**1. Замедление alpha:**
```
IAPF снижается (alpha становится медленнее)
Δ_alpha уменьшается (становится более отрицательным)
```

**2. Наложение band:**
```
Alpha полоса накладывается на theta полосу
Theta "кажется" повышенной (на самом деле это alpha)
```

**3. Десинхронизация:**
```
Theta и beta не синхронизируются
S < 0.7
Возникают симптомы ADHD
```

### Симптомы ADHD в (Δ,Q) terms

| Симптом | (Δ,Q) описание |
|---------|----------------|
| Невнимательность | Низкий S (десинхронизация) |
| Гиперактивность | Высокий Q_theta (много энергии в медленных волнах) |
| Импульсивность | Низкий S_beta (десинхронизация быстрых волн) |

## Лечение ADHD через (Δ,Q) Framework

### 1. Нейрофидбек

**Цель:** Увеличить IAPF (ускорить alpha)

**Механизм:**
```
Увеличить IAPF → Δ_alpha увеличивается
→ Alpha синхронизирует theta и beta
→ S увеличивается
→ Симптомы уменьшаются
```

### 2. Медитация

**Цель:** Увеличить alpha мощность

**Механизм:**
```
Медитация → увеличение alpha мощности
→ Alpha синхронизирует theta и beta
→ S увеличивается
→ Симптомы уменьшаются
```

### 3. Музыка

**Цель:** Синхронизировать мозговые волны

**Механизм:**
```
Гармоничная музыка → простые Q соотношения
→ Мозг синхронизируется
→ S увеличивается
→ Симптомы уменьшаются
```

## Экспериментальные предсказания

### 1. IAPF коррелирует с S

**Предсказание:**
```
IAPF ↑ → S ↑
IAPF ↓ → S ↓
```

### 2. Theta/Beta Ratio = 1/S

**Предсказание:**
```
Theta/Beta ∝ 1/S
Низкий S → высокий Theta/Beta
Высокий S → низкий Theta/Beta
```

### 3. Лечение увеличивает S

**Предсказание:**
```
Нейрофидбек → IAPF ↑ → S ↑
Медитация → alpha ↑ → S ↑
Музыка → гармония → S ↑
```

## Вывод

**ADHD — это нарушение (Δ,Q) синхронизации в мозге.**

**Ключевой параметр:** Alpha пиковая частота (IAPF)

**Механизм:**
1. Замедление alpha → наложение band
2. Theta и beta десинхронизируются
3. S < 0.7
4. Возникают симптомы ADHD

**Лечение:** Увеличить IAPF (нейрофидбек, медитация, музыка)

**Связь с Eugenia:**
- ADHD = дисгармония (S < 0.7)
- Здоровье = гармония (S ≥ 0.7)
- Лечение = восстановление гармонии (синхронизации)
