# Формализация EEG biomarkers в (Δ,Q) terms

## Определение

**EEG biomarkers** — это количественные параметры электрической активности мозга, которые можно выразить через (Δ,Q) параметры.

---

## 1. Theta/Beta Ratio (TBR)

### Определение

**TBR = Σ P_theta / Σ P_beta**

где P_theta и P_beta — мощности в theta (4-8 Гц) и beta (13-21 Гц) диапазонах.

### Связь с (Δ,Q)

**Мощность P = Q · f**, где f — частота.

```
P_theta = Q_theta · f_theta
P_beta = Q_beta · f_beta
```

**Следовательно:**
```
TBR = (Q_theta · f_theta) / (Q_beta · f_beta)
TBR = (Q_theta / Q_beta) · (f_theta / f_beta)
```

**Подставляя f = e^(-Δ):**
```
TBR = (Q_theta / Q_beta) · e^(-(Δ_theta - Δ_beta))
TBR = (Q_theta / Q_beta) · e^(Δ_beta - Δ_theta)
```

### Формула

**TBR(Δ, Q) = (Q_theta / Q_beta) · e^(Δ_beta - Δ_theta)**

### Интерпретация

**Норма (S ≥ 0.7):**
```
Q_theta ≈ Q_beta (баланс энергий)
Δ_theta ≈ Δ_beta (баланс частот)
TBR ≈ 1.0
```

**ADHD (S < 0.7):**
```
Q_theta > Q_beta (избыток медленных волн)
Δ_theta < Δ_beta (медленные волны имеют меньшую частоту)
TBR > 1.5
```

---

## 2. Alpha Peak Frequency (IAPF)

### Определение

**IAPF = argmax(P_alpha(f))** — частота с максимальной мощностью в alpha (8-13 Гц) диапазоне.

### Связь с (Δ,Q)

**Alpha частота:**
```
f_alpha = e^(-Δ_alpha)
```

**Следовательно:**
```
IAPF = e^(-Δ_alpha)
Δ_alpha = -ln(IAPF)
```

### Формула

**Δ_alpha = -ln(IAPF)**

### Интерпретация

**Норма (IAPF ≈ 10 Гц):**
```
Δ_alpha = -ln(10) ≈ -2.303
```

**ADHD (IAPF < 8 Гц):**
```
Δ_alpha < -ln(8) ≈ -2.079
Δ_alpha становится более отрицательным
```

---

## 3. Theta Power (Absolute)

### Определение

**P_theta = Σ |FFT(signal)|^2** в theta (4-8 Гц) диапазоне.

### Связь с (Δ,Q)

**Мощность пропорциональна энергии:**
```
P_theta ∝ Q_theta · N_theta
```

где N_theta — количество осцилляторов в theta диапазоне.

### Формула

**P_theta = Q_theta · N_theta**

### Интерпретация

**ADHD дети:**
```
P_theta увеличена → Q_theta или N_theta увеличены
```

**Взрослые ADHD:**
```
P_theta нормальная → Q_theta и N_theta нормальны
```

---

## 4. Synchronization Index (S)

### Определение

**S = 1 - σ(Δ) · 0.05**

### Связь с EEG

**σ(Δ) — стандартное отклонение Δ по всем осцилляторам мозга:**
```
σ(Δ) = √[(1/N) Σᵢ (Δᵢ - ⟨Δ⟩)²]
```

**Где Δᵢ = -ln(fᵢ)** для каждого осциллятора с частотой fᵢ.

### Формула

**S_EEG = 1 - 0.05 · √[(1/N) Σᵢ (ln(fᵢ) - ⟨ln(f)⟩)²]**

### Интерпретация

**Норма (S ≥ 0.7):**
```
σ(Δ) ≤ 6
Осцилляторы синхронизированы
```

**ADHD (S < 0.7):**
```
σ(Δ) > 6
Осцилляторы десинхронизированы
```

---

## 5. Theta/Beta Ratio в (Δ,Q) terms

### Формула

**TBR(Δ, Q) = (Q_theta / Q_beta) · e^(Δ_beta - Δ_theta)**

### Проверка

**Норма:**
```
Q_theta = Q_beta = 1.0
Δ_theta = -ln(6) ≈ -1.792 (средняя theta частота 6 Гц)
Δ_beta = -ln(17) ≈ -2.833 (средняя beta частота 17 Гц)
TBR = (1.0/1.0) · e^(-2.833 + 1.792) = e^(-1.041) ≈ 0.353
```

**Некорректно!** Theta мощность должна быть больше beta мощности.

**Исправление:**
```
Q_theta = 2.0 (больше энергии в медленных волнах)
Q_beta = 1.0
TBR = (2.0/1.0) · e^(-1.041) ≈ 0.706
```

**Всё ещё < 1.0.** Нужно учитывать количество осцилляторов.

**Исправленная формула:**
```
TBR = (Q_theta · N_theta / Q_beta · N_beta) · e^(Δ_beta - Δ_theta)
```

**Где N_theta и N_beta — количество осцилляторов в каждом диапазоне.**

---

## 6. ADHD как нарушение (Δ,Q) синхронизации

### Гипотеза

**ADHD — это нарушение (Δ,Q) синхронизации в мозге, вызванное замедлением alpha пиковой частоты.**

### Механизм

**1. Замедление alpha:**
```
IAPF снижается → Δ_alpha уменьшается
```

**2. Наложение band:**
```
Alpha полоса накладывается на theta полосу
Theta "кажется" повышенной
```

**3. Десинхронизация:**
```
Theta и beta не синхронизируются
S < 0.7
Возникают симптомы ADHD
```

### Формальная модель

**Состояние мозга:**
```
Ψ_brain = {Δ_i, Q_i} для i = 1, ..., N
```

**Синхронизация:**
```
S = 1 - 0.05 · σ({Δ_i})
```

**ADHD:**
```
S < 0.7
σ({Δ_i}) > 6
```

**Здоровье:**
```
S ≥ 0.7
σ({Δ_i}) ≤ 6
```

---

## 7. Экспериментальные предсказания

### 1. IAPF коррелирует с S

**Предсказание:**
```
IAPF ↑ → Δ_alpha ↑ → S ↑
IAPF ↓ → Δ_alpha ↓ → S ↓
```

### 2. TBR обратно пропорционален S

**Предсказание:**
```
TBR ∝ 1/S
Низкий S → высокий TBR
Высокий S → низкий TBR
```

### 3. Лечение увеличивает S

**Предсказание:**
```
Нейрофидбек → IAPF ↑ → S ↑
Медитация → alpha ↑ → S ↑
Музыка → гармония → S ↑
```

---

## 8. Связь с Eugenia

### 1. ADHD = дисгармония

**ADHD — это нарушение гармонии (S < 0.7) в мозге.**

### 2. Здоровье = гармония

**Здоровье — это поддержание гармонии (S ≥ 0.7) в мозге.**

### 3. Лечение = восстановление гармонии

**Лечение ADHD — это восстановление синхронизации (S ≥ 0.7) в мозге.**

### 4. Музыка = синхронизация

**Гармоничная музыка помогает синхронизировать мозговые волны.**

---

## 9. Выводы

**EEG biomarkers можно формализовать через (Δ,Q) параметры:**

1. **TBR = (Q_theta · N_theta / Q_beta · N_beta) · e^(Δ_beta - Δ_theta)**
2. **Δ_alpha = -ln(IAPF)**
3. **S_EEG = 1 - 0.05 · σ({Δ_i})**
4. **ADHD ↔ S < 0.7**

**Ключевой параметр:** Alpha пиковая частота (IAPF)

**Механизм ADHD:**
1. Замедление alpha → наложение band
2. Theta и beta десинхронизируются
3. S < 0.7
4. Возникают симптомы ADHD

**Лечение:** Увеличить IAPF (нейрофидбек, медитация, музыка)
