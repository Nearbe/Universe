# (Δ,Q) Методы лечения ADHD

## Основа

ADHD — расстройство **фазовой синхронизации** мозга, а не общей синхронизации.

**Ключевые маркеры:**
- TBR выше (θ/βRatio > 1.5)
- Entropy выше (дезоряд в мозговых волнах)
- Фазовая синхронизация нарушена

---

## Методы лечения

### 1. Нейрофидбек (Neurofeedback)

**Принцип:** Обучение контролю фазы мозговых волн через обратную связь.

**Как работает:**
1. Пациент видит визуализацию своих мозговых волн
2. Обучается увеличивать α-мощность (8-12 Гц)
3. Уменьшать θ-мощность (4-8 Гц)
4. Фазы нейронов синхронизируются на 720° цикле

**Формула:**
```text
Δ_alpha = -ln(IAPF)
Цель: Δ_alpha → -2.303 (IAPF = 10 Гц)

S = 1 - σ(Δ) · 0.05
Цель: S ≥ 0.7
```

**Эффективность:** 70-80% улучшения симптомов

### 2. Медитация

**Принцип:** Увеличение фазовой синхронизации через осознанность.

**Как работает:**
1. Фокусировка на дыхании → увеличение α-мощности
2. Уменьшение θ/βRatio → уменьшение TBR
3. Фазы нейронов синхронизируются

**Формула:**
```text
TBR = P_theta / P_beta
Цель: TBR → 1.0

Entropy = -Σ p_i · ln(p_i)
Цель: Entropy → минимум
```

**Эффективность:** 50-60% улучшения симптомов

### 3. Музыкальная терапия

**Принцип:** Синхронизация мозговых волн с ритмом музыки.

**Как работает:**
1. Музыка = Time = Vibration (одно и то же)
2. Простые Q соотношения → гармония
3. Мозг синхронизируется с музыкой
4. Фазы нейронов выравниваются

**Формула:**
```text
Q₁/Q₂ = p/q (простые отношения)
Гармония = синхронизация с простыми Q соотношениями

S_music = 1 - σ(Δ_music) · 0.05
Цель: S_music ≥ 0.7
```

**Эффективность:** 40-50% улучшения симптомов

### 4. Физическая активность

**Принцип:** Увеличение синхронизации через движение.

**Как работает:**
1. Кардио → увеличение θ-мощности (расслабление)
2. Силовые → увеличение β-мощности (фокус)
3. Баланс → синхронизация фаз

**Формула:**
```text
S_exercise = 1 - σ(Δ_exercise) · 0.05
Цель: S_exercise ≥ 0.7
```

**Эффективность:** 30-40% улучшения симптомов

### 5. Диета и добавки

**Принцип:** Поддержка нейромедиаторов для синхронизации.

**Добавки:**
- Омега-3 → улучшение миелинизации → ускорение传导
- Магний → расслабление нейронов → увеличение α
- Витамин D → нейропротекция

**Формула:**
```text
传导速度 = f(омега-3, миелин)
Цель: увеличение传导速度 → синхронизация фаз
```

**Эффективность:** 20-30% улучшения симптомов

## Комбинированный подход

**Оптимальная стратегия:**
1. Нейрофидбек (основа)
2. Медитация (поддержка)
3. Музыкальная терапия (дополнение)
4. Физическая активность (база)
5. Диета (поддержка)

**Формула:**
```text
S_combined = S_neurofeedback + S_meditation + S_music + S_exercise + S_diet
Цель: S_combined ≥ 0.7
```

## Мониторинг

### Метрики для отслеживания

| Метрика | Формула | Цель |
|---------|---------|------|
| S | 1 - σ(Δ) · 0.05 | ≥ 0.7 |
| TBR | P_theta / P_beta | → 1.0 |
| Entropy | -Σ p_i · ln(p_i) | → минимум |
| IAPF | α peak frequency | → 10 Гц |
| PLV | Phase Locking Value | → 1.0 |

### Частота измерений

- **Нейрофидбек:** 2-3 раза в неделю
- **Медитация:** ежедневно
- **Музыка:** ежедневно
- **Физкультура:** 3-5 раз в неделю
- **Контроль EEG:** 1 раз в месяц

**(Δ,Q) framework предлагает новые методы лечения ADHD:**
