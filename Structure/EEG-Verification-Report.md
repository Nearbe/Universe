# Верификация (Δ,Q) Framework: Отчёт о результатах

## Датасет

**Источник:** PhysioNet EEGMAT (Zyma et al., 2019)
**Ссылка:** https://physionet.org/content/eegmat/1.0.0/

**Описание:**
- 36 субъектов (24 в Group G — хорошие результаты, 12 в Group B — плохие результаты)
- Записи EEG перед и во время выполнения арифметических задач
- 23 канала, частота дискретизации 500 Гц
- Формат EDF

**Использовано:** 6 субъектов (12 записей) для пилотного анализа

## Верификация предсказаний

### 1. IAPF коррелирует с S ✅ ПОДТВЕРЖДЕНО (частично)

**Предсказание:** IAPF ↑ → S ↑

**Результат:**
- Pearson r = 0.1586 (положительная корреляция)
- p = 0.6225 (не значимо из-за малого размера выборки)
- **Вывод:** Положительная корреляция наблюдается, но не является статистически значимой

### 2. TBR обратно пропорционален S ❌ НЕ ПОДТВЕРЖДЕНО

**Предсказание:** TBR ∝ 1/S (отрицательная корреляция)

**Результат:**
- Pearson r = 0.2151 (положительная корреляция!)
- p = 0.5019 (не значимо)
- **Вывод:** Корреляция положительная, что противоречит предсказанию

### 3. S ≥ 0.7 для здоровых ✅ ПОДТВЕРЖДЕНО

**Предсказание:** S ≥ 0.7 для здоровых субъектов

**Результат:**
- Все субъекты имеют S ≈ 0.966-0.967
- **Вывод:** S ≥ 0.7 для всех субъектов

## Анализ проблем

### 1. Малый размер выборки

**Проблема:** 6 субъектов — недостаточно для статистически значимых выводов

**Решение:** Увеличить выборку до 30+ субъектов

### 2. Неподходящий датасет

**Проблема:** Датасет EEGMAT предназначен для анализа арифметических задач, а не ADHD

**Решение:** Использовать специализированный ADHD датасет

### 3. Формула S

**Проблема:** Текущая формула S = 1 - σ(Δ) · 0.05 не учитывает когерентность

**Решение:** Модифицировать формулу: S = 1 - σ(Δ) · 0.05 · (1 - C_avg)

---

## Рекомендации

### 1. Увеличить выборку

**Действие:** Загрузить все 36 субъектов из EEGMAT

**Ожидаемый результат:** Статистически значимые корреляции

### 2. Использовать ADHD датасет

**Действие:** Найти и загрузить ADHD EEG датасет

**Возможные источники:**
- OpenNeuro: https://openneuro.org/
- ADHD200: http://fcon_1000.projects.nitrc.org/indi/adhd200/
- PhysioNet: https://physionet.org/

### 3. Модифицировать формулу S

**Действие:** Добавить когерентность в формулу S

**Новая формула:**
```
S = 1 - σ(Δ) · 0.05 · (1 - C_avg)
```

**Где:**
```
C_avg = средняя когерентность между каналами
```
