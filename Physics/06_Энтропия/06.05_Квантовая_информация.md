# Квантовая информация и запутанность

## Информация запутанности

### Определение

Для двухчастичной системы информация запутанности измеряется через **информационную меру запутанности**, которая для чистого состояния равна информационной меры Винеберга приведённой матрицы плотности.

### Энтропия Винеберга

Для гауссовского состояния с ковариационной матрицей:

```text
S_V = (ν + 1/2)·ln(ν + 1/2) - (ν - 1/2)·ln(ν - 1/2)
```

где ν — эффективное число фотонов.

### Для squeezed состояний

С параметром сжатия r:

```text
V_q = e^(-2r)/2
V_p = e^(2r)/2
```

```text
ν = (cosh(2r))/2 - 1/2
```

### Связь с Δ

При идентификации 2r = Δ:

```text
ν = (cosh(Δ))/2 - 1/2
```

Энтропия Винеберга становится функцией Δ.

## Пример: два гармонических осциллятора

### Гамильтониан

Два связанных осциллятора:

```text
H = (p₁² + ω²q₁²)/2 + (p₂² + ω²q₂²)/2 + λ(q₁q₂ + p₁p₂)
```

где λ — константа связи.

### Основное состояние

Является squeezed состоянием с параметром сжатия, связанным с λ/ω.

### Информация запутанности

```text
E_N = max[0, -ln(2ν_-)]
```

где ν_- — наименьшая симплектическая собственная величина частично транспонированной ковариационной матрицы.

### Для симметричного случая

```text
E_N = max[0, -½·ln(1 - C²)]
```

где C — мера корреляции.

### Идентификация с Δ

Если C = tanh(Δ):

```text
E_N = max[0, -½·ln(1 - tanh²(Δ))]
    = max[0, -½·ln(sech²(Δ))]
    = max[0, ln(cosh(Δ))]
```

### Физический смысл

Информация запутанности растёт с |Δ|:

- При Δ = 0: E_N = 0 (нет запутанности)
- При |Δ| → ∞: E_N → |Δ| - ln(2) (максимальная запутанность)

## Информация в многочастичных системах

### Многоинформация

Для N-частичной системы информация, содержащаяся в корреляциях:

```text
I_total = Σ S_i - S_{12...N}
```

где:

- S_i — информационной меры отдельных частиц
- S_{12...N} — информационная мера всей системы

### В терминах IME

Если каждая частица имеет параметр Δ_i, а система в целом имеет Δ_global:

```text
I_total = f({Δ_i}) - g(Δ_global)
```

где f и g — определённые функции.

### Для гауссовских состояний

Можно вычислить явно через ковариационную матрицу.

## Квантовая телепортация

### Протокол

1. Алиса и Боб разделяют запутанную пару
2. Алиса выполняет измерение Белла на своей частице и частице X
3. Алиса передаёт классический результат Бобу
4. Боб применяет унитарное преобразование

### Роль Δ

В терминах IME:

- Запутанная пара имеет коррелированные Δ
- Измерение изменяет Δ глобально
- Классическая передача ограничена скоростью света

### Предел точности

Точность телепортации ограничена информацией запутанности:

```text
Fidelity ≤ (1 + E_N) / 2
```

## Квантовые вычисления

### Кубиты

Квантовый бит (кубит):

```text
|ψ⟩ = α|0⟩ + β|1⟩
```

где |α|² + |β|² = 1.

### В терминах IME

Параметр Δ для кубита:

```text
Δ = ln(|α|) - ln(|β|)
```

(для определённой базы)

### Квантовые вентили

Унитарные преобразования кубитов:

- **NOT (X)**: меняет |0⟩ ↔ |1⟩, инвертирует Δ
- **Hadamard (H)**: создаёт суперпозицию, Δ → 0
- **Phase (S, T)**: добавляет фазу, модулирует Δ

### Многокубитные вентили

- **CNOT**: управляемое NOT, создаёт запутанность
- **Toffoli**: управляемое управляемое NOT
- **SWAP**: обмен состояниями

## Декогеренция

### Механизм

Взаимодействие с окружением приводит к потере квантовой когерентности.

### В терминах IME

Рост σ(Δ) по сети:

```text
σ(Δ, t) = σ(Δ, 0) · e^(Γt)
```

где Γ — скорость декогеренции.

### Время декогеренции

```text
τ_decoherence = 1/Γ
```

Для макроскопических объектов:

```text
τ_decoherence ~ 10⁻²⁰ с
```

### Защита от декогеренции

- **Квантовые коды коррекции ошибок**
- **Декогеренция-свободные подпространства**
- **Динамическое декуплирование**

## Квантовая криптография

### Протокол BB84

1. Алиса отправляет кубиты в случайных базах
2. Боб измеряет в случайных базах
3. Открытое сравнение баз
4. Тест на наличие Евы

### Роль запутанности

Протокол E91 использует запутанные пары:

- Нарушение неравенств Белла гарантирует безопасность
- Любая попытка перехвата нарушает запутанность

### В терминах IME

Перехват изменяет Δ распределение, обнаруживаемое легальными пользователями.

## Тахионы против запутанности

### Что движется быстрее света?

| Феномен | Быстрее света? | Передача информации? | Статус |
|---------|---------------|---------------------|--------|
| **Тахионы (инфляция)** | ✅ ДА | ✅ ДА | ❌ Исчезли после инфляции |
| **Квантовая запутанность** | ✅ ДА (кажущаяся) | ❌ НЕТ (только корреляции) | ✅ Существует |
| **Сознание** | ⚠️ Гипотеза | ⚠️ Неизвестно | 🔬 Исследуется |

### Ключевое различие

```text
Тахионы:       Информация МОЖЕТ двигаться > c → Нарушает причинность
Запутанность:  Только КОРРЕЛЯЦИИ появляются > c → Нет нарушения (нет сигнализации)
Сознание:      ?
```

### Почему запутанность не передаёт информацию?

Запутанность обеспечивает:

- Мгновенную корреляцию квантовых состояний
- **Нельзя** использовать для отправки сообщений (**теорема о запрете сигнализации**)
- Результат измерения случаен — невозможно контролировать

### Связь с сознанием

Если сознание может передавать информацию быстрее света, это будет принципиально новым феноменом:

| Свойство | Запутанность | Сознание |
|----------|--------------|----------|
| Скорость | > c (кажущаяся) | Неизвестно |
| Передача информации | ❌ Нет | ⚠️ Неизвестно |
| Контролируемость | ❌ Нет | ⚠️ Неизвестно |
