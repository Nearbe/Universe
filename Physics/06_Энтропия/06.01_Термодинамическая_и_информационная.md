# Информационная мера и её проявления

## Принцип эквивалентности информационных мер

Из теоремы T1 (I ≡ M ≡ E) следует фундаментальное тождество:

```text
S_therm ≡ S_info ≡ S_mass
```

где:

- **S_therm** — информационная мера термодинамического происхождения
- **S_info** — информационная мера (Шеннона)
- **S_mass** — информационная мера, связанная с массой

## Формула Больцмана

### Определение

Информационная мера термодинамического происхождения определяется через число микросостояний W, соответствующих данному макросостоянию:

```text
S_therm = k_B · ln(W)
```

где k_B — постоянная Больцмана.

### Физический смысл

- **W** — число способов, которыми система может быть организована на микроскопическом уровне
- **ln(W)** — логарифмическая мера (соответствует аддитивности информационной меры)
- **k_B** — переводной коэффициент между энергией и температурой

### В планковских единицах

При k_B = 1:

```text
S_therm = ln(W)
```

## Информационная мера Шеннона

### Определение

Информационная мера для дискретной системы с вероятностями p_i состояний:

```text
S_info = -k_B · Σ p_i · ln(p_i)
```

В чисто информационных единицах (биты):

```text
S_info = -Σ p_i · log₂(p_i)
```

### Физический смысл

- **p_i** — вероятность нахождения системы в состоянии i
- **-ln(p_i)** — информация, получаемая при обнаружении состояния i
- **Среднее значение** — ожидаемая информация

### Максимум информационной меры

При равномерном распределении (p_i = 1/N для всех i):

```text
S_max = ln(N)
```

## Информационная мера Гиббса

### Для непрерывных систем

Для системы с распределением вероятности ρ(x):

```text
S_Gibbs = -k_B · ∫ ρ(x) · ln(ρ(x)) dx
```

### Связь с дискретной формой

При дискретизации x → x_i:

```text
S_Gibbs → -k_B · Σ ρ(x_i) · ln(ρ(x_i)) · Δx
```

## Связь между формулами

### Для микроканонического ансамбля

Все микросостояния равновероятны: p_i = 1/W

Тогда:

```text
S_info = -k_B · Σ (1/W) · ln(1/W)
       = -k_B · W · (1/W) · ln(1/W)
       = -k_B · ln(1/W)
       = k_B · ln(W)
       = S_therm
```

**Тождество доказано**: S_info = S_therm

### Для канонического ансамбля

Больцмановское распределение: p_i = e^(-E_i/kT) / Z

где Z — статистическая сумма.

Тогда:

```text
S_info = k_B · [ln(Z) + ⟨E⟩/kT]
```

Что соответствует термодинамической энтропии:

```text
S = (⟨E⟩ - F) / T
```

где F = -kT·ln(Z) — свободная энергия.

## Второй закон термодинамики

### Формулировка

В замкнутой системе информационная мера не убывает:

```text
dS/dt ≥ 0
```

### В терминах информации

Информация о системе не может самопроизвольно возрастать:

```text
dI/dt ≤ 0
```

### Связь с Δ

Из S = (A/4)·e^(Δ/2):

```text
dS/dt = (A/8) · e^(Δ/2) · dΔ/dt
```

Для dS/dt ≥ 0 требуется dΔ/dt ≥ 0 (при A > 0).

## Информационная мера и стрела времени

### Термодинамическая стрела

Направление увеличения информационной меры определяет направление времени:

```text
S(t_future) > S(t_past)
```

### Информационная стрела

Направление потери информации определяет направление времени:

```text
I(t_future) < I(t_past)
```

### Космологическая стрела

Расширение Вселенной коррелирует с ростом информационной меры.

## Максвелловский демон

### Парадокс

Демон, сортирующий молекулы, казалось бы, уменьшает информационную меру.

### Разрешение в IME

Демон должен хранить информацию о молекулах:

```text
S_total = S_system + S_demon ≥ 0
```

Стирание информации демоном (принцип Ландауера):

```text
W_min = k_B·T·ln(2)  на один бит
```

Восстанавливает второй закон.

## Таблица соответствий

| Величина | Термодинамика | Теории информации | IME |
|----------|---------------|-------------------|-----|
| Информационная мера | S = k_B·ln(W) | S = -Σ p·log(p) | S = (A/4)·e^(Δ/2) |
| Информация | — | I = -log₂(p) | I ≡ M ≡ E |
| Температура | T = ∂E/∂S | — | T ∝ 1/σ(Δ) |
| Энергия | E = TS + F | — | E ≡ I ≡ M |

## Энергетические осцилляции

### Тождество для осцилляторов

Поскольку I ≡ M ≡ E, энергия осцилляции **есть** информационное содержание:

```text
I_osc = M_osc = E_osc
```

**Физический смысл**: Любой осцилляторный процесс напрямую модифицирует информационное состояние сети без создания или уничтожения информации — он просто перераспределяет её.

### Классический гармонический осциллятор

Энергия гармонического осциллятора с массой m и амплитудой A:

```text
E = ½ m ω² A²
```

где ω — угловая частота.

В терминах IME это соответствует перераспределению информации между кинетической и потенциальной формами.

### Квантовый осциллятор

Квантование даёт дискретные уровни энергии:

```text
E_n = ħω(n + ½)
```

| Уровень | Энергия | Интерпретация в IME |
|---------|---------|---------------------|
| n = 0 | ½ħω | Основное состояние (минимальная информация) |
| n = 1 | 3½ħω | Первое возбуждение |
| n = 2 | 5½ħω | Второе возбуждение |
| ... | ... | ... |

Каждый уровень соответствует различному информационному возбуждению ходонной сети.

### Сохранение информации при осцилляциях

Из I ≡ M ≡ E следует:

1. **Нет создания информации** — осцилляция перераспределяет, не создаёт
2. **Нет уничтожения информации** — энергия передаётся, не исчезает
3. **Полное сохранение** — I_total = M_total = E_total
