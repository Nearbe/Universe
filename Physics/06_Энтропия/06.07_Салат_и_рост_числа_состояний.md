# Рост числа состояний

## Пример

Представьте процесс приготовления салата на кухне.

1. **Исходное состояние** – имеется n разных ингредиентов (помидор, огурец, лук, перец, оливки, сыр, зелень и т.д.).
2. **Действие** – для каждого ингредиента вы делаете логический выбор: добавить в салат либо не добавить.
3. **Результат** – после обработки всех n ингредиентов число различных возможных салатов равно  

   \[
   \text{Варианты салата} = 2^n .
   \]

Этот рост экспоненциален по числу выполненных решений.

## Краткая формула поясняющая логику

Для системы с n независимыми бинарными выборами число возможных состояний  

\[
W = 2^n,
\]

а мера роста состояний (в планковских единицах)  

\[
\mathcal{G} = \ln W = n \ln 2 .
\]

Таким образом, рост числа состояний возникает непосредственно из правила действия, без необходимости введения отдельной сущности вроде энтропии.
