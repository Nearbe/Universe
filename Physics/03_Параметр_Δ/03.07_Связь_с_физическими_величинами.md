# Связь параметра Δ с физическими величинами

## Связь с информационной энтропией Шеннона

### Двухсостояная система

Для системы с двумя состояниями с вероятностями p и q = 1-p:

Информационная энтропия:

```text
H = -p·log₂(p) - q·log₂(q)
```

### Связь с Δ

Можно показать, что для определённой параметризации:

```text
Δ ∝ log(p/q)
```

Это делает Δ **логарифмическим шансом** (log-odds) отношения между состояниями.

### Физическая интерпретация

- **Δ > 0**: состояние p доминирует
- **Δ < 0**: состояние q доминирует
- **Δ = 0**: состояния равновероятны (p = q = 0.5)

## Связь с информацией Фишера

### Определение

Информация Фишера для параметра θ в распределении p(x|θ):

```text
F(θ) = E[(∂log p/∂θ)²]
```

### Для гауссовского распределения

Для гауссовского распределения с дисперсией, зависящей от Δ:

```text
F ∝ 1/(cosh²(Δ))
```

### Физический смысл

Это показывает, что **информативность измерений**:

- **Минимальна** при больших |Δ|
- **Максимальна** при Δ = 0

## Связь с относительной энтропией (Kullback-Leibler divergence)

### Определение

Kullback-Leibler divergence между двумя гауссовскими распределениями с дисперсиями σ₁² и σ₂²:

```text
D_KL = ½[(σ₁²/σ₂²) - 1 - ln(σ₁²/σ₂²)]
```

### При идентификации σ₁²/σ₂² = e^Δ

```text
D_KL = ½[e^Δ - 1 - Δ]
```

### Свойства

- **D_KL ≥ 0** всегда (неравенство Гиббса)
- **D_KL = 0** только при Δ = 0
- Это мера «расстояния» между распределениями

## Связь с термодинамической температурой

### Больцмановское распределение

Для двухуровневой системы с разностью энергий ΔE:

```text
p₁/p₂ = exp(-ΔE/(kT))
```

### Идентификация с Δ

Если отождествить:

```text
Δ = -ΔE/(kT)
```

Тогда:

- **Δ → +∞**: T → 0⁺ (полное основное состояние)
- **Δ → -∞**: T → 0⁻ (инверсная популяция)
- **Δ = 0**: T → ∞ (равномерное распределение)

### Физическая интерпретация

Параметр Δ можно рассматривать как **безразмерную обратную температуру**.

## Связь с квантовой вероятностью

### Правило Борна

Квадрат модуля волновой функции даёт вероятность:

```text
P = |Ψ|² = Re² + Im²
```

### Выражение через Δ

Можно выразить Re и Im через P и Δ:

```text
Re = √P · cosh(Δ/2) · ε_Re
Im = √P · sinh(Δ/2) · ε_Im
```

### Проверка

Подставляя:

```text
P = P·[cosh²(Δ/2) + sinh²(Δ/2)] = P·cosh(Δ)
```

Требуется нормировка: cosh(Δ) = 1, что выполняется только при Δ = 0.

### Корректное соотношение

С фазовым параметром φ:

```text
Re = |Ψ| · cos(φ)
Im = |Ψ| · sin(φ)
Δ = ln(|cos(φ)|) - ln(|sin(φ)|) = ln(|cot(φ)|)
```

Тогда вероятность сохраняется тождественно:

```text
P = |Ψ|²·[cos²(φ) + sin²(φ)] = |Ψ|²
```

## Связь с энергией и массой

### Из тождества I ≡ M ≡ E

Поскольку информация эквивалентна энергии и массе:

```text
Δ ∝ ln(E_re/E_im) ∝ ln(M_re/M_im)
```

где E_re, E_im — энергия, связанная с реальной и мнимой компонентами.

### Для релятивистских частиц

Для частицы с энергией E и массой m:

```text
E² = p²c² + m²c⁴
```

В терминах IME это можно связать с Δ через волновую функцию.

## Связь с частотой и периодом

### Для осцилляторов

Для гармонического осциллятора с частотой ω:

```text
Ψ(t) = A·e^(iωt) = A·(cos(ωt) + i·sin(ωt))
```

Тогда:

```text
Δ(t) = ln(|cos(ωt)|) - ln(|sin(ωt)|) = ln(|cot(ωt)|)
```

### Периодичность

Δ периодичен с периодом π/ω:

```text
Δ(t + π/ω) = Δ(t)
```
