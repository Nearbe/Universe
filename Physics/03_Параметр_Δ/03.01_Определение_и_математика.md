# Определение и математическая форма параметра Δ

## Основное определение

Параметр отклонения Δ quantifies the relative magnitude of the real and imaginary components of the wave function через разность их натуральных логарифмов:

```text
Δ = ln(|Re|) - ln(|Im|) ∈ ℝ ∪ {±∞}
```

## Эквивалентные формы

### При Re ≠ 0 и Im ≠ 0

```text
Δ = ln(|Re|/|Im|)
```

### Предпочтительная форма

Форма **разности логарифмов** предпочтительнее для избежания сингулярностей:

- При Im = 0: ln(|Re|) - ln(0) = ln(|Re|) - (-∞) = +∞
- При Re = 0: ln(0) - ln(|Im|) = -∞ - ln(|Im|) = -∞

## Онтологические интерпретации предельных случаев

### Δ → +∞ (Сингулярность)

**Условие**: Im → 0 при fixed Re ≠ 0

**Математика**:

```text
ln(|Im|) → -∞
Δ = ln(|Re|) - (-∞) = +∞
```

**Физическая интерпретация**:

- Полная предсказуемость
- Исчезновение мнимой части как информационного понятия
- Состояние чёрной дыры

### Δ → -∞ (Тёмная энергия)

**Условие**: Re → 0 при fixed Im ≠ 0

**Математика**:

```text
ln(|Re|) → -∞
Δ = (-∞) - ln(|Im|) = -∞
```

**Физическая интерпретация**:

- Доминирование скрытой информации
- Ультрафиолетовая турбулентность
- Состояние тёмной энергии

### Δ = 0 (Равновесие)

**Условие**: |Re| = |Im|

**Математика**:

```text
ln(|Re|) - ln(|Im|) = 0
```

**Физическая интерпретация**:

- Баланс между видимой и скрытой информацией
- Обычное квантовое состояние

### Δ = 0 и Re = Im = 0 (Информационная пустота)

**Условие**: Обе части волновой функции равны нулю

**Математика**:

```text
|Ψ| = 0
I = M = E = 0
```

**Физическая интерпретация**:

- Состояние вакуума без квантовых флуктуаций
- Истинная информационная пустота

## Диапазон и связность

### Два режима Δ

**1. Обычная физика (5% видимая Вселенная):**

```text
Δ ∈ [-1, +1]
```

- Стабильные вычисления
- Математика не ломается
- Все наблюдаемые состояния в этом диапазоне

**2. Фазовый переход (другой масштаб):**

```text
Δ ∈ ℝ  (все конечные вещественные числа)
```

- Сингулярность (Δ → +∞): Re → ∞, Im → 0 — **предел, не точка**
- Тёмная энергия (Δ → -∞): Re → 0, Im → ∞ — **предел, не точка**
- Экстремальные состояния за пределами обычной физики

**Важно:** ±∞ — это **предельные состояния**, не достижимые значения!

**Полный диапазон:**

```text
Δ ∈ ℝ  (все конечные вещественные числа)
```

Это полная вещественная прямая **без** точек на бесконечности.

## Связь с другими величинами

### С модулем и фазой

Представляя волновую функцию в полярной форме:

```text
Ψ = |Ψ| · e^(iθ) = |Ψ| · (cosθ + i·sinθ)
```

Тогда:

```text
Re = |Ψ| · cosθ
Im = |Ψ| · sinθ
```

Подставляя в определение Δ:

```text
Δ = ln(|Ψ| · |cosθ|) - ln(|Ψ| · |sinθ|)
  = ln(|cosθ|) - ln(|sinθ|)
  = ln(|cotθ|)
```

**Важно**: Δ зависит **только от фазы θ**, а не от амплитуды |Ψ|.

### С вероятностью

Квадрат модуля волновой функции даёт вероятность:

```text
P = |Ψ|² = Re² + Im²
```

Можно выразить Re и Im через P и Δ, но требуется осторожность с квадрантами комплексной плоскости.
