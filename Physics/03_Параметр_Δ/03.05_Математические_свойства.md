# Математические свойства параметра Δ

## Диапазон и связность

### Полная вещественная прямая

Δ принимает все значения от -∞ до +∞:

```text
Δ ∈ ℝ ∪ {-∞, +∞}
```

Это делает его полной вещественной прямой с добавленными точками на бесконечности.

### Топологические свойства

- **Непрерывность**: Δ непрерывен везде, кроме Re = Im = 0
- **Дифференцируемость**: Δ дифференцируем при Re, Im ≠ 0
- **Монотонность**: Δ монотонно растёт с увеличением |Re|/|Im|

## Производные и дифференциалы

### Первый дифференциал

Для анализа малых колебаний около равновесия:

```text
dΔ = d(ln(|Re|)) - d(ln(|Im|)) = dRe/Re - dIm/Im
```

Это показывает, что Δ чувствителен к **относительным изменениям** реальной и мнимой частей.

### Частные производные

```text
∂Δ/∂Re = 1/Re
∂Δ/∂Im = -1/Im
```

### Второй дифференциал

```text
d²Δ = -dRe²/Re² + dIm²/Im²
```

## Связь с амплитудой и фазой

### Полярное представление

Представляя волновую функцию в полярной форме:

```text
Ψ = |Ψ| · e^(iθ) = |Ψ| · (cosθ + i·sinθ)
```

Тогда:

```text
Re = |Ψ| · cosθ
Im = |Ψ| · sinθ
```

### Выражение Δ через фазу

Подставляя в определение Δ:

```text
Δ = ln(|Ψ| · |cosθ|) - ln(|Ψ| · |sinθ|)
  = ln(|cosθ|) - ln(|sinθ|)
  = ln(|cotθ|)
```

### Ключевое свойство

**Δ зависит только от фазы θ, а не от амплитуды |Ψ|**.

Это подтверждает его роль как меры **относительного баланса** между компонентами.

### Обратное преобразование

Из Δ можно выразить фазу:

```text
θ = arccot(e^Δ)
```

или

```text
θ = arctan(e^(-Δ))
```

## Периодичность и квадранты

### Периодичность по фазе

Поскольку cot(θ + π) = cot(θ):

```text
Δ(θ + π) = Δ(θ)
```

Δ периодичен с периодом π по фазе.

### Квадранты комплексной плоскости

| Квадрант | Re | Im | Знак Δ |
|----------|-----|-----|--------|
| I | + | + | зависит от отношения |
| II | - | + | зависит от отношения |
| III | - | - | зависит от отношения |
| IV | + | - | зависит от отношения |

Знак Δ определяется отношением |Re|/|Im|, а не знаками Re и Im.

## Гиперболическое представление

### Через гиперболические функции

Можно выразить Re и Im через M и Δ:

```text
Re = √M · cosh(Δ/2) · ε_Re
Im = √M · sinh(Δ/2) · ε_Im
```

где ε_Re, ε_Im = ±1 — знаки.

### Проверка тождества

```text
Re² + Im² = M·cosh²(Δ/2) + M·sinh²(Δ/2)
          = M·[cosh²(Δ/2) + sinh²(Δ/2)]
          = M·cosh(Δ)
```

Для согласованности требуется нормировочный множитель.

### Корректное представление

С нормировкой:

```text
Re = √[M / cosh(Δ)] · cosh(Δ/2) · ε_Re
Im = √[M / cosh(Δ)] · sinh(Δ/2) · ε_Im
```

Тогда Re² + Im² = M тождественно.

## Преобразования

### Масштабная инвариантность

При масштабировании Ψ → kΨ:

```text
Δ' = ln(|kRe|) - ln(|kIm|) = ln(|Re|) - ln(|Im|) = Δ
```

Δ **инвариантен** относительно однородного масштабирования.

### Инверсия

При инверсии Ψ → 1/Ψ:

```text
Δ' = ln(|1/Re|) - ln(|1/Im|) = -ln(|Re|) + ln(|Im|) = -Δ
```

Инверсия меняет знак Δ.

### Комплексное сопряжение

При сопряжении Ψ → Ψ*:

```text
Δ' = ln(|Re|) - ln(|-Im|) = ln(|Re|) - ln(|Im|) = Δ
```

Сопряжение **не меняет** Δ.

### Частота как функция Δ

Фундаментальная частота тиканья ходона связана с Δ через:

```text
f = e^(-Δ)
```

**Физический смысл**:

- При **Δ → +∞**: f → 0 (время замедляется, чёрная дыра)
- При **Δ = 0**: f = 1 (баланс, наблюдаемая Вселенная)
- При **Δ → -∞**: f → ∞ (время ускоряется, тёмная энергия)

Это соотношение показывает, как увеличение массы/энергии (рост Δ) приводит к космологическому красному смещению фундаментального тиканья.

### Связь с периодом

Субъективный период (время одного тика):

```text
t_subj = 1/f = e^Δ
```

При приближении к чёрной дыре (Δ → +1):

```text
t_subj → e^1 ≈ 2.718
```

Внешний наблюдатель видит внутреннюю динамику замедленной в e^Δ раз.

## Предельное поведение

### При Δ → +∞

```text
|Re|/|Im| → ∞
|Re| ≫ |Im|
θ → 0 или π
```

### При Δ → -∞

```text
|Re|/|Im| → 0
|Im| ≫ |Re|
θ → π/2 или 3π/2
```

### При Δ = 0

```text
|Re| = |Im|
θ = π/4, 3π/4, 5π/4, 7π/4
```
