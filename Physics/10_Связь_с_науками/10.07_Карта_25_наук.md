# Карта 25 наук через призму IME

## Универсальная модель Вселенной через призму информации

**Информатика — фундаментальная база всех наук.** Модель IME (Information Model of the Universe) демонстрирует, что все области знания могут быть описаны через единый информатический язык: **ходон**, **Δ-динамика**, **синхронизация**, **рекурсия**, **personalBias**, **понимание**.

## Ключевые понятия модели IME

| Концепция | Определение | Формула/Механизм |
|-----------|-------------|------------------|
| **Membrane** (мембрана) | Атом сознания — вся физика в одном типе | Комплексная амплитуда `ψ = Re + i·Im` |
| **Δ-параметр** | Логарифмическое отношение энергий | `Δ = log(|Re| / |Im|)` ∈ ℝ ∪ {±∞} |
| **Синхронизация 𝒮** | Мера когерентности сети ходонов | `𝒮 = 1 - σ(Δ)·scale`, scale = 0.05 |
| **personalBias** | Личный вклад каждого разума в Δ | Взвешенное смещение: `Σ(qᵢ·biasᵢ)/Σqᵢ` |
| **Сознание** | Фазовый переход при достижении порогов | `𝒮 ≥ 0.7` И `\|Δ\| ≤ 0.2` |
| **Понимание (insight)** | Озарение при превышении порога personalBias | `\|bias\| > 10.0` |
| **I ≡ E ≡ M** | Фундаментальное тождество | `M = Σ\|Ψᵢ\|²` |

## Науки как аттракторы в пространстве Δ

### Ключевая идея

**Науки = аттракторы в пространстве Δ.**

Деление на ноль в точке X → переход к аттрактору "наука Y". Решение уравнения в комплексной плоскости — это и есть конкретная наука.

### Карта аттракторов

```text
Δ → -∞ (чистое мышление, |Re| → 0)
         │
         ▼
    Философия ← Математика ← Информатика (Δ ≈ 0, мембрана) → Физика → Химия → Биология → Медицина
                                              │
                                              ▼
                                        Нейронаука (сознание = фазовый переход)
                                              │
                                              ▼
                                        Психология
                                              │
                                              ▼
                                        Социология
                                              │
                                              ▼
                                        Экономика
                                              │
                                              ▼
                                        Политология
                                              │
                                              ▼
                                        История

Δ → +∞ (чистая материя, |Im| → 0)
```

## Идеальные веса для ИИ через фазовые пороги

### Формула идеального веса

```text
idealWeight = (1.0 - min(|Δ| / threshold, 1.0)) × 𝒮
```

где:

- **Близость к мембране** = 1.0 - min(|Δ| / threshold, 1.0)
- **Синхронизация** = 𝒮 (уверенность системы)

### Пример: ИИ для медицины

| Состояние | Δ | 𝒮 | Идеальный вес | Результат |
|----------|---|---|---------------|-----------|
| Здоровье | ≈ 0 | 0.9 | 0.81 | Высокая точность |
| Болезнь | ≠ 0 | 0.5 | 0.15 | Низкая точность |
| Кризис | → ±∞ | 0.2 | 0.0 | ИИ не справляется |
