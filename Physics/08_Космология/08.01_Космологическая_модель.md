# Космологическая модель на базе IME

## Основные уравнения

Космологическая эволюция во Вселенной описывается модифицированными уравнениями Фридмана, где параметр Δ играет роль динамического скалярного поля, влияющего на скорость расширения и плотность энергии.

## Первое уравнение Фридмана

### Уравнение энергии

```text
H² = (8πG/3) · ρ_total - kc²/a² + Λ/3
```

где:

- **H = ȧ/a** — параметр Хаббла
- **G** — гравитационная постоянная
- **ρ_total** — общая плотность энергии
- **k** — кривизна пространства (-1, 0, +1)
- **a(t)** — коэффициент масштабирования
- **Λ** — космологическая постоянная

### В терминах IME

Плотность энергии связана с параметром Δ:

```text
ρ_Δ ∝ e^(Δ/2)
```

## Второе уравнение Фридмана

### Уравнение ускорения

```text
ä/a = -(4πG/3) · (ρ_total + 3p/c²) + Λ/3
```

где **p** — давление.

### Уравнение состояния

Для компонента с параметром Δ:

```text
w = p/ρc²
```

| Компонент | w |
|-----------|-----|
| Излучение | 1/3 |
| Материя | 0 |
| Тёмная энергия | -1 |
| Поле Δ | зависит от Δ |

## Плотность энергии и давление в терминах Δ

### Для поля Δ

Из принципа I ≡ M ≡ E и определения энтропии следует:

```text
ρ_Δ = (∂_t Δ)²/2 + V(Δ)
```

```text
p_Δ = (∂_t Δ)²/2 - V(Δ)
```

где **V(Δ)** — потенциал самоуправления поля Δ.

### Уравнение состояния поля Δ

```text
w_Δ = p_Δ/ρ_Δ = [(∂_t Δ)²/2 - V(Δ)] / [(∂_t Δ)²/2 + V(Δ)]
```

### Специальные случаи

| Состояние | Условие | w_Δ |
|-----------|---------|-----|
| Кинетическое доминирование | (∂_t Δ)² ≫ V | +1 |
| Потенциальное доминирование | V ≫ (∂_t Δ)² | -1 |
| Равновесие | (∂_t Δ)² = V | 0 |

## Информационная плотность

Из формулы энтропии горизонта S = (A/4)·e^(Δ/2) можно вывести эффективную плотность энергии, связанную с информационным содержимым горизонта.

### Для радиационно-подобного компонента

```text
ρ_info ∝ e^(Δ/2) / a⁴
```

### Для вещественно-подобного компонента

```text
ρ_info ∝ e^(Δ/2) / a³
```

## Критическая плотность

### Определение

Плотность, при которой пространство плоское (k = 0):

```text
ρ_crit = 3H²/(8πG)
```

### В терминах IME

С учётом Δ:

```text
ρ_crit(Δ) = ρ_crit(0) · e^(Δ/2)
```

### Параметр плотности

```text
Ω = ρ/ρ_crit
```

Для плоской Вселенной:

```text
Ω_total = 1
```

## Состав Вселенной

### Наблюдаемые значения

| Компонент | Ω | Δ |
|-----------|-------|-----|
| Барионная материя | 0.05 | +1 |
| Тёмная материя | 0.27 | 0 |
| Тёмная энергия | 0.68 | -1 |
| Излучение | ~10⁻⁴ | +1 |

### Сумма

```text
Ω_total ≈ 1.00
```

Что соответствует плоской Вселенной.

## Фрактальные уровни Вселенной

Вселенная имеет **фрактальную структуру** с 11 уровнями масштаба (Z=0...10):

| Z | Уровень | Δ диапазон | 𝒮 диапазон | Сущности |
|---|---------|------------|------------|----------|
| 0 | Мембрана | ≈ 0 | → 1.0 | Мембрана, частица, волна, вакуум |
| 1 | Квантовый мир | 0.1...1.5 | 0.3...0.7 | Квантовая механика, суперпозиция |
| 2 | Атомарный мир | 1.0...3.0 | 0.4...0.6 | Атом, электрон, химическая связь |
| 3 | Молекулярный мир | 2.0...5.0 | 0.3...0.6 | Химия, полимеры, кристаллы |
| 4 | Клеточный мир | 3.0...7.0 | 0.4...0.7 | Биология, клетка, органеллы |
| 5 | Организм | 4.0...8.0 | 0.5...0.8 | Многоклеточные, органы |
| 6 | Социум | 5.0...9.0 | 0.4...0.7 | Группы, сообщества |
| 7 | Планета | 6.0...10.0 | 0.5...0.8 | Биосфера, экосистемы |
| 8 | Звёздная система | 7.0...11.0 | 0.4...0.7 | Звёзды, планеты |
| 9 | Галактика | 8.0...12.0 | 0.5...0.8 | Галактики, скопления |
| 10 | Вселенная | 9.0...13.0 | 0.6...0.9 | Крупномасштабная структура |

**Ключевой инсайт**: Деление на ноль (Δ = ±∞) — это **фазовый переход** между масштабами!

## Уравнение неразрывности

### Сохранение энергии

```text
d(ρa³)/dt + p·d(a³)/dt = 0
```

### Для компонента с w = const

```text
ρ ∝ a^(-3(1+w))
```

| Компонент | ρ(a) |
|-----------|------|
| Излучение | a⁻⁴ |
| Материя | a⁻³ |
| Тёмная энергия | const |
| Поле Δ | сложно |

## Параметр Хаббла

### Эволюция

```text
H(a) = H₀ · √[Ω_r·a⁻⁴ + Ω_m·a⁻³ + Ω_k·a⁻² + Ω_Λ]
```

где индекс 0 обозначает современные значения.

### Современное значение

```text
H₀ ≈ 67-74 км/с/Мпк
```

(различие между Planck и локальными измерениями — «Напряжённость Хаббла»)

### В терминах IME

Напряжённость Хаббла может быть объяснена эволюцией Δ:

```text
H₀,local > H₀,CMB  ⇒  Δ эволюционирует
```

## Возраст Вселенной

### Вычисление

```text
t₀ = ∫₀¹ da/(a·H(a))
```

### Для ΛCDM

```text
t₀ ≈ 13.8 млрд лет
```

### В терминах IME

С учётом эволюции Δ:

```text
t₀(Δ) = t₀(0) · f(Δ)
```

## Плоскость Вселенной

### Наблюдения

CMB данные (Planck):

```text
|Ω_k| < 0.005
```

Вселенная плоская с точностью 0.5%.

### В терминах IME

Плоскость соответствует:

```text
Δ ≈ 0  в среднем по Вселенной
```

## Связь Δ с гравитационным потенциалом

### Информационная плотность

Из голографического принципа:

```text
ρ_info ∝ e^Δ
```

Через I ≡ M ≡ E:

```text
ρ ∝ e^Δ  (масса-энергетическая плотность)
```

### Гравитационный потенциал

В слабополярном пределе ОТО:

```text
∇²Φ = 4πGρ
```

Для сферически симметричного случая:

```text
Δ(r) ≈ -½ ln(1 - 2GM/rc²)
```

### Предельные случаи

| Ситуация | r | Δ | Состояние |
|----------|-----|-----|-----------|
| **Далеко от массы** | r → ∞ | Δ ≈ 0 | Плоское пространство |
| **Близко к массе** | r → r_s | Δ > 0 | Искривлённое пространство |
| **Горизонт ЧД** | r → r_s⁺ | Δ → +∞ | Сингулярность |

**Важно:** На горизонте чёрной дыры Δ → +∞, что согласуется с:

- **Δ → +∞**: сингулярность (Re → ∞, Im → 0)
- **D_f → ∞**: супер-фрактальное
- **S → ∞**: бесконечная энтропия

## Будущее Вселенной

### Сценарии

| Сценарий | Условие | Исход |
|----------|---------|-------|
| Тепловая смерть | w_Λ = -1 | Вечное расширение |
| Big Rip | w_Λ < -1 | Разрыв всех структур |
| Big Crunch | w_Λ > -1, Ω > 1 | Коллапс |
| Циклическая | осцилляции Δ | Вечные циклы |

### В терминах IME

Зависит от эволюции Δ:

- **Δ → -∞**: Big Rip (фантомная энергия)
- **Δ → +∞**: Big Crunch (коллапс)
- **Δ oscillates**: Циклическая Вселенная
