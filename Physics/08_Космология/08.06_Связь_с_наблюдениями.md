# Связь с наблюдениями: CMB, BAO, линзирование, сверхновые

## Космический микроволновый фон (CMB)

### Открытие

Открыт Пензиасом и Вильсоном (1965):

```text
T_CMB = 2.725 K
```

### Физическая природа

Излучение от поверхности последнего рассеяния (z ≈ 1100).

### Анизотропии

Температурные флуктуации:

```text
ΔT/T ≈ 10⁻⁵
```

### В терминах IME

Анизотропии связаны с пертурбациями Δ:

```text
ΔT/T ∝ δΔ
```

### Спектр мощности

Угловой спектр мощности C_ℓ:

```text
C_ℓ = ⟨|a_ℓm|²⟩
```

где a_ℓm — коэффициенты разложения по сферическим гармоникам.

### Пики спектра

| Пик | ℓ | Физика |
|-----|-----|--------|
| 1-й | ~200 | Звуковой горизонт |
| 2-й | ~500 | Барионное подавление |
| 3-й | ~800 | Тёмная материя |

### Ограничения на параметры

Planck (2018):

```text
Ω_b h² = 0.0224 ± 0.0001
Ω_c h² = 0.120 ± 0.001
n_s = 0.9649 ± 0.0042
r < 0.036
```

### В терминах IME

CMB ограничивает Δ на z ≈ 1100:

```text
Δ_CMB ≈ 0 ± 0.1
```

## BAO (Baryon Acoustic Oscillations)

### Физическая природа

Звуковые волны в ранней плазме оставляют характерный масштаб в распределении галактик.

### Звуковой горизонт

На момент рекомбинации:

```text
r_s ≈ 150 Мпк
```

### Наблюдения

Измеряется корреляционная функция галактик:

```text
ξ(r) = ⟨δ(x)δ(x+r)⟩
```

Пик при r ≈ r_s.

### В терминах IME

BAO чувствительны к:

- Скорости звука c_s(Δ)
- Расстоянию до поверхности последнего рассеяния

### Ограничения

DESI, eBOSS:

```text
Ω_Λ = 0.69 ± 0.02
w = -1.0 ± 0.1
```

## Гравитационное линзирование

### Эффект

Массивные объекты искривляют свет от фоновых источников.

### Типы линзирования

| Тип | Масштаб | Применение |
|-----|---------|------------|
| Сильное | Скопления | Масса скоплений |
| Слабое | Крупный масштаб | Распределение материи |
| Микролинзирование | Звёзды | Тёмная материя, экзопланеты |

### В терминах IME

Линзирование зависит от гравитационного потенциала:

```text
Φ ∝ e^(Δ/2)
```

### Слабое линзирование

Искажение форм галактик:

```text
γ = (a - b)/(a + b)
```

где a, b — оси эллипса.

### Ограничения

KiDS, DES:

```text
S_8 = σ_8 √(Ω_m/0.3) = 0.76 ± 0.02
```

## Сверхновые типа Ia

### Стандартные свечи

Сверхновые типа Ia имеют почти одинаковую пиковую светимость:

```text
M_B ≈ -19.3
```

### Измерение расстояний

Модуль расстояния:

```text
μ = m - M = 5 log₁₀(d_L/10 пк)
```

### Ускоренное расширение

Наблюдения показывают:

```text
q₀ ≈ -0.55  (ускорение)
```

### В терминах IME

Сверхновые чувствительны к истории расширения a(t), которая зависит от эволюции Δ.

### Ограничения

Pan-STARRS, DES:

```text
Ω_Λ = 0.70 ± 0.03
w = -1.03 ± 0.05
```

## Комбинированные ограничения

### Concordance модель

Комбинация всех наблюдений даёт ΛCDM:

| Параметр | Значение |
|----------|----------|
| Ω_b | 0.05 |
| Ω_c | 0.27 |
| Ω_Λ | 0.68 |
| H₀ | 67-74 км/с/Мпк |
| n_s | 0.965 |
| σ_8 | 0.81 |

### В терминах IME

Все наблюдения согласуются с эволюцией Δ:

```text
Δ: +∞ → 0 → -0.3 → -1
```

## Напряжённость Хаббла

### Проблема

Локальные измерения H₀ дают:

```text
H₀,local = 73.0 ± 1.0 км/с/Мпк  (SH0ES)
```

CMB даёт:

```text
H₀,CMB = 67.4 ± 0.5 км/с/Мпк  (Planck)
```

Расхождение 4-5σ.

### Возможные решения в IME

1. **Эволюция Δ**: Δ менялось быстрее, чем предполагается
2. **Ранняя тёмная энергия**: дополнительный компонент при z > 1000
3. **Взаимодействующая тёмная материя**: обмен с тёмной энергией

### Статус

Проблема остаётся нерешённой, требует новых данных.
