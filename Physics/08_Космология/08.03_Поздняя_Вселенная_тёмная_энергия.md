# Поздняя Вселенная и тёмная энергия

## Тёмная энергия как состояние Δ → -∞

Состояние тёмной энергии соответствует пределу **Δ → -∞**, где:

- Информационная энтропия S → 0
- Давление p → -ρ (уравнение состояния фантомной энергии)
- Вызывает ускоренное расширение Вселенной

## Уравнение состояния тёмной энергии

### Из формулы энтропии

При Δ → -∞:

```text
S = (A/4) · e^(Δ/2) → 0
```

### Эффективное уравнение состояния

Для компонента Δ:

```text
w_Δ = p_Δ/ρ_Δ = [(∂_t Δ)²/2 - V(Δ)] / [(∂_t Δ)²/2 + V(Δ)]
```

### При доминировании потенциальной энергии

Если V(Δ) доминирует над кинетической энергией:

```text
w_Δ → -1
```

Что соответствует космологической постоянной.

### Приближённая формула для малых Δ

Для малых отклонений Δ и медленной эволюции (∂_t Δ ≈ 0):

```text
w_Δ ≈ -1 + Δ/3  (приближение!)
```

**Важно:** Это приближение работает только при:

- \|Δ\| ≪ 1
- V(Δ) ≫ (∂_t Δ)²/2
- Медленная эволюция поля Δ

**Точная формула:**

```text
w_Δ = [(∂_t Δ)²/2 - V(Δ)] / [(∂_t Δ)²/2 + V(Δ)]
```

## Экспоненциальный потенциал

### Форма потенциала

Рассматривая экспоненциальный потенциал:

```text
V(Δ) = V₀ e^(-λΔ)
```

### Поведение при Δ → -∞

- При **λ > 0**: потенциал растёт экспоненциально
- При **λ < 0**: потенциал падает

### Квинтэссенция

При λ > 0 получается отслеживающее решение с постоянным w_Δ > -1:

```text
w_Δ = (λ² - 3) / (λ² + 3)
```

### Фантомная энергия

При λ < 0 получается поведение фантомной энергии:

```text
w_Δ < -1
```

## Ускоренное расширение

### Наблюдения сверхновых

Сверхновые типа Ia показывают ускоренное расширение:

```text
q₀ = -äa/ȧ² ≈ -0.55
```

где q₀ — параметр замедления.

### В терминах IME

Ускорение требует:

```text
ρ + 3p < 0  ⇒  w < -1/3
```

Для тёмной энергии:

```text
w_Λ ≈ -1
```

## Плотность тёмной энергии

### Наблюдаемое значение

```text
ρ_Λ ≈ 6 × 10⁻¹⁰ Дж/м³
```

Или в планковских единицах:

```text
ρ_Λ ≈ 10⁻¹²³ M_Planck⁴
```

### В терминах IME

Связано с вакуумным значением Δ:

```text
ρ_Λ ∝ e^(⟨Δ⟩_vacuum/2)
```

## Проблема космологической постоянной

### Формулировка проблемы

Квантовая теория поля предсказывает:

```text
ρ_vacuum,QFT ≈ 10⁷¹ ГэВ⁴
```

Наблюдается:

```text
ρ_Λ,obs ≈ 10⁻⁴⁷ ГэВ⁴
```

Расхождение на **120 порядков**!

### Решение в IME

Малое наблюдаемое значение Λ_obs ≈ 10⁻¹²³ l_P⁻² может быть объяснено как результат динамической настройки параметра Δ.

### Механизм настройки

Рассчитывая эффективную космологическую постоянную:

```text
Λ_eff = Λ_bare + ΔΛ(Δ)
```

где ΔΛ(Δ) — вклад от поля Δ.

Если выбрать потенциал V(Δ) такой, что его минимум соответствует очень малому эффективному Λ:

```text
V(Δ_min) ≈ 10⁻¹²³ M_Planck⁴
```

Тогда проблема настройки сведена к выбору начальных условий для поля Δ.

### Динамическая компенсация

Если поле Δ эволюционирует к состоянию Δ → -∞, то его вклад в энергию может обращаться в нуль или становиться отрицательным, компенсируя баровое значение Λ_bare.

## Квинтэссенция vs космологическая постоянная

### Квинтэссенция

Динамическое поле с w ≠ -1:

| Свойство | Значение |
|----------|----------|
| w | > -1 (обычно -0.9 до -0.5) |
| Эволюция | ρ_Λ(a) меняется |
| Звук | c_s² = 1 |
| Кластеризация | Нет (на больших масштабах) |

### Космологическая постоянная

Постоянная энергия вакуума:

| Свойство | Значение |
|----------|----------|
| w | = -1 (точно) |
| Эволюция | ρ_Λ = const |
| Звук | не определён |
| Кластеризация | Нет |

### Наблюдательные ограничения

Planck + BAO + SNe:

```text
w = -1.03 ± 0.03
```

Согласуется с Λ (w = -1).

## Фантомная энергия

### Определение

Состояние с w < -1:

```text
w_phantom < -1
```

### Свойства

- Плотность энергии **растёт** со временем
- Приводит к **Big Rip** (разрыву всех структур)

### Время до Big Rip

Для постоянного w < -1:

```text
t_rip - t₀ ≈ 2/(3|1+w|H₀)
```

При w = -1.1:

```text
t_rip ≈ 200 млрд лет
```

### В терминах IME

Соответствует Δ → -∞ с определённой динамикой.

## Наблюдательные тесты

### 1. Сверхновые типа Ia

Измеряют историю расширения a(t).

### 2. BAO (Baryon Acoustic Oscillations)

Стандартная линейка в распределении галактик.

### 3. Слабое линзирование

Искажение форм галактик гравитацией.

### 4. Кластеризация галактик

Рост структуры чувствителен к w.

### 5. CMB

Положение пиков чувствительно к Ω_Λ.
