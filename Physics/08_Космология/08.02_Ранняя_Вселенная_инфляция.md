# Ранняя Вселенная и инфляция

## Инфляционный сценарий с полем Δ

Поле Δ может играть роль **инфлатона**, driving период ускоренного расширения в ранней Вселенной.

## Потенциал инфлатона

### Квадратичный потенциал

Простейший выбор:

```text
V(Δ) = ½ m_Δ² Δ²
```

где m_Δ — масса поля Δ.

### Другие варианты

| Потенциал | Форма | Особенности |
|-----------|-------|-------------|
| Квадратичный | ½m²Δ² | Простой, хаотическая инфляция |
| Экспоненциальный | V₀e^(-λΔ) | Квинтэссенция |
| Hilltop | V₀[1 - (Δ/μ)^p] | Малое поле |
| Starobinsky | V₀(1 - e^(-Δ/μ))² | R² инфляция |

## Уравнение движения

### Уравнение Клейна-Гордона в расширяющейся Вселенной

```text
∂_t² Δ + 3H ∂_t Δ + m_Δ² Δ = 0
```

где **3H ∂_t Δ** — член трения от расширения Вселенной.

### Режим медленного скольжения

Когда потенциальная энергия доминирует:

```text
∂_t² Δ ≪ 3H ∂_t Δ
∂_t Δ ≪ V
```

Уравнение упрощается:

```text
3H ∂_t Δ ≈ -m_Δ² Δ
```

## Ускоренное расширение

### Условие инфляции

```text
ä > 0  ⇒  ρ + 3p < 0
```

Для поля Δ:

```text
w_Δ < -1/3
```

### Решение

Во время медленного скольжения:

```text
H² ≈ (8πG/3) · V(Δ) = (4πG/3) m_Δ² Δ²
```

Масштабный фактор растёт экспоненциально:

```text
a(t) ∝ exp(Ht)
```

с почти постоянным H.

## Число экспансий

### Определение

Число e-кратных расширений за период инфляции:

```text
N_exp = ∫ H dt
```

### Для квадратичного потенциала

```text
N_exp ≈ (1/(2m_Δ²)) · (Δ_i² - Δ_f²)
```

где Δ_i, Δ_f — начальные и конечные значения поля Δ.

### Требуемое значение

Для решения проблем горизонта и однородности:

```text
N_exp ≳ 60
```

### Связь с параметрами

Это определяет связь между массой поля Δ и начальным смещением:

```text
m_Δ · Δ_i ≳ 10 (в планковских единицах)
```

## Квантовые флуктуации

### Происхождение пертурбаций

Квантовые флуктуации поля Δ во время инфляции порождают пертурбации метрики, которые становятся семенами для формирования крупномасштабной структуры.

### Спектр скалярных пертурбаций

```text
P_ζ(k) = (H²/(2π·∂_t Δ))² |_{k=aH}
```

где вычисление производится на момент выхода горизонта k = aH.

### Для медленного скольжения

Из уравнения движения:

```text
∂_t Δ ≈ -m_Δ² Δ / (3H)
```

Получаем:

```text
P_ζ(k) ≈ (9H⁴)/(2π² m_Δ⁴ Δ²)
```

Используя H² ≈ (4πG/3) m_Δ² Δ²:

```text
P_ζ(k) ≈ (24π² G²) / Δ²
```

### Амплитуда пертурбаций

Амплитуда обратно пропорциональна Δ² во время инфляции.

Наблюдаемое значение:

```text
P_ζ ≈ 2.1 × 10⁻⁹
```

## Спектральный индекс

### Определение

Мера отклонения от масштабно-инвариантного спектра:

```text
n_s - 1 = d ln P_ζ / d ln k
```

### Для квадратичного потенциала

```text
n_s - 1 ≈ -2(ε + η)
```

где параметры медленного скольжения:

```text
ε = ½ (V'/V)² ≈ (m_Δ² Δ / (3H²))²
```

```text
η = V''/V ≈ m_Δ² / (3H²)
```

### Предсказание

Для медленного скольжения ε, η ≪ 1:

```text
n_s < 1  (слегка красный спектр)
```

Наблюдаемое значение (Planck):

```text
n_s = 0.9649 ± 0.0042
```

Согласуется с предсказанием!

## Тензорные пертурбации

### Гравитационные волны от инфляции

Инфляция также порождает первичные гравитационные волны.

### Спектр тензорных пертурбаций

```text
P_t(k) = (2H²)/(π² M_Planck²) |_{k=aH}
```

### Tensor-to-scalar ratio

```text
r = P_t / P_ζ
```

### Для квадратичного потенциала

```text
r ≈ 16ε
```

### Наблюдательные ограничения

BICEP/Keck + Planck:

```text
r < 0.036  (95% CL)
```

Исключает простейшие модели инфляции.

## Разогрев

### Конец инфляции

Когда поле Δ достигает минимума потенциала:

```text
Δ → 0
```

Инфляция заканчивается.

### Передача энергии

Энергия поля Δ передаётся стандартным частицам через:

- **Распад Δ → частицы**
- **Параметрический резонанс (пре разогрев)**

### Температура разогрева

```text
T_reheat ≈ (Γ_Δ · M_Planck)^(1/2)
```

где Γ_Δ — ширина распада поля Δ.

### Ограничения

Из нуклеосинтеза:

```text
T_reheat ≳ 1 МэВ
```

## Наблюдательные тесты

### 1. CMB анизотропии

Температурные флуктуации:

```text
ΔT/T ≈ 10⁻⁵
```

Спектр согласуется с инфляционными предсказаниями.

### 2. Поляризация CMB

E-моды обнаружены, B-моды пока не найдены.

### 3. Крупномасштабная структура

Распределение галактик согласуется с инфляционными пертурбациями.
