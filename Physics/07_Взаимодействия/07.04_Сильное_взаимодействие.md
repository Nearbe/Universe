# Сильное взаимодействие

## Глюоны как кванты цветовых осцилляций

Глюоны — это кванты **осцилляций цветового заряда** (генераторы SU(3)) в ходонной сети.

### Число глюонов

Есть **восемь независимых цветовых мод**, каждая несёт цвет и антицвет, что позволяет глюонам самовзаимодействовать.

### Цветовой заряд

Кварки несут цветовой заряд трёх типов:

- **r** (red, красный)
- **g** (green, зелёный)
- **b** (blue, синий)

Антикварки несут антицвет:

- **r̄** (anti-red)
- **ḡ** (anti-green)
- **b̄** (anti-blue)

## Механизм конфайнмента

### Образование струны

Попытка разделить цветовые заряды создаёт **струну** возбуждений цветовой моды:

```text
кварк ←——— струна ———→ антикварк
```

### Линейный потенциал

Энергия в струне растёт линейно с разделением:

```text
V(r) ≈ σr
```

где σ — натяжение струны (~1 ГэВ/фм).

### Разрыв струны

Когда энергия достаточна для создания новой пары кварк-антикварк:

```text
E_струны > 2m_q c²
```

Струна рвётся, производя адроны (мезоны или барионы), но **не изолированный цветовой заряд**.

### В терминах ходонов

Струна — это когерентная цепочка возмущений цветовой моды, проходящая через ходоны сети.

## Асимптотическая свобода

### Бегущая константа связи

Сильная связь α_s зависит от масштаба зондирования (расстояние/импульс):

#### Низкие энергии / большие расстояния (~1 ГэВ)

```text
α_s ~ 1
```

Цветовые моды сильно связаны → **конфайнмент**.

#### Высокие энергии / короткие расстояния (>>100 ГэВ)

```text
α_s << 1
```

Цветовые моды отделяются → **асимптотическая свобода**.

### Уравнение эволюции

```text
α_s(Q²) = α_s(μ²) / [1 + (β₀/4π) · α_s(μ²) · ln(Q²/μ²)]
```

где:

- Q — переданный импульс
- μ — масштаб нормировки
- β₀ = 11 - (2/3)n_f — коэффициент (n_f — число ароматов)

### В терминах IME

Это поведение следует из того, как цветовые корреляции ходонной сети изменяются с длиной волны возбуждения.

## Квантовая хромодинамика (КХД)

### Лагранжиан

```text
L_QCD = Σ_q Ψ̄_q(iγ^μ D_μ - m_q)Ψ_q - (1/4)G^a_μν G^μν_a
```

где:

- Ψ_q — поле кварка аромата q
- G^a_μν — тензор напряжённости глюонного поля
- a = 1...8 — индекс цвета

### Ковариантная производная

```text
D_μ = ∂_μ + ig_s T^a A^a_μ
```

где:

- g_s — сильная константа связи
- T^a — генераторы SU(3)
- A^a_μ — глюонное поле

### Тензор напряжённости

```text
G^a_μν = ∂_μ A^a_ν - ∂_ν A^a_μ + g_s f^abc A^b_μ A^c_ν
```

где f^abc — структурные константы SU(3).

### Нелинейность

Последний член (неабелевский) приводит к самовзаимодействию глюонов.

## Адроны

### Мезоны

Кварк-антикварковые пары:

```text
мезон = q + q̄
```

Примеры:

- **π⁺** = u + d̄
- **K⁰** = d + s̄
- **J/ψ** = c + c̄

### Барионы

Три кварка:

```text
барион = q + q + q
```

Примеры:

- **протон p** = u + u + d
- **нейтрон n** = u + d + d
- **Λ** = u + d + s

### Цветовая нейтральность

Все наблюдаемые адроны бесцветны:

- **Мезоны**: цвет + антицвет = 0
- **Барионы**: r + g + b = 0 (синглет)

## Ядерные силы

### Остаточное сильное взаимодействие

Сильное ядерное взаимодействие между нуклонами — это остаточный эффект цветовых сил:

```text
цветовые поля кварков внутри нуклонов "вытекают" наружу
```

### Обмен мезонами

Нуклоны взаимодействуют через обмен виртуальными мезонами (преимущественно π-мезонами):

```text
N + N → N + N + π (виртуальный)
```

### Потенциал Юкавы

```text
V(r) = -g² · (e^(-mr) / r)
```

где m — масса пиона (~140 МэВ).

### Характерный масштаб

```text
r_0 ≈ ħ/(mc) ≈ 1.4 фм
```

## Экспериментальные проверки

### 1. Глубоко неупругое рассеяние

Эксперименты в SLAC (1968) показали:

- Протоны имеют точечную структуру (партоны)
- Партоны идентифицированы как кварки

### 2. Открытие глюона

PETRA (DESY, 1979):

- Наблюдение 3-струйных событий e⁺e⁻ → qq̄g
- Прямое доказательство существования глюона

### 3. Измерение α_s

Различные процессы дают согласованные значения:

```text
α_s(M_Z) ≈ 0.118
```
