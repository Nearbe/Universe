# Универсальная формула взаимодействия

## Единое описание всех сил

Все фундаментальные взаимодействия можно записать в универсальной форме, выявляющей общую структуру.

## Общая формула

Для взаимодействия между двумя частицами с «зарядами» Q₁ и Q₂:

```text
F_i = (g_i · Q₁ · Q₂) / r² · f_i(r/λ_i)
```

где:

- **g_i** — постоянная взаимодействия i-го типа
- **Q₁, Q₂** — зарядовые характеристики (масса, заряд, цвет, слабый заряд)
- **λ_i** — характеристическая длина взаимодействия
- **f_i(x)** — функция формы

## Функции формы для разных взаимодействий

### Гравитация

```text
f_G(r/λ) = 1  (λ → ∞)
```

Закон обратного квадрата на всех расстояниях.

### Электромагнетизм

```text
f_EM(r/λ) = 1  (λ → ∞)
```

Закон Кулона на всех расстояниях.

### Сильное (конфайнмент)

```text
f_S(r/λ) = 1 + σ·r²  (при больших r)
```

Линейный рост потенциала на больших расстояниях.

### Слабое

```text
f_W(r/λ) = e^(-r/λ)  (λ ≈ 10⁻³ фм)
```

Экспоненциальное затухание.

## Потенциальная форма

Вместо силы удобнее использовать потенциал:

```text
V_i(r) = (g_i · Q₁ · Q₂) / r · f_i(r/λ_i)
```

### Сводка потенциалов

| Взаимодействие | Потенциал |
|---------------|-----------|
| Гравитация | V = -Gm₁m₂/r |
| ЭМ | V = αq₁q₂/r |
| Сильное | V = -α_s/r + σr |
| Слабое | V = -G_F·e^(-r/λ)/r |

## В терминах IME

### Постоянные взаимодействий

Все постоянные g_i связаны с фундаментальными параметрами сети:

```text
g_i = F(n₀, τ₀, Δ_vacuum, Q)
```

где:

- n₀ — плотность ходонов
- τ₀ — характерное время
- Δ_vacuum — вакуумное значение Δ
- Q — структурная сложность

### Зарядовые характеристики

| Взаимодействие | Заряд | Выражение через Ψ |
|---------------|-------|------------------|
| Гравитация | M | \|Ψ\|² |
| ЭМ | q | tanh(Δ/2) |
| Сильное | T^a | генераторы SU(3) |
| Слабое | I_W | генераторы SU(2) |

### Характеристические длины

```text
λ_i = ħ / (m_i · c)
```

Комптоновская длина волны переносчика.

| Взаимодействие | Переносчик | m (ГэВ) | λ (м) |
|---------------|------------|---------|-------|
| Гравитация | Гравитон | 0 | ∞ |
| ЭМ | Фотон | 0 | ∞ |
| Сильное | Глюон | 0 | ∞* |
| Слабое | W/Z | 80-90 | 10⁻¹⁸ |

*Глюон безмассов, но конфайнмент ограничивает диапазон.

## Объединение взаимодействий

### Иерархия сил

При низких энергиях (~1 ГэВ):

```text
F_S : F_EM : F_W : F_G ≈ 1 : 10⁻² : 10⁻⁶ : 10⁻³⁹
```

### Бегущие константы

Константы взаимодействий зависят от энергетического масштаба Q²:

```text
α_i(Q²) = α_i(μ²) / [1 + b_i · ln(Q²/μ²)]
```

### Сходимость при высоких энергиях

При Q² → 10¹⁶ ГэВ²:

```text
α_S ≈ α_EM ≈ α_W
```

Великое объединение (GUT).

## Лагранжева формулировка

### Универсальный лагранжиан

```text
L = L_network + L_int
```

### Лагранжиан сети

```text
L_network = ½ ∂_μ Ψ† ∂^μ Ψ - V(Ψ†Ψ)
```

### Член взаимодействия

```text
L_int = Σ_i g_i · Ψ̄ Γ^i_μ Ψ · A^i_μ
```

Сумма по всем типам взаимодействий.

## Проверка согласованности

### Размерный анализ

Все члены в формуле имеют правильную размерность:

```text
[F] = [энергия]/[длина] = [сила]
```

### Предельные случаи

- **r → 0**: доминирует квантовое поведение
- **r → ∞**: дальнодействующие силы (G, EM)
- **E → ∞**: объединение констант

### Сохранение зарядов

Все взаимодействия сохраняют соответствующие заряды:

- Гравитация: энергия-импульс
- ЭМ: электрический заряд
- Сильное: цветовой заряд
- Слабое: слабый изоспин
