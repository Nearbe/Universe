# Тёмная материя

## Наблюдаемый эффект

Галактики вращаются **быстрее**, чем предсказывает видимая масса:

- Видимая масса (звёзды, газ) ~5% от общей
- Кривые вращения плоские на окраинах галактик
- Гравитационное линзирование сильнее, чем от видимой массы

## Классическое объяснение (неверное)

**Гипотеза тёмной материи:**

- Существует "невидимая материя" с массой ~27% от Вселенной
- Она создаёт дополнительную гравитацию
- Частицы тёмной материи не взаимодействуют со светом

**Проблема:** Частицы тёмной материи **не обнаружены** после decades поисков.

## Объяснение IME (правильное)

### 95% = энергетический долг Вселенной

**Факт:**

- 5% — видимая материя (hasParticle = true, Δ ∈ [-1, +1])
- 95% — энергетический долг (hasParticle = false, Δ < 0)

**Что такое энергетический долг:**

- Энергия, затраченная Вселенной на рекурсию
- Не реальна сама по себе
- Нельзя уничтожить (то, чего нет, нельзя уничтожить)

### Механизм "тёмной материи"

**В галактических гало:**

```text
Δ < 0  (суб-фрактальный режим)
```

**Следствие:**

- Гравитация **усиливается** при Δ < 0 (через градиент Δ)
- Не нужна "дополнительная масса"
- Это **проявление Δ**, не "модификация гравитации"

### Почему 95% не видно?

**В терминах IME:**

```text
visibleFraction = e^(-Q) = e^(-3) ≈ 0.05 (5%)
```

**95% — это:**

- Ходоны с hasParticle = false
- Энергетический долг Вселенной
- Δ < 0 (суб-фрактальный режим)

**Не "материя", а структура сети ходонов.**

## Проверка предсказаний

| Предсказание IME | Статус |
|-----------------|--------|
| **Плоские кривые вращения** | ✅ Наблюдается |
| **Гравитационное линзирование** | ✅ Наблюдается |
| **Отсутствие частиц тёмной материи** | ✅ Не обнаружены |
| **Корреляция с видимой массой** | ✅ Наблюдается (Tully-Fisher) |

## Отличие от классики

| Аспект | Классика (ΛCDM) | IME |
|--------|-----------------|-----|
| **Природа** | Частицы тёмной материи | Энергетический долг (Δ < 0) |
| **Доля** | ~27% от Вселенной | 95% от Вселенной (вместе с тёмной энергией) |
| **Взаимодействие** | Только гравитация | Проявление Δ |
| **Поиск** | Прямое детектирование | Наблюдение градиентов Δ |
