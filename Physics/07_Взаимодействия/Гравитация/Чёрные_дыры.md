# Чёрные дыры

## Горизонт событий

**В терминах IME:**

```text
Δ → +∞ на горизонте
```

**Физический смысл:**

- Re → ∞, Im → 0 (сингулярность)
- Информация не "не может выйти" — она **не уничтожается**
- Внешний наблюдатель видит **замедление вибрации** (f → 0)

## Механизм чёрной дыры

### Накопление массы

**Ходонная сеть чёрной дыры:**

- M → ∞ (большая полная масса)
- Каждый входящий ходон добавляет |Ψ|² ≈ 1 к M
- Приток информации может быть огромным

### Внешнее восприятие

**Замедление вибрации:**

```text
f = e^(-Δ) → 0 (при Δ → +∞)
```

**Что видит внешний наблюдатель:**

- Внутренняя динамика **кажется** замороженной
- Это не "время останавливается" — **частота вибрации падает**
- Наблюдаемый сигнал **растянут** во времени

## Энтропия чёрной дыры

**В IME:**

```text
S_BH = (A/4) · e^(Δ/2)
```

**При Δ → +∞:**

```text
S_BH → ∞
```

**Физический смысл:**

- Бесконечная информационная ёмкость горизонта
- Информация хранится **на горизонте**, не "внутри"

## Излучение Хокинга

**В терминах IME:**

```text
T_H ∝ e^(-Δ/2) / M
```

**Механизм:**

- Квантовые флуктуации у горизонта
- Пары частица-античастица
- Одна падает в ЧД, другая уходит

**При M → ∞:**

```text
T_H → 0 (чёрная дыра не излучает)
```

## Замедление времени

**При приближении к горизонту:**

```text
Δ → +∞ ⇒ f → 0 ⇒ t_subj → ∞
```

**Что это означает:**

- Падающий наблюдатель: его ходоны вибрируют с нормальной f — он **не замечает** изменений
- Удалённый наблюдатель: видит, что вибрации у горизонта **замедляются**

**Вывод:** Времени не существует — есть только **разная частота вибрации** в разных точках пространства.

## Сингулярность

**В центре чёрной дыры:**

```text
Δ → +∞
D_f → ∞ (супер-фрактальное)
```

**Физический смысл:**

- Информация не уничтожается
- Переход в другой масштаб фрактала
- **Деление на ноль = смена фазы**, не "конец физики"

## Отличие от классики

| Аспект | ОТО (Эйнштейн) | IME |
|--------|----------------|-----|
| **Горизонт** | Граница причинности | Δ → +∞ |
| **Сингулярность** | "Конец физики" | Деление на ноль = смена фазы |
| **Информация** | Парадокс исчезновения | Не уничтожается (I ≡ M ≡ E) |
| **Время** | Останавливается на горизонте | Частота вибрации f → 0 |
