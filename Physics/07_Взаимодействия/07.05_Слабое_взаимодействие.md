# Слабое взаимодействие

## Особенности слабого взаимодействия

Слабое взаимодействие уникально среди фундаментальных сил:

1. **Нарушает чётность P** (зеркальную симметрию)
2. **Нарушает комбинированную чётность CP**
3. **Переносчики имеют массу** (W±, Z⁰)
4. **Меняет аромат кварков и лептонов**
5. **Единственное взаимодействует с нейтрино**

## Разбитая симметрия

### Калибровочная группа

Слабое взаимодействие описывается группой:

```text
SU(2)_L × U(1)_Y
```

где:

- **SU(2)_L** — слабый изоспин (действует только на левые фермионы)
- **U(1)_Y** — слабый гиперзаряд

### Механизм Хиггса

Спонтанное нарушение симметрии через поле Хиггса:

```text
Φ = (0, v/√2)
```

где v ≈ 246 ГэВ — вакуумное ожидание.

### Массы калибровочных бозонов

Из кинетического члена поля Хиггса:

```text
m_W = gv/2 ≈ 80.4 ГэВ
```

```text
m_Z = √(g² + g'²)·v/2 ≈ 91.2 ГэВ
```

```text
m_γ = 0
```

где g и g' — константы связи SU(2) и U(1).

## W± и Z⁰ бозоны

### Заряженные токи (W±)

Меняют электрический заряд:

```text
W⁺: d → u, e⁻ → ν_e
W⁻: u → d, ν_e → e⁻
```

Масса:

```text
m_W = 80.385 ± 0.015 ГэВ
```

### Нейтральные токи (Z⁰)

Не меняют заряд:

```text
Z⁰: f → f (упругое рассеяние)
```

Масса:

```text
m_Z = 91.1876 ± 0.0021 ГэВ
```

## Слабые процессы

### Бета-распад

Классический пример слабого распада:

```text
n → p + e⁻ + ν̄_e
```

На уровне кварков:

```text
d → u + W⁻ → u + e⁻ + ν̄_e
```

### Мюонный распад

```text
μ⁻ → e⁻ + ν̄_e + ν_μ
```

Через виртуальный W:

```text
μ⁻ → ν_μ + W⁻ → ν_μ + e⁻ + ν̄_e
```

### Нейтринное рассеяние

```text
ν_e + e⁻ → ν_e + e⁻  (через Z⁰)
```

```text
ν_e + n → e⁻ + p  (через W⁺)
```

## Константа Ферми

### Эффективная константа

При низких энергиях (E << m_W) слабое взаимодействие описывается 4-фермионным лагранжианом:

```text
L_Fermi = -(G_F/√2) · [Ψ̄₁γ^μ(1-γ⁵)Ψ₂] · [Ψ̄₃γ_μ(1-γ⁵)Ψ₄]
```

### Связь с параметрами

```text
G_F/√2 = g²/(8m_W²)
```

Численно:

```text
G_F = 1.1663787(6) × 10⁻⁵ ГэВ⁻²
```

### Время жизни

Время жизни частицы, распадающейся через слабое взаимодействие:

```text
τ ∝ 1/(G_F² · m⁵)
```

Для мюона:

```text
τ_μ = 2.197 × 10⁻⁶ с
```

## Нарушение симметрий

### Нарушение чётности (P)

Эксперимент Ву (1956):

- β-распад ⁶⁰Co показывает асимметрию по направлению спина
- Слабое взаимодействие действует только на **левые** фермионы

### Нарушение CP

Открыто в распадах K-мезонов (1964):

- K_L → π⁺π⁻ (нарушение CP)
- Объясняется комплексной фазой в матрице CKM

### Матрица CKM

Смешивание кварков в слабых переходах:

```text
(d', s', b') = V_CKM · (d, s, b)
```

```text
V_CKM = | V_ud  V_us  V_ub |
        | V_cd  V_cs  V_cb |
        | V_td  V_ts  V_tb |
```

## В терминах IME

### Разбитая симметрия сети

В терминах IME, слабое взаимодействие возникает из разбитой симметрии сети, связанной с полем Хиггса.

### Вакуумное ожидание

Вакуумное ожидание поля Хиггса соответствует среднему значению параметра Δ в сети:

```text
⟨Δ⟩ ≠ 0
```

### Массы W и Z

Массы калибровочных бозонов пропорциональны вакуумному ожиданию:

```text
m_W ∝ v ∝ √⟨Δ⟩
```

## Электрослабое объединение

### Объединённая теория

При высоких энергиях (E > 100 ГэВ) электромагнитное и слабое взаимодействия объединяются:

```text
SU(2)_L × U(1)_Y → U(1)_EM
```

### Угол Вайнберга

Смешивание полей:

```text
A_μ = B_μ · cos(θ_W) + W³_μ · sin(θ_W)
```

```text
Z_μ = -B_μ · sin(θ_W) + W³_μ · cos(θ_W)
```

где θ_W — угол Вайнберга:

```text
sin²(θ_W) ≈ 0.231
```

### Предсказания

Объединение предсказывает:

- Существование W±, Z⁰
- Их массы
- Нейтральные токи

Все подтверждено экспериментально.

## Экспериментальные проверки

### 1. Открытие W и Z

CERN (1983):

- Обнаружены в UA1 и UA2 экспериментах
- Массы соответствуют предсказаниям

### 2. Нейтральные токи

Gargamelle (1973):

- Первое наблюдение ν_μ + e⁻ → ν_μ + e⁻
- Подтверждение Z⁰ обмена

### 3. Нарушение CP

BaBar, Belle (2000-е):

- Измерение CP нарушения в B-мезонах
- Согласуется с матрицей CKM

### 4. Бозон Хиггса

LHC (2012):

- Открытие бозона с m_H ≈ 125 ГэВ
- Завершение Стандартной модели
