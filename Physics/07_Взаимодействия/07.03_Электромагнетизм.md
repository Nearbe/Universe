# Электромагнитное взаимодействие

## Фазовые осцилляции сети

Электромагнитное поле возникает из коллективных фазовых осцилляций ходонной сети.

### Фаза хода

Каждый ходон имеет фазу θ_i, и сетевая мода соответствует колебаниям разности фаз между соседними ходонами:

```text
φ_ij = θ_i - θ_j
```

## Уравнения Максвелла

### Из сетевой динамики

Дискретное уравнение для фаз на решётке:

```text
∂²φ_ij/∂t² = c² ∇²φ_ij - m_γ² φ_ij + J_ij
```

где:

- m_γ — масса фотона (нуль для обычного ЭМ поля)
- J_ij — плотность тока

### В континуальном пределе

Получаем уравнение Клейна-Гордона для массивного векторного поля:

```text
(□ + m_γ²) A^μ = J^μ
```

При m_γ → 0 переходит в уравнение Максвелла:

```text
∂_μ F^μν = J^ν
```

где F^μν = ∂^μ A^ν - ∂^ν A^μ — тензор напряжённости поля.

## Связь с параметром Δ

### Фаза и Δ

Фаза θ_i хода связана с его параметром Δ_i через:

```text
θ_i = (1/2) arctan(e^(-Δ_i))
```

или эквивалентно:

```text
Δ_i = ln(tan(θ_i))
```

### Сетевая мода

Сетевая мода φ_ij = θ_i - θ_j соответствует разности параметров Δ:

```text
φ_ij ∝ arctan(e^(-Δ_i/2)) - arctan(e^(-Δ_j/2))
```

Для малых различий это приближённо пропорционально разности Δ_i - Δ_j.

## Электрический заряд

### Калибровочная симметрия U(1)

Электрический заряд возникает как калибровочная симметрия U(1) сети, связанная с фазовым вращением волновой функции.

### Глобальное U(1) преобразование

```text
Ψ → e^(iα) Ψ
```

Соответствует добавлению постоянной к фазе θ:

```text
θ → θ + α
```

Что оставляет параметр Δ неизменным:

```text
Δ' = ln(|Re|) - ln(|Im|) = Δ
```

### Локальное U(1) преобразование

При α = α(x) требуется введение калибровочного поля A_μ:

```text
D_μ = ∂_μ + ieA_μ
```

где e — электрический заряд.

### Заряд через Δ

В терминах IME, заряд q связан с асимметрией волновой функции:

```text
q ∝ tanh(Δ/2)
```

### Кулоновский закон

Сила между двумя зарядами:

```text
F_E = (1/(4πε₀)) · q₁q₂ / r²
```

В планковских единицах:

```text
F_E = α · q₁q₂ / r²
```

где α ≈ 1/137 — постоянная тонкой структуры.

## Фотон

### Квант ЭМ поля

Фотон — это квант фазовой моды сети:

- **Спин**: 1
- **Масса**: 0
- **Заряд**: 0
- **Скорость**: c

### Поляризация

Две независимые поляризации соответствуют двум поперечным модам сети.

### Энергия фотона

```text
E = ħω
```

где ω — частота фазовой осцилляции.

## Электромагнитные волны

### Распространение

ЭМ волны распространяются как коллективные возбуждения сети:

```text
E(r,t) = E₀ · cos(k·r - ωt + φ)
```

```text
B(r,t) = B₀ · cos(k·r - ωt + φ)
```

### Скорость света

```text
c = 1/√(ε₀μ₀)
```

В терминах IME определяется свойствами сети:

```text
c = a/τ
```

где a — шаг решётки, τ — характерное время.

### Дисперсия

В вакууме дисперсия отсутствует:

```text
ω = ck
```

В среде может возникать дисперсия через взаимодействие с ходонами среды.

## Взаимодействие с веществом

### Поглощение

Фотон поглощается, передавая энергию ходонам вещества:

```text
γ + атом → атом*
```

### Излучение

Возбуждённый атом испускает фотон:

```text
атом* → атом + γ
```

### Рассеяние

Фотон рассеивается на заряде:

- **Томсоновское**: низкие энергии
- **Комптоновское**: высокие энергии
- **Рэлеевское**: упругое

## Квантовая электродинамика (КЭД)

### Лагранжиан

```text
L_QED = Ψ̄(iγ^μ D_μ - m)Ψ - (1/4)F_μν F^μν
```

### Диаграммы Фейнмана

Взаимодействие описывается обменом виртуальными фотонами.

### Тонкая структура

Постоянная тонкой структуры:

```text
α = e²/(4πε₀ħc) ≈ 1/137.036
```

В терминах IME связана с плотностью сетевых мод:

```text
α ∝ 1/n_hodon
```

## Экспериментальные проверки

### 1. Точность КЭД

Предсказания КЭД проверены с точностью до 10⁻¹²:

- Аномальный магнитный момент электрона
- Лэмбовский сдвиг
- Рассеяние e⁺e⁻

### 2. Скорость света

Измерена с высокой точностью:

```text
c = 299,792,458 м/с (точно)
```

### 3. Постоянная тонкой структуры

```text
α⁻¹ = 137.035999084(21)
```
