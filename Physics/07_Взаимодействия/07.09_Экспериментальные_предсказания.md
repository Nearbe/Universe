# Экспериментальные предсказания для взаимодействий

## 1. Вариация постоянных взаимодействий

### Предсказание

Если параметр Δ вакуума изменяется во времени или пространстве, то постоянные взаимодействий должны меняться:

```text
Ġ/G ∝ dΔ/dt
```

```text
α̇/α ∝ dΔ/dt
```

```text
ġ_s/g_s ∝ dΔ/dt
```

```text
Ġ_F/G_F ∝ dΔ/dt
```

### Экспериментальные ограничения

#### Постоянная тонкой структуры

Квазарные спектры (z ~ 1-3):

```text
|Δα/α| < 10⁻⁵ - 10⁻⁶
```

Метеоритные данные (Oklo, 2×10⁹ лет):

```text
|Δα/α| < 10⁻⁷
```

#### Гравитационная постоянная

Лунная лазерная дальнометрия:

```text
|Ġ/G| < 10⁻¹³ год⁻¹
```

Пульсарные тайминги:

```text
|Ġ/G| < 10⁻¹² год⁻¹
```

### Интерпретация в IME

Ограничения на вариацию постоянных дают ограничения на скорость изменения Δ:

```text
|dΔ/dt| < 10⁻¹² - 10⁻¹⁵ год⁻¹
```

## 2. Связь между постоянными взаимодействий

### Предсказание

Поскольку все постоянные происходят из одного источника (ходонной сети), должны существовать соотношения между ними:

```text
α ∝ G^β · g_s^γ · G_F^δ
```

с определёнными показателями β, γ, δ.

### Проверка

Сравнение измеренных значений с предсказанными соотношениями.

## 3. Отклонения от закона обратного квадрата

### На малых расстояниях

На расстояниях, сравнимых с характерным размером сетевой ячейки (предположительно планковской длиной или немного больше), должны наблюдаться отклонения:

```text
F(r) = (g·Q₁·Q₂)/r² · [1 + ε(r/l_P)]
```

### Эксперименты

#### Крутильные весы

Поиск отклонений от закона Ньютона на масштабах ~100 мкм:

```text
V(r) = -Gm₁m₂/r · [1 + α·e^(-r/λ)]
```

Ограничения:

```text
α < 10⁻²  при λ ~ 100 мкм
```

#### Атомная спектроскопия

Поиск отклонений от закона Кулона:

```text
V(r) = -α/r · [1 + δ·e^(-r/λ)]
```

Ограничения:

```text
δ < 10⁻¹⁰  при λ ~ 1 Å
```

## 4. Дополнительные взаимодействия при высоких энергиях

### Предсказание

При энергиях, сравнимых с характерной энергией сети (планковской энергией или энергией сетевого резонанса), могут появляться новые взаимодействия.

### Проявления

#### LHC

Поиск:

- Z' бозонов (дополнительные U(1) симметрии)
- Лептокварков
- Гравитонных резонансов

Пока не обнаружено.

#### Космические лучи

Частицы сверхвысоких энергий (E > 10²⁰ эВ):

- Нарушение предела ГЗК (Грейзена-Зацепина-Кузьмина)
- Новые каналы распада

Наблюдаются, но статистика низкая.

## 5. Анизотропия взаимодействий

### Предсказание

Внешние гравитационные или электромагнитные поля могут анизотропно модулировать сеть, приводя к направлению-зависимым постоянным взаимодействий.

### Эксперименты

#### Сравнение атомных часов

Часы с разной ориентацией:

```text
Δν/ν < 10⁻¹⁸
```

Ограничения на анизотропию α.

#### Интерферометрия

Опыт Майкельсона-Морли с современной точностью:

```text
Δc/c < 10⁻¹⁷
```

## 6. Гравитационные волны

### Предсказания IME

#### Поляризация

Гравитационные волны должны иметь тензорную моду (спин 2):

- **Два режима поляризации**: + и ×
- **Скорость**: c

Подтверждено LIGO/Virgo.

#### Модификации

При наличии Δ:

```text
h ∝ e^(Δ/2)
```

Может приводить к отклонениям от ОТО.

### Тесты

#### Скорость гравитации

GW170817 (гравитационные волны + гамма-всплеск):

```text
|c_g - c|/c < 10⁻¹⁵
```

#### Затухание орбит

Двойные пульсары:

```text
dP/dt согласуется с ОТО в пределах 0.1%
```

## 7. Нарушение эквивалентности

### Предсказание

Если гравитация зависит от Δ, а Δ может различаться для разных форм материи, то принцип эквивалентности может нарушаться.

### Эксперименты

#### Eötvös эксперимент

Сравнение ускорения разных материалов:

```text
η = 2(a₁ - a₂)/(a₁ + a₂) < 10⁻¹³
```

#### MICROSCOPE спутник

```text
η < 10⁻¹⁵
```

#### Лунная лазерная дальнометрия

Сравнение ускорения Земли и Луны к Солнцу:

```text
η < 10⁻¹⁴
```

## 8. Квантовые эффекты гравитации

### Предсказания

#### Излучение Хокинга

Чёрные дыры должны испускать тепловое излучение:

```text
T_H = ħc³/(8πGMk_B)
```

Для солнечных масс чёрных дыр:

```text
T_H ~ 10⁻⁸ К  (не наблюдаемо)
```

Для мини-чёрных дыр (если существуют):

```text
T_H ~ 10¹² К  (гамма-всплески)
```

#### Гравитационная запутанность

Гравитация должна запутывать квантовые состояния.

Предложенные эксперименты:

- Запутывание массивных зеркал
- Гравитационно-индуцированная декогеренция
