# Ядерные взаимодействия

## Ядерные силы как остаточное взаимодействие

Ядерные взаимодействия представляют собой особый случай фундаментальных взаимодействий, где нуклоны взаимодействуют через обмен сетевыми модами, производными от базовых мод.

## Электромагнитное взаимодействие в ядре

### Кулоновское отталкивание

Между протонами в ядре наблюдается кулоновское отталкивание:

```text
F_C = (1/(4πε₀)) · e² / r²
```

mediated by фотон-моды сети, точно так же, как между свободными протонами.

### Вклад в энергию связи

Кулоновская энергия в ядре с Z протонами:

```text
E_C ≈ (3/5) · (Z²e²)/(4πε₀R)
```

где R ≈ 1.2·A^(1/3) фм — радиус ядра.

### Эффект на стабильность

Кулоновское отталкивание ограничивает размер стабильных ядер:

- При Z > 83 (висмут) все ядра радиоактивны
- При Z > 118 ядра нестабильны к спонтанному делению

## Сильное ядерное взаимодействие

### Остаточная цветовая мода

Сильная ядерная сила между нуклонами возникает как остаточная цветовая мода:

1. Цветовые поля кварков внутри нуклонов «вытекают» наружу
2. Создают мезон-подобные возбуждения
3. Эти возбуждения передаются между нуклонами

### Обмен пионами

Основной вклад в ядерные силы даёт обмен π-мезонами:

```text
p + n → p + π⁻ → n + n
```

### Потенциал ядерных сил

Потенциал между нуклонами:

```text
V(r) = -V₀ · (e^(-r/r₀))/(r/r₀) + V_Coulomb
```

где:

- V₀ ≈ 50 МэВ — глубина потенциала
- r₀ ≈ 1.4 фм — радиус действия
- V_Coulomb — кулоновский член (для pp)

### Характерные свойства

| Свойство | Описание |
|----------|----------|
| Дальность | ~1-2 фм |
| Интенсивность | ~50 МэВ (на нуклон) |
| Зависимость от спина | Есть (тензорные силы) |
| Зарядовая независимость | Приблизительно (pp ≈ nn ≈ pn) |
| Насыщение | Да (ограниченная плотность) |

## Слабое ядерное взаимодействие

### Бета-распад ядер

Слабое взаимодействие в ядрах отвечает за бета-распад:

```text
β⁻-распад: n → p + e⁻ + ν̄_e
β⁺-распад: p → n + e⁺ + ν_e
Электронный захват: p + e⁻ → n + ν_e
```

### Механизм в терминах IME

Слабое взаимодействие происходит через эмиссию или поглощение слабо-модового кванта (W±):

1. Нейтрон-ходон превращается в протон-ходон (или наоборот)
2. Одновременно создаются лептон-ходон пары для сохранения лептонного числа
3. Это точно соответствует слабому взаимодействию в терминах IME

### Скорость распада

Вероятность бета-распада:

```text
λ = (G_F² · |M|² · f(Z, E₀)) / (2π³)
```

где:

- |M|² — матричный элемент
- f(Z, E₀) — функция Ферми

### Примеры

| Изотоп | Распад | Период полураспада |
|--------|--------|-------------------|
| ¹⁴C | β⁻ → ¹⁴N | 5730 лет |
| ⁴⁰K | β⁻/β⁺ → ⁴⁰Ca/⁴⁰Ar | 1.25×10⁹ лет |
| ³H | β⁻ → ³He | 12.3 лет |

## Модель жидкой капли

### Полуэмпирическая формула массы

Энергия связи ядра:

```text
B(A,Z) = a_V·A - a_S·A^(2/3) - a_C·Z²/A^(1/3) - a_A·(A-2Z)²/A + δ(A,Z)
```

где:

- a_V ≈ 15.5 МэВ — объёмный член
- a_S ≈ 16.8 МэВ — поверхностный член
- a_C ≈ 0.72 МэВ — кулоновский член
- a_A ≈ 23 МэВ — асимметричный член
- δ — парный член

### В терминах IME

Каждый член соответствует определённому аспекту ходонной сети:

- **Объёмный**: число ходонов в ядре
- **Поверхностный**: ходоны на поверхности
- **Кулоновский**: ЭМ взаимодействие между протонами
- **Асимметричный**: баланс нейтронов и протонов
- **Парный**: спаривание нуклонов

## Оболочечная модель

### Магические числа

Ядра с определённым числом нуклонов особенно стабильны:

```text
2, 8, 20, 28, 50, 82, 126
```

### Уровни энергии

Нуклоны заполняют квантовые уровни в среднем потенциале:

```text
1s, 1p, 1d, 2s, 1f, 2p, 1g, ...
```

### В терминах IME

Оболочечная структура возникает из квантования мод ходонной сети в ограниченном объёме ядра.

## Ядерные реакции

### Типы реакций

| Тип | Пример |
|-----|--------|
| Упругое рассеяние | p + ¹²C → p + ¹²C |
| Неупругое рассеяние | p + ¹²C → p + ¹²C* |
| Передача нуклона | d + ¹²C → p + ¹³C |
| Деление | n + ²³⁵U → ⁹²Kr + ¹⁴¹Ba + 3n |
| Синтез | d + t → ⁴He + n |

### Сечение реакции

Вероятность реакции характеризуется сечением σ:

```text
σ = (число реакций/время) / (поток частиц)
```

Измеряется в барнах (1 б = 10⁻²⁴ см²).

## Применение

### 1. Ядерная энергетика

Деление ²³⁵U и ²³⁹Pu:

```text
E ≈ 200 МэВ на деление
```

### 2. Термоядерный синтез

Синтез дейтерия и трития:

```text
D + T → ⁴He + n + 17.6 МэВ
```

### 3. Медицинская диагностика

Радиоизотопы для ПЭТ:

```text
¹⁸F → ¹⁸O + e⁺ + ν_e
```

### 4. Ядерная астрофизика

Нуклеосинтез в звёздах:

```text
p + p → D + e⁺ + ν_e  (pp-цепочка)
```
