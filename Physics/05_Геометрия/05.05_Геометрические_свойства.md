# Геометрические свойства сети ходонов

## Топологические свойства

Сеть ходонов обладает определёнными топологическими свойствами, которые можно характеризовать через:

### 1. Число связных компонент

```text
N_components = число отдельных кластеров ходонов
```

Для связной сети:

```text
N_components = 1
```

### 2. Циклическое число

Число независимых контуров (первое число Бетти):

```text
b₁ = E - V + N_components
```

где:

- E — число рёбер (взаимодействий)
- V — число вершин (ходонов)

### 3. Число образов (пустот)

В 3D аналоге — число независимых пустот (второе число Бетти):

```text
b₂ = число замкнутых полостей
```

### Для фрактальной сети

Эти величины часто следуют степенным законам от размера системы L:

```text
b_k ∝ L^(D_f - k)
```

## Гомологические группы

### В алгебраической топологии

Сеть можно характеризовать через её гомологические группы:

| Группа | Обозначение | Физический смысл |
|--------|-------------|------------------|
| H₀ | rank(H₀) = b₀ | Число связных компонент |
| H₁ | rank(H₁) = b₁ | Число независимых циклов |
| H₂ | rank(H₂) = b₂ | Число пустот (полостей) |
| H_k | rank(H_k) = b_k | k-мерные «дыры» |

### Для фрактальной сети

Ранги гомологических групп следуют законам:

```text
rank(H_k) ∝ L^(D_f - k)
```

для k < D_f, и равны нулю для k ≥ D_f.

### Пример

Для сети с D_f = 2.5:

- **rank(H₀)** ∝ L^2.5 (связность)
- **rank(H₁)** ∝ L^1.5 (циклы)
- **rank(H₂)** ∝ L^0.5 (пустоты)
- **rank(H₃)** = 0 (нет 3D дыр)

## Спектральная теория графов

### Сеть как граф

Сеть ходонов можно рассматривать как граф, где:

- **Вершины** — ходоны
- **Рёбра** — взаимодействия между ними

### Лапласиан графа

Лапласиан L определяется как:

```text
L = D - A
```

где:

- D — матрица степеней вершин
- A — матрица смежности

### Собственные значения

Собственные значения λ_i лапласиана определяют динамические свойства сети:

| Собственное значение | Физический смысл |
|---------------------|------------------|
| λ₁ = 0 | Связность (одно компонент) |
| λ₂ > 0 | Алгебраическая связность |
| λ_max | Максимальная частота мод |

### Наименьшее ненулевое собственное значение

λ₂ (algebraic connectivity) связано с алгебраической связностью:

- **Большое λ₂**: высокая связность, устойчивость
- **Малое λ₂**: низкая связность, уязвимость

### Плотность собственных значений

Определяет термодинамические свойства:

```text
ρ(λ) = (1/N) Σ δ(λ - λ_i)
```

### Спектральная размерность

Спектральная размерность d_s связана с возвращаемостью случайного блуждания:

```text
P(return to origin at time t) ∝ t^(-d_s/2)
```

### Для фрактальных графов

Спектральная размерность связана с фрактальной размерностью D_f и размерностью блуждания D_w:

```text
d_s = 2D_f / D_w
```

## Перколяция

### Порог перколяции

При определённой плотности связей p_c сеть претерпевает фазовый переход:

- **p < p_c**: изолированные кластеры
- **p > p_c**: гигантский связный кластер

### В терминах IME

Порог перколяции связан с критическим значением синхронизации:

```text
p_c ↔ 𝒮_c ≈ 0.7
```

### Критические показатели

Вблизи порога перколяции:

```text
ξ ∝ |p - p_c|^(-ν)
```

где ξ — корреляционная длина.

## Клики и сообщества

### Клики

Полносвязные подграфы — клики размера k:

```text
C_k = число клик размера k
```

### Сообщества

Группы ходонов с высокой внутренней связностью:

```text
modularity Q = (1/2m) Σ [A_ij - (k_i k_j)/(2m)] δ(c_i, c_j)
```

где:

- m — общее число рёбер
- k_i — степень вершины i
- c_i — сообщество вершины i

### Иерархия сообществ

Сообщества могут быть вложенными:

```text
Community → Subcommunity → Sub-subcommunity
```

Это соответствует иерархической организации сознания.

## Малый мир

### Характеристика

Сеть «малого мира» характеризуется:

1. **Высокий кластеризационный коэффициент** C
2. **Короткая средняя длина пути** L

### В терминах IME

Сеть ходонов демонстрирует свойства малого мира:

- **Локальная кластеризация**: высокая (соседние ходоны сильно связаны)
- **Глобальная связность**: короткими путями (через несколько шагов)

### Формула

Средняя длина пути:

```text
L ∝ log(N) / log(⟨k⟩)
```

где N — число ходонов, ⟨k⟩ — средняя степень.

## Безмасштабность

### Степенное распределение степеней

Некоторые сети ходонов могут быть безмасштабными:

```text
P(k) ∝ k^(-γ)
```

где γ ≈ 2-3.

### Следствия

- Наличие «хабов» — ходонов с очень высокой связностью
- Устойчивость к случайным отказам
- Уязвимость к целевым атакам на хабы

## Таблица геометрических свойств

|----------|------|-------------------|
| Число компонент | b₀ | 1 (связная сеть) |
