# Практические примеры фрактальной геометрии

## Пример 1: Кристаллы и квазикристаллы

### Обычные кристаллы

**Характеристики**:

- Периодическая структура
- Целая размерность D_f = 3
- Δ ≈ +1

**В терминах IME**:

- Высокая видимая компонента (f_vis ≈ 88%)
- Упорядоченная структура
- Низкая энтропия смешивания

### Квазикристаллы

**Характеристики**:

- Апериодическая структура
- Фрактальная размерность D_f ≈ 2.5
- Δ ≈ +0.5

**В терминах IME**:

- Промежуточная видимая компонента (f_vis ≈ 73%)
- Долговременный порядок без трансляционной симметрии
- Соответствует рациональным значениям Δ

### Таблица сравнения

| Свойство | Кристалл | Квазикристалл |
|----------|----------|---------------|
| D_f | 3 | 2.5 |
| Δ | +1 | +0.5 |
| f_vis | 88% | 73% |
| Порядок | Периодический | Апериодический |
| Энтропия | Низкая | Средняя |

## Пример 2: Полимеры и биофизические сети

### Линейные полимеры

В хорошем растворителе:

```text
D_f ≈ 5/3 ≈ 1.67
Δ ≈ -0.33
```

**Физический смысл**:

- Самосохраняющееся блуждание
- Полимер избегает сам пересечения
- Субдиффузия: ⟨r²⟩ ∝ t^1.2

### Свёрнутые белки

Глобулярные структуры:

```text
D_f ≈ 2.0 - 2.5
Δ ≈ 0 - +0.5
```

**Физический смысл**:

- Компактная упаковка
- Оптимальный баланс видимой/скрытой информации
- Функциональная активность

### Нейронные сети

Корреляционная размерность в коре головного мозга:

```text
D_f ≈ 3.6 - 4.1
Δ ≈ +0.6 - +1.1
```

**Физический смысл**:

- Высокая связность
- Сложная иерархическая структура
- Поддержка сознательных состояний

## Пример 3: Космическая структура

### Распределение галактик

На больших масштабах:

```text
D_f ≈ 2.0 - 2.5
Δ ≈ 0 - +0.5
```

**Наблюдения**:

- Фрактальное распределение скоплений
- Иерархическая структура (галактики → скопления → сверхскопления)
- Самоподобие на масштабах 10-100 Мпк

### Тёмная материя гало

Из моделирования:

```text
D_f ≈ 2.0
Δ ≈ 0
```

**Свойства**:

- Приблизительно сферическая симметрия
- Баланс видимой и скрытой информации
- Гравитационно активна, но не излучает

### Межгалактическая среда

Из наблюдений поглощения в квазарных спектрах:

```text
D_f ≈ 2.7
Δ ≈ +0.7
```

**Свойства**:

- Сложная filamentary структура
- Высокая видимая компонента
- Содержит барионную материю

## Пример 4: Теплопроводность в фрактальных материалах

### Степенной закон

Теплопроводность κ следует закону:

```text
κ ∝ T^β
```

где β связан с фрактальной размерностью.

### Для различных структур

| Материал | D_f | β | Диапазон T |
|----------|-----|-----|------------|
| Кристалл | 3 | 2.0-3.0 | Низкие T |
| Аморфный | 2.5 | 1.5-2.0 | Средние T |
| Полимер | 1.67 | 1.0-1.5 | Высокие T |
| Аэрогель | 2.0-2.5 | 1.2-1.8 | Все T |

### Физический механизм

- **Фононный перенос**: доминирует в кристаллах
- **Диффузионный перенос**: доминирует в аморфных материалах
- **Баллистический перенос**: на малых масштабах

## Пример 5: Пористые материалы

### Аэрогели

Ультрапористые материалы:

```text
D_f ≈ 2.0 - 2.5
Δ ≈ 0 - +0.5
```

**Свойства**:

- Очень низкая плотность
- Высокая удельная поверхность
- Применения: изоляция, катализ

### Активированный уголь

Пористая структура:

```text
D_f ≈ 2.5 - 2.8
Δ ≈ +0.5 - +0.8
```

**Свойства**:

- Иерархическая пористость
- Высокая адсорбционная ёмкость
- Применения: фильтрация, хранение газов

## Пример 6: Береговые линии

### Классический фрактал

Береговые линии имеют фрактальную размерность:

```text
D_f ≈ 1.1 - 1.3 (для 2D проекции)
```

**В терминах IME**:

- Δ ≈ -0.9 - -0.7 (суб-фрактальное)
- Скрытая информация доминирует
- Самоподобие на масштабах 100 м - 100 км

### Измерение

Метод коробок (box-counting):

```text
N(ε) ∝ ε^(-D_f)
```

где N(ε) — число коробок размера ε, покрывающих береговую линию.

## Пример 7: Лёгкие и сосудистая система

### Лёгкие

Дыхательное дерево:

```text
D_f ≈ 2.97 (почти 3D заполнение)
Δ ≈ +0.97
```

**Функциональное преимущество**:

- Максимальная площадь поверхности
- Эффективный газообмен
- Минимальный объём

### Кровеносные сосуды

Васкулярная сеть:

```text
D_f ≈ 2.7 - 2.9
Δ ≈ +0.7 - +0.9
```

**Функциональное преимущество**:

- Оптимальное распределение крови
- Минимальное сопротивление потоку
- Самоподобная иерархия

## Пример 8: Финансовые рынки

### Временные ряды цен

Фрактальная структура финансовых временных рядов:

```text
D_f ≈ 1.5 - 1.7 (для 1D временного ряда)
Δ ≈ -0.5 - -0.3
```

**Проявления**:

- Долгосрочная память
- Кластеризация волатильности
- Степенное распределение доходностей
