# Q-структура и энергия вибрации

## Определение Q

Q — это **энергия вибрации пространства** (networkEnergy), определяющая структурную сложность системы:

```text
Q = energy
```

Где energy — амплитуда осцилляций пространства, нормированная на базовую величину.

## Физический смысл Q

### Что такое вибрация

Пространство — не пустота, а информационная сеть ходонов. **Вибрация** — это колебания этой сети. Каждая "частица" — это не частица, а **устойчивая частота вибрации** пространства.

- **Частиц нет** — есть только вибрации пространства
- **Масса** — это энергия вибрации: m ∝ e^(Δ/2) · f(Q)
- **Частота** вибрации: f = e^(-Δ)

### Q как мера сложности

```text
Q = energy ∈ [0, ∞)
```

- Q = 0: нет вибраций, вакуум (нет энергии)
- Q = 2: мезоны (2 кварка — 2 связанные вибрации)
- Q = 3: барионы (3 кварка — 3 связанные вибрации)
- Q = 4: тетракварки (4 связанные вибрации)
- Q = 5: пентакварки (5 связанных вибраций)
- Q > 5: экзотические состояния с высокой энергией

## Диапазон Q

### Минимальное значение

```text
Q_min = 0
```

**Состояние**: Нет вибраций — абсолютный вакуум без энергии.

### Максимальное значение

Теоретически ограничений нет:

```text
Q_max = ∞
```

Практически наблюдаемые значения в диапазоне [0, ~7] для известных частиц.

## Связь с кварками

| Состояние | Число кварков N | Q | Комментарий |
|-----------|-----------------|---|-------------|
| Мезоны (q q̄) | 2 | 2 | Две связанные вибрации |
| Барионы (q q q) | 3 | 3 | Три связанные вибрации |
| Тетракварки | 4 | 4 | Четыре связанные вибрации |
| Пентакварки | 5 | 5 | Пять связанных вибраций |

**Формула**:

```text
Q = N
```

Где N — число кварков (для экзотических состояний — эффективное число связанных вибраций).

## Связь с массой

### Закон масс

```text
m ∝ e^(Δ/2) · f(Q)
```

Где f(Q) — монотонно возрастающая функция от Q.

При фиксированном Δ масса растёт с Q (больше вибраций → больше энергия).

### Примеры

- **Мезоны (Q=2)**: массы в диапазоне ~0.1–10 ГэВ
- **Барионы (Q=3)**: массы в диапазоне ~0.9–6.4 ГэВ
- **Тетракварки (Q=4)**: массы в диапазоне ~2.9–7.2 ГэВ
- **Пентакварки (Q=5)**: массы в диапазоне ~4.3–4.5 ГэВ

## Связь с Δ

Параметр отклонения Δ и энергия вибрации Q напрямую связаны:

```text
Q = e^(|Δ|)
```

### Физический смысл

- **Δ → +∞** (сингулярность/чёрная дыра): f → 0, Q → ∞
- **Δ → -∞** (тёмная энергия): f → ∞, Q → 0
- **Δ = 0** (равновесие): f = 1, Q = 1

### Частота вибрации

```text
f = e^(-Δ)
```

Частота вибрации обратно пропорциональна энергии:

- Больше энергии (Q) → меньше частота
- Меньше энергии (Q) → больше частота

Для "обычного космоса" (Δ ∈ [-1, +1]):

- Q = e^(Δ) ∈ [e^(-1), e^(+1)] ≈ [0.37, 2.72]
- Это объясняет почему типичные значения Q ≈ 2-3 для видимой материи

## Связь с видимой Вселенной

### Видимая доля

```text
visibleFraction = e^(-Q)
```

При Q = 3 (барионы): visibleFraction = e^(-3) ≈ 5%

Это соответствует наблюдаемому:

- **5%** — барионная (видинамая) материя
- **27%** — тёмная материя (низкоэнергетические вибрации)
- **68%** — тёмная энергия (высокоэнергетические вибрации)

## Динамика Q

### Изменение во времени

```text
dQ/dt = ... (зависит от процессов в сети)
```

### Источники изменений

1. **Рождение частиц**: increase of vibrational energy
2. **Аннигиляция**: decrease of vibrational energy
3. **Фазовые переходы**: скачкообразные изменения

### Сохранение

В замкнутой системе суммарная энергия вибраций сохраняется:

```text
d(ΣQ)/dt = 0
```

Энергия может перераспределяться между модами колебаний.

## Отличие от старой модели

**Старая модель**:

```text
Q = existence + particle + networkEnergy
Q ∈ [1, 3]
```

**Новая модель**:

```text
Q = e^|Δ|
Q ∈ [0, ∞)
```

### Почему изменение

Старая модель была приближением. Новая модель:

1. **Экспоненциальная связь** с Δ — физически обоснована
2. **Расширенный диапазон** — объясняет чёрные дыры (Q → ∞) и тёмную энергию (Q → 0)
3. **Единая формула** — Q = e^|Δ| работает для всех состояний
