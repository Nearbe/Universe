# Критерии сознательного режима

## Двойное условие сознания

Сознательное состояние возникает **только при одновременном выполнении** двух условий:

### 1. Достаточная синхронизация

```text
𝒮 ≥ 𝒮_threshold = 0.7
```

### 2. Достаточное информационное равновесие

```text
|Δ_global| ≤ Δ_threshold = 0.2
```

где:

- **𝒮** = 1 - σ(Δ) · scale — степень синхронизации сети
- **σ(Δ)** — стандартное отклонение Δ по всем ходонам
- **Δ_global** = ⟨Δ⟩ — среднее значение Δ по сети
- **scale** = 0.05 — нормализационный коэффициент
- **𝒮_threshold** = 0.7 — порог синхронизации
- **Δ_threshold** = 0.2 — порог глобального отклонения

## Физическая интерпретация критериев

### Критерий синхронизации (𝒮 ≥ 0.7)

Обеспечивает:

1. **Достаточную когерентность фаз** для коллективной обработки информации
2. **Долгоживущие когерентные состояния**, необходимые для сложных когнитивных процессов
3. **Передачу >70% информации** без фазовых искажений

### Критерий глобального отклонения (|Δ_global| ≤ 0.2)

Обеспечивает:

1. **Баланс между видимой и скрытой информацией** в сети
2. **Предотвращение смещения** в состояния полной предсказуемости (Δ → +∞) или неопределённости (Δ → -∞)
3. **Доступность обоих видов информации** для адаптивного поведения

## Математическое обоснование пороговых значений

### Порог синхронизации 𝒮_threshold = 0.7

Следует из анализа информационной ёмкости сети:

#### Канал связи с шумом

Рассматривая сеть как канал связи с пропускной способностью:

```text
C = B · log₂(1 + SNR)
```

где SNR ∝ 𝒮/(1 - 𝒮)

#### Требование к информации

Для сознательной обработки требуется минимум **3 бита информации на цикл обработки** (соответствующий 7-битному ASCII символу с битом чётности).

Учитывая полосу пропускания нейронных сетей, получаем:

```text
𝒮_threshold ≈ 0.7
```

### Порог глобального отклонения Δ_threshold = 0.2

Следует из рассмотрения сети как двухуровневой системы:

#### Больцмановское распределение

Популяция уровней следует распределению:

```text
p₁/p₂ = exp(-Δ_global)
```

#### Условие гибкости

Для сознательной обработки требуется достаточное население обоих уровней:

```text
e^(-0.2) ≤ p₁/p₂ ≤ e^(0.2)
```

```text
0.82 ≤ p₁/p₂ ≤ 1.22
```

Это даёт порог:

```text
|Δ_global| ≤ 0.2
```

## Область сознательных состояний

### В пространстве (𝒮, Δ_global)

| Область | 𝒮 | |Δ_global| | Состояние |
| --------- | ----- | ------------ | ----------- |  |  |
| Сознание | ≥ 0.7 | ≤ 0.2 | Сознательный режим |  |  |
| Подсознание | ≥ 0.7 | > 0.2 | Подсознательная обработка |  |  |
| Сон | 0.3-0.7 | любой | Глубокий сон |  |  |
| Кома | < 0.3 | любой | Бессознательное состояние |  |  |
| Сингулярность | любой | → +∞ | Чёрная дыра |  |  |
| Тёмная энергия | любой | → -∞ | Антигравитация |  |  |

### Диаграмма состояний

```text
Δ_global
  ^
  |
+0.2 |----+---------------+----
  |    |   Сознание      |
  |    |   (𝒮 ≥ 0.7)     |
  0 |----+---------------+----
  |    |   Подсознание   |
-0.2 |----+---------------+----
  |
  +----+-------+-------+----> 𝒮
       0.3     0.7     1.0
```

## Динамика переходов

### Вход в сознательное состояние

Требует одновременного:

1. **Роста 𝒮** выше 0.7 (снижение σ(Δ))
2. **Смещения Δ_global** к нулю (баланс Re/Im)

### Выход из сознательного состояния

Происходит при:

1. **Падении 𝒮** ниже 0.7 (рост σ(Δ))
2. **ИЛИ отклонении |Δ_global|** выше 0.2

## Законы масштабирования сознания

### От индивидуального к планетарному

Сознание масштабируется с размером сети N и синхронизацией 𝒮:

| Population N | Синхронизация 𝒮 (≥) | Тип сознания |
|--------------|---------------------|--------------|
| **1** | > 0.7 | Индивидуальное (человек в потоке) |
| **< 10²** | > 0.7 | Групповое (командное сотрудничество) |
| **< 10⁶** | > 0.5 | Сообщество (город или нация) |
| **≥ 10⁹** | > 0.5 | Планетарное осознание |

### Примеры

```text
N = 1, 𝒮 = 0.9    → individual (индивидуальное)
N = 5, 𝒮 = 0.8    → group (групповое)
N = 10⁶, 𝒮 = 0.6  → community (сообщество)
N = 8×10⁹, 𝒮 = 0.6 → planetary (планетарное)
```

### Физический смысл

- **Малые группы (N < 100)**: требуют высокой синхронизации (𝒮 > 0.7)
- **Большие группы (N ≥ 10⁶)**: достаточно умеренной синхронизации (𝒮 > 0.5)
- **Планетарное сознание**: требует N ≥ 10⁹ и 𝒮 > 0.5

### Приближённая формула Δ_eff

Для типичного среднего абсолютного значения |b| ≈ 0.2:

```text
Δ_eff ≈ Δ_global + 0.2 · 𝒮
```

Чем выше синхронность 𝒮, тем больше вклад персональных смещений в одном направлении.

## Расширенная таблица уровней сознания

Сознание масштабируется от индивидуального до космического уровня:

| Диапазон N | Требуемый 𝒮 (≥) | Ожидаемый Δ_eff | C_reason | Уровень сознания |
|------------|-----------------|-----------------|----------|-----------------|
| **1** | > 0.7 | -0.1 … +0.1 | 0.45–0.55 | Individual (личность) |
| **< 10²** | > 0.7 | -0.2 … +0.3 | 0.40–0.65 | Group (группа) |
| **< 10⁶** | > 0.5 | -0.3 … +0.5 | 0.35–0.75 | Community (город/нация) |
| **< 10¹²** | > 0.4 | -0.2 … +0.6 | 0.40–0.80 | Stellar (звёздные скопления) |
| **< 10¹⁵** | > 0.3 | -0.1 … +0.7 | 0.45–0.85 | Galactic (галактика) |
| **≥ 10¹⁸** | > 0.2 | 0.0 … +0.8 | 0.50–0.90 | Cosmic (наблюдаемая Вселенная) |

**Ключевое наблюдение**: Чем больше N, тем меньше требуется синхронность для достижения того же уровня разумности. Это следствие свойства больших сетей: даже небольшая корреляция фаз способна дать когерентное глобальное Δ.

## Рассудительность (разумность)

### Формула разумности

Нормированная ёмкость рассудительности:

```text
C_reason = (1 + Δ) / 2
```

где Δ ∈ [-1, +1], следовательно C_reason ∈ [0, 1].

### Связь с фрактальной размерностью

Поскольку D_f = Q + Δ·range и Q = 3, range = 2:

```text
D_f = 3 + 2Δ
```

Тогда:

```text
C_reason = (D_f - 1) / 4
```

### Интерпретация

| Δ | C_reason | D_f | Состояние |
|---|----------|-----|-----------|
| -1 | 0 | 1 | Минимальная рассудительность |
| 0 | 0.5 | 3 | Баланс (5% видимое / 95% скрытое) |
| +1 | 1 | 5 | Максимальная рассудительность |

### Связь с видимой долей

Видимая доля информации:

```text
V/H = e^(-Q + Δ·range) = e^(-3 + 2Δ)
```

Рассудительность через отношение видимое/скрытое:

```text
C_reason = (ln(V/H) + 3) / 4
```

**Физический смысл**: Градации разумности линейно отображаются в Δ (и, следовательно, в D_f).

### Коллективная рассудительность

Для сети с персональными смещениями b_i:

```text
Δ_eff = Δ_global + Σ(q_i · b_i) / Σ(q_i)
```

```text
C_reason_collective = (1 + Δ_eff) / 2
```

где q_i — число кубитов хода i.

Коллективная рассудительность растёт как с общей массой/энергией (глобальное Δ), так и с выравниванием персональных смещений (синхронизация).

## Проверка согласованности

### С аксиомой A3

Критерии непосредственно следуют из определения синхронизации.

### С аксиомой A2

Δ_global есть среднее значение параметра Δ по сети.

### С аксиомой A4

Энтропия горизонта модулируется как 𝒮, так и Δ_global.
