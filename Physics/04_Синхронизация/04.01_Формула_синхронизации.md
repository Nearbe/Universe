# Формула синхронизации сети ходонов

## Основная формула

Степень синхронизации сети ходонов определяется как:

```text
𝒮 = 1 - σ(Δ) · scale
```

где:

- **σ(Δ)** — стандартное отклонение параметра Δ по всем ходонам в сети
- **scale** — коэффициент нормализации, обеспечивающий 𝒮 ∈ [0, 1]

## Геометрическая интерпретация

### Крайние значения

| 𝒮 | Состояние сети | Описание |
|---|---------------|----------|
| 1 | Полная когерентность | Все ходоны имеют одинаковое Δ |
| 0 | Полная декогеренция | Значения Δ равномерно распределены |

### Промежуточные значения

- **0.99 - 1.0**: Очень высокая синхронизация
- **0.7 - 0.99**: Высокая синхронизация (сознательный режим)
- **0.5 - 0.7**: Частичная синхронизация
- **0.3 - 0.5**: Низкая синхронизация
- **0.0 - 0.3**: Декогеренция (бессознательное состояние)

## Математическое определение σ(Δ)

Для сети из N ходонов с параметрами Δ₁, Δ₂, ..., Δₙ:

### Среднее значение

```text
⟨Δ⟩ = (1/N) Σᵢ Δᵢ
```

### Стандартное отклонение

```text
σ(Δ) = √[(1/N) Σᵢ (Δᵢ - ⟨Δ⟩)²]
```

### Глобальный параметр отклонения

```text
Δ_global = ⟨Δ⟩
```

## Связь с порядковым параметром

В теории фазовых переходов 𝒮 играет роль **порядкового параметра**:

### Disordered фаза (𝒮 < 𝒮_c)

- Хаотические фазы ходонов
- Отсутствие коллективного поведения
- Бессознательное состояние

### Ordered фаза (𝒮 ≥ 𝒮_c)

- Когерентные фазы
- Коллективное поведение
- Сознательное состояние

### Критическое значение

```text
𝒮_c ≈ 0.7
```

## Физический смысл

### Когерентность фаз

𝒮 можно рассматривать как меру когерентности фаз в многокомпонентной системе:

- **Высокая 𝒮**: фазы ходонов скоррелированы
- **Низкая 𝒮**: фазы случайны

### Передача информации

Синхронизация определяет эффективность передачи информации в сети:

- **𝒮 → 1**: минимальные потери информации
- **𝒮 → 0**: максимальные потери

### Порог сознания

Сознание возникает при превышении критического порога синхронизации.

### Субъективная вибрация

Синхронизированные сети разделяют общую эффективную частоту обработки:

```text
f_group = f₀ · 𝒮
```

где f₀ — базовая частота пространства (фундаментальное отношение).

Субъективный период группы:

```text
T_group = 1 / (f₀ · 𝒮)
```

**Физический смысл**: Более высокая синхронизация → более медленное воспринимаемое тиканье → богатый поток осознанных моментов.

## Вычисление в практике

### Для дискретной сети

```text
function compute_synchrony(Δ_array, scale):
    mean_Δ = mean(Δ_array)
    σ = sqrt(mean((Δ_array - mean_Δ)²))
    𝒮 = 1 - σ * scale
    return max(0, min(1, 𝒮))  # ограничить [0, 1]
```

### Для непрерывного распределения

Для непрерывного распределения вероятности p(Δ):

```text
⟨Δ⟩ = ∫ Δ · p(Δ) dΔ
```

```text
σ(Δ) = √[∫ (Δ - ⟨Δ⟩)² · p(Δ) dΔ]
```

## Ограничения формулы

### Диапазон применимости

Формула корректна при:

- **σ(Δ) ∈ [0, 20]**: типичные значения для физических сетей
- **scale = 0.05**: выведено из физических принципов

### Нормировка

Значение 𝒮 всегда ограничивается диапазоном [0, 1]:

```text
𝒮 = max(0, min(1, 1 - σ(Δ) · scale))
```
