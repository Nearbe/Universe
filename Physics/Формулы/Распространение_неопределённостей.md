# Распространение неопределённостей

## Определение

Формулы для оценки погрешности производных величин.

## Основное правило

```text
δf = |∂f/∂x| · δx
```

## Для функции нескольких переменных

```text
δf = √[(∂f/∂x · δx)² + (∂f/∂y · δy)² + ...]
```

## Сохранение информации

```text
ΔI = ΔE = ΔM
```

Каждое увеличение информации должно соответствовать равному увеличению энергии или массы.
