# Лагранжиан

## Определение

L — функция Лагранжа, характеризующая динамику системы.

## Формулы

### Для свободной частицы

```text
L = ½ m v²
```

### Для частицы в потенциале

```text
L = ½ m v² - V(x)
```

### Для поля Ψ

```text
L = ½ ∂_μ Ψ† ∂^μ Ψ - V(Ψ†Ψ)
```

## Энергия (функция Гамильтона)

```text
E = Σ q̇_i (∂L/∂q̇_i) - L = const
```
