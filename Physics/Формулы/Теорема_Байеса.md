# Теорема Байеса

## Определение

Теорема Байеса — фундаментальное соотношение в теории вероятностей, позволяющее обновлять вероятности гипотез при получении новых данных.

## Формула

```text
P(A|B) = P(B|A) · P(A) / P(B)
```

где:

- `P(A|B)` — апостериорная вероятность A при условии B
- `P(B|A)` — функция правдоподобия
- `P(A)` — априорная вероятность A
- `P(B)` — маргинальная вероятность B

## Применение в IME

Оценка параметров Δ и 𝒮 из экспериментальных данных.
