# Доверительные интервалы

## Определение

Доверительный интервал — интервальная оценка параметра, которая с заданной вероятностью содержит истинное значение параметра.

## Для среднего

При известном σ:

```text
⟨x⟩ ± z_(α/2) · σ/√N
```

где z_(α/2) — квантиль нормального распределения.

## Для σ = 95% доверительный интервал

```text
z_(0.025) ≈ 1.96
```

## Применение в IME

Оценка погрешности среднего значения параметра отклонения ⟨Δ⟩.
