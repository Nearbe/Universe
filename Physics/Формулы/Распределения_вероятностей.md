# Стандартные распределения вероятностей

## Определения

Основные распределения, используемые в IME.

## Дискретные распределения

### Биномиальное

```text
P(k) = C(n,k) · p^k · (1-p)^(n-k)
```

### Пуассона

```text
P(k) = (λ^k / k!) · e^(-λ)
```

## Непрерывные распределения

### Гауссовское (нормальное)

```text
P(x) = (1/√(2πσ²)) · exp[-(x-μ)²/(2σ²)]
```

где:

- **μ** — среднее
- **σ²** — дисперсия

### Равномерное

```text
P(x) = 1/(b-a)  для x ∈ [a,b]
```

## Распределение Δ

Обычно предполагается гауссовским:

```text
P(Δ) = (1/√(2πσ²)) · exp[-(Δ-⟨Δ⟩)²/(2σ²)]
```
