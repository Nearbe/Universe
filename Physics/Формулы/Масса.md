# Масса

## Определение

M — масса системы, определяемая через волновую функцию.

## Формула

```text
M = Σ |Ψᵢ|² = Σ (Reᵢ² + Imᵢ²)
```

## Описание

Масса вычисляется как сумма квадратов модулей волновой функции по всем состояниям.
