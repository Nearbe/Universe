# Решение систем линейных уравнений

## Определение

Методы решения системы Ax = b.

## Метод Гаусса

Прямое решение системы.

## Итерационные методы

### Метод Якоби

```text
x_i^{(k+1)} = (b_i - Σ_{j≠i} A_ij x_j^{(k)}) / A_ii
```

### Метод Гаусса-Зейделя

Использует обновлённые значения:

```text
x_i^{(k+1)} = (b_i - Σ_{j<i} A_ij x_j^{(k+1)} - Σ_{j>i} A_ij x_j^{(k)}) / A_ii
```

## Вычисление собственных значений

### Степенной метод

Нахождение наибольшего собственного значения:

```text
v_{n+1} = A · v_n / ||A · v_n||
```

### QR-алгоритм

Для всех собственных значений.
