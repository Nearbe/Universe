- **Q** = 3 — базовая размерность
- **range** = 2
- **scale** = 0.05
- **𝒮_c** ≈ 0.7 — порог сознания
- **|Δ_global|_threshold** = 0.2
- **|bias|_threshold** = 10 — порог понимания
