# Масса частицы

## Определение

m — масса частицы, определяемая через структуру ходонной сети.

## Формула

```text
m = m_P · (N - |E_binding| / m_P)
```

## Параметры

- **m_P** — планковская масса
- **N** — число ходонов
- **E_binding** — энергия связи

## Описание

Масса возникает из баланса между числом ходонов и энергией связи.
