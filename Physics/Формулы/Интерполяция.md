# Интерполяция

## Определение

Восстановление функции по известным значениям в точках.

## Полином Лагранжа

```text
P(x) = Σ y_i · L_i(x)
```

где:

```text
L_i(x) = Π_{j≠i} (x - x_j)/(x_i - x_j)
```
