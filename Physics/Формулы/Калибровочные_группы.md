# Калибровочные группы

## Определение

Группы симметрии, соответствующие фундаментальным взаимодействиям.

## Таблица групп

| Группа | Размерность | Взаимодействие |
|--------|-------------|----------------|
| U(1) | 1 | Электромагнетизм |
| SU(2) | 3 | Слабое взаимодействие |
| SU(3) | 8 | Сильное взаимодействие |
| SO(3,1) | 6 | Группа Лоренца |
| ISO(3,1) | 10 | Группа Пуанкаре |

## U(1) — электромагнетизм

```text
U = e^(iα), α ∈ ℝ
```

Преобразование поля:

```text
Ψ → e^(iα) Ψ
```

## SU(2) — слабое взаимодействие

Генераторы — матрицы Паули:

```text
σ₁ = (0 1; 1 0)
σ₂ = (0 -i; i 0)
σ₃ = (1 0; 0 -1)
```

## SU(3) — сильное взаимодействие

Генераторы — матрицы Гелл-Манна λ_a (a = 1...8).

## Группа Лоренца

Преобразования, сохраняющие интервал:

```text
ds² = c²dt² - dx² - dy² - dz²
```

Генераторы:

- **Вращения**: J_i (3 генератора)
- **Бусты**: K_i (3 генератора)

## Группа Пуанкаре

Группа Лоренца + сдвиги пространства-времени.

Генераторы:

- **Импульс**: P_μ (4 генератора)
- **Момент**: M_μν (6 генераторов)
