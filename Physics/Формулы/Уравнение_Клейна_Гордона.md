# Уравнение Клейна-Гордона

## Определение

Уравнение для скалярного поля с массой.

## Формула

```text
(□ + m²)φ = 0
```

## Даламбертиан

```text
□ = ∂_μ∂^μ = (1/c²)∂²/∂t² - ∇²
```

## Описание

Описывает распространение возмущений Δ в сети ходонов.
