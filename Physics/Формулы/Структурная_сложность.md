# Структурная сложность

## Определение

Q — структурная сложность (энергия вибрации), определяющая количество связанных состояний.

## Формула

```text
Q = e^|Δ|
```

## Значения

| Δ | Q |
|---|---|
| 0 | 1 |
| ±1 | ≈ 2.72 ≈ 3 |
| ±2 | ≈ 7.39 |

## При Q = 3

При Q = 3 (барионы):

```text
visibleFraction = e^(-3) ≈ 5%
```
