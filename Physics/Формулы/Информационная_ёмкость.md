# Информационнаяёмкость

## Определение

D_f — фрактальная размерность, определяющая как информация масштабируется с размером системы.

## Формула

```text
D_f = Q + Δ · range
```

## Параметры

- **Q** = 3 — базовая размерность
- **range** = 2

## Частные случаи

| Δ | D_f |
|---|-----|
| 0 | Q = 3 |
| +1 | Q + range = 5 |
| -1 | Q - range = 1 |
