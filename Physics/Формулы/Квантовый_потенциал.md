# Квантовый потенциал

## Определение

Q — квантовый потенциал, возникающий из неоднородностей плотности вероятности.

## Формула

```text
Q = -(ħ²/(2m)) · (∇²√ρ)/√ρ
```

## Описание

Вводится в преобразовании Маделунга для уравнения Шрёдингера.
