# Типы сетей

## Определение

Классификация сетей по структурным свойствам.

## Регулярные сети

Все вершины имеют одинаковую степень.

## Случайные сети (Эрдёша-Реньи)

Рёбра добавляются с вероятностью p.

## Безмасштабные сети

Распределение степеней следует степенному закону:

```text
P(k) ∝ k^(-γ)
```

где γ ≈ 2-3.

## Малый мир

Высокая кластеризация при короткой средней длине пути.

## Перколяция

### Порог перколяции

Критическая вероятность p_c, при которой возникает гигантский кластер.

### Для квадратной решётки

```text
p_c ≈ 0.5927
```

В терминах IME перколяция соответствует возникновению глобальной синхронизации.
