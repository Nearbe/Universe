# Метод максимального правдоподобия

## Определение

Метод максимального правдоподобия (ММП) — метод оценки параметров распределения, основанный на максимизации функции правдоподобия.

## Функция правдоподобия

```text
L(θ) = P(data|θ)
```

## Оценка параметра

```text
θ̂ = argmax L(θ)
```

## Для гауссовского распределения

Оценки максимального правдоподобия:

```text
μ̂ = (1/N) Σ xᵢ
```

```text
σ̂² = (1/N) Σ (xᵢ - μ̂)²
```

## Применение в IME

Оценка параметров Δ и 𝒮 из экспериментальных данных.
