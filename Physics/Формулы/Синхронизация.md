# Синхронизация

## Определение

Синхронизация 𝒮 — мера когерентности сети ходонов, порядковый параметр фазового перехода.

## Формула

```text
𝒮 = 1 - σ(Δ) · scale
```

## Параметры

- **σ(Δ)** — стандартное отклонение Δ по сети
- **scale** = 0.05

## Диапазон

𝒮 ∈ [0, 1]

## Критическое значение

𝒮_c ≈ 0.7 — порог сознания

При 𝒮 ≥ 𝒮_c возникает коллективное поведение.
