# Уравнение Дирака

## Определение

Уравнение для фермионов со спином 1/2.

## Формула

```text
(iγ^μ∂_μ - m)Ψ = 0
```

## Свойства матриц Дирака

```text
{γ^μ, γ^ν} = 2g^μν
```

где g^μν — метрический тензор.
