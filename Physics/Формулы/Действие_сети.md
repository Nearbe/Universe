# Действие для сети ходонов

## Определение

Действие для ходонной сети с учётом параметра Δ.

## Формула

```text
S_network = ∫ d⁴x [½ ∂_μ Ψ† ∂^μ Ψ - V(Ψ†Ψ, Δ)]
```

## Описание

Потенциал зависит от Δ:

```text
V = V(Ψ†Ψ, Δ)
```

Вариация по Δ даёт уравнение движения для поля Δ.
