# Спектральная теория графов

## Определение

Методы анализа сетей через собственные значения матриц.

## Матрица смежности

```text
A_ij = 1, если (i,j) ∈ E
A_ij = 0, иначе
```

## Лапласиан графа

```text
L = D - A
```

где D — диагональная матрица степеней.

## Собственные значения

Собственные значения лапласиана λ_i определяют динамику сети.

### Наименьшее ненулевое

λ₂ (algebraic connectivity) характеризует связность сети.
