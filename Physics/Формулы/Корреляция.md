# Корреляция Пирсона

## Определение

Коэффициент корреляции Пирсона — мера линейной связи между двумя переменными.

## Формула

```text
ρ_XY = Cov(X,Y) / (σ_X · σ_Y)
```

## Ковариация

```text
Cov(X,Y) = ⟨(X-⟨X⟩)(Y-⟨Y⟩)⟩
```

## Диапазон

```text
ρ_XY ∈ [-1, +1]
```

- ρ = +1: положительная корреляция
- ρ = -1: отрицательная корреляция
- ρ = 0: нет линейной связи

## В терминах IME

Корреляция между Δ в разных точках сети характеризует синхронизацию.
