# Частные производные Δ

## Определение

Частные производные параметра Δ по Re и Im.

## Формулы

```text
∂Δ/∂Re = 1/Re
```

```text
∂Δ/∂Im = -1/Im
```

## Второй дифференциал

```text
d²Δ = -dRe²/Re² + dIm²/Im²
```
