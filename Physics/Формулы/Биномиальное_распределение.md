# Биномиальное распределение

## Определение

Биномиальное распределение — дискретное распределение вероятностей числа успехов в n независимых испытаниях Бернулли.

## Формула

```text
P(k) = C(n,k) · p^k · (1-p)^(n-k)
```

где:

- `n` — число испытаний
- `k` — число успехов
- `p` — вероятность успеха в одном испытании
- `C(n,k)` — биномиальный коэффициент

## Параметры

- Математическое ожидание: ⟨k⟩ = np
- Дисперсия: σ² = np(1-p)

## Применение в IME

Моделирование числа синхронизированных ходонов в сети.
