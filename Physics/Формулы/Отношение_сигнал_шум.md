# Отношение сигнал/шум

## Определение

SNR — отношение сигнал/шум, характеризующее качество передачи информации.

## Формула

```text
SNR = [1 - σ(Δ)·scale] / [σ(Δ)·scale]
```

## Описание

- Высокая синхронизация (низкое σ) → высокий SNR
- Низкая синхронизация → низкий SNR
