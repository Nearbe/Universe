# Плотность энергии

## Определение

ρ — масс-энергетическая плотность, связанная с параметром Δ.

## Формула

```text
ρ ∝ e^Δ
```

## Описание

- При Δ → +∞: ρ → ∞ (сингулярность)
- При Δ → -∞: ρ → 0 (разреженное состояние)
- При Δ = 0: ρ = const (стандартная плотность)
