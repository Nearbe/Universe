# Иерархические сети

## Определение

Иерархические сети — сети с многоуровневой структурой организации.

## Уровни иерархии

```text
Level 0: отдельные ходоны
Level 1: локальные кластеры
Level 2: глобальные структуры
```

## Масштабирование

Свойства на разных уровнях связаны степенными законами.
