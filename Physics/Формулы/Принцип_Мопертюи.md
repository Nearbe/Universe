# Принцип Мопертюи

## Определение

Вариационный принцип для консервативных систем.

## Формула

Действие в форме Мопертюи:

```text
S₀ = ∫ p dq
```

где p — обобщённый импульс.

## Условие

При фиксированной энергии E:

```text
δS₀ = 0
```
