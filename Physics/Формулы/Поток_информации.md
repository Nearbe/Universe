# Поток информации

## Определение

j — поток информации в квантовой системе.

## Формула

```text
j = (ħ/m) · ρ · ∇θ
```

## Параметры

- **ħ** — постоянная Планка
- **m** — масса
- **ρ** = |Ψ|² — плотность вероятности
- **θ** — фаза волновой функции
