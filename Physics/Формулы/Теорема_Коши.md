# Теорема Коши для групп

## Формулировка

Число неприводимых представлений конечной группы равно числу классов сопряжённости.

## Сумма квадратов размерностей

```text
Σ (dim R_i)² = |G|
```

где:

- `dim R_i` — размерность i-го неприводимого представления
- `|G|` — порядок группы

## Применение в теории групп

Эта теорема используется для:

- Определения количества неприводимых представлений
- Проверки полноты набора представлений
- Вычисления размерностей представлений
