# Байесовская статистика

## Определение

Байесовская статистика — подход к статистическому анализу, использующий теорему Байеса для обновления вероятностей параметров.

## Основные понятия

### Априорная вероятность

```text
P(θ) — до получения данных
```

### Апостериорная вероятность

```text
P(θ|data) ∝ P(data|θ) · P(θ)
```

## Применение в IME

Оценка параметров Δ и 𝒮 из экспериментальных данных.
