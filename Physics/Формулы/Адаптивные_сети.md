# Адаптивные сети

## Определение

Адаптивные сети — сети, в которых структура связей эволюционирует во времени в зависимости от состояния узлов.

## Эволюция связей

```text
dw_ij/dt = f(Δ_i, Δ_j)
```

## Пластичность

### Правило Хебба

```text
Δw_ij ∝ pre_i · post_j
```

Связь усиливается, когда оба узла активны одновременно.

## Применение в IME

Моделирование динамики ходонной сети с изменяющимися связями.
