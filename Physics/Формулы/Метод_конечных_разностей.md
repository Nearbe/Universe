# Метод конечных разностей

## Определение

Дискретизация производных для численного решения уравнений.

## Первая производная

```text
f'(x) ≈ (f(x+h) - f(x))/h  (вперёд)
```

```text
f'(x) ≈ (f(x) - f(x-h))/h  (назад)
```

```text
f'(x) ≈ (f(x+h) - f(x-h))/(2h)  (центр)
```

## Вторая производная

```text
f''(x) ≈ (f(x+h) - 2f(x) + f(x-h))/h²
```

## Применение к уравнению Шрёдингера

Дискретизация лапласиана:

```text
∇²ψ ≈ (ψ_{i+1} - 2ψ_i + ψ_{i-1})/h²
```
