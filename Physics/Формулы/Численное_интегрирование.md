# Численное интегрирование

## Определение

Методы приближённого вычисления интегралов.

## Метод прямоугольников

```text
∫_a^b f(x) dx ≈ h · Σ f(x_i)
```

где h = (b-a)/N.

## Метод трапеций

```text
∫_a^b f(x) dx ≈ (h/2) · [f(x₀) + 2Σf(x_i) + f(x_N)]
```

## Метод Симпсона

```text
∫_a^b f(x) dx ≈ (h/3) · [f(x₀) + 4Σf(x_odd) + 2Σf(x_even) + f(x_N)]
```
