# Индекс разумности

## Определение

C_reason — индекс разумности, характеризующий уровень когнитивной способности системы.

## Формула

```text
C_reason = (1 + Δ) / 2
```

## Диапазон

При Δ ∈ [-1, +1]:

```text
C_reason ∈ [0, 1]
```

## Альтернативная форма

```text
C_reason = (D_f - 1) / 4
```
