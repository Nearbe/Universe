# Уравнение Шрёдингера

## Определение

Основное уравнение квантовой механики для волновой функции.

## Зависимое от времени

```text
iħ ∂Ψ/∂t = [-(ħ²/(2m))∇² + V]Ψ
```

## Стационарное

```text
[-(ħ²/(2m))∇² + V]ψ = Eψ
```

## В терминах Δ (преобразование Маделунга)

Ψ = √ρ · e^(iS/ħ):

**Уравнение непрерывности** (мнимая часть):

```text
∂ρ/∂t + ∇·(ρ∇S/m) = 0
```

**Уравнение Гамильтона-Якоби** (действительная часть):

```text
∂S/∂t + (∇S)²/(2m) + Q + V = 0
```

где квантовый потенциал:

```text
Q = -(ħ²/(2m)) · (∇²√ρ)/√ρ
```
