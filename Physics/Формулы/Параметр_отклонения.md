# Параметр отклонения

## Определение

Параметр Δ — безразмерная величина, определяющая структуру квантового состояния через соотношение действительной и мнимой частей волновой функции.

## Формула

```text
Δ = ln(|Re|) - ln(|Im|)
```

## Эквивалентные формы

- `Δ = ln(|Re|/|Im|)`
- `Δ = ln(|cot(θ)|)` — через фазу θ

## Диапазон

| Режим | Диапазон |
|-------|----------|
| Обычная физика | Δ ∈ [-1, +1] |
| Полный | Δ ∈ ℝ ∪ {±∞} |

## Предельные случаи

| Условие | Δ |
|---------|---|
| Im → 0 | +∞ |
| Re → 0 | -∞ |
| \|Re\| = \|Im\| | 0 |
