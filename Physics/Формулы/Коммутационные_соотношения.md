# Коммутационные соотношения групп

## Определение

Коммутационные соотношения определяют структуру алгебры Ли группы.

## Формула генераторов

```text
[T_a, T_b] = i f_abc T_c
```

где:

- `T_a` — генераторы группы
- `f_abc` — структурные константы группы

## Для матриц Паули (SU(2))

```text
[σ_a/2, σ_b/2] = i ε_abc (σ_c/2)
```

где `ε_abc` — символ Леви-Чивиты.

## Для группы Лоренца

```text
[J_i, J_j] = i ε_ijk J_k
[J_i, K_j] = i ε_ijk K_k
[K_i, K_j] = -i ε_ijk J_k
```

где:

- `J_i` — генераторы вращений
- `K_i` — генераторы бустов
