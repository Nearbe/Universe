# Даламбертиан

## Определение

□ — даламбертиан (волновой оператор).

## Формула

```text
□ = ∂_μ∂^μ = (1/c²)∂²/∂t² - ∇²
```

## Использование

В уравнении Клейна-Гордона:

```text
(□ + m²)φ = 0
```
