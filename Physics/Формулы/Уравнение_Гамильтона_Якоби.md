# Уравнение Гамильтона-Якоби

## Определение

Уравнение для действия S в квантовой механике.

## Формула

```text
∂S/∂t + (∇S)²/(2m) + Q + V = 0
```

## Параметры

- **S** — действие
- **m** — масса
- **Q** — квантовый потенциал
- **V** — классический потенциал
