# Оператор эволюции IME

## Определение

Оператор эволюции — унитарный оператор, описывающий эволюцию ходонной сети во времени.

## Унитарная эволюция

```text
|Ψ(t+1)⟩ = U_Δ |Ψ(t)⟩

U_Δ = exp(-i·H_Δ·t/ℏ)
```

где H_Δ — гамильтониан с Δ-зависимостью.

## Гамильтониан с Δ

```text
H_Δ = H_0 + Δ·V

H_0 = ∫ d³x [½ π² + ½ (∇Ψ)² + m² |Ψ|²]

V = ∫ d³x |Ψ|²
```

где π — канонически сопряжённый импульс.

## Δ-эволюция

Параметр Δ эволюционирует без деления на ноль:

```text
Δ(t) = (M(t) - 1) / (M(t) + 1)

M(t+1) = M(t) + δM_cascade
```

### Границы Δ

**Для обычной физики (5% видимая Вселенная):**

- При M → 0: Δ → -1 (горизонт, конечное)
- При M → ∞: Δ → +1 (супер-фрактальное, конечное)
- При M = 1: Δ = 0 (равновесие)

**Для фазовых переходов:**

- При Re → ∞, Im → 0: Δ → +∞ (сингулярность)
- При Re → 0, Im → ∞: Δ → -∞ (тёмная энергия)
