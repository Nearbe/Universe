# Динамика на сетях

## Определение

Уравнения, описывающие эволюцию параметров сети ходонов.

## Распространение возмущений

Уравнение для возмущения в узле i:

```text
dΔ_i/dt = -Γ Δ_i + Σ_j w_ij Δ_j
```

где:

- **Γ** — скорость затухания
- **w_ij** — веса связей

## Синхронизация на сетях

Условие синхронизации:

```text
𝒮 = 1 - σ(Δ) · scale ≥ 0.7
```
