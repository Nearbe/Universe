# Интегрированная информация

## Определение

Φ — интегрированная информация (мера сознания по IIT).

## Формула

```text
Φ ∝ 𝒮 · (1 - σ(Δ)/σ_max) · f(Δ_global)
```

## Описание

Φ измеряет количество информации, генерируемое системой как целым, превышающим сумму информации её частей.
