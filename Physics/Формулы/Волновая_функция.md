# Волновая функция

## Определение

Ψ — волновая функция, описывающая состояние ходона (комплексное число).

## Формула

```text
Ψ = Re + i·Im
```

## Полярная форма

```text
Ψ = |Ψ| · e^(iθ) = |Ψ| · (cosθ + i·sinθ)
```

## Модуль

```text
|Ψ| = √(Re² + Im²)
```
