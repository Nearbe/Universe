# Метод конечных элементов

## Определение

Метод конечных элементов — численный метод решения дифференциальных уравнений, основанный на разбиении области на малые элементы.

## Разбиение области

Область разбивается на конечные элементы (треугольники, четырёхугольники в 2D; тетраэдры, параллелепипеды в 3D).

## Базисные функции

Решение ищется в виде:

```text
u(x) ≈ Σ c_i · φ_i(x)
```

где φ_i(x) — базисные функции (обычно полиномы).

## Слабая форма

Интегральная форма дифференциального уравнения:

```text
∫ Ω (∇v · ∇u) dΩ = ∫ Γ v·g dΓ
```

## Применение в IME

Решение уравнений для поля Δ в сложных геометриях.
