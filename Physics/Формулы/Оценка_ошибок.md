# Оценка ошибок

## Определение

Оценка погрешности численных методов — важная часть вычислительного эксперимента.

## Погрешность округления

```text
ε_machine ≈ 10⁻¹⁶ (double precision)
```

Определяется разрядностью используемого типа данных.

## Погрешность усечения

Зависит от метода:

- Метод Эйлера: O(h)
- Метод Рунге-Кутты 4-го порядка: O(h⁴)
- Конечные разности (центральные): O(h²)

## Накопление ошибок

При длительных вычислениях ошибки могут накапливаться (экспоненциальный рост для неустойчивых задач).
