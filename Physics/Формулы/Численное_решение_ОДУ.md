# Численное решение ОДУ

## Определение

Методы решения обыкновенных дифференциальных уравнений.

## Метод Эйлера

Для уравнения dy/dt = f(y,t):

```text
y_{n+1} = y_n + h · f(y_n, t_n)
```

## Метод Рунге-Кутты 4-го порядка

```text
k₁ = h · f(y_n, t_n)
k₂ = h · f(y_n + k₁/2, t_n + h/2)
k₃ = h · f(y_n + k₂/2, t_n + h/2)
k₄ = h · f(y_n + k₃, t_n + h)

y_{n+1} = y_n + (k₁ + 2k₂ + 2k₃ + k₄)/6
```

## Применение в IME

Интегрирование уравнения движения для Δ:

```text
∂_t² Δ + 3H ∂_t Δ + m_Δ² Δ = 0
```
