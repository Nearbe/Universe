# Функционал действия

## Определение

S — функционал от траектории системы.

## Формула

```text
S[q(t)] = ∫ L(q, q̇, t) dt
```

где L — лагранжиан системы.

## Принцип наименьшего действия

Истинная траектория системы доставляет экстремум действия:

```text
δS = 0
```

## Уравнения Эйлера-Лагранжа

```text
∂L/∂q - d/dt(∂L/∂q̇) = 0
```
