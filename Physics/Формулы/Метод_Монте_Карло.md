# Метод Монте-Карло

## Определение

Статистический метод численного интегрирования.

## Интегрирование Монте-Карло

```text
∫ f(x) dx ≈ (V/N) · Σ f(x_i)
```

где x_i — случайные точки в объёме V.

## Марковские цепи Монте-Кaрло (MCMC)

Генерация выборки из распределения P(x):

1. Предложить новое состояние x'
2. Принять с вероятностью min(1, P(x')/P(x))

## Применение в IME

Оценка параметров сети ходонов из экспериментальных данных.
