# Преобразование Фурье

## Определение

Разложение функции на частотные компоненты.

## Дискретное преобразование Фурье (DFT)

```text
X_k = Σ_{n=0}^{N-1} x_n · e^(-2πink/N)
```

## Быстрое преобразование Фурье (FFT)

Эффективный алгоритм для DFT:

```text
O(N log N) вместо O(N²)
```

## Применение в IME

Анализ частотных мод сети ходонов.
