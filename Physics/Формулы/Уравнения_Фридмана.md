# Уравнения Фридмана

## Определение

Уравнения, описывающие динамику однородной изотропной Вселенной.

## Первое уравнение (энергии)

```text
H² = (8πG/3) · ρ - kc²/a² + Λ/3
```

## Второе уравнение (ускорения)

```text
ä/a = -(4πG/3) · (ρ + 3p/c²) + Λ/3
```

## Параметры

- **H** = ȧ/a — параметр Хаббла
- **a(t)** — коэффициент масштабирования
- **k** — кривизна
- **Λ** — космологическая постоянная

## В терминах IME

С учётом зависимости ρ от Δ:

```text
ρ_Δ ∝ e^(Δ/2)
```
