# Матрицы Паули

## Определение

Матрицы Паули — набор из трёх комплексных матриц 2×2, являющихся генераторами группы SU(2).

## Формулы

```text
σ₁ = | 0  1 |
     | 1  0 |

σ₂ = | 0 -i |
     | i  0 |

σ₃ = | 1  0 |
     | 0 -1 |
```

## Свойства

- Эрмитовость: σ_i = σ_i†
- След: Tr(σ_i) = 0
- Определитель: det(σ_i) = -1
- Коммутатор: [σ_a, σ_b] = 2i ε_abc σ_c

## Применение

Генераторы SU(2): T_a = σ_a/2

Используются для описания:

- Слабого взаимодействия (изospin)
- Спина электрона
- Квантовых двухуровневых систем
