# Спонтанное нарушение симметрии

## Определение

Спонтанное нарушение симметрии — ситуация, когда гамильтониан системы симметричен, но основное состояние (вакуум) этой симметрией не обладает.

## Механизм Хиггса

Потенциал с нарушенной симметрией:

```text
V(Φ) = -μ² Φ†Φ + λ(Φ†Φ)²
```

Минимум при:

```text
⟨Φ⟩ = v/√2 ≠ 0
```

где v — вакуумное среднее значения поля Хиггса.

## В терминах IME

Соответствует ненулевому среднему значению ⟨Δ⟩ ≠ 0.
