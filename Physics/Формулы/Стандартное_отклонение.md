# Стандартное отклонение

## Определение

σ(Δ) — стандартное отклонение параметра Δ по сети ходонов.

## Формула (дискретная)

```text
σ(Δ) = √[(1/N) Σᵢ (Δᵢ - ⟨Δ⟩)²]
```

где ⟨Δ⟩ — среднее значение.

## Формула (непрерывная)

```text
σ(Δ) = √[∫ (Δ - ⟨Δ⟩)² · p(Δ) dΔ]
```
