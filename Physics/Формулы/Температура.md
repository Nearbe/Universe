# Температура

## Определение

T — температура системы, связанная со степенью синхронизации.

## Формула

```text
T ∝ 1/σ(Δ)
```

## Описание

- Низкая синхронизация (высокое σ) → низкая температура
- Высокая синхронизация (низкое σ) → высокая температура
