# Уравнение непрерывности

## Определение

Уравнение сохранения информационной плотности.

## Формула

```text
∂ρ/∂t + ∇·j = 0
```

## Параметры

- **ρ** = |Ψ|² — информационная плотность
- **j** — поток информации

## Поток информации

```text
j = (ħ/m) · ρ · ∇θ
```
