# Уравнение движения поля Δ

## Определение

Уравнение, описывающее эволюцию параметра Δ в расширяющейся Вселенной.

## Формула

```text
∂_t² Δ + 3H ∂_t Δ + V'(Δ) = 0
```

## Параметры

- **H** = ȧ/a — параметр Хаббла
- **3H ∂_t Δ** — член трения от расширения
- **V(Δ)** — потенциал поля Δ

## Для квадратичного потенциала

```text
V(Δ) = ½ m_Δ² Δ²
```

Уравнение принимает вид:

```text
∂_t² Δ + 3H ∂_t Δ + m_Δ² Δ = 0
```
