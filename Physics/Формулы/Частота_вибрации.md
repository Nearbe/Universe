# Частота вибрации

## Определение

Частота вибрации f — характеристика ходона, определяющая темп вычисления (субъективное "тиканье" пространства-времени).

## Формула

```text
f = e^(-Δ)
```

## Описание

- При Δ → +∞: f → 0 (остановка вычислений, горизонт событий)
- При Δ → -∞: f → ∞ (очень быстрые процессы)
- При Δ = 0: f = 1 (базовая частота)
