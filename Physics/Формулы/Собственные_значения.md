# Вычисление собственных значений

## Определение

Нахождение собственных значений матрицы — ключевая задача в численных методах для анализа динамики систем.

## Степенной метод

Нахождение наибольшего собственного значения:

```text
v_{n+1} = A · v_n / ||A · v_n||
```

Сходится к собственному вектору, соответствующему наибольшему по модулю собственному значению.

## QR-алгоритм

Итерационный метод для нахождения всех собственных значений.

## Применение в IME

- Анализ устойчивости сети ходонов
- Спектральный анализ матрицы Лапласа графа
