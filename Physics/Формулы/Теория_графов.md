# Основы теории графов

## Определение

Граф G = (V, E), где:

- **V** — множество вершин (ходонов)
- **E** — множество рёбер (взаимодействий)

## Степень вершины

Число рёбер, инцидентных вершине:

```text
k_i = |{j : (i,j) ∈ E}|
```

## Средняя степень

```text
⟨k⟩ = (1/N) Σ k_i
```
