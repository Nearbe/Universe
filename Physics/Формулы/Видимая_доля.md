# Видимая доля

## Определение

visibleFraction — доля информации в "видимой" части системы (в противоположность "скрытой").

## Формула

```text
visibleFraction = e^(-Q)
```

## Значение

При Q = 3:

```text
visibleFraction ≈ 5%
```

## Описание

- Q = 3 соответствует барионной материи
- 95% — скрытая информация
