# Перколяция

## Определение

Перколяция — фазовый переход, описывающий возникновение связанных кластеров в случайной среде.

## Порог перколяции

Критическая вероятность p_c, при которой возникает гигантский кластер.

### Для квадратной решётки

```text
p_c ≈ 0.5927
```

## В терминах IME

Перколяция соответствует возникновению глобальной синхронизации в ходонной сети.
