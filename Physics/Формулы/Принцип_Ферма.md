# Принцип Ферма

## Определение

Принцип Ферма — вариационный принцип в оптике, утверждающий, что луч света между двумя точками распространяется по траектории, для которой оптическая длина пути минимальна (стационарна).

## Формулировка

```text
δ∫ n ds = 0
```

где:

- `n` — показатель преломления среды
- `ds` — элемент длины пути
- `δ` — вариация

## Связь с другими принципами

Принцип Ферма является оптическим аналогом принципа Мопертюи:

| Принцип | Формулировка | Область |
|---------|--------------|---------|
| Ферма | δ∫ n ds = 0 | Оптика |
| Мопертюи | δ∫ p dq = 0 | Механика |

## Связь с принципом наименьшего действия

Для электромагнитной волны принцип Ферма получается из принципа наименьшего действия:

- Оптическая длина пути пропорциональна фазе волны
- Минимум фазы соответствует стационарному действию
