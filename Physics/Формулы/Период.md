# Период

## Определение

Период T — время одного вычислительного шага ходона (субъективное время).

## Формула

```text
T = e^Δ
```

## Описание

- При Δ → +∞: T → ∞ (замедление, горизонт событий)
- При Δ → -∞: T → 0 (очень быстрые процессы)
- При Δ = 0: T = 1
