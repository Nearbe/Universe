# Дифференциал параметра Δ

## Определение

dΔ — первый дифференциал параметра отклонения.

## Формула

```text
dΔ = d(ln(|Re|)) - d(ln(|Im|)) = dRe/Re - dIm/Im
```

## Частные производные

```text
∂Δ/∂Re = 1/Re
```

```text
∂Δ/∂Im = -1/Im
```

## Второй дифференциал

```text
d²Δ = -dRe²/Re² + dIm²/Im²
```
