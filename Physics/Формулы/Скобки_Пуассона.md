# Скобки Пуассона

## Определение

Классические скобки Пуассона для функций f(q,p) и g(q,p).

## Формула

```text
{f, g} = Σ (∂f/∂q_i · ∂g/∂p_i - ∂f/∂p_i · ∂g/∂q_i)
```

## Свойства

- **Антисимметрия**: {f, g} = -{g, f}
- **Линейность**: {af + bg, h} = a{f, h} + b{g, h}
- **Тождество Якоби**: {f, {g, h}} + {g, {h, f}} + {h, {f, g}} = 0

## Канонические скобки

```text
{q_i, p_j} = δ_ij
{q_i, q_j} = {p_i, p_j} = 0
```

## Переход к квантовой механике

Замена скобок Пуассона на коммутаторы:

```text
{f, g} → (1/iħ)[f̂, ĝ]
```

## Канонические коммутаторы

```text
[x̂, p̂] = iħ
```
