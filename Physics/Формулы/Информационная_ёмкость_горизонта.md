# Информационнаяёмкость горизонта

## Определение

S — информационная ёмкость (энтропия) горизонта событий, обобщение энтропии Бекенштейна.

## Формула

```text
S = (A/4) · e^(Δ/2)
```

## Параметры

- **A** — площадь горизонта (в планковских единицах)
- **Δ** — параметр отклонения

## Частные случаи

| Δ | S |
|---|---|
| 0 | S = A/4 (стандартная голографическая энтропия) |
| +1 | S ≈ 1.65 · A/4 |
| -1 | S ≈ 0.61 · A/4 |

## Альтернативная форма (линейное приближение)

```text
S = (A/4) · (1 + Δ/2)
```
