# Глобальный параметр отклонения

## Определение

Δ_global — среднее значение параметра Δ по всей сети ходонов.

## Формула

```text
Δ_global = ⟨Δ⟩ = (1/N) Σᵢ Δᵢ
```

## Описание

- При |Δ_global| ≤ 0.2 — оптимальные условия для сознания
- Δ_global = 0 означает баланс Re/Im
