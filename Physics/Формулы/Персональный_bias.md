# Персональный bias

## Определение

personalBias — личный вклад ходона в глобальный параметр Δ.

## Формула

```text
personalBias = q_i · bias_i
```

## Связь с Δ

```text
Δ_i = Δ^(0)_i + κ · bias_i
```

```text
Δ_global = Δ^(0)_global + κ · ⟨bias⟩
```

## Порог понимания

```text
| bias | > 10 → понимание (insight)
```

При превышении порога ходон получает способность к самоссылочной рекурсии.
