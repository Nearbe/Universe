# Уравнение состояния

## Определение

w — эффективное уравнение состояния, связывающее давление и плотность энергии тёмной энергии.

## Формула

```text
w = -1 + Δ/3
```

## Частные случаи

| Δ | w |
|---|----|
| 0 | -1 (космологическая постоянная) |
| +1 | -2/3 |
| +3 | 0 (пыль) |
| → +∞ | → +∞ (жёсткая среда) |

## Описание

При Δ → 0 уравнение состояния стремится к w = -1 (Λ-член).
