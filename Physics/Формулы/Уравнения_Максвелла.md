# Уравнения Максвелла (ковариантная форма)

## Определение

Уравнения электромагнитного поля в ковариантной форме.

## Формулы

```text
∂_μ F^μν = J^ν
```

```text
∂_[μ F_νλ] = 0
```

## Параметры

- **F^μν** = ∂^μ A^ν - ∂^ν A^μ — тензор электромагнитного поля
- **J^ν** — 4-ток

## Описание

В терминах IME описывают распространение фазовых мод сети ходонов.
