# Линейная регрессия

## Определение

Линейная регрессия — метод моделирования зависимости между переменными путём подгонки линейной функции.

## Модель

```text
y = ax + b + ε
```

где ε — случайная ошибка.

## Метод наименьших квадратов

Минимизация суммы квадратов ошибок:

```text
Σ (yᵢ - axᵢ - b)² → min
```

## Оценки параметров

```text
â = Cov(X,Y) / Var(X)
```

```text
b̂ = ⟨y⟩ - â · ⟨x⟩
```

## Применение в IME

Анализ зависимости между параметрами Δ в разных частях сети.
