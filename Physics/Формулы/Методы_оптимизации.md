# Методы оптимизации

## Определение

Методы нахождения экстремумов функций.

## Градиентный спуск

Минимизация функции f(x):

```text
x_{n+1} = x_n - η · ∇f(x_n)
```

где η — скорость обучения.

## Метод сопряжённых градиентов

Более быстрая сходимость для квадратичных функций.

## Генетические алгоритмы

Эволюционная оптимизация:

1. Популяция решений
2. Скрещивание и мутация
3. Отбор лучших
