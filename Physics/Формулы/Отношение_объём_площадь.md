# Отношение объём/площадь

## Определение

V/H — отношение объёма к площади, характеризующее масштабирование пространства.

## Формула

```text
V/H = e^(-Q + Δ·range)
```

При Q = 3, range = 2:

```text
V/H = e^(-3 + 2Δ)
```

## Описание

- При Δ = 0: V/H = e^(-3) ≈ 0.05 (5% видимое)
- Связывает видимую долю с фрактальной размерностью
