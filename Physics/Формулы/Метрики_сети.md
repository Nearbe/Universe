# Метрики сети

## Определение

Основные метрики для анализа сетей ходонов.

## Средняя длина пути

```text
L = (1/N(N-1)) Σ_{i≠j} d(i,j)
```

где d(i,j) — кратчайшее расстояние между i и j.

## Кластеризационный коэффициент

Для вершины i:

```text
C_i = (число треугольников с i) / (число троек с i)
```

Средний по сети:

```text
C = (1/N) Σ C_i
```

## Центральность

### Степенная центральность

```text
C_D(i) = k_i / (N-1)
```

### Посредническая центральность

```text
C_B(i) = Σ_{s≠i≠t} (σ_st(i) / σ_st)
```

где σ_st — число кратчайших путей между s и t.
