# Теорема T5: Фрактальная размерность сети ходонов

## Формулировка

Фрактальная размерность ходонной сети зависит от базовой размерности Q и параметра Δ:

```text
D_f = Q + Δ · range
```

где:

- **Q** = e^|Δ| — энергия вибрации (связь с Δ)
- **range** = Δ_max - Δ_min — диапазон флуктуаций Δ
- Δ ∈ (-∞, +∞) — полная Вселенная
- Δ ∈ [-1, +1] — видимая часть (5%)

## Структура Q

### Компоненты

```text
Q = energy
```

| Компонента | Обозначение | Диапазон | Описание |
|------------|-------------|----------|----------|
| energy | n | 0 - ∞ | Энергия вибраций пространства |

### Диапазон Q

```text
Q_min = 0  (нет вибраций)
Q_max = ∞  (теоретически без ограничений)
Q ∈ [0, ∞)
```

## Связь с видимой долей Вселенной

Видимая доля Вселенной связана с Q через:

```text
visibleFraction = e^(-Q)
```

При **Q ≈ 3** (типичная барионная материя):

```text
visibleFraction = e^(-3) ≈ 0.05 = 5%
```

Это объясняет, почему только ~5% энергии-массы Вселенной — видимая (барионная) материя, а ~95% — тёмная материя и тёмная энергия.

## Обобщённая формула Δ

Из структуры Q можно получить обобщённую формулу:

```text
Δ = (Q - M) / (Q + M)
```

где M — нормализованная масса/энергия/информация системы.

### Частные случаи

**При M = Q** (масса равна структурной сложности):

```text
Δ = 0  (точка равновесия)
```

**При M → ∞**:

```text
Δ → -1  (сжатие, чёрные дыры)
```

**При M → 0**:

```text
Δ → +1  (расширение, вакуум)
```

### Классическая формула как частный случай

При **Q = 1** (минимальная сложность):

```text
Δ = (1 - M) / (1 + M)  (классическая формула)
```

При **Q = 3** (полная структура):

```text
Δ = (3 - M) / (3 + M)
```

## Специальные случаи D_f

### При равновесии (Δ = 0)

```text
D_f = Q
```

### При Δ = +1

```text
D_f = Q + range  (максимальная видимая компонента)
```

### При Δ = -1

```text
D_f = Q - range  (максимальная скрытая компонента)
```

## Физическая интерпретация компонентов Q

### existence = 1

Фундаментальная единица бытия, присутствующая в каждом ходоне. Константа.

### particle = 0-1

Наличие вещественной компоненты или возбуждения:

- **1**: ходон содержит частицу/возбуждение
- **0**: ходон в вакуумном состоянии

### networkEnergy = 0-1

Энергия взаимодействий с ближайшими соседями:

- Пропорциональна амплитуде осцилляций
- Отражает кинетическую и потенциальную энергию сети

## Проверка согласованности

### С аксиомой A1

Изменения D_f приводят к эквивалентным изменениям I, E, M:

```text
I ∝ L^D_f,  E ∝ L^D_f,  M ∝ L^D_f
```

### С аксиомой A2

D_f зависит от Δ, который определён через Re и Im.

### С аксиомой A4

Энтропия горизонта масштабируется как:

```text
S ∝ L^D_f
```

Для D_f = 2 (обычная поверхность): S ∝ L² (голографический принцип)
Для D_f ≠ 2: модифицированный голографический принцип
