# Теорема T2: Параметр отклонения Δ

## Формулировка

Параметр отклонения Δ определяется как разность логарифмов абсолютных значений действительной и мнимой частей волновой функции:

```text
Δ = ln(|Re|) - ln(|Im|) ∈ ℝ ∪ {±∞}
```

## Альтернативные записи

При Re ≠ 0 и Im ≠ 0 эквивалентно:

```text
Δ = ln(|Re|/|Im|)
```

Однако форма **разности логарифмов** предпочтительнее для избежания сингулярностей при Im=0 или Re=0.

## Онтологические состояния

### Δ → +∞ (Сингулярность)

- **Условие**: Im → 0, Re ≠ 0
- **Физика**: полная предсказуемость, исчезновение мнимой части
- **Проявление**: чёрная дыра, гравитационная сингулярность

### Δ → -∞ (Тёмная энергия)

- **Условие**: Re → 0, Im ≠ 0
- **Физика**: доминирование скрытой информации, УФ-турбулентность
- **Проявление**: тёмная энергия, антигравитация

### Δ = 0 (Равновесие)

- **Условие**: |Re| = |Im|
- **Физика**: баланс видимой и скрытой информации
- **Проявление**: обычное квантовое состояние

### Δ = 0 и Re = Im = 0 (Информационная пустота)

- **Условие**: обе части равны нулю
- **Физика**: |Ψ| = 0, информация = энергия = масса = 0
- **Проявление**: истинный вакуум без флуктуаций

## Почему разность логарифмов?

### 1. Информационная аддитивность

Информация в независимых системах складывается:

```text
I_total = I₁ + I₂  →  Δ_total = Δ₁ + Δ₂
```

### 2. Масштабная инвариантность

Δ не меняется при однородном масштабировании Ψ:

```text
Ψ → kΨ  ⇒  Δ' = ln(|kRe|) - ln(|kIm|) = Δ
```

### 3. Избежание вычислительных сингулярностей

Форма ln(|Re|) - ln(|Im|) корректно обрабатывает:

- Im = 0 → Δ = +∞ (сингулярность)
- Re = 0 → Δ = -∞ (тёмная энергия)
- Re = Im = 0 → Δ = NaN (истинная неопределённость)

### 4. Естественная мера информации

Логарифмическая шкала соответствует битам информации — фундаментальной единице информации по Шеннону.

## Математические свойства

### Связь с модулем и фазой

Представляя Ψ в полярной форме:

```text
Ψ = |Ψ| · e^(iθ) = |Ψ| · (cosθ + i·sinθ)
```

Тогда:

```text
Δ = ln(|cosθ|) - ln(|sinθ|) = ln(|cotθ|)
```

Это показывает, что **Δ зависит только от фазы θ**, а не от амплитуды |Ψ|.

### Дифференциал

Для малых колебаний:

```text
dΔ = dRe/Re - dIm/Im
```

Чувствительность к относительным изменениям компонент.

## Выражение Re и Im через M и Δ

Из системы:

```text
M = Re² + Im²
Δ = ln(|Re|) - ln(|Im|)
```

Следует:

```text
Re = √M · e^(Δ/2) / √(e^Δ + e^(-Δ))
Im = √M · e^(-Δ/2) / √(e^Δ + e^(-Δ))
```

Или через гиперболические функции:

```text
Re = √M · cosh(Δ/2) · ε_Re
Im = √M · sinh(Δ/2) · ε_Im
```

где ε_Re, ε_Im = ±1 — знаки в зависимости от квадранта.

## Проверка согласованности

### С теоремой T1 (I ≡ M ≡ E)

Из T1: M = |Ψ|² = Re² + Im²
Из T2: Δ = ln(|Re|) - ln(|Im|)

Подстановка показывает согласованность при учёте фазового множителя.

### С теоремой T3 (Синхронизация)

Синхронизация 𝒮 зависит от σ(Δ) — стандартного отклонения Δ по сети, что естественно следует из определения.

## Вычислительные аспекты

### Стабильный алгоритм

```text
function compute_delta(Re, Im):
    absRe = abs(Re)
    absIm = abs(Im)

    if absIm == 0:
        if absRe == 0:
            return NaN  # истинная неопределённость 0/0
        else:
            return +inf  # сингулярность

    if absRe == 0:
        return -inf  # тёмная энергия

    return log(absRe) - log(absIm)  # избегаем деления
```

### Совместимость с IEEE 754

Стандарт плавающей точки корректно обрабатывает:

- log(0) = -∞
- log(+∞) = +∞
- конечное - ∞ = -∞
- ∞ - конечное = +∞
