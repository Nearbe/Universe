# Заряд и калибровочные симметрии

## Электрический заряд

### Калибровочная симметрия U(1)

Электрический заряд возникает как калибровочная симметрия U(1) сети, связанная с фазовым вращением волновой функции.

### Глобальное U(1) преобразование

```text
Ψ → e^(iα) Ψ
```

Соответствует добавлению постоянной к фазе θ:

```text
θ → θ + α
```

Что оставляет параметр Δ неизменным:

```text
Δ' = ln(|Re|) - ln(|Im|) = Δ
```

### Локальное U(1) преобразование

При α = α(x) требуется введение калибровочного поля A_μ:

```text
D_μ = ∂_μ + ieA_μ
```

где e — электрический заряд.

### Сохранение заряда

Из теоремы Нётер следует закон сохранения электрического заряда:

```text
∂_μ j^μ = 0
```

где j^μ — ток вероятности.

## Связь с параметром Δ

В терминах IME, заряд q связан с асимметрией волновой функции:

```text
q ∝ √(|Re|² - |Im|²) = |Ψ| · cos(2θ)
```

или более точно через параметр Δ:

```text
q ∝ tanh(Δ/2)
```

### Физический смысл

- **Δ > 0**: положительный заряд (избыток Re)
- **Δ < 0**: отрицательный заряд (избыток Im)
- **Δ = 0**: нейтрально (баланс)

## Цветовой заряд

### Калибровочная симметрия SU(3)

Цветовой заряд возникает из SU(3) калибровочной симметрии сильного взаимодействия.

### Три цвета

Кварки несут цветовой заряд трёх типов:

- **r** (red, красный)
- **g** (green, зелёный)
- **b** (blue, синий)

### В терминах IME

Это соответствует более сложной внутренней структуре ходонов, где сетевые моды имеют три независимые степени свободы, преобразующиеся как тройка под SU(3).

### Генераторы SU(3)

8 генераторов T^a (a = 1...8) соответствуют 8 глюонам.

### Конфайнмент

Возникает из неабелевской природы SU(3), приводящей к линейному росту энергии с расстоянием для цветовых зарядов.

В терминах IME конфайнмент объясняется тем, что цветовые моды локализованы в пределах адронов.

## Слабый изоспин

### Калибровочная симметрия SU(2)

Слабый изоспин возникает из SU(2) калибровочной симметрии слабого взаимодействия.

### Дублеты

Левые фермионы организованы в дублеты:

```text
(ν_e, e⁻)_L  (лептонный дублет)
```

```text
(u, d)_L  (кварковый дублет)
```

### В терминах IME

Это связано с двойной структурой определённых мод сети, которые преобразуются как двойка под SU(2).

### Разбитая симметрия

Разбитая симметрия слабого взаимодействия возникает из вакуумного ожидания поля Хиггса, которое в терминах IME связано со средним значением параметра Δ или его дисперсией в сети.

## Гиперзаряд

### Определение

Слабый гиперзаряд Y связан с электрическим зарядом Q и третьим компонентом слабого изоспина T₃:

```text
Q = T₃ + Y/2
```

### Калибровочная группа

Полная электрослабая симметрия:

```text
SU(2)_L × U(1)_Y
```

## Спонтанное нарушение симметрии

### Механизм Хиггса

Спонтанное нарушение симметрии через поле Хиггса:

```text
Φ = (0, v/√2)
```

где v ≈ 246 ГэВ — вакуумное ожидание.

### Массы калибровочных бозонов

Из кинетического члена поля Хиггса:

```text
m_W = gv/2 ≈ 80.4 ГэВ
```

```text
m_Z = √(g² + g'²)·v/2 ≈ 91.2 ГэВ
```

```text
m_γ = 0
```

### В терминах IME

Вакуумное ожидание соответствует среднему значению параметра Δ:

```text
⟨Δ⟩ ≠ 0
```

## Аномалии

### Треугольные аномалии

Квантовые эффекты могут нарушать классические симметрии.

### Условие непротиворечивости

Сумма зарядов фермионов в поколении должна обнуляться:

```text
Σ Q = 0
```

Для одного поколения:

```text
Q(ν_e) + Q(e⁻) + 3Q(u) + 3Q(d) = 0 + (-1) + 3(2/3) + 3(-1/3) = 0
```

### В терминах IME

Аномалии соответствуют топологическим дефектам в конфигурациях Δ сети.
