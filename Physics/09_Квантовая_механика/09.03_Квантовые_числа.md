# Квантовые числа и симметрии

## Главное квантовое число n

### Определение

Главное квантовое число связано с числом узлов волновой функции и, следовательно, с числом изменений знака в параметре Δ вдоль определённого направления.

### Для водородоподобного атома

Энергетические уровни:

```text
E_n = -(m_e e⁴)/(8ε₀²h²) · 1/n²
```

где n = 1, 2, 3, ...

### В терминах IME

Это соответствует уровням энергии, определяющимся глубиной потенциальной ямы, созданной структурой сети с определённой симметрией.

### Связь с Δ

Число узлов волновой функции соответствует числу раз, когда Δ проходит через ноль:

```text
n = число нулей Δ + 1
```

## Орбитальное квантовое число l

### Определение

Орбитальное квантовое число связано с моментом импульса:

```text
L² = ħ² l(l+1)
```

где l = 0, 1, 2, ..., n-1.

### Происхождение из симметрии

Возникает из сферической симметрии потенциала.

### В терминах IME

Сферическая симметрия соответствует изотропному распределению параметра Δ вокруг центра, что приводит к сохранению момента импульса.

### Обозначения

| l | Обозначение |
|---|-------------|
| 0 | s |
| 1 | p |
| 2 | d |
| 3 | f |

## Магнитное квантовое число m_l

### Определение

Магнитное квантовое число связано с проекцией момента импульса на выбранную ось:

```text
L_z = ħ m_l
```

где m_l = -l, -l+1, ..., l-1, l.

### Происхождение

Возникает из разбитой аксиальной симметрии (например, во внешнем магнитном поле).

### В терминах IME

Соответствует предпочтительному направлению градиента Δ или направлению спиновой поляризации сети.

## Спиновое квантовое число s

### Определение

Спин — внутренний момент импульса частицы:

```text
S² = ħ² s(s+1)
```

### Для электрона

```text
s = 1/2
```

### Проекция спина

```text
S_z = ħ m_s
```

где m_s = ±1/2 для электрона.

### Происхождение в IME

Спин возникает из внутренней структуры волновой функции и связан с поведением при вращении на 360°.

В терминах IME спин может быть связан с **топологическими свойствами вихревых конфигураций** параметра Δ в сети, где поворот на 360° приводит к изменению знака волновой функции для полуцелых спинов:

```text
Ψ(θ + 2π) = -Ψ(θ)  (для фермионов)
```

```text
Ψ(θ + 4π) = Ψ(θ)  (полное восстановление)
```

### Визуализация спина ½

| Вращение | Волновая функция | Состояние |
|----------|-----------------|-----------|
| 0° | Ψ | Исходное |
| 360° (2π) | -Ψ | Знак перевёрнут |
| 720° (4π) | Ψ | Снова исходное |

**Физический смысл**: требуется два полных оборота (720°) для восстановления исходного состояния — отличительный признак спина ½.

## Полное угловое число j

### Сложение моментов

Для частицы с орбитальным и спиновым моментами:

```text
j = l + s
```

Возможные значения:

```text
j = |l - s|, |l - s| + 1, ..., l + s
```

### Для электрона в атоме

При l > 0:

```text
j = l ± 1/2
```

## Квантовые числа в таблице Менделеева

### Электронная конфигурация

Каждый электрон в атоме описывается набором:

```text
(n, l, m_l, m_s)
```

### Принцип запрета Паули

Два электрона не могут иметь одинаковый набор всех четырёх квантовых чисел.

### В терминах IME

Это следует из антисимметричности волновой функции фермионов при обмене частиц, что связано с топологическими свойствами конфигураций Δ.

## Симметрии и законы сохранения

### Теорема Нётер

Каждой непрерывной симметрии соответствует закон сохранения.

| Симметрия | Сохранение |
|-----------|------------|
| Сдвиг во времени | Энергия |
| Сдвиг в пространстве | Импульс |
| Вращение | Момент импульса |
| U(1) калибровочная | Электрический заряд |
| SU(3) цветовая | Цветовой заряд |

### В терминах IME

Симметрии соответствуют инвариантности конфигураций ходонной сети относительно определённых преобразований.

## Нарушенные симметрии

### Спонтанное нарушение симметрии

Симметрия лагранжиана не является симметрией основного состояния.

Пример: поле Хиггса.

### Явное нарушение симметрии

Симметрия нарушена в лагранжиане.

Пример: слабое взаимодействие нарушает P и CP.

### В терминах IME

Нарушенные симметрии соответствуют асимметричным конфигурациям Δ в сети.
