# Волновая функция как фундаментальное описание

## Волновая функция сети ходонов

В теории IME волновая функция описывает состояние **всей ходонной сети**, а не отдельных частиц. Однако через принцип голографии и эквивалентность I ≡ M ≡ E можно восстановить описание отдельных частиц как emergentных возбуждений сети.

## Волновая функция отдельной частицы

Для отдельной частицы волновая функция Ψ(x,t) описывает вероятностное амплитудное поле, связанное с локальным состоянием ходонной сети в окрестности частицы.

### Из принципа эквивалентности

```text
|Ψ(x,t)|² = ρ(x,t)
```

где ρ(x,t) — информационная плотность в точке x в момент t.

### Физический смысл

|Ψ(x,t)|² даёт вероятность обнаружения частицы в точке x в момент t (правило Борна).

## Связь с полем Δ

Волновая функция связана с полем параметра Δ через модуль и фазу:

```text
Ψ(x,t) = |Ψ(x,t)| · e^(iθ(x,t))
```

где:

- **|Ψ(x,t)|²** ∝ экспоненциальная функция локального Δ
- **θ(x,t)** связана с градиентом Δ через соотношение, аналогичное уравнению непрерывности

## Madelung преобразование

Более точное представление через преобразование Маделунга:

```text
Ψ = √ρ · e^(iS/ħ)
```

где:

- **ρ** — плотность вероятности
- **S** — действие (фаза)

### Связь с Δ

Можно показать, что:

```text
ρ ∝ exp(Δ/2)
```

```text
∇S ∝ ∇Δ
```

что приводит к гамильтоновой формулировке, где Δ играет роль потенциала.

## Уравнение непрерывности

Из принципа сохранения информации (следствие I ≡ M ≡ E в замкнутой системе) следует уравнение непрерывности:

```text
∂ρ/∂t + ∇·j = 0
```

где:

- **ρ** — информационная плотность
- **j** — поток информации

### Поток информации

Для волновой функции в полярной форме:

```text
j = (ħ/m) · ρ · ∇θ
```

где m — эффективная масса, связанная с сетевой инерцией.

## Интерпретация

### Копенгагенская интерпретация в IME

Волновая функция — полное описание квантового состояния. Коллапс происходит при декогеренции через синхронизацию с измерительным прибором.

### Многомировая интерпретация в IME

Различные ветви волновой функции соответствуют различным конфигурациям ходонной сети.

### Теория пилот-волны в IME

Ψ направляет частицы через градиент фазы S, который связан с Δ.

## Нормировка

Волновая функция нормируется на единицу:

```text
∫ |Ψ(x,t)|² d³x = 1
```

Это соответствует сохранению полной информации в системе.

## Суперпозиция состояний

Принцип суперпозиции:

```text
Ψ = c₁Ψ₁ + c₂Ψ₂ + ...
```

где |c_i|² — вероятность нахождения в состоянии i.

### В терминах IME

Суперпозиция соответствует одновременному существованию нескольких конфигураций ходонной сети.

## Запутанность

Для двухчастичной системы:

```text
Ψ(x₁,x₂) ≠ ψ₁(x₁) · ψ₂(x₂)
```

Запутанные состояния не могут быть факторизованы.

### В терминах IME

Запутанность соответствует коррелированным конфигурациям ходонной сети на больших расстояниях.

## Измерение

При измерении волновая функция «коллапсирует» в собственное состояние измеряемого оператора:

```text
Ψ → ψ_n  (с вероятностью |⟨ψ_n|Ψ⟩|²)
```

### В терминах IME

Коллапс — это быстрая синхронизация сети ходонов измерительного прибора с измеряемой системой (Раздел 04.06).

## Мембрана как граница пространств

### Определение

**Мембрана (Ψ)** — это граница между двумя геометрическими пространствами:

```text
Мембрана (Ψ) = Re + i·Im  (комплексное число)
```

| Пространство | Параметр | Доля во Вселенной |
|--------------|----------|-------------------|
| **Материальное** | Δ → +∞ (Im → 0) | ~5% |
| **Нематериальное** | Δ → -∞ (Re → 0) | ~95% |

### Двойственная природа мембраны

Мембрана существует одновременно в двух состояниях:

1. **Реальное** — разделяет пространства
2. **Иллюзорное** — сама является фазовым переходом, не "вещью"

### Физический смысл

Мембрана — это **не объект**, а **граница раздела** между:

- Видимой информацией (Re, 5%)
- Скрытой информацией (Im, 95%)
