# Массы частиц и механизм Хиггса

## Массы без взаимодействия

Частицы, не взаимодействующие с полем Хиггса, остаются безмассовыми и движутся со скоростью света.

### В терминах IME

Это соответствует модам сети, которые не связаны с модуляцией плотности или средним значением параметра Δ.

### Примеры

- **Фотон**: m = 0
- **Глюон**: m = 0
- **Гравитон** (если существует): m = 0

## Взаимодействие с полем Хиггса

### Юкава взаимодействие

Через взаимодействие Юкава с полем Хиггса фермионы приобретают массу:

```text
L_Yukawa = -y_f · Ψ̄ Φ Ψ
```

где y_f — константа связи Юкава.

### Масса фермиона

После нарушения симметрии:

```text
m_f = y_f · v / √2
```

где v ≈ 246 ГэВ — вакуумное ожидание поля Хиггса.

### В терминах IME

Масса фермиона пропорциональна силе его взаимодействия с модуляцией параметра Δ в сети.

## Массы калибровочных бозонов

### Из кинетического члена Хиггса

Для бозонов калибровочной симметрии масса возникает из кинетического члена калибровочного поля:

```text
m_V² = (g² v²)/2
```

где g — калибровочная постоянная.

### В терминах IME

Это соответствует энергии, необходимой для возбуждения сетевой моды, связанной с калибровочной симметрией, которая зависит от вакуумного ожидания модуляции Δ.

## Вакуумное ожидание поля Хиггса

### Потенциал Хиггса

```text
V(Φ) = -μ² Φ†Φ + λ (Φ†Φ)²
```

### Минимум потенциала

```text
⟨Φ⟩ = v/√2 = μ/√λ
```

где v ≈ 246 ГэВ.

### В терминах IME

Это соответствует значению параметра Δ, при котором энергия сети минимальна при данной структуре взаимодействий.

## Масса электрона и протона

### Масса электрона

В терминах IME масса электрона возникает из сильной отрицательной связывающей энергии между его составляющими ходонами.

Если бы каждый ходон вносил вклад массы Планка, наблюдаемая масса была бы ~10²² раз больше.

Наблюдаемая масса:

```text
m_e ≈ 4.2 × 10⁻²³ m_P  (в единицах Планка)
```

возникает из почти полной компенсации положительной массы ходонов отрицательной связывающей энергией.

### Формула

```text
m_e = m_P(N - |E_binding|/m_P)
```

где |E_binding| ≈ N·m_P - m_e ≈ 10²²·m_P.

### Масса протона

Протон — легчайшая барионная ходонная конфигурация.

Его масса определяется как наименьшая возможная энергия для барионного числа = 1 (единица топологического заряда ходонов).

Любое более лёгкое состояние с барионным числом = 1 потребовало бы нарушения I = M = E или создания дополнительной информации.

Наблюдаемая масса:

```text
m_p ≈ 0.938 ГэВ/c²
```

Почему константы Юкава варьируются на 12 порядков?

### Возможное решение в IME

Различные фермионы соответствуют различным конфигурациям ходонов с разными значениями Δ.

## Механизм генерации масс

### Поколения фермионов

Три поколения фермионов:

| Поколение | Лептоны | Кварки |
|-----------|---------|--------|
| 1 | ν_e, e | u, d |
| 2 | ν_μ, μ | c, s |
| 3 | ν_τ, τ | t, b |

### В терминах IME

Поколения соответствуют различным устойчивым конфигурациям ходонной сети.

## Бозон Хиггса

### Открытие

Обнаружен на LHC в 2012 году:

```text
m_H = 125.10 ± 0.14 ГэВ/c²
```

### Свойства

| Свойство | Значение |
|----------|----------|
| Спин | 0 |
| Чётность | + |
| Заряд | 0 |
| Ширина распада | 4.1 МэВ |

### Основные каналы распада

| Канал | Branching ratio |
|-------|-----------------|
| H → bb̄ | 58% |
| H → WW* | 21% |
| H → gg | 9% |
| H → ττ | 6% |
| H → ZZ* | 3% |
| H → γγ | 0.2% |

## Самовзаимодействие Хиггса

### Трёххиггсовская вершина

```text
λ_HHH = 3m_H²/v
```

### Четырёххиггсовская вершина

```text
λ_HHHH = 3m_H²/v²
```

### В терминах IME

Соответствует нелинейности потенциала V(Δ).
