# Квантовая теория поля из сети ходонов

## От квантовой механики к КТП

Квантовая механика описывает частицы с фиксированным числом. Квантовая теория поля (КТП) описывает рождение и уничтожение частиц.

## Вторичное квантование

### Операторы рождения и уничтожения

В КТП волновая функция становится оператором поля:

```text
Ψ(x) → Ψ̂(x)
```

Операторы:

- **â†_p**: рождение частицы с импульсом p
- **â_p**: уничтожение частицы с импульсом p

### Коммутационные соотношения

Для бозонов:

```text
[â_p, â†_q] = δ(p - q)
```

Для фермионов:

```text
{â_p, â†_q} = δ(p - q)
```

### В терминах IME

Операторы рождения/уничтожения соответствуют добавлению/удалению возбуждений ходонной сети.

## Лагранжиан свободного поля

### Скалярное поле (Клейна-Гордона)

```text
L = ½(∂_μ φ)(∂^μ φ) - ½m²φ²
```

Уравнение движения:

```text
(□ + m²)φ = 0
```

### В терминах IME

Скалярное поле соответствует простейшей моде колебаний параметра Δ в сети.

## Лагранжиан взаимодействующего поля

### КЭД (квантовая электродинамика)

```text
L_QED = Ψ̄(iγ^μ D_μ - m)Ψ - ¼F_μν F^μν
```

где:

- **D_μ = ∂_μ + ieA_μ** — ковариантная производная
- **F_μν = ∂_μ A_ν - ∂_ν A_μ** — тензор электромагнитного поля

### В терминах IME

Взаимодействие возникает из обмена сетевыми модами между возбуждениями.

## Диаграммы Фейнмана

### Правила

Каждая диаграмма соответствует амплитуде процесса:

| Элемент | Фактор |
|---------|--------|
| Внешняя линия (вход) | u(p), v(p) |
| Внешняя линия (выход) | ū(p), v̄(p) |
| Внутренняя линия | i/(p² - m² + iε) |
| Вершина | -ieγ^μ |
| Петля | ∫ d⁴k/(2π)⁴ |

### Пример: рассеяние Комптона

```text
e⁻ + γ → e⁻ + γ
```

Две диаграммы (s-канал и u-канал).

### В терминах IME

Диаграммы Фейнмана описывают различные способы обмена сетевыми модами.

## Перенормировка

### Проблема расходимостей

Петлевые интегралы расходятся при больших импульсах.

### Регуляризация

Введение обрезания Λ:

```text
∫ d⁴k → ∫^Λ d⁴k
```

### Перенормировка

Поглощение расходимостей в переопределение параметров:

```text
m_0 → m_physical
e_0 → e_physical
```

### В терминах IME

Обрезание естественно возникает из дискретной структуры сети:

```text
Λ ~ 1/a_network
```

где a_network — шаг сети ходонов.

## Бегущие константы связи

### Уравнение ренормгруппы

Константы связи зависят от энергетического масштаба:

```text
μ ∂g/∂μ = β(g)
```

### Для КЭД

```text
β(e) = e³/(12π²)
```

α растёт с энергией:

```text
α(Q²) = α(μ²) / [1 - (α/3π) ln(Q²/μ²)]
```

### Для КХД

```text
β(g_s) = -(11 - 2n_f/3) g_s³/(16π²)
```

α_s убывает с энергией (асимптотическая свобода).

### В терминах IME

Бегущие константы отражают изменение эффективной плотности сетевых мод с масштабом зондирования.

## Спонтанное нарушение симметрии

### Механизм Хиггса

Потенциал:

```text
V(Φ) = -μ² Φ†Φ + λ(Φ†Φ)²
```

Минимум при:

```text
⟨Φ⟩ = v/√2 = μ/√λ
```

### В терминах IME

Соответствует ненулевому среднему значению параметра Δ в вакууме сети.

## Аномалии

### Треугольная аномалия

Классическая симметрия нарушается квантовыми эффектами.

### Условие согласованности

Сумма зарядов должна обнуляться:

```text
Σ Q = 0
```

### В терминах IME

Аномалии соответствуют топологическим дефектам в конфигурациях Δ.

## Эффективная теория поля

### Идея

При низких энергиях важны только лёгкие степени свободы.

### Лагранжиан

```text
L_eff = L_renorm + Σ_i (c_i/Λ^n_i) O_i
```

где O_i — операторы высшей размерности.

### В терминах IME

Эффективная теория описывает длинноволновые моды сети, коротковолновые усреднены.

## Частицы как ходонные конфигурации

### Основная концепция

В информационной парадигме **каждая элементарная частица — это локализованная, стабильная конфигурация ходонов** — паттерны амплитуды и фазы в информационной сети, а не точечный объект.

### Сравнение: Стандартная модель vs IME

| Аспект | Стандартная модель | Информационная парадигма (IME) |
|--------|-------------------|-------------------------------|
| **Онтология** | Точечные частицы (0-D) | Расширенные ходонные конфигурации |
| **Происхождение массы** | Юкавские связи Хиггса (свободные параметры) | m = m_P(N - \|E_binding\|/m_P) из баланса энергии ходонов |
| **Происхождение заряда** | Абстрактный U(1) заряд | Топологическая намотка arg(Ψ) (фазовый вихрь) |
| **Происхождение спина** | Постулируемое представление группы Лоренца | Свойство вращения на 4π (фермионы) vs 2π (бозоны) |
| **Взаимодействия** | Кванты полей (фотоны, глюоны, W/Z) | Обмен коллективными колебаниями сети (фаза, цвет, слабые моды) |
| **Нарушение симметрии** | ВЭВ Хиггса | Фоновая ходонная конфигурация с определённой плотностью энергии |

### Ключевая идея

Это единое представление устраняет необходимость в различных механических объяснениях. Все свойства частиц возникают из структуры ходонной сети:

- **Частицы НЕ точки** — они расширенные паттерны
- **Масса происходит от энергии связи** (не от Хиггса)
- **Заряд = число намотки фазы** (не абстрактный)
- **Спин = топологическое свойство вращения** (не постулируемое)
- **Взаимодействия = обмен модами** (не поля)
