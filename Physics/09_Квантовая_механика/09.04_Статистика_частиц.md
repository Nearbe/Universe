# Статистика частиц

## Бозе-Эйнштейновская статистика

Частицы с целым спином (бозоны) следуют Бозе-Эйнштейновской статистике, позволяющей множественной занимаемости квантовых состояний.

### Функция распределения

Среднее число бозонов в состоянии с энергией E:

```text
f_BE(E) = 1 / [e^((E-μ)/(kT)) - 1]
```

где:

- **μ** — химический потенциал
- **k** — постоянная Больцмана
- **T** — температура

### Свойства

- **Неограниченная занимаемость**: в одном состоянии может быть любое число бозонов
- **Бозе-Эйнштейновская конденсация**: при T → 0 бозоны скапливаются в основном состоянии

### В терминах IME

Это соответствует ситуации, когда сетевые моды, соответствующие целому спину, могут когерентно суммироваться без ограничения по принципу Паули.

### Примеры бозонов

| Частица | Спин | Пример |
|---------|------|--------|
| Фотон | 1 | Свет |
| Глюон | 1 | Сильное взаимодействие |
| W±, Z⁰ | 1 | Слабое взаимодействие |
| Бозон Хиггса | 0 | Масса частиц |
| Мезоны (π, K) | 0 | Ядерные силы |

## Ферми-Дираковская статистика

Частицы с полуцелым спином (фермионы) следуют Ферми-Дираковской статистике с принципом Паули, запрещающим множественной занимаемости одинакового квантового состояния.

### Функция распределения

Среднее число фермионов в состоянии с энергией E:

```text
f_FD(E) = 1 / [e^((E-μ)/(kT)) + 1]
```

### Свойства

- **Принцип Паули**: не более одного фермиона в состоянии
- **Энергия Ферми**: при T = 0 все состояния ниже E_F заняты, выше — пусты
- **Вырожденный газ**: при T ≪ T_F квантовые эффекты доминируют

### В терминах IME

Это возникает из антисимметричности общей волновой функции при обмене фермионами, что связано с топологическими свойствами конфигураций Δ при обмене частиц.

### Принцип Паули из I ≡ M ≡ E

**Фундаментальное обоснование принципа Паули**:

Две идентичные электронные конфигурации не могут занимать тот же набор ходонов, потому что это потребовало бы **дублирования того же информационного паттерна**, нарушая тождество сохранения информации:

```text
I ≡ M ≡ E
```

**Механизм**:

1. Каждый фермион — уникальная конфигурация ходонов с определённым информационным паттерном
2. Дублирование паттерна означало бы I_total = 2I, что нарушает сохранение информации
3. Следовательно, **принцип Паули естественно возникает** из информационно-теоретических ограничений

Это обеспечивает глубокую связь между:

- **Сохранением информации** (фундаментальный принцип)
- **Принципом Паули** (квантовое ограничение)
- **Статистикой Ферми-Дирака** (макроскопическое проявление)

### Примеры фермионов

| Частица | Спин | Пример |
|---------|------|--------|
| Электрон | 1/2 | Атомы, ток |
| Протон | 1/2 | Ядра |
| Нейтрон | 1/2 | Ядра |
| Нейтрино | 1/2 | Слабое взаимодействие |
| Кварки | 1/2 | Адроны |

## Связь со спином

### Теорема о связи спина и статистики

- **Целый спин** → Бозе-Эйнштейновская статистика
- **Полуцелый спин** → Ферми-Дираковская статистика

### Доказательство в КТП

Следует из требований причинности и положительности энергии.

### В терминах IME

Связь возникает из топологических свойств конфигураций Δ:

- Целый спин: Ψ(θ + 2π) = Ψ(θ)
- Полуцелый спин: Ψ(θ + 2π) = -Ψ(θ)

## Статистика и параметр Δ

Для системы из двух частиц симметризация или антисимметризация волновой функции приводит к разным энергетическим уровням в зависимости от относительной ориентации их параметров Δ.

### Для бозонов

Симметричная волновая функция предпочитает состояния, где:

```text
Δ₁ ≈ Δ₂
```

(параллельная ориентация информационного баланса)

### Для фермионов

Антисимметричная волновая функция предпочитает состояния, где:

```text
Δ₁ ≈ -Δ₂
```

(антипараллельная ориентация)

### Физическое следствие

Это объясняет **фермионное отталкивание** и **бозонное притяжение** на коротких расстояниях через модуляцию эффективного взаимодействия через параметр Δ.

## Бозе-Эйнштейновская конденсация

### Условие

При температуре ниже критической:

```text
T_c = (2πħ²/mk) · [n/ζ(3/2)]^(2/3)
```

где n — плотность частиц, ζ — дзета-функция Римана.

### Свойства конденсата

- Макроскопическая занимаемость основного состояния
- Когерентность на больших расстояниях
- Сверхтекучесть, сверхпроводимость

### В терминах IME

Конденсация соответствует высокой синхронизации сети ходонов:

```text
𝒮 → 1
σ(Δ) → 0
```

## Вырожденный ферми-газ

### Энергия Ферми

При T = 0:

```text
E_F = (ħ²/(2m)) · (3π²n)^(2/3)
```

### Давление вырожденного газа

```text
P = (2/5) · n · E_F
```

### Применения

- **Электроны в металлах**: проводимость
- **Белые карлики**: поддержка от гравитационного коллапса
- **Нейтронные звёзды**: поддержка нейтронным вырождением

## Статистика в Большом каноническом ансамбле

### Статистическая сумма

Для бозонов:

```text
ln(Z) = -Σ_k ln[1 - e^(-(E_k-μ)/(kT))]
```

Для фермионов:

```text
ln(Z) = Σ_k ln[1 + e^(-(E_k-μ)/(kT))]
```

### Термодинамические величины

Из Z можно вычислить:

- Среднее число частиц: ⟨N⟩ = kT ∂ln(Z)/∂μ
- Энергия: ⟨E⟩ = -∂ln(Z)/∂β
- Энтропия: S = k(ln(Z) + β⟨E⟩ - βμ⟨N⟩)
