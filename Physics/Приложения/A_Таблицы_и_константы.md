# Таблицы и отношения

## Фундаментальные отношения

В безразмерной физике IME нет констант — только отношения:

| Отношение | Символ | Значение | Описание |
|-----------|--------|----------|----------|
| **Базовая частота** | f₀ | ~10⁴³ Гц | Фундаментальная вибрация пространства |
| **Базовая энергия** | E₀ | mc² | Энергия покоя ходона |
| **Базовая длина** | ℓ₀ | ~10⁻³⁵ м | Минимальный масштаб |
| **Базовый период** | T₀ | 1/f₀ | Период осцилляции ходона |

**Важно**: Эти значения — не константы природы, а фундаментальные отношения, возникающие из структуры ходонной сети.

## Параметры IME

| Параметр | Обозначение | Значение/Диапазон | Вывод |
|----------|-------------|-------------------|-------|
| Параметр отклонения | Δ | ℝ ∪ {-∞, +∞} | Δ = ln\|Re/Im\| |
| Синхронизация | 𝒮 | [0, 1] | 𝒮 = 1 - σ(Δ)·scale |
| Порог сознания (синхронизация) | 𝒮_c | 0.7 | Фазовый переход |
| Порог сознания (Δ) | Δ_c | 0.2 | Близость к мембране |
| Коэффициент scale | scale | 0.05 | scale = 1/(4×\|b\|), \|b\|≈0.2 |
| Фрактальная размерность | D_f | [0, ∞) | D_f = e^ |
| Энергия вибрации | Q | [0, ∞) | Q = e^ |
| Видимая доля | visibleFraction | e^(-Q) | При Q≈3: ~5% |
