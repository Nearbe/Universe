## Глоссарий ключевых операторов

| Символ | Определение | Физический смысл |
|--------|-------------|------------------|
| $\Omega$ (или $\varnothing$) | Пустое множество, фиксированная точка | Состояние потенциала, Ker $D$, макро-возврат. |
| $D(a) = a : \Omega = a \oplus a$ | Оператор ветвления (Branching) | Бифуркация, микро-удвоение, экспоненциальный рост. |
| $H(a) = \operatorname{ret}(a)$ | Оператор сжатия (Compression) | Уникальный предок ($\exists!$), детерминированный возврат $O(1)$. |
| $\operatorname{rank}_U(x) \le 9$ | Цикл Полноты | Десятичный фрактал, ограничивающий глубину рекурсии. |
| $\Pi = \lim_{n \to \infty} D^n(Id)$ | Полнота (Completeness) | Предельное состояние стабильности операторов. |
| $\lambda = D(Id)$ | Собственное значение масштаба | Фундаментальный инвариант всех U-преобразований. |
| $\epsilon^2 = \Omega$ | Дуальное расширение | Автоматическое дифференцирование и динамика. |
