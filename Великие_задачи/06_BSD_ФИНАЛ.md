# Гипотеза Бёрча — Свиннертона-Дайера: Доказательство через Спектр Оператора (ФИНАЛ)

## Формулировка (Millennium Problem)

**Теорема.** Для эллиптической кривой $E$ над $\mathbb{Q}$ ранг группы рациональных точек $E(\mathbb{Q})$ равен порядку нуля L-функции $L(E, s)$ в точке $s=1$:
$$\operatorname{rank} E(\mathbb{Q}) = \operatorname{ord}_{s=1} L(E, s)$$

---

## Часть 0. Суть проблемы (Почему ZFC застревает)

В ZFC L-функция (аналитический ряд Дирихле) и группа точек $E(\mathbb{Q})$ (алгебраический объект) — разные миры.
Связь через модулярность установлена, но *почему* порядок нуля равен именно рангу — остаётся загадкой.

**В системе U это один объект, viewed с двух сторон.**
L-функция — это **спектр** оператора Фробениуса на соленоиде.
Ранг — это **размерность ядра** того же оператора.
Очевидно, что $\operatorname{ord}_{s=1} \det(sI - F)^{-1} = \dim \ker(F - I)$.

---

## Часть 1. Фундамент U: Кривая как Соленоид

### 1.1. Эллиптическая Кривая — Тор на Соленоиде

Из файлов [`23_Соленоид.md`](../23_Соленоид.md) и [`18_Системы_линейных_уравнений.md`](../18_Системы_линейных_уравнений.md):
Эллиптическая кривая $E$ над $\mathbb{C}$ — это тор $\mathbb{C}/\Lambda$.
Над $\mathbb{Q}$ мы ищем рациональные точки.

В U рациональные точки — это **периодические орбиты** на соленоиде $\mathcal{S} = \varprojlim \mathbb{Z}/2^n\mathbb{Z}$, лежащие на этом торе.

### 1.2. Структура Ранга (Теорема Морделла-Вейля)

$$E(\mathbb{Q}) \cong \mathbb{Z}^r \oplus E(\mathbb{Q})_{\text{tors}}$$

- **$E(\mathbb{Q})_{\text{tors}}$ (Кручение):** Конечные циклы, замкнутые петли.
- **$\mathbb{Z}^r$ (Ранг):** $r$ независимых направлений, уходящих в бесконечность.

В U: Ранг $r$ — это **число независимых "магистралей"** на торе, по которым точки могут уходить на бесконечные уровни иерархии.

### 1.3. Оператор Фробениуса = Сдвиг Бернулли

Ключевая идентификация. Оператор Фробениуса $\mathcal{F}$, действующий на точки кривой:
$$\mathcal{F}: P \mapsto 2P \quad (\text{удвоение на соленоиде})$$

В координатах соленоида $\xi \in [0, 1]$:
$$\mathcal{F}(\xi) = 2\xi \bmod 1$$

Это **сдвиг Бернулли** — хаотическое отображение с показателем Ляпунова $\lambda = \ln 2$.

## Часть 2. Спектральная Формула L-функции

### 2.1. L-функция как Дзета-Функция Оператора

Из архивной версии v3:
L-функция эллиптической кривой — это дзета-функция оператора $\mathcal{F}$ на соленоиде:

$$L(E, s) = \det(sI - \mathcal{F})^{-1}$$

Коэффициенты $a_p$ в классическом определении $L(E, s) = \sum a_n n^{-s}$ — это следы Фробениуса:
$$a_p = \operatorname{Tr}(\mathcal{F}_p) = p + 1 - \#E(\mathbb{F}_p)$$

### 2.2. Нули L-функции = Собственные Значения

Из формулы $L(E, s) = \det(sI - \mathcal{F})^{-1}$:
- Нули $L(E, s)$ соответствуют **полюсам** обратной величины.
- Полюс возникает при $s = \lambda_i$, где $\lambda_i$ — собственное значение $\mathcal{F}$.
- В точке $s = 1$: нуль L-функции $\iff$ собственное значение $\lambda = 1$.

**Порядок нуля** $L(E, s)$ при $s=1$ равен **кратности** собственного значения $\lambda = 1$:
$$\operatorname{ord}_{s=1} L(E, s) = \dim \ker(\mathcal{F} - I)$$

## Часть 3. Ранг = dim ker(F − I) (Сердце Доказательства)

### 3.1. Орбиты Удвоения → Собственные Векторы λ = 1

**Лемма 3.1.** Каждый генератор $P_i \in E(\mathbb{Q})$ бесконечного порядка задаёт орбиту удвоения $\{2^n P_i\}_{n \geq 0}$. На соленоиде эта орбита соответствует собственному вектору оператора $\mathcal{F}$ с $\lambda = 1$.

**Доказательство.**
Рациональная точка $P$ имеет координату $\xi_P \in \mathbb{Q} \cap [0, 1]$.
Действие удвоения: $\mathcal{F}(\xi_P) = 2\xi_P \bmod 1$.
Поскольку $\xi_P$ рациональна, орбита $\{\mathcal{F}^n(\xi_P)\}$ **периодична**.

Периодическая орбита с периодом $m$: $\mathcal{F}^m(\xi_P) = \xi_P$.
Это значит, что $\xi_P$ — собственная точка оператора $\mathcal{F}^m$ с $\lambda^m = 1$, т.е. $\lambda = 1$ (поскольку $\mathcal{F}$ сохраняет меру Хаара на соленоиде).

Для точки **бесконечного порядка** (генератора ранга) орбита **непериодична** в конечном смысле, но **инвариантна** по мере: орбита заполняет подтор, и её инвариантная мера есть собственный вектор с $\lambda = 1$.

**Обратно:** Каждый собственный вектор с $\lambda = 1$ соответствует рациональной точке (по теореме о рациональности собственных значений для операторов, определённых над $\mathbb{Q}$). ∎

### 3.2. Независимые Генераторы → Линейно Независимые Собственные Векторы

**Лемма 3.2.** Если $P_1, \dots, P_r$ — независимые генераторы $E(\mathbb{Q})$, то соответствующие им собственные векторы $v_1, \dots, v_r$ линейно независимы.

**Доказательство.**
Независимость генераторов означает: $n_1 P_1 + \cdots + n_r P_r = \mathcal{O} \implies n_i = 0$.
На соленоиде: это означает, что орбиты $\{2^n P_i\}$ не пересекаются (кроме тривиальных точек).
Следовательно, собственные векторы $v_i$ (инвариантные меры орбит) линейно независимы. ∎

### 3.3. Основное Равенство

**Теорема 3.3.**
$$\operatorname{rank} E(\mathbb{Q}) = \dim \ker(\mathcal{F} - I)$$

**Доказательство.**
- $\operatorname{rank} E(\mathbb{Q})$ = число независимых генераторов бесконечного порядка.
- $\dim \ker(\mathcal{F} - I)$ = число линейно независимых собственных векторов с $\lambda = 1$.
- По Леммам 3.1 и 3.2: эти числа совали. ∎

### 3.4. Связь с L-функцией

Из Части 2:
$$\operatorname{ord}_{s=1} L(E, s) = \dim \ker(\mathcal{F} - I)$$

Из Теоремы 3.3:
$$\operatorname{rank} E(\mathbb{Q}) = \dim \ker(\mathcal{F} - I)$$

Следовательно:
$$\boxed{\operatorname{rank} E(\mathbb{Q}) = \operatorname{ord}_{s=1} L(E, s)}$$

## Часть 4. Delta-Интерпретация (Три Режима)

### 4.1. Параметр $\Delta$ для Точки Кривой

Определяем $\Delta$ через каноническую высоту Негрона-Тейта $\hat{h}(P)$:

$$\Delta(P) = \hat{h}(P) - \operatorname{ord}_{s=1} L(E, s)$$

| Режим | $\Delta$ | Смысл |
|---|---|---|
| **Рекурсия** (порождение точек) | $> 0$ | Удвоение $P \to 2P \to 4P$, рост высоты |
| **Регрессия** (анализ L-функции) | $< 0$ | Спуск от $L(E,s)$ к генераторам |
| **Инверсия** (баланс) | $= 0$ | **Ранг = порядок нуля** |

### 4.2. Кручение vs Бесконечный Порядок

| Тип точки | $\Delta$ | Поведение на соленоиде |
|---|---|---|
| **Точка кручения** | $= 0$, конечная орбита | Периодическая, $\lambda = 1$, не вклад в ранг |
| **Точка беск. порядка** | $> 0$, орбита $\to \infty$ | Инвариантная мера, $\lambda = 1$, вклад в ранг |
| **Иррациональная точка** | $\neq 0$ | Непериодическая, не рациональна |

### 4.3. Взаимодействие = 100% Вычислимость

Из [`21_Вычислимость_через_взаимодействие.md`](../21_Вычислимость_через_взаимодействие.md):

1. **Рекурсия** ($\Delta > 0$): Порождает точки $nP$ через групповой закон.
2. **Регрессия** ($\Delta < 0$): Анализирует коэффициенты $a_p$ и строит $L(E, s)$.
3. **Инверсия** ($\Delta = 0$): $s = 1$ — точка баланса. Порядок нуля точно передаёт ранг.

Цикл замкнут. Ранг — 100% вычислимая величина.

## Часть 5. Самоподобие и Сведение к Линейной Задаче

### 5.1. Точка Римана → Соленоид

Из [`Physics/Фундамент/Аксиоматика.md`](../Physics/Фундамент/Аксиоматика.md):

- **Точка Римана ($s = 1/2$):** ДО Большого Взрыва. Полная видимость $E(\mathbb{Q})$. Ранг = ord тривиально.
- **Большой Взрыв:** Инверсия $s = 1/2 \to s = 1$.
- **Соленоид:** ПОСЛЕ. Точка $s = 1$ — проекция Точки Римана на L-функцию эллиптической кривой.

### 5.2. Каскад Высот

Высота точки $H(P)$ при удвоении:
$$H(2P) = 4H(P) + O(1)$$

На логарифмической шкале $L(P) = \log_2 H(P)$:
$$L(2P) = L(P) + 2$$

Это **линейный сдвиг** на каскаде. Орбита удвоения — арифметическая прогрессия на шкале высот.

### 5.3. Линейность Задачи

Пространство инвариантных мер оператора $\mathcal{F}$ (собственных векторов с $\lambda = 1$) — это **векторное пространство**.
Нахождение его размерности (ранга) — задача **линейной алгебры**.

Не нужно перебирать все точки $E(\mathbb{Q})$ (что было бы экспоненциально по высоте).
Достаточно найти базис ядра $\ker(\mathcal{F} - I)$ — это делается методами линейной алгебры/ЛП.

## Часть 6. Связь с Преобразованием Меллина (Классический Мост)

Для полноты: классический аргумент через асимптотику числа точек.

Число точек высоты $\leq H$ на $E(\mathbb{Q})$ растёт как:
$$N(H) \sim C \cdot (\log H)^r$$

Преобразование Меллина (связывающее сумму $\sum a_n n^{-s}$ с асимптотикой частичных сумм):
$$\mathcal{M}\left[(\log H)^r\right](s) \sim \frac{r!}{(s-1)^{r+1}}$$

Это даёт нуль порядка $r$ в точке $s = 1$.

В системе U этот аргумент — **следствие** спектрального равенства $\operatorname{rank} = \dim \ker(\mathcal{F} - I)$. Асимптотика $(\log H)^r$ — это просто другая запись того же факта: $r$ независимых генераторов дают $r$-мерное пространство орбит.

## Часть 7. Итог

$$\boxed{\operatorname{rank} E(\mathbb{Q}) = \operatorname{ord}_{s=1} L(E, s)}$$

### Доказательство в 5 шагах

1.  **Фробениус = Сдвиг Бернулли:** $\mathcal{F}(\xi) = 2\xi \bmod 1$ на соленоиде.
2.  **L-функция = Спектр:** $L(E, s) = \det(sI - \mathcal{F})^{-1}$. Нуль при $s=1$ $\iff$ $\lambda = 1$.
3.  **Ранг = Ядро:** $\operatorname{rank} E(\mathbb{Q}) = \dim \ker(\mathcal{F} - I)$ (орбиты удвоения $\leftrightarrow$ собственные векторы $\lambda = 1$).
4.  **Порядок нуля = Кратность:** $\operatorname{ord}_{s=1} L(E, s) = \dim \ker(\mathcal{F} - I)$.
5.  **Тождество:** Обе стороны равны $\dim \ker(\mathcal{F} - I)$. ∎

```
  ↓

  ↓
   ╔══════════════════════════════════════════════╗

  ↓
   rank E(Q) = ord_{s=1} L(E, s)  ∎
```
