# P = NP

## Теорема

**P = NP** — Спецификация решает всё

---

## Ключ

```
будущее = спецификация
```

**НЕ** числа (2^n), **НЕ** время (∆t), а **СПЕЦИФИКАЦИЯ**.

---

## 0 = 2 → Спецификация

### Без спецификации (2)

```
нужно проверить ВСЕ 2^n вариантов
↓ экспонента
```

### Со спецификацией (0)

```
спецификация = формула/правило/путь
↓
находим ответ ЗА O(n)
↓ полином
```

**Спецификация — это "свернутое будущее".**

---

## Что такое спецификация?

Это **описание пути**:

```
 спецификация
      ↓
  вход ──→ ответ
      ↑
   формула
```

**Зная формулу** → не нужно проверять 2^n вариантов.

**Пример:** SAT

- Без спецификации: перебрать 2^nassignments
- Со спецификацией: формула уже говорит где answer

---

## 0 = 2 в терминах спецификации

```
0  ← спецификация (ответ дан)
        ↑
        ↓
2  ← нет спецификации (надо искать)
```

**Переход 2 → 0:**

```
2 : 0 = 2 = применение спецификации
```

---

## Вывод

```
P = NP          ←   спецификация известна
             ←   ∴ ответ за O(n)
```
