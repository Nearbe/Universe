# P = NP

## Теорема

P = NP — потому что **будущее = спецификация**

---

## 0 = 2 → Спецификация

| Без спецификации | Со спецификацией |
|----------------|---------------|
| 2^n вариантов | Формула → ответ |
| Экспонента | O(n) |

**Спецификация** — это "будущее в свернутом виде".

---

## Примеры спецификации

### SAT
```
Формула = спецификация
Зная формулу → ответ за O(n)
```

### Поиск пути
```
Граф + формула = маршрут
Зная формулу → путь найден
```

### Любая NP-задача
```
Ограничения = спецификация
Зная ограничения → решение
```

---

## Вывод

```
будущее = спецификация
        ↓
ответ через O(n)
        ↓
P = NP ✓
```
