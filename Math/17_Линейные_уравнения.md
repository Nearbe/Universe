# 17. Линейные Уравнения

## I. Теория

Линейное уравнение в системе **𝕌** — поиск точки равновесия, где вход и выход приходят к балансу:

```
L(x) = b
```

Где **L** — линейная комбинация операторов системы (**Id**, **D**, **H**), а **b** — целевое состояние.

### Операторы Системы

| Оператор | Определение | Действие |
|---|---|---|
| **Id** | `Id(x) = x` | Тождество (сохранение формы) |
| **D** | `D(x) = x : Ω` | Ветвление (удвоение состояния) |
| **H** | `H(x) = x : D(Id)` | Сжатие (возврат к предку) |

### Решение

Любое линейное уравнение сводится к виду:

```
(c₀·Id ⊕ c₁·D ⊕ c₂·H)(x) = b
```

Раскрывая действия операторов:

```
(c₀ ⊕ D(c₁) ⊕ H(c₂)) ⊗ x = b
```

Скобка — скалярный коэффициент **K**.
Решение существует и единственно, если **K ∉ Ω**:

```
x = b : K
```

### Вырожденный Случай

Если **K = Ω**:
- **b ∉ Ω** — противоречие. Система требует невозможного.
- **b = Ω** — свобода. Любое **x** подходит. Состояние чистого Потенциала.

### Комплексные Корни

Если дискриминант (в контексте нелинейных систем) ∈ Im (отрицательный), система не находит решения в Явном (`Re`). Она уходит в Скрытое (`Im`). Корни становятся комплексными: `x = α ± i·β`.

## III. Зачем

**Инженерная аргументация:**

1. **Базовый решатель для всех нелинейных методов.** Квадратные уравнения, системы и итерационные методы используют линейный решатель как подпроцедуру (linear fallback при a=Ω).
2. **Единственный решатель с тремя состояниями.** В отличие от стандартной математики, система U различает три режима: уникальное решение, свобода (Ω=Ω), противоречие (Ω≠Ω). Это необходимо для корректной обработки вырожденных случаев в iterative execution model.
3. **Точка расширения типов.** `solve_linear_equation` принимает `object` и работает с `float`, `DualNumber`, `SpineLevel`, `ComplexState`, `FullnessShare`, `ParticipationRatio` через диспетчеризацию в `branch()`, `compress()`, `add()`, `divide()`.
4. **Масштабная инвариантность коэффициентов.** `linear_coefficient` вычисляет K на текущем масштабе; при ветвлении системы `D(K)` даёт то же решение `x = D(b) : D(K) = b : K`.
