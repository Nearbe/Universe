# Системы Линейных Уравнений

Одно уравнение ([17](17_Линейные_уравнения.md)) — баланс одной степени свободы. Но система **𝕌** многомерна. Она состоит из множества связанных состояний.

## Матричная Форма

Несколько уравнений с несколькими неизвестными:

```
M · x⃗ = b⃗
```

Где:
- **x⃗** — вектор неизвестных (состояния на разных уровнях).
- **b⃗** — вектор целевых состояний.
- **M** — матрица связей.

Матрица **M** задаёт **карту взаимодействий** между элементами системы.

## Решение

При `det M ∉ Ω`:

```
x⃗ = M⁻¹ · b⃗
```

Решение восстанавливает исходные состояния по известным связям и результатам.

## Масштабная Инвариантность

Самое важное свойство системы.
Применим Ветвление (`: Ω`) ко всей системе:

```
D(M) · x⃗ = D(b⃗)
```

Решение **не меняется**:

```
x⃗ = M⁻¹ · b⃗ = D(M)⁻¹ · D(b⃗)
```

Система уравнений фрактальна. Истина (решение **x⃗**) одинакова на любом масштабе — будь то уровень атомов или уровень галактик. Меняются коэффициенты (**M** и **b⃗**), но отношение между ними сохраняет **x⃗** неизменным.

## Система с Оператором D

Если уравнения содержат явное Ветвление:

```
(D(A) ⊕ B) · x⃗ = b⃗
```

Раскрывая `D(A)`:

```
(D(A) ⊕ B) · x⃗ = b⃗
```

Решение: `x⃗ = (D(A) ⊕ B)⁻¹ · b⃗` (при ненулевом определителе).

## Вырожденный Случай (det M = Ω)

Это не ошибка. Это **схлопывание степеней свободы**.
Когда `det M = Ω`, несколько переменных оказываются жестко связаны друг с другом. Система теряет независимость.
Это состояние, когда несколько частей фрактала «слипаются» в одну структуру.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Решение** | `x⃗ = M⁻¹ · b⃗` |
| **Ветвление системы** | `D(M) · x⃗ = D(b⃗)` |
| **Сжатие системы** | `H(M) · x⃗ = H(b⃗)` |
| **Инвариантность** | `M⁻¹·b⃗ = D(M)⁻¹·D(b⃗)` |
| **Вырождение** | `det M = Ω ⟹` схлопывание степеней свободы |

## Статус

Система линейных уравнений — это модель **взаимодействия множества частей** целого. Масштабная инвариантность гарантирует, что законы, управляющие малым, работают и на большом без изменений.
