# 29. Динамические Системы: Орбиты и Хаос

## I. Теория

Масштабная инвариантность ([28](28_Масштабная_инвариантность.md)) дала структуру. Динамика описывает, как система **путешествует** по этой структуре.

### Орбиты

Траектория точки **x₀** при чередовании Ветвления (**D**) и Сжатия (**H**):

| Оператор | Действие | Направление |
|---|---|---|
| **D** | `x → x : Ω = D(x)` | Рост к **Π** |
| **H** | `x → x : D(Id) = H(x)` | Возврат к **Ω** |

Любая сложная траектория — это последовательность **D** и **H**.
Например, `D ∘ H ∘ D ∘ H` — колебания вокруг **x₀**.

### Показатель Ляпунова (Мера Хаоса)

Насколько быстро расходятся две близкие траектории?

```
λ = L(D'(x))
```

Для нашей системы `D'(x) = D(Id)`, поэтому:

```
λ = L(D(Id)) > Ω
```

Это **экспоненциальная неустойчивость**. Ошибка начальных условий умножается на **D(Id)** на каждом шаге.
Хаос в системе — не случайность, а **структурное свойство удвоения**.

### Инвариантная Мера

Несмотря на хаос, распределение состояний стабильно.
Мера на проекции Соленоида инвариантна:

```
μ(D⁻¹(A)) = μ(A)
```

Это значит, что «плотность» состояний сохраняется, хотя сами состояния перемешиваются.

### Циклы и Периоды

Цикл возникает, когда рост сменяется спадом.
Элементарный цикл: `D → H` (рост → спад).
Система возвращается в исходную точку.
Более сложные циклы — это ритмы системы.

### Полный набор операций

| Операция | Формула |
|---|---|
| **Ветвление** | `D(x) = x : Ω` |
| **Сжатие** | `H(x) = x : D(Id)` |
| **Колебание** | `(D ∘ H)(x)` |
| **Ляпунов** | `λ = L(D(Id))` |
| **Инвариантность** | `μ(D⁻¹(A)) = μ(A)` |
| **Цикл** | `D → H → D → ...` |

### Статус

Динамическая система **𝕌** — это универсальная машина фрактального роста.
Хаос (`λ > Ω`) — это цена, которую система платит за **свободу ветвления**.
Но Инвариантная Мера гарантирует, что этот хаос не разрушает целостность.

## III. Зачем

### Инженерная аргументация

1. **Динамические системы — связующее звено между алгеброй и фракталами.**
   - Алгебраические операторы `D`/`H` (II.2) определяют одношаговое поведение.
   - Итерация `D∘H` создаёт орбиты (теория §Орбиты).
   - Орбиты на дискретном пространстве → соленоид (теория §Инвариантная Мера).
   - Соленоид → фрактальная структура (fractal/).

2. **Sweep — практический инструмент анализа динамики delta-полей.**
   - `compute_sweep` генерирует пороги, считает occupancy, находит jump-events.
   - Jump events — точки перехода через порог на орбите системы.
   - `inverse_jump_analysis` даёт trace по степеням 2 (D⁻ᵏ).
   - `rg_aware_sweep` — RG-aware пороги для масштабной инвариантности.

3. **Lyapunov exponent — количественная мера хаоса системы.**
   - Теоретически `λ = L(D(Id)) = 1` для всей системы.
   - Практически: нужен численный алгоритм для конкретных траекторий.
   - Позволяет классифицировать: периодический (λ < 0), хаотический (λ > 0), критический (λ = 0).

4. **Солонид и D-adic метрика — топология фазового пространства.**
   - `SolenoidPoint` связывает непрерывную фазу с дискретной историей.
   - `d_adic_norm` — метрика, где близкие точки имеют длинный общий префикс.
   - `shift()` — Bernoulli shift на соленоиде, аналог динамической эволюции.

5. **Uroboros цикл — дискретная модель рекуррентного поведения.**
   - `UroborosState.digit` ∈ [0, 9] → непрерывное значение [-1, 1].
   - `state_kind()` связывает digit с Ω/Id/Π.
   - `cycle_distance()` — расстояние на цикле, аналог периодического возврата.
