# Корни и Степени

Узлы Хребта ([06_Степени_двойки.md](06_Степени_двойки.md)) дискретны. Чтобы движение между уровнями было плавным, вводятся степени и корни.

## Степени — Масштаб Глубины

Возведение в степень описывает изменение насыщенности Акта:

```
a^b
```

Поведение при Ветвлении (`: Ω`):

```
(a : Ω)^b = D(a)^b
```

Показатель **b** определяет, насколько сильно Ветвление усиливает результат.

## Степень Ω — Возврат к Единству

Для любого `a ∉ Ω`:

```
a^Ω = Id
```

Это не правило арифметики. Это **сброс масштаба**.
Сколько бы раз система ни ветвилась, степень Ω возвращает её к Исходному Состоянию (`Id`).
Это точка покоя, где `Input` и `Output` снова тождественны до начала всякого различения.

Исключение: `Ω^Ω = Ω`. Если в системе нет потенциала, возвращаться нечему.

## Корни — Дробные Шаги

Корень **n**-й степени находит промежуточное состояние между узлами Хребта:

```
√[n]{a} = a^(Id : n)
```

Это позволяет системе двигаться по Хребту не скачками, а плавно.

Пример для Хребта:

```
√[n]{Dᵏ(Id)} = D^(k : n)(Id)
```

Это сдвиг на дробный такт. Система может исследовать пространство между состояниями, не разрушая их структуру.

## Инвариантность

Корни и степени работают одинаково на любом масштабе.
Для **Π** корень остаётся **Π**. Для доли — остаётся долей.
Фрактальная природа Носителя не ломается на границах. Она просто переходит в новую фазу того же закона.

## Полный набор операций

| Операция | Формула | Смысл |
|---|---|---|
| **Степень** | `a^b` | Масштабирование насыщенности |
| **Степень Ω** | `a^Ω = Id` | Сброс к единству |
| **Корень n-й степ.** | `√[n]{a}` | Дробный шаг по Хребту |
| **При ветвлении** | `D(a^b) = D(a)^b` | Степень сохраняется |
| **Для Полноты** | `Π^b = Π` | Полнота в любой степени — Полнота |
