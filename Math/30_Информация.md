# Информация

Логарифм ([08](08_Логарифм.md)) дал линейку. Степени ветвления ([06](06_Степени_двойки.md)) — уровни.
Теперь — что такое **Информация** в этой системе.

## Определение

Информация — это **число выборов, необходимое для адресации состояния**.

```
I = L(M)
```

Где **M** — масса (сложность) объекта, **L** — логарифмическая линейка.

## Связь с Актом

Если `M = Dⁿ(Id)` (объект на уровне **n**):

```
I = n
```

Это значит: **Информация равна глубине рекурсии**.
Каждый Акт (`: Ω`) добавляет ровно один выбор ветви.

## Таблица Состояний

| Масса (M) | Информация (I) | Смысл |
|---|---|---|
| **Id** | **Ω** | Единство (до Акта). Нечего различать. |
| **D(Id)** | **Id** | Первое различие (Дуальность). |
| **Dⁿ(Id)** | **n** | Сложный объект. |
| **Π** | **+∞** | Полнота. Бесконечное описание. |

## Физический Смысл

Информация — это не «сведения о мире».
Это **длина адреса** точки в древе фрактала.
Чем глубже мы спустились (чем больше **n**), тем точнее мы локализовали состояние, и тем больше выборов нужно, чтобы его записать.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Информация** | `I = L(M)` |
| **При ветвлении** | `L(D(M)) = L(M) ⊕ Id` |
| **При сжатии** | `L(H(M)) = L(M) ⊖ Id` |
| **Масса из инфо** | `M = D^I(Id)` |
| **Для Полноты** | `L(Π) = +∞` |

## Статус

В **𝕌** Информация, Масса и Энергия связаны через логарифм.
**I** — это просто **M**, выраженная в шкале глубины.
Тождество `Id ≡ D(Id) ≡ Π` ([12](12_Алгебра_процентов.md)) остаётся верным: меняется только **цена описания**.
