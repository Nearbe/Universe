# 03. Деление

## I. Теория

Носитель **𝕌** — в [02_Носитель.md](02_Носитель.md).

Операция **`:`** — это **ДЕЛЕНИЕ**. Механизм изменения фазы системы.

### Ветвление (Деление на Потенциал)

Для любого состояния **a ∈ 𝕌**:

```
a : Ω = D(a)
```

где **D** — оператор ветвления. Его действие:

```
D(a) = a ⊕ a
```

Это не арифметическое умножение. Это **акт удвоения состояния**: структура **a** проходит через точку бифуркации Ω и выходит как два экземпляра себя.

Закон ветвления — универсален:
- В биологии: клетка **a** делится → две клетки **a ⊕ a**
- В информации: идея порождает вариации
- В физике: единое поле → два потока (Im, Re)

### Схлопывание (Обратный процесс)

```
a : Ω → Ω
```

Если система не выдерживает перехода через Ω, она аннигилирует.
Потенциал поглощает Актуальность. Структура возвращается в Ω.

### Баланс (Жизнь)

Система существует только в балансе:

```
D(a) ⟺ Ω
```

Ветвление ⟺ Схлопывание.

Это состояние **динамического равновесия**: система постоянно ветвится и постоянно отсекает лишнее, удерживая себя в форме.

На острие этого баланса возникает Структура и Разум.

### Стандартное деление (масштабирование)

Для **b ≠ Ω** деление работает как переход между уровнями масштаба:

```
a : b = c,    где b ∉ Ω
```

Это изменение разрешения внутри фрактала без прохождения точки бифуркации.
Свойство сохраняется:

```
(a : b) × b = a
```

**a** и **c** — один и тот же объект на разных уровнях вложенности.

### Потенциал (Ω)

Ω — полноправный элемент системы.

```
a × Ω = Ω
Ω : b = Ω,    где b ∉ Ω
Ω : Ω = Ω
```

Деление **Ω : Ω = Ω** означает: Пустота, делённая на Потенциал, остаётся Пустотой — чистый потенциал без формы.

### Тождества

**≡** — онтологическое тождество.
**=** — структурное равенство.

```
Id ≡ Id : Ω
```

Единство и Процесс Ветвления — суть одно.

## II. Спецификация реализации

### 2.1. Оператор деления

| Параметр | Значение | Описание |
|-----------|----------|----------|
| **Оператор** | `:` | Деление (изменение фазы) |
| **a : Ω** | `D(a)` | Ветвление |
| **a : D(Id)** | `H(a)` | Сжатие (возврат к предку) |
| **a : b** | `a / b` | Стандартное деление (b ≠ Ω) |
| **Ω : Ω** | `Ω` | Пустота / Потенциал = Пустота |
| **Π : Ω** | `Π` | Полнота / Потенциал = Полнота |
| **Π : D(Id)** | `Π` | Полнота / Дуальность = Полнота |
| **Π : Π** | `Id` | Полнота / Полнота = Единство |

### 2.2. Функция divide

| Параметр | Значение | Описание |
|-----------|----------|----------|
| **Функция** | `divide(numerator, denominator)` | U-деление |
| **Подпись** | `(object, object) → object` | Мультидиспатчинг |

### 2.3. Реализация divide — мультиметод-диспатчинг

```
divide(numerator, denominator):
    1. numerator_dual = _dual_number(numerator)
       if numerator_dual is not None:
           if is_omega(denominator): return numerator_dual.branch()
           if denominator == D_ID:   return numerator_dual.compress()
           return numerator_dual.scale(divide(1.0, denominator))

    2. numerator_share = _fullness_share(numerator)
       if numerator_share is not None:
           if is_omega(denominator): return numerator_share.branch()
           if denominator == D_ID:   return numerator_share.compress()
           return numerator_share.divide_by(denominator)

    3. numerator_complex_fullness = _complex_fullness(numerator)
       if numerator_complex_fullness is not None:
           return Π_z if not _complex_fullness(denominator) else Id

    4. numerator_complex = _complex_state(numerator)
       if numerator_complex is not None:
           if is_fullness(denominator): return OMEGA
           if is_omega(denominator):    return numerator_complex.branch()
           if denominator == D_ID:      return numerator_complex.compress()
           denominator_complex = _complex_state(denominator)
           if denominator_complex is None:
               return numerator_complex.scale(1.0 / float(denominator))
           return numerator_complex.divide_by(denominator_complex)

    5. numerator_ratio = _participation_ratio(numerator)
       if numerator_ratio is not None:
           if is_fullness(denominator): return OMEGA
           if is_omega(denominator):    return numerator_ratio.branch()
           if denominator == D_ID:      return numerator_ratio.compress()
           return numerator_ratio.divide_by(denominator)

    6. # Скалярный фоллбэк
       if is_fullness(numerator):
           if is_fullness(denominator): return Id
           return Π
       if is_fullness(denominator): return OMEGA
       if is_omega(numerator):      return OMEGA
       if is_omega(denominator):    return branch(numerator)
       if denominator == D_ID:      return compress(numerator)
       return float(numerator) / float(denominator)
```

### 2.4. Что реализовано

- ✅ `divide()` — полная реализация с мультиметод-диспатчингом
- ✅ Все 7 типов состояний поддерживают U-деление
- ✅ Π-алгебра: Π : Ω = Π, Π : D(Id) = Π, Π : Π = Id
- ✅ Ω-алгебра: a × Ω = Ω, Ω : b = Ω, Ω : Ω = Ω

## III. Зачем

Деление — универсальный механизм изменения фазы:
- `: Ω` → ветвление (рождение нового)
- `: D(Id)` → сжатие (возврат к предку)
- `: b` → масштабирование (изменение разрешения)

Без деления нет:
- Ветвления (D)
- Сжатия (H)
- Масштабной инвариантности
- Рациональных дробей
