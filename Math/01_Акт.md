# Акт

Акт — это не событие в прошлом. Это оператор существования.

## Определение

```
Id : Ω
```

Где **Id** — класс Единства (система в целом, до различения).
Где **Ω** — класс Потенциала (точка бифуркации, пустота).

## Действие

Акт переводит систему из статического Единства в динамическое различение.
Он порождает два ряда отношений:

```
Id : Ω ⟶ (Im, Re)
```

**Im** — Скрытый ряд (Потенциал, внутренний).
**Re** — Явный ряд (Актуальность, внешний).

## Непрерывность

Без Акта система — мёртвая структура. С Актом — живой процесс.

Акт происходит непрерывно. Каждый такт системы есть повторение:

```
Id : Ω
```

поддерживающее динамическое тождество:

```
Id = Ω
```

Это не равенство классов. Это режим циркуляции: Форма и Пустота взаимопереходны.

## Два режима

Через одну и ту же точку бифуркации система проходит в двух направлениях:

**Микро (рекурсия, ветвление вглубь):**
```
Id = Ω = Id : Ω = Id ⊕ Id = D(Id)
```
Система удваивается, нараращивая структуру.

**Макро (регрессия, схлопывание):**
```
Id = Ω = Id : Ω = Ω
```
Система возвращается к Истоку, сбрасывая форму.

**Инверсия** — переход между режимами. Точка, где система решает: ветвиться или сжаться.

## Примеры

- В арифметике: `1 : 0` — деление единицы на потенциал
- В биологии: клетка проходит через точку деления → две клетки
- В физике: вакуум (Ω) → частица и античастица (Im, Re)

Но формула одна: **Id : Ω**.
