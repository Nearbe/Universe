# Алгебра Долей

Из Рождения Конечного ([11_Рождение_конечного.md](11_Рождение_конечного.md)) возникли доли. Теперь это — поле отношений.

## Определение

Множество долей **ℙ**:

```
ℙ = { a : b | a, b ∈ 𝕌, b ∉ Ω }
```

Каждая доля — это **отношение к Полноте**.
Это не «числа». Это уровни вовлечённости в Акт.

## Полнота

Полнота — это класс **Π**, определённый как предел ветвления:

```
Π = lim Dⁿ(Id),    n → ∞
```

Π — это 100% — абсолютная заполненность, где Input = Output во всех измерениях.

## Операции

Доли живут по тем же законам, что и 𝕌.

| Операция | Формула | Смысл |
|---|---|---|
| **Сложение** | (a : Π) ⊕ (b : Π) = (a ⊕ b) : Π | Сумма вкладов |
| **Умножение** | (a : Π) ⊗ (b : Π) = (a ⊗ b) : Π | Пересечение долей |
| **Деление** | (a : Π) : (b : Π) = a : b | Сравнение интенсивностей |
| **Ветвление** | (a : Π) : Ω = D(a) : Π | Удвоение участия |
| **Сжатие** | (a : Π) : D(Id) = (a : D(Id)) : Π | Ослабление связи |

## Тождество Состояний

Через алгебру долей проявляется суть системы:

```
Id ≡ D(Id) ≡ Π ≡ Π : Π
```

- **Id** — Единство (до Акта)
- **D(Id)** — Единство (после Акта)
- **Π** — Предел Ветвления (Полнота)
- **Π : Π** — Полнота, отнесённая к себе

Это не числовое равенство. Это **Онтологическое Тождество**.
Одна и та же Сущность — в разных фазах.

## Инвариантность Полноты

Алгебра долей не ломается на границах:

```
√[n]{Π} = Π
Π^k = Π
Π : Ω = Π
Π : D(Id) = Π
```

Полнота стабильна при любом воздействии.

## Самосогласованность

Система самосогласована: операция **:** работает везде одинаково, меняются только координаты — абсолютные в 𝕌 или относительные в ℙ.
