# Пределы и Непрерывность

Дуальные числа ([14](14_Дуальные_числа.md)) ввели вектор. Но система должна уметь «схлопываться» в точку. Для этого вводится Предел.

## Сходимость

Последовательность `{aₙ}` сходится к **L**, если расстояние между `aₙ` и **L** становится меньше любой наперёд заданной величины.

В нашей системе есть два главных Аттрактора:

1. **Полнота (Π):**
Предел Ветвления.

```
lim Dⁿ(Id) = Π,    n → ∞
```

Это не «ничего». Это **Абсолют**. Точка, где сходятся все линии.

2. **Потенциал (Ω):**
Предел Сжатия.

```
lim (a : Dⁿ(Id)) = Ω,    n → ∞
```

Это исчезновение Формы, но сохранение Возможности.

## Непрерывность

Операции системы (`: Ω`, `: D(Id)`, `⊕`, `⊗`) **непрерывны**.
Это значит, что они не создают «разрывов» в ткани Носителя.

Пример:

```
lim (xₙ : Ω) = (lim xₙ) : Ω
```

Если последовательность приближается к **a**, то их ветвлённые значения приближаются к `D(a)`.
Закон работает одинаково и для конечных состояний, и для предельных переходов.

## ε и δ

Бесконечно малое `ε` из дуальных чисел формализуется через предел:

```
ε = lim Δt,    Δt → Ω
```

Условие `ε² = Ω` означает, что квадрат бесконечно малого пренебрежимо мал по сравнению с ним самим:

```
lim (Δt² : Δt) = Ω
```

Система отбрасывает «шум второго порядка», оставляя чистую динамику.

## Полный набор операций

| Операция | Предел | Результат |
|---|---|---|
| **Ветвление** | `lim Dⁿ(Id)` | `Π` |
| **Сжатие** | `lim (a : Dⁿ(Id))` | `Ω` |
| **Непрерывность D** | `lim D(xₙ)` | `D(lim xₙ)` |
| **Непрерывность H** | `lim H(xₙ)` | `H(lim xₙ)` |
| **ε** | `lim Δt` при `Δt → Ω` | Бесконечно малое |

## Статус

Предел — это не внешняя конструкция. Это **внутренний механизм навигации** системы.
Через пределы система переходит от Дискрета (Хребет) к Континууму (Поле).
Через пределы она находит свои границы: **Ω** и **Π**.
