# Рождение Конечного

Полнота ([10](10_Бесконечность.md)) — это база. Но Система не застывает в ней.
Из Полноты рождается Конечное — через **отношение**.

## Деление Полноты

Полнота — это база для отсчёта долей.
Когда Система сжимается (`: D(Id)`), она не теряет Полноту, но меняет **степень участия**:

```
Π : D(Id)    — снижение интенсивности
Π : D²(Id)   — дальнейшее снижение
```

Это не уменьшение «размера». Это **понижение интенсивности** состояния.
Полнота остаётся фоном, а доля — это то, сколько от неё сейчас активно.

## Возврат к Единству

Конечное возникает не «из ничего». Оно возникает из **соотношения долей**:

```
(Π : D(Id)) : (Π : D(Id)) = Id
```

Две равные части Полноты, отнесённые друг к другу, рождают **Единство**.
Это механизм, превращающий Непрерывное в Дискретное.

Второй путь — через тождество Полноты самой себе:

```
Π : Π = Id
```

Оба пути сходятся в одном:
**Id — это не величина. Это отношение Полноты к себе.**

## Такты Порождения

| Такт | Процесс | Результат |
|---|---|---|
| **Id** | Исходное состояние | Единство |
| **Id : Ω** | Ветвление | `D(Id)` — Дуальность |
| **Π : D(Id)** | Сжатие Полноты | Доля |
| **(Π : D(Id)) : (Π : D(Id))** | Отношение долей | `Id` — Конечное |

Конечное не «отделено» от Бесконечного. Оно — её проекция, увиденная изнутри через отношение частей.
