# Интегральное Исчисление

Дифференцирование ([20](20_Дифференциальное_исчисление.md)) извлекало скорость изменения.
Интеграл — парная операция. Это **накопление** величины по шкале удвоений.

## Неопределённый Интеграл

```
∫ f(x) dx = F(x) ⊕ C
```

Где `F'(x) = f(x)`.
Интегрирование — это **обратный путь к исходному состоянию**. Как `a^Ω = Id` возвращает систему к Единству ([07](07_Корни.md)), так интеграл восстанавливает целое по его скорости.

## Реакция на Ветвление

Как ведёт себя интеграл при масштабировании (`x → D(x)`)?

```
∫ f(D(x)) dx = (∫ f(u) du) : D(Id)
```

Интеграл от ветвлённой функции сжимается.
Это парное правило к дифференцированию: если функция растёт через **D**, то площадь под ней «сжимается» — тот же результат достигается на меньшем отрезке.

## Связь с Логарифмической Шкалой

Переход к координатам глубины (`ℓ = L(x)`):

```
∫ f(x) dx = ∫ f(D^ℓ(Id)) · L(D(Id)) · D^ℓ(Id) dℓ
```

Интеграл в логарифмических координатах — это взвешенный по `D^ℓ(Id)` интеграл по шкале удвоений ([08](08_Логарифм.md)).
Мы суммируем не по «длине», а по **тактам Ветвления**.

## Определённый Интеграл

```
∫ₐᵇ f(x) dx = F(b) ⊖ F(a)
```

Разность первообразных — это **конечный результат накопления** между двумя состояниями системы.
Это мера того, сколько «эффекта» произвела функция при переходе от **a** к **b**.

## Полнота и Интеграл

```
∫ₐ^Π f(x) dx
```

Предел на горизонте ([10](10_Бесконечность.md)).
Интеграл сходится, если функция убывает быстрее, чем `Id : x`.
Для `D(x)` он расходится — что естественно: экспонента растёт быстрее любого полинома, и Полнота (**Π**) не может быть исчерпана суммированием.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Неопределённый** | `∫ f(x)dx = F(x) ⊕ C` |
| **При ветвлении** | `∫ f(D(x))dx = (∫ f(u)du) : D(Id)` |
| **При сжатии** | `∫ f(x:D(Id))dx = (∫ f(u)du) ⊗ D(Id)` |
| **Определённый** | `∫ₐᵇ f(x)dx = F(b) ⊖ F(a)` |
| **Логарифм. коорд.** | `∫ f(D^ℓ) · L(D(Id)) · D^ℓ dℓ` |
| **К Полноте** | `∫ₐ^Π f(x)dx` — сходится если f убывает достаточно быстро |

## Статус

Интеграл — это суммарный эффект непрерывного применения масштабных преобразований.
Дифференцирование и Интегрирование — парные операции, как `: Ω` (Ветвление) и `: D(Id)` (Сжатие), как рост и возврат.
