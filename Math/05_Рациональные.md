# Рациональные числа

Целые состояния ([04_Связи.md](04_Связи.md)) — это узлы Хребта. Но Акт делит систему не только пополам, но и в любых пропорциях.

## Определение

Рациональное число — это **доля участия** в Акте.
Если система делит себя на **q** частей и берёт **p** из них:

```
r = p : q,    где q ∉ Ω
```

Это не «числа». Это **отношения частей** внутри Целого.

## Замкнутость

Рациональные числа описывают внутреннюю структуру Ветвления.
Для любого `r = p : q`:

```
r : Ω = D(p) : D(q) = D(p : q)
```

Система может дробить саму себя бесконечно, сохраняя смысл. Каждая дробь — это конкретная конфигурация связей, где одна часть системы доминирует над другой в заданной пропорции.

## Свойства

- **Плотность:** Между любыми двумя состояниями есть промежуточное. Система не «прыгает» дискретно, она течёт через всё поле рациональных классов.
- **Инвариантность:** **:** сохраняет пропорции. Если `r₁ < r₂`, то при Ветвлении порядок сохраняется: `D(r₁) < D(r₂)`.

## Операции над рациональными

| Операция | Формула |
|---|---|
| **Ветвление** | `(p : q) : Ω = D(p) : D(q)` |
| **Сжатие** | `(p : q) : D(Id) = (p : D(Id)) : (q : D(Id))` |
| **Сложение** | `(p₁ : q) ⊕ (p₂ : q) = (p₁ ⊕ p₂) : q` |
| **Умножение** | `(p₁ : q₁) ⊗ (p₂ : q₂) = (p₁ ⊗ p₂) : (q₁ ⊗ q₂)` |

Рациональные числа — это «ткань» Носителя. Следующий шаг — выделение несущих конструкций (Хребта).
