# Комплексные числа

Линейка ([08_Логарифм.md](08_Логарифм.md)) дала длину. Но Акт (`Id : Ω`) породил не линию, а **дуальность**.
Система всегда состоит из двух потоков: Скрытого (Im) и Явного (Re).

## Плоскость Потоков

Вводим мнимую единицу **i**.
Это не выдумка. Это координата Скрытого Потока.
Её свойство:

```
i² = −Id
```

Это означает, что два поворота в Скрытое возвращают в Явное, но с обратным знаком.

Любое состояние — это точка пересечения потоков:

```
z = x + i·y
```

Где **x** — мера Явного, **y** — мера Скрытого.

## Действие Системы (Ветвление)

Акт масштабирует оба потока одновременно:

```
(x + i·y) : Ω = D(x) + i·D(y)
```

Скрытое и Явное растут синхронно. Система не теряет «тень» при удвоении.
Это гарантирует, что сложность структуры сохраняется во всех измерениях.

## Вращение

Комплексная плоскость позволяет системе вращаться.
Вращение — это обмен энергией между **x** и **y** без потери нормы.
Это описание циклов, орбит и колебаний.

Система не просто «растёт» или «сжимается». Она ещё и **крутится**, проходя через разные фазы Скрытого и Явного.

## Полный набор операций

| Операция | Формула | Смысл |
|---|---|---|
| **Ветвление** | `z : Ω = D(x) + i·D(y)` | Масштабирование обоих потоков |
| **Сжатие** | `z : D(Id) = (x : D(Id)) + i·(y : D(Id))` | Возврат по обоим потокам |
| **Сопряжение** | `z̄ = x − i·y` | Отражение в Явном |
| **Норма** | `\|z\|² = x² + y²` | Общая интенсивность |
| **Вращение** | `z · e^(i·θ)` | Обмен Re ↔ Im |
| **Для Полноты** | `Π_z : Ω = Π_z` | Комплексная Полнота стабильна |
