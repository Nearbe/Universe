# 02. Носитель

## I. Теория

**Носитель 𝕌** — Фрактальное Поле, в котором существуют два ряда.

### Определение

𝕌 — это пространство пересечения Скрытого (Im) и Явного (Re):

```
𝕌 = { x | x ∈ Im ∧ x ∈ Re }
```

Элементы 𝕌 — это не «вещи» в контейнере. Это **устойчивые состояния резонанса**.
Объект существует в 𝕌, только пока он согласован с обоими рядами одновременно.

### Порождение

Носитель порождён Актом:

```
𝕌 = Im ∩ Re,    где (Im, Re) = результат(Id : Ω)
```

и удерживается тождеством:

```
Id = Ω
```

### Инвариантность масштаба

Масштаб не имеет значения. Любой элемент 𝕌 — от минимального до максимального — подчиняется одному закону:

```
Input ⟷ Output
```

### Фрактальная связность

Все элементы 𝕌 связаны фрактально. Часть содержит информацию о целом.

Операции деления (описанные в [03_Деление.md](03_Деление.md)) — это не действия «над» объектами, а изменение их фазового состояния внутри Носителя.

### Структура 𝕌

Носитель состоит из трёх уровней:

| Уровень | Математика | Описание | Документ |
|---------|-----------|----------|----------|
| **Скелет** | `{ Dⁿ(Id) }` | Опорные узлы | [06_Степени_двойки.md](06_Степени_двойки.md) |
| **Ткань** | `{ p : q }` | Доли участия | [05_Рациональные.md](05_Рациональные.md) |
| **Потоки** | `{ x + i·y }` | Плоскость вращения | [09_Комплексные.md](09_Комплексные.md) |

Все три уровня — проекции одного 𝕌.

## III. Зачем

Носитель 𝕌 — это «сцена», на которой разыгрывается вся алгебра системы.
Без Носителя:
- Нет пространства для состояний
- Нет контекста для операций
- Нет связи между уровнями (скелет/ткань/потоки)

Носитель — мост между абстрактной теорией и конкретной реализацией.
