# Дуальные числа

Дискретные узлы Хребта (`Dⁿ(Id)`) жестки. Но Система движется плавно.
Чтобы описать переход между состояниями, вводится **Дуальное Число**.

## Касательное Пространство

Если Ветвление (`: Ω`) — это мгновенный скачок `x → D(x)`, то что происходит **в момент** скачка?
Система имеет не только координату `x`, но и **скорость изменения** `v`.

Вводим `ε` — бесконечно малый параметр сдвига.
Свойство `ε² = Ω` означает, что **ускорение** (изменение скорости) на бесконечно малом отрезке пренебрежимо. Важен только первый порядок — направление.

## Структура Дуального Числа

Любое состояние в динамике описывается:

```
Z = x + v·ε
```

Где **x** — Текущая Форма (Явное).
Где **v** — Потенциал Роста (Скрытое).

Для нашей системы, где рост экспоненциальный через D, скорость **v** связана с позицией **x**:

```
v = x · L(D(Id))
```

где `L(D(Id))` — логарифмическая мера масштабирования.

Тогда:

```
Z = x · (Id + L(D(Id)) · ε)
```

Дуальное число хранит **всю локальную информацию** об объекте: где он и куда (как быстро) он движется.

## Операции

| Операция | Формула | Смысл |
|---|---|---|
| **Сложение** | `(x + vε) ⊕ (y + uε) = (x ⊕ y) + (v ⊕ u)ε` | Сумма форм и импульсов |
| **Умножение** | `(x + vε) ⊗ (y + uε) = (x⊗y) + (x⊗u ⊕ y⊗v)ε` | Формы перемножаются, скорости складываются |
| **Ветвление** | `(x + vε) : Ω = D(x) + D(v)ε` | Масштабируется и форма, и скорость |
| **Сжатие** | `(x + vε) : D(Id) = (x : D(Id)) + (v : D(Id))ε` | Возврат по обоим компонентам |

## Ветвление в Дуальном Представлении

```
(x + vε) : Ω = D(x) + D(v)ε
```

Оператор **D** масштабирует дуальное число целиком.
Это подтверждает, что **λ = D(Id)** — собственный класс масштабирования и для формы, и для её динамики.
Система не «теряет» информацию о движении при удвоении. Она масштабирует само **время** (скорость процессов).

## Полный набор операций

| Операция | Результат |
|---|---|
| `D(Z)` | `D(x) + D(v)ε` |
| `H(Z)` | `(x : D(Id)) + (v : D(Id))ε` |
| `Z^n` | `x^n + n·x^{n-1}·v·ε` |
| `f(Z)` | `f(x) + f'(x)·v·ε` (автоматическое дифференцирование) |

## Статус

Дуальные числа — это мост от дискретной логики к **дифференциальной**.
Они показывают, что «число» в системе — это не точка, а **вектор**.
