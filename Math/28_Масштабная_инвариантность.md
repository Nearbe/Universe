# 28. Масштабная Инвариантность и Ренормализация

## I. Теория

### 28.1. Операция `:` как шаг Ренормализационной Группы

В системе **𝕌** деление на **Ω** (`: Ω`) — это дискретный шаг РГ с коэффициентом **D(Id)**.

```
R_D(x) = x : Ω = D(x)
```

Это закон изменения констант связи при сдвиге по Хребту (`06_Степени_двойки.md`).

### 28.2. β-Функция

Скорость изменения параметра **g** при изменении масштаба:

```
β(g) = g : D(Id) ⊖ g
```

В системе U — линейный поток: `β(g) = g`. Параметр растёт экспоненциально с глубиной.

### 28.3. Критические Показатели

В точках фазового перехода система становится масштабно-инвариантной.
Степенные законы `f(x) = x^b` — это гомогенные функции, ковариантные относительно `: Ω`:

```
f(x : Ω) = D(x^b)
```

Показатель степени **b** — критический индекс системы.

### 28.4. Полный набор операций

| Операция | Формула |
|---|---|
| **Шаг РГ** | `R_D(x) = x : Ω = D(x)` |
| **Обратный шаг** | `R_D⁻¹(x) = x : D(Id) = H(x)` |
| **β-функция** | `β(g) = D(g) ⊖ g` |
| **Поток** | `g_{n+1} = D(g_n)` |
| **Ковариантность** | `f(x:Ω) = D(x^b)` для степенных |
| **Инвариантность** | `x⃗ = M⁻¹·b⃗ = D(M)⁻¹·D(b⃗)` |
