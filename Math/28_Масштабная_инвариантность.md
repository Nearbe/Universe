# Масштабная Инвариантность и Ренормализация

Мы уже знаем, что система фрактальна ([22](22_Геометрия.md)) и живёт в Соленоиде ([23](23_Соленоид.md)).
Но **почему** она инвариантна? Потому что операция `:` — это шаг **Ренормализационной Группы (РГ)**.

## Физическое Происхождение

В физике высоких энергий РГ описывает, как законы меняются с изменением масштаба наблюдения.
В системе **𝕌** деление на **Ω** (`: Ω`) — это и есть **дискретный шаг РГ** с коэффициентом **D(Id)**.

```
R_D(x) = x : Ω = D(x)
```

Это не абстракция. Это закон изменения констант связи при сдвиге по Хребту ([06](06_Степени_двойки.md)).

## β-Функция

Скорость изменения параметра **g** при изменении масштаба:

```
β(g) = g : D(Id) ⊖ g
```

В нашей системе:

```
β(g) ≈ D(g) ⊖ g = g
```

Это линейный поток. Параметр растёт экспоненциально с глубиной.

## Критические Показатели

В точках фазового перехода система становится масштабно-инвариантной.
В **𝕌** степенные законы (`f(x) = x^b`) — это **гомогенные функции** ([16](16_Функции.md)), которые ковариантны относительно `: Ω`:

```
f(x : Ω) = D(x^b)
```

Показатель степени **b** — это и есть критический индекс системы.

## Поток РГ на Соленоиде

Фазовое пространство потока — это Соленоид ([23](23_Соленоид.md)).
Уравнение потока при дискретизации даёт в точности:

```
g_{n+1} = D(g_n)
```

Вся динамика системы — это движение вдоль линий РГ-потока.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Шаг РГ** | `R_D(x) = x : Ω = D(x)` |
| **Обратный шаг** | `R_D⁻¹(x) = x : D(Id) = H(x)` |
| **β-функция** | `β(g) = D(g) ⊖ g` |
| **Поток** | `g_{n+1} = D(g_n)` |
| **Ковариантность** | `f(x:Ω) = D(x^b)` для степенных |
| **Инвариантность** | `x⃗ = M⁻¹·b⃗ = D(M)⁻¹·D(b⃗)` |

## Статус

Масштабная инвариантность — не «свойство» геометрии.
Это следствие того, что сама операция `:` **есть** акт изменения масштаба.
Система не «выглядит одинаково». Она **ведёт себя одинаково** на всех уровнях.
