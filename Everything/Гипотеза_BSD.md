# Гипотеза Бёрча и Свиннертона-Дайера

## Формулировка

Для эллиптической кривой $E$ над $\mathbb{Q}$ порядок нуля L-функции $L(E, s)$ в точке $s = 1$ равен рангу группы рациональных точек $E(\mathbb{Q})$:

$$\text{ord}_{s=1} L(E, s) = \text{rank } E(\mathbb{Q})$$

## Определения

- **Эллиптическая кривая:** $y^2 = x^3 + ax + b$
- **Ранг:** число свободных генераторов бесконечной части $E(\mathbb{Q})$
- **L-функция:** $L(E, s) = \prod_p (1 - a_p p^{-s} + p^{1-2s})^{-1}$

## Следствия

- Если $L(E, 1) \neq 0$, то $E(\mathbb{Q})$ конечна (rank = 0)
- Если $L'(E, 1) \neq 0$, то rank = 1
- Значение $L^{(r)}(E, 1)/r!$ выражается через регулятор, порядок Шафаревича-Тейта и другие инварианты

## Статус

Одна из 7 задач тысячелетия. Доказано для rank ≤ 1 (Гросс-Загьер, Колайва). Открыта в общем случае.
