# Электричество

Родитель: Электромагнетизм ([158](158_Электромагнетизм.md)), Калибровка U(1) ([55](55_Калибровочные_группы.md))

## Определение

```
E = ⊖∇φ ⊖ ∂A : ∂t
∇·E = ρ_q : ε₀
∇×E = ⊖∂B : ∂t
```

где `φ` — скалярный потенциал, `A` — векторный потенциал, `ρ_q` — плотность заряда.

## Физический смысл

Электрическое поле — градиент фазового потенциала `φ` и временная производная калибровочного поля `A`. В `𝕌` заряд `ρ_q` есть мера рассогласования локального потока с глобальной калибровкой U(1). Ток `j = ρ_q ⊗ v` описывает перенос фазового дисбаланса. Электричество — это макроскопическое проявление выравнивания фаз в проводящей сети.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Поле** | `E = ⊖∇φ ⊖ ∂A:∂t` |
| **Закон Гаусса** | `∇·E = ρ_q:ε₀` |
| **Закон Фарадея** | `∇×E = ⊖∂B:∂t` |
| **Сила** | `F = q ⊗ (E ⊕ v×B)` |
| **Энергия поля** | `u_E = ε₀ ⊗ E² : D(D(Id))` |
