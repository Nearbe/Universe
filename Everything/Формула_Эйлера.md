# Формула Эйлера

## Определение

$$e^{ix} = \cos x + i \sin x$$

## Следствия

**Тождество Эйлера:**
$$e^{i\pi} + 1 = 0$$

Связывает пять фундаментальных констант: $0, 1, e, i, \pi$.

**Полярная форма комплексного числа:**
$$z = r e^{i\theta}$$

**Формулы Муавра:**
$$(\cos x + i\sin x)^n = \cos(nx) + i\sin(nx)$$

## Доказательство через ряды Тейлора

$$e^{ix} = \sum_{n=0}^{\infty} \frac{(ix)^n}{n!} = \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k}}{(2k)!} + i \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k+1}}{(2k+1)!}$$
$$= \cos x + i \sin x$$

## Применения

- Упрощение тригонометрических вычислений
- Теория колебаний и волн
- Квантовая механика (фазовый множитель)
- Преобразование Фурье
