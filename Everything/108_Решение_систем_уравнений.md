# Решение систем уравнений

Системы линейных уравнений ([18](18_Системы_линейных_уравнений.md)) — матричная форма. Соленоид ([23](23_Соленоид.md)) — дискретные шаги.

## Методы

- Гаусс: прямое решение
- Якоби: итеративное
- Гаусс-Зейдель: с обновлёнными значениями
- Степенной метод: наибольшее собственное значение

## Физический смысл

Решение дискретных уравнений динамики сети. Каждый метод — свой «темп» нахождения равновесия.

## Полный набор операций

| Метод | Формула | Смысл |
|---|---|---|
| **Гаусс** | Прямое исключение | Точное решение |
| **Якоби** | `x(n+1) = M⁻¹⊗(b ⊖ N⊗x(n))` | Итеративное |
| **Гаусс-Зейдель** | Обновление на каждой итерации | Быстрее Якоби |
| **Степенной** | `v(n+1) = A⊗v(n) : \|A⊗v(n)\|` | Максимальное λ |
