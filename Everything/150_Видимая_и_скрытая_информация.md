# Видимая и скрытая информация

Родитель: Мембрана (Δ) ([122](122_Дифференциал_Delta.md), Синхронизация (𝒮) ([45](45_Синхронизация.md))

## Определение

```
I_vis = I ⊗ f_vis
I_hid = I ⊗ f_hid
```

где `f_vis = Id:(Id⊕e^Q)`, `f_hid = e^Q:(Id⊕e^Q)`, `I = L(M)` — полная информация системы.

## Физический смысл

Разделение полного информационного объёма `I` на декодируемую (`I_vis`) и недоступную (`I_hid`) части. Граница определяется пропускной способностью канала Регрессии (5%). При `𝒮 → 0.95`: `f_vis → Id`, `I_hid → Ω` (полная прозрачность). При `𝒮 → Ω`: `f_vis → Ω`, `I_hid → I` (полная непрозрачность). Динамика `I_vis` и `I_hid` описывается уравнениями потока через Мембрану.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Видимая** | `I_vis = I ⊗ f_vis` |
| **Скрытая** | `I_hid = I ⊗ f_hid` |
| **Сохранение** | `I_vis ⊕ I_hid = I` |
| **Поток** | `dI_vis:dt = −dI_hid:dt = κ(I_hid:I_vis ⊖ I_vis:I_hid)` |
| **Предел 𝒮→0.95** | `I_hid → Ω, I_vis → I` |
