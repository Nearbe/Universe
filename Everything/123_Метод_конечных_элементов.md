# Метод конечных элементов

Метод конечных разностей ([104](104_Метод_конечных_разностей.md)) — дискретизация. Линейные уравнения ([17](17_Линейные_уравнения.md)) — решение.

## Определение

```
u(x) ≈ Σ c_i ⊗ φ_i(x)
```

## Физический смысл

Решение уравнений для поля **Δ** в сложных геометриях. Область разбивается на элементы, решение ищется как комбинация базисных функций.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Аппроксимация** | `u(x) ≈ Σ c_i⊗φ_i(x)` |
| **Базис** | `φ_i(x)` — функции элементов |
| **Коэффициенты** | `c_i` — решаются из системы |
| **Смысл** | Сложная геометрия → простые элементы |
