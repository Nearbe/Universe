# Ёмкость состояний

Родитель: Синхронизация ([45](45_Синхронизация.md)), Энтропия ([166](166_Термодинамика.md))

## Определение

```
C(𝒮) = C_max ⊗ 𝒮^{D_f ⊖ D(D(Id))}
```

где `C_max` — максимальная информационная ёмкость сети, `𝒮` — параметр синхронизации.

## Физический смысл

Ёмкость сети пропорциональна степени её когерентности. При `𝒮 → Id`: `C → C_max` (все узлы участвуют в обработке). При `𝒮 → Ω`: `C → Ω` (сеть дезинтегрирована, передача информации невозможна). Зависимость от `D_f` показывает, что системы с большей фрактальной размерностью эффективнее используют синхронизацию для хранения данных.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Ёмкость** | `C(𝒮) = C_max ⊗ 𝒮^{D_f⊖D(D(Id))}` |
| **𝒮 → Id** | `C → C_max` |
| **𝒮 → Ω** | `C → Ω` |
| **Производная** | `dC:d𝒮 = C_max ⊗ (D_f⊖D(D(Id))) ⊗ 𝒮^{D_f⊖D(D(Id))⊖Id}` |
| **Смысл** | Когерентность ⟹ ёмкость |
