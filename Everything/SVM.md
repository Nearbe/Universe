# Метод опорных векторов (SVM)

## Определение

Алгоритм классификации, находящий гиперплоскость, максимизирующую зазор (margin) между классами.

## Оптимизационная задача

$$\min_{w, b} \frac{1}{2}\|w\|^2 \quad \text{при} \quad y_i(w^T x_i + b) \geq 1$$

## Мягкий зазор (soft margin)

Для линейно неразделимых данных вводятся переменные ослабления $\xi_i$:
$$\min \frac{1}{2}\|w\|^2 + C \sum \xi_i, \quad \xi_i \geq 0$$

$C$ — гиперпараметр, контролирующий компромисс между шириной зазора и ошибками.

## Ядровой трюк (Kernel Trick)

Для нелинейной разделяющей поверхности данные отображаются в пространство большей размерности $\phi(x)$:
$$K(x_i, x_j) = \langle \phi(x_i), \phi(x_j) \rangle$$

Популярные ядра: RBF (гауссово), полиномиальное, сигмоидальное.

## Свойства

- Зависит только от опорных векторов (точки на границе зазора)
- Устойчив к переобучению при больших margin
- Вычислительная сложность обучения: $O(n^2)$–$O(n^3)$
