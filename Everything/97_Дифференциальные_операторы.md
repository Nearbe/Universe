# Дифференциальные операторы

Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — производные. Геометрия ([22](22_Геометрия.md)) — метрика.

## Таблица

| Оператор | Обозначение | Что делает |
|---|---|---|
| Градиент | **∇f** | Скорость изменения |
| Дивергенция | **∇·F** | Поток из точки |
| Ротор | **∇×F** | Вращение поля |
| Лапласиан | **∇²f** | Неоднородность |
| Даламбертиан | **□** | Волновой оператор |

## Физический смысл

Все операторы применяются к полям **Δ** и **Ψ**. `□Ψ = Ω` — свободное распространение вибрации.

## Полный набор операций

| Оператор | Формула |
|---|---|
| **∇f** | `(∂f:∂x_i)` — вектор скоростей |
| **∇·F** | `Σ ∂F_i:∂x_i` — скаляр потока |
| **∇×F** | Антисимметричная часть `∂_i F_j ⊖ ∂_j F_i` |
| **∇²f** | `Σ ∂²f:∂x_i²` |
| **□** | `(Id:c²)⊗∂²:∂t² ⊖ ∇²` |
