# Фазовый переход сознания

Синхронизация ([45](45_Синхронизация.md)) — **𝒮**. Температура ([40](40_Температура.md)) — `T ∝ σ`.

## Фазовые переходы Ландау

Свободная энергия:

```
F(𝒮) = F₀ ⊕ a⊗(T ⊖ T_c)⊗𝒮² ⊕ b⊗𝒮⁴
```

| Фаза | Условие | 𝒮 |
|---|---|---|
| Беспорядок | `T > T_c` | **Ω** |
| Порядок | `T < T_c` | `∝ √(T_c ⊖ T)` |

## Критические явления

Вблизи критической точки:

- **Дивергенция корреляционной длины**: `ξ ∝ \|T ⊖ T_c\|^(⊖ν)`
- **Степенной закон**: `P(size) ∝ size^(⊖α)`

## Нейронные лавины

Наблюдаемые каскады нейронной активности:

```
α ≈ D(D(Id)) ⊖ 0.5
```

Проявление критического состояния сети вблизи перехода сознания.

## Гистерезис

```
𝒮_вход(Δ) ≠ 𝒮_выход(Δ)
```

- Инерция сознания: трудность изменения паттернов
- Эффект прайминга: предыдущие стимулы влияют на последующие

## Полный набор операций

| Операция | Формула |
|---|---|
| **F(𝒮)** | `F₀ ⊕ a⊗(T⊖T_c)⊗𝒮² ⊕ b⊗𝒮⁴` |
| **Порядок** | `𝒮 ∝ √(T_c⊖T)` |
| **Корреляция** | `ξ ∝ \|T⊖T_c\|^(⊖ν)` |
| **Лавины** | `P(size) ∝ size^(⊖α)`, `α≈1.5-2.0` |
| **Гистерезис** | `𝒮_вход ≠ 𝒮_выход` |
