# Экспериментальные проявления Δ

Синхронизация ([45](45_Синхронизация.md)) — **𝒮**. Кубиты и квантовые состояния — волновая функция ([31](31_Волновая_функция.md)).

## Квантовая оптика

Squeezed состояния:

```
Δ = D(D(Id)) ⊗ r
```

где **r** — параметр сжатия. Гомодинная детекция напрямую измеряет Δ.

## Сверхпроводники

Фаза параметра порядка:

```
Δ = ln(|cot θ|)
```

Джозефсоновский эффект: `J = J_c ⊗ sin(θ)`.

## Космология

Инфлатон:

- Реальная часть — кинетическая энергия
- Мнимая часть — потенциальная энергия

Инфляция: `Δ → +Π`. Тёмная энергия: `Δ → ⊖Π`.

## Нейронауки

| Ритм | Частота | Δ |
|---|---|---|
| Гамма | 30-100 Гц | `≈ Ω` |
| Альфа | 8-12 Гц | `∈ [⊖0.2, +0.2]` |
| Дельта | 0.5-4 Гц | `< Ω` |

## Биофизика

| Система | D_f | Δ |
|---|---|---|
| Белки | `D(D(Id))-D⁵(D(Id))` | `∈ [Ω, +0.5]` |
| Нейронные сети | `≈ D⁴(Id)` | `∈ [+0.6, +D(Id)]` |

## Предсказания

- Вариация постоянных: `Ġ:G ∝ dΔ:dt`
- Отклонения от `Id:r²` на планковских расстояниях
- Критические явления при `𝒮 ≈ 0.7`
