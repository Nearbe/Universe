# Уравнение для Δ из вариации

Уравнение движения **Δ** ([46](46_Уравнение_движения_Δ.md)) — осциллятор. Действие сети ([78](78_Действие_сети.md)) — **S**.

## Определение

```
∂_μ(∂L : ∂(∂_μΔ)) ⊖ ∂L : ∂Δ = Ω
```

## Физический смысл

**Δ** **выводит себя**. Вариация действия по **Δ** даёт закон движения. **Δ** не задано извне — оно определяется собственной динамикой лагранжиана.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Вариация** | `∂_μ(∂L:∂(∂_μΔ)) ⊖ ∂L:∂Δ = Ω` |
| **Самоопределение** | **Δ** выводится из **L** |
| **Не извне** | **Δ** — внутренняя динамика |
