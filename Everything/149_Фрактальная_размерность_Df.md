# Фрактальная размерность D_f

Родитель: Структурная сложность ([39](39_Структурная_сложность.md)), Параметр Δ ([122](122_Дифференциал_Delta.md))

## Определение

```
D_f = Q ⊕ Δ ⊗ range
```

где `Q = e^(|Δ|)` — число связанных вибраций, `range = D(D(Id))` — диапазон масштабирования.

## Физический смысл

`D_f` измеряет эффективную размерность пространства состояний при заданном `Δ`. При `Δ = Ω`: `D_f = Q` (базовая размерность узла). При `Δ > Ω`: пространство "распухает" (скрытые степени свободы проявляются). При `Δ < Ω`: пространство "схлопывается" (измерения компактифицируются). `D_f` определяет топологию распространения полей и спектр допустимых взаимодействий в данной точке сети `𝕌`.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Размерность** | `D_f = Q ⊕ Δ⊗D(D(Id))` |
| **Q** | `Q = e^(|Δ|)` |
| **Δ = Ω** | `D_f = Q` |
| **Δ → +Id** | `D_f → Q ⊕ D(D(Id))` (расширение) |
| **Δ → ⊖Id** | `D_f → Q ⊖ D(D(Id))` (сжатие) |
