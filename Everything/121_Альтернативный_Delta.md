# Альтернативный параметр Δ

Деление ([03](03_Деление.md)) — режимы. Комплексные числа ([09](09_Комплексные.md)) — Re/Im.

## Определение

```
Δ_alg = (M ⊖ Id) : (M ⊕ Id)
```

где `M = |Re| : |Im|`.

## Связь с логарифмическим Δ

```
Δ_alg = tanh(Δ_log : D(Id))
Δ_log = D(Id) ⊗ atanh(Δ_alg)
```

## Диапазоны

| Форма | Диапазон | Когда |
|---|---|---|
| **Δ_log** | `ℝ ∪ {±Π}` | Геометрия, сингулярности |
| **Δ_alg** | `[⊖Id, +Id]` | Масса, эволюция сети |

## Физический смысл

`Δ_alg` — **сжатая версия Δ**. Логарифмический уходит в `±Π`, алгебраический ограничен. Один параметр, две «линзы».

## Полный набор операций

| Операция | Формула |
|---|---|
| **Алгебр.** | `Δ_alg = (M⊖Id):(M⊕Id)` |
| **Логарифмич.** | `Δ_log = D(Id)⊗atanh(Δ_alg)` |
| **Обратная** | `Δ_alg = tanh(Δ_log:D(Id))` |
| **M** | `M = |Re|:|Im|` |
