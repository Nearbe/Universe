# Оператор

## Определение

Оператор — отображение из одного векторного пространства в другое (часто в себя).

## Типы операторов

- **Линейный:** $T(ax + by) = aT(x) + bT(y)$
- **Самосопряжённый:** $T = T^*$
- **Унитарный:** $T^*T = TT^* = I$
- **Компактный:** переводит ограниченные множества в относительно компактные
- **Проекционный:** $P^2 = P$

## Основные операторы в анализе

- **Дифференциальный:** $D = \frac{d}{dx}$
- **Лапласиан:** $\Delta = \nabla^2$
- **Интегральный:** $(Kf)(x) = \int k(x, y)f(y)\,dy$
- **Сдвига:** $(Tf)(x) = f(x - a)$

## Норма оператора

$$\|T\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|}$$

## Спектр

Множество $\lambda \in \mathbb{C}$, для которых $T - \lambda I$ не имеет ограниченного обратного.
