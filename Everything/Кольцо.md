# Кольцо

## Определение

Кольцо $(R, +, \cdot)$ — множество с двумя операциями:

1. $(R, +)$ — абелева группа
2. Умножение ассоциативно: $(ab)c = a(bc)$
3. Дистрибутивность: $a(b+c) = ab + ac$, $(a+b)c = ac + bc$

## Дополнительные свойства

- **Единица:** существует $1$: $1 \cdot a = a \cdot 1 = a$
- **Коммутативность:** $ab = ba$
- **Область целостности:** коммутативное кольцо с 1, без делителей нуля
- **Поле:** коммутативное кольцо, где каждый ненулевой элемент обратим

## Примеры

- $\mathbb{Z}$ — целые числа (коммутативное кольцо с 1)
- $\mathbb{Q}, \mathbb{R}, \mathbb{C}$ — поля
- $M_n(\mathbb{R})$ — матрицы (некоммутативное кольцо)
- $\mathbb{Z}_n$ — целые по модулю $n$

## Идеал

Подмножество $I \subset R$:
- $a, b \in I \implies a+b \in I$
- $a \in I, r \in R \implies ra \in I$

## Факторкольцо

$$R/I = \{a + I \mid a \in R\}$$

Операции: $(a+I) + (b+I) = (a+b)+I$, $(a+I)(b+I) = ab+I$.
