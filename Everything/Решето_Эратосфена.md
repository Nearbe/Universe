# Решето Эратосфена

## Определение

Алгоритм нахождения всех простых чисел до заданного $N$.

## Алгоритм

```
sieve(N):
    is_prime[2..N] = true
    for p = 2 to √N:
        if is_prime[p]:
            for k = p² to N step p:
                is_prime[k] = false
    return все p, где is_prime[p]
```

## Сложность

- **Время:** $O(N \log \log N)$
- **Память:** $O(N)$ бит

## Модификации

**Решето Аткина:** $O(N / \log \log N)$ — быстрее, но сложнее.

**Сегментированное решето:** $O(\sqrt{N})$ памяти, обрабатывает блоки.

## Связь с теорией чисел

- Принцип включения-исключения → формула Лежандра для $\pi(x)$
- Обобщается на решето Бруна, решето Сельберга, большое решето
- Оценка остаточного члена в $\pi(x)$ через анализ решёт

## Применения

Генерация таблиц простых, криптография (поиск простых для RSA), тестирование алгоритмов.
