# Лагранжиан

Второй закон Ньютона ([49](49_Второй_закон_Ньютона.md)) — классический предел. Теперь — откуда он следует.

## Определение

Для частицы:

```
L = ½ ⊗ m ⊗ v² ⊖ V(x)
```

Для поля **Ψ**:

```
L = ½ ⊗ ∂_μΨ† ⊗ ∂^μΨ ⊖ V(Ψ†⊗Ψ)
```

## Физический смысл

Лагранжиан — **разница между кинетическим и потенциальным**. Не сумма — разница.

**L** — «недовольство» системы: чем больше **L**, тем сильнее система стремится измениться. Природа «выбирает» путь, где эта разница минимальна.

Уравнение Ньютона — следствие из минимизации действия `∫ L dt`. Не фундаментальный закон, а вариационный принцип.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Частица** | `L = ½⊗m⊗v² ⊖ V(x)` |
| **Поле** | `L = ½⊗∂_μΨ†⊗∂^μΨ ⊖ V(Ψ†⊗Ψ)` |
| **Действие** | `S = ∫ L dt` |
| **Минимизация** | `δS = Ω ⟹` Эйлера-Лагранжа |
| **Из Э-Л** | `d/dt(∂L/∂v) ⊖ ∂L/∂x = Ω` |
