# Ранняя Вселенная и инфляция

Родитель: Космология ([163](163_Космология.md)), Уравнение движения Δ ([46](46_Уравнение_движения_Δ.md))

## Поле Δ как инфлатон

```
V(Δ) = D(D(Id)) ⊗ m_Δ² ⊗ Δ²
```

## Уравнение движения

```
∂_t²Δ ⊕ D(D(D(Id)))⊗H⊗∂_tΔ ⊕ m_Δ²⊗Δ = Ω
```

`D(D(D(Id)))⊗H⊗∂_tΔ` — трение от расширения Вселенной.

## Режим медленного скольжения

```
D(D(D(Id)))⊗H⊗∂_tΔ ≈ ⊖m_Δ²⊗Δ
a(t) ∝ exp(H⊗t)
```

Число экспансий: `N_exp ≳ 60`.

## Квантовые флуктуации

```
P_ζ ≈ 2.1×10⁻⁹

```
