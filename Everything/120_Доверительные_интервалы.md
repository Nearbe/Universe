# Доверительные интервалы

Вероятность ([27](27_Вероятность.md)) — **σ**. Глобальный параметр ([52](52_Глобальный_параметр.md)) — `⟨Δ⟩`.

## Определение

Для среднего при известном **σ**:

```
⟨x⟩ ± z_(α:D(Id)) ⊗ σ : √N
```

`z_(Id:D(Id)) ≈ 1.96` для 95% доверия.

## Физический смысл

Оценка погрешности `⟨Δ⟩`: «истинное среднее отклонение лежит в интервале с долей 95%».

## Полный набор операций

| Операция | Формула |
|---|---|
| **Интервал** | `⟨x⟩ ± z_(α:D(Id))⊗σ:√N` |
| **95%** | `z ≈ 1.96` |
| **99%** | `z ≈ 2.58` |
| **Смысл** | Истинное значение в интервале с заданной долей |
