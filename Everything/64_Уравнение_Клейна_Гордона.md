# Уравнение Клейна-Гордона

Свет ([36](36_Свет.md)) — `□A^μ = Ω`. Звук ([37](37_Звук.md)) — `□p = Ω`. Теперь — массивное скалярное поле.

## Определение

```
(□ ⊕ m²) ⊗ φ = Ω
```

Дискретная форма (такт **n**):

```
(φ(n⊕Id) ⊖ D(φ(n)) ⊕ φ(n⊖Id)) : τ² ⊖ ∇²φ(n) ⊕ m²⊗φ(n) = Ω
```

## Физический смысл

Клейн-Гордон — **скалярная вибрация**. Простейшее волновое уравнение с массой.

**φ** — амплитуда возмущения, **m²** — «жёсткость» связи. Описывает распространение возмущений **Δ** в сети.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Непрерывное** | `(□ ⊕ m²)⊗φ = Ω` |
| **Дискретное** | `(φ(n+1)−2φ(n)+φ(n−1)):τ² − ∇²φ + m²φ = Ω` |
| **Без массы** | `m = Ω ⟹ □φ = Ω` (свет) |
| **С массой** | `m ≠ Ω ⟹` массивная волна |
