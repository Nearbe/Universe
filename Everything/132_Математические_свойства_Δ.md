# Математические свойства Δ

Комплексные числа ([09](09_Комплексные.md)) — фаза **θ**. Дифференциал Δ ([122](122_Дифференциал_Delta.md)) — производные.

## Связь с фазой

```
Δ = ln(|cot θ|)
θ = arctan(e^(⊖Δ))
```

**Δ зависит только от фазы θ, не от амплитуды \|Ψ\|.** Это мера относительного баланса между компонентами.

## Преобразования

| Преобразование | Эффект на Δ |
|---|---|
| Масштабирование `Ψ → k⊗Ψ` | Не меняется |
| Инверсия `Ψ → Id:Ψ` | Меняет знак |
| Сопряжение `Ψ → Ψ̄` | Не меняется |

## Производные

```
dΔ = dRe:Re ⊖ dIm:Im
∂Δ:∂Re = Id:Re
∂Δ:∂Im = ⊖Id:Im
```

Δ чувствителен к **относительным изменениям**, не абсолютным.

## Периодичность

```
Δ(θ ⊕ π) = Δ(θ)
```

Период **π** по фазе.

## Физический смысл

- `Δ → +Π`: `f → Ω` (время замедляется)
- `Δ = Ω`: `f = Id` (баланс)
- `Δ → ⊖Π`: `f → Π` (время ускоряется)

## Полный набор операций

| Операция | Формула |
|---|---|
| **Δ от θ** | `Δ = ln(\|cot θ\|)` |
| **θ от Δ** | `θ = arctan(e^(⊖Δ))` |
| **Период** | `Δ(θ+π) = Δ(θ)` |
| **Масштаб** | `Ψ→k⊗Ψ ⟹ Δ` не меняется |
| **Инверсия** | `Ψ→Id:Ψ ⟹ Δ→⊖Δ` |
