# Граничные условия

Действие сети ([78](78_Действие_сети.md)) — **S**. Интегральное исчисление ([21](21_Интегральное_исчисление.md)) — вариация.

## Определение

```
δq(t₁) = Ω,    δq(t₂) = Ω
```

## Физический смысл

Граничные условия — **концы пути зафиксированы**. Вариация ищет оптимальный путь между двумя точками, не трогая концы.

Начальное и конечное состояния заданы — вариация ищет, как сеть «думает» между ними.

## Полный набор операций

| Тип | Формула | Смысл |
|---|---|---|
| **Дирихле** | `q(t₁)=Ω, q(t₂)=Ω` | Значения зафиксированы |
| **Неймана** | `∂q:∂n = Ω` | Производная зафиксирована |
| **Смешанные** | Комбинация | Частичная фиксация |
