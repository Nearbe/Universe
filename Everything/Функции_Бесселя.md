# Функции Бесселя

## Определение

Функции Бесселя — решения дифференциального уравнения Бесселя:

$$x^2 y'' + x y' + (x^2 - n^2)y = 0$$

## Виды

**Функция Бесселя первого рода:**
$$J_n(x) = \sum_{k=0}^{\infty} \frac{(-1)^k}{k! \, \Gamma(k+n+1)} \left(\frac{x}{2}\right)^{2k+n}$$

**Функция Бесселя второго рода:**
$Y_n(x)$ — линейно независимое решение, сингулярное в $x = 0$.

**Модифицированные функции:**
$I_n(x)$, $K_n(x)$ — решения уравнения с $-x^2$ вместо $x^2$.

## Свойства

- Рекуррентные соотношения:
  $$J_{n-1}(x) + J_{n+1}(x) = \frac{2n}{x} J_n(x)$$
- Ортогональность: $\int_0^1 x J_n(\alpha_{nm} x) J_n(\alpha_{nk} x) \, dx = 0$ при $m \neq k$
- Асимптотика при больших $x$: $J_n(x) \sim \sqrt{\frac{2}{\pi x}} \cos\left(x - \frac{n\pi}{2} - \frac{\pi}{4}\right)$

## Применения

- Задачи с цилиндрической симметрией
- Распространение волн в цилиндрических волноводах
- Тепловые процессы в цилиндрах
- Акустика мембран
