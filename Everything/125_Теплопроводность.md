# Теплопроводность

Температура ([40](40_Температура.md)) — `T ∝ Id:σ`. Огонь ([48](48_Огонь.md)) — переход.

## Определение

```
∂T : ∂t = α ⊗ ∇²T
```

где **α** — коэффициент температуропроводности.

## Физический смысл

Теплопроводность — **выравнивание рассинхронизации**.

Тёплое = хаотичное (высокий **σ**). Холодное = упорядоченное (низкий **σ**). Тепло течёт от горячего к холодному, потому что элементы «учат» соседей своему состоянию.

Тепло ≠ огонь. Тепло — статистика (**σ**). Огонь — процесс (переход **σ** от низкого к высокому).

| **Тёплое** | `σ` высокое |
| **Холодное** | `σ` низкое |
| **Поток** | `q = ⊖k⊗∇T` |
