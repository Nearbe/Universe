# Уравнение непрерывности

Поток информации ([41](41_Поток_информации.md)) — `j = (ħ:m) ⊗ ρ ⊗ ∇θ`. Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — дивергенция. Теперь — сохранение.

## Определение

```
∂ρ/∂t ⊕ ∇·j = Ω
```

Дискретная форма (такт **n**):

```
ρ(n⊕Id) = ρ(n) ⊖ τ ⊗ ∇·j(n)
```

## Физический смысл

Непрерывность — **закон сохранения смысла**. Информация не возникает из ничего и не исчезает — она перетекает.

- `∇·j > Ω`: информация «утекает» из точки
- `∇·j < Ω`: информация «накапливается»

## Полный набор операций

| Операция | Формула |
|---|---|
| **Непрерывное** | `∂ρ/∂t ⊕ ∇·j = Ω` |
| **Дискретное** | `ρ(n+1) = ρ(n) ⊖ τ⊗∇·j(n)` |
| **Утечка** | `∇·j > Ω` |
| **Накопление** | `∇·j < Ω` |
| **Стационарное** | `∇·j = Ω ⟹ ∂ρ/∂t = Ω` |
