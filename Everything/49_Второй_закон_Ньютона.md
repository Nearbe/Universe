# Второй закон Ньютона

Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — производные. Уравнение Шрёдингера ([44](44_Уравнение_Шрёдингера.md)) — полная эволюция. Теперь — классический предел.

## Определение

```
m ⊗ a = ⊖∇V
```

## Физический смысл

Ньютон — **классический предел**. Когда `Δ ≈ Ω` и квантовый потенциал `Q ≈ Ω`, сеть ведёт себя как классическая механика.

`m ⊗ a = ⊖∇V` — частный случай уравнения Эйлера-Лагранжа. Не фундаментальный закон, а **предельный случай** полной эволюции **Ψ**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Определение** | `m⊗a = ⊖∇V` |
| **Ускорение** | `a = ∂²x/∂t²` |
| **Класс. предел** | `Q → Ω, Δ → Ω` |
| **Из Лагранжиана** | `d/dt(∂L/∂v) ⊖ ∂L/∂x = Ω` |
| **Из Шрёдингера** | `ħ → Ω ⟹` Ньютон |
