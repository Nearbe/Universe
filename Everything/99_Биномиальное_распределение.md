# Биномиальное распределение

Комбинаторика ([26](26_Комбинаторика.md)) — **C(n,k)**. Вероятность ([27](27_Вероятность.md)) — **p**.

## Определение

```
P(k) = C(n,k) ⊗ p^k ⊗ (Id ⊖ p)^(n⊖k)
```

```
⟨k⟩ = n ⊗ p,    σ² = n ⊗ p ⊗ (Id ⊖ p)
```

## Физический смысл

Оценка числа синхронизированных элементов в сети из **n** при доле синхронизации **p**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Вероятность** | `P(k) = C(n,k)⊗p^k⊗(Id⊖p)^(n⊖k)` |
| **Среднее** | `⟨k⟩ = n⊗p` |
| **Дисперсия** | `σ² = n⊗p⊗(Id⊖p)` |
| **При p = Id:D(Id)** | Симметричное распределение |
