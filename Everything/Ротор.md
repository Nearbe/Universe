# Ротор (вихрь)

## Определение

Ротор векторного поля $\mathbf{F} = (F_1, F_2, F_3)$:

$$\text{rot}\, \mathbf{F} = \nabla \times \mathbf{F} = \left(\frac{\partial F_3}{\partial y} - \frac{\partial F_2}{\partial z}, \frac{\partial F_1}{\partial z} - \frac{\partial F_3}{\partial x}, \frac{\partial F_2}{\partial x} - \frac{\partial F_1}{\partial y}\right)$$

## Физический смысл

Ротор характеризует локальное вращение поля. Вектор ротора направлен по оси вращения, модуль — удвоенная угловая скорость.

## Теорема Стокса

$$\iint_S (\nabla \times \mathbf{F}) \cdot d\mathbf{S} = \oint_{\partial S} \mathbf{F} \cdot d\mathbf{r}$$

Циркуляция по замкнутому контуру = поток ротора через поверхность.

## Свойства

- $\text{rot}(\nabla f) = 0$ (ротор градиента = 0)
- $\text{div}(\text{rot}\, \mathbf{F}) = 0$ (дивергенция ротора = 0)
- $\text{rot}(f\mathbf{F}) = \nabla f \times \mathbf{F} + f\,\text{rot}\,\mathbf{F}$

## В уравнениях Максвелла

$$\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t}$$
$$\nabla \times \mathbf{B} = \mu_0 \mathbf{J} + \mu_0 \varepsilon_0 \frac{\partial \mathbf{E}}{\partial t}$$
