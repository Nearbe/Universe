# Эффективный лагранжиан

Родитель: Лагранжиан ([50](50_Лагранжиан.md)), Ренормализация ([28](28_Масштабная_инвариантность.md))

## Определение

```
L_eff = L_0 ⊕ Σ_{i} (c_i : Λ^{d_i⊖D(D(Id))}) ⊗ O_i
```

где `Λ` — масштаб обрезания (глубина Хребта), `O_i` — операторы высшей размерности, `c_i` — безразмерные коэффициенты связи.

## Физический смысл

Низкоэнергетическое описание системы `𝕌`. На масштабах `E ≪ Λ` вклад операторов `O_i` подавляется степенным фактором `(E:Λ)^{d_i}`. Это позволяет аппроксимировать полную динамику конечным числом степеней. При `E → Λ` ряд расходается — требуется полная теория (глубина `∞`). Эффективный лагранжиан — математический инструмент работы на конечной глубине фрактала.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Разложение** | `L_eff = L_0 ⊕ Σ(c_i:Λ^{d_i⊖D(D(Id))})⊗O_i` |
| **Подавление** | `O_i ∝ (E:Λ)^{d_i⊖D(D(Id))}` |
| **Предел E≪Λ** | `L_eff ≈ L_0` |
| **Предел E→Λ** | `Σ → Π` (требуется полная теория) |
| **Смысл** | Аппроксимация на конечной глубине |
