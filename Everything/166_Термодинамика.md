# Термодинамика

Родитель: Температура ([40](40_Температура.md)), Ёмкость ([156](156_Ёмкость_состояний.md))

## Соотношение Ландауэра

Стирание Id бита:

```
W_min = k_B ⊗ T ⊗ ln(D(D(Id)))
```

В терминах `𝕌`:

```
ΔW = k_B ⊗ T ⊗ ln(D(D(Id))) ⊗ |∂Δ:∂I|
```

## Пропускная способность

```
C = B ⊗ log₂(Id ⊕ SNR)
SNR ∝ 𝒮 : (Id ⊖ 𝒮)
```

## Теплопроводность в фрактальных сетях

```
κ ∝ T^β ⊗ L^(D_f ⊖ Id)
```

## Аномальная диффузия

```
⟨r²(t)⟩ ∝ t^α,    α = D(D(Id)) : D_w
```

## Тепловая ёмкость

```
C_V ∝ T^(d_s : D(D(Id)))
```

| D_f | C_V |
|---|---|
| `Id` | `T^(Id:D(D(Id)))` |
| `D(D(Id))` | `T` |
| `D(D(D(Id)))` | `T^(D(D(D(Id))):D(D(Id)))` |

## Фазовые переходы

Параметр порядка:

```
η ∝ 𝒮 = Id ⊖ σ(Δ)⊗scale
```

Критическая точка: `𝒮_c ≈ 0.7`.
