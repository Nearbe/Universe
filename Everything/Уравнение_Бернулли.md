# Уравнение Бернулли

## Определение

Дифференциальное уравнение вида:

$$\frac{dy}{dx} + P(x)y = Q(x)y^n$$

где $n \neq 0, 1$ — действительное число.

## Метод решения

Подстановка $z = y^{1-n}$ приводит к линейному уравнению:

$$\frac{dz}{dx} + (1-n)P(x)z = (1-n)Q(x)$$

## Общее решение

$$y(x) = \left[Ce^{-(1-n)\int P(x)dx} + (1-n)\int Q(x)e^{(1-n)\int P(x)dx} dx\right]^{\frac{1}{1-n}}$$

## Частные случаи

- При $n = 0$: линейное неоднородное уравнение
- При $n = 1$: уравнение с разделяющимися переменными

## Применения

- Гидродинамика (уравнение Бернулли для идеальной жидкости)
- Популяционная динамика
- Финансовая математика
