# Метод максимального правдоподобия

Вероятность ([27](27_Вероятность.md)) — мера. Глобальный параметр ([52](52_Глобальный_параметр.md)) — `⟨Δ⟩`.

## Определение

```
θ̂ = argmax L(θ),    где L(θ) = P(data|θ)
```

Для гауссовского: `μ̂ = (Id : N) ⊗ Σ x_i`.

## Физический смысл

Оценка `⟨Δ⟩` и `σ(Δ)` из данных сети. Параметры выбираются так, чтобы данные были наиболее вероятны.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Правдоподобие** | `L(θ) = P(data\|θ)` |
| **Оценка** | `θ̂ = argmax L(θ)` |
| **Гауссовское μ** | `μ̂ = (Id:N)⊗Σx_i` |
| **Смысл** | Параметры ⟹ данные наиболее вероятны |
