# Гипотеза Ходжа

## Формулировка

Для гладкого проективного комплексного многообразия $X$ каждый класс Ходжа типа $(p, p)$ является рациональной линейной комбинацией классов алгебраических циклов.

## Определения

**Когомологии Де Рама:** $H^k(X, \mathbb{C})$ — классы замкнутых дифференциальных форм.

**Разложение Ходжа:**
$$H^k(X, \mathbb{C}) = \bigoplus_{p+q=k} H^{p,q}(X)$$

**Класс Ходжа:** элемент $H^{p,p}(X) \cap H^{2p}(X, \mathbb{Q})$.

**Алгебраический цикл:** линейная комбинация подмногообразий размерности $\dim X - p$.

## Статус

Одна из 7 задач тысячелетия. Доказана для $p = 1$ (теорема Лефшеца). Открыта для $p \geq 2$.

## Значение

Связывает топологию (когомологии) с алгебраической геометрией (подмногообразия).
