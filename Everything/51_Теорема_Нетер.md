# Теорема Нётер

Лагранжиан ([50](50_Лагранжиан.md)) — разница кинетического и потенциального. Теперь — откуда берутся законы сохранения.

## Формулировка

Каждой непрерывной симметрии лагранжиана соответствует закон сохранения.

| Симметрия | Сохраняющаяся величина |
|---|---|
| Сдвиг во времени | Энергия (**E**) |
| Сдвиг в пространстве | Импульс (**p**) |
| Вращение | Момент импульса (**L**) |
| **U(1)** калибровочная | Заряд (**q**) |

## Физический смысл

Нётер — **симметрия порождает сохранение**. Каждое «ничего не меняется» даёт «что-то сохраняется».

В системе **𝕌**: каждая симметрия сети порождает сохраняющуюся величину. Энергия, импульс, заряд — не «фундаментальные законы», а **следствия симметрий** лагранжиана.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Временной сдвиг** | `∂L/∂t = Ω ⟹ E = const` |
| **Пространств. сдвиг** | `∂L/∂x = Ω ⟹ p = const` |
| **Вращение** | `∂L/∂θ = Ω ⟹ L = const` |
| **U(1)** | `Ψ → e^(i·α)⊗Ψ ⟹ q = const` |
