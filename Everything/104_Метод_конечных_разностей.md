# Метод конечных разностей

Линейные уравнения ([17](17_Линейные_уравнения.md)) — дискретные шаги. Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — производные.

## Формулы

Первая производная:

```
f'(x) ≈ (f(x⊕h) ⊖ f(x⊖h)) : D(h)
```

Вторая производная:

```
f''(x) ≈ (f(x⊕h) ⊖ D(f(x)) ⊕ f(x⊖h)) : h²
```

## Физический смысл

Дискретизация лапласиана:

```
∇²ψ ≈ (ψ_{i⊕Id} ⊖ D(ψ_i) ⊕ ψ_{i⊖Id}) : h²
```

Точность: `O(h²)`. Дискретная форма всех уравнений поля.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Первая** | `f'(x) ≈ (f(x+h)⊖f(x−h)):D(h)` |
| **Вторая** | `f''(x) ≈ (f(x+h)⊖D(f(x))+f(x−h)):h²` |
| **Лапласиан** | `∇²ψ ≈ (ψ_{i+1}⊖D(ψ_i)+ψ_{i−1}):h²` |
