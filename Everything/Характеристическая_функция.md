# Характеристическая функция

## Определение

Фурье-образ распределения вероятностей:

$$\varphi_X(t) = \mathbb{E}[e^{itX}] = \int_{-\infty}^{\infty} e^{itx} \, dF_X(x)$$

## Свойства

- $\varphi(0) = 1$
- $|\varphi(t)| \leq 1$
- $\varphi(-t) = \overline{\varphi(t)}$
- Равномерно непрерывна
- Взаимно однозначно соответствует функции распределения

## Связь с моментами

$$\mathbb{E}[X^n] = \frac{1}{i^n} \varphi^{(n)}(0)$$

## Для суммы независимых

$$\varphi_{X+Y}(t) = \varphi_X(t) \cdot \varphi_Y(t)$$

## Примеры

| Распределение | $\varphi(t)$ |
|---|---|
| Нормальное($\mu, \sigma^2$) | $e^{i\mu t - \sigma^2 t^2/2}$ |
| Коши | $e^{-|t|}$ |
| Пуассона($\lambda$) | $e^{\lambda(e^{it}-1)}$ |
| Равномерное($a, b$) | $\frac{e^{itb} - e^{ita}}{it(b-a)}$ |

## Теорема обращения

$$f(x) = \frac{1}{2\pi} \int_{-\infty}^{\infty} e^{-itx} \varphi(t) \, dt$$

## Применения

- Доказательство ЦПТ
- Теория устойчивости распределений
- Статистика (оценка параметров)
