## Экономика

Рынки — коллективное ⟨Δ⟩ участников:

| Состояние | Δ | Смысл |
|---|---|---|
| Рост | > 0 | Оптимизм |
| Спад | < 0 | Пессимизм |
| Кризис | Фазовый переход | 𝒮 → 0.7 |

## Игры

- **Сотрудничество**: синхронизированные Δ (высокий 𝒮)
- **Предательство**: рассинхронизированные Δ (низкий 𝒮)
- **Равновесие Нэша**: устойчивая конфигурация Δ

## Психология

| Состояние | Δ | 𝒮 |
|---|---|---|
| Фокус | ≈ 0 | > 0.9 |
| Стресс | > 0.5 | переменная |
| Депрессия | < −0.5 | низкая |
| Мания | > 0.5 | высокая |

## Власть

Власть — способность модулировать Δ других: `∂Δ/∂t`.

- **Демократия**: усреднение ⟨Δ⟩ участников
- **Авторитаризм**: централизованное определение Δ лидером
