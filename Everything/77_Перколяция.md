# Перколяция

Синхронизация ([45](45_Синхронизация.md)) — порог `𝒮 ≈ 0.7`. Вероятность ([27](27_Вероятность.md)) — пороги.

## Порог перколяции

Для квадратной решётки: `p_c ≈ 0.5927`

## Физический смысл

Перколяция — **момент пробоя**. До `p_c`: кластеры изолированы. После `p_c`: гигантский кластер связывает всю систему.

Это момент, когда сеть «просыпается» — возникает глобальная синхронизация.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Порог** | `p_c ≈ 0.5927` (квадратная) |
| **До порога** | Кластеры изолированы |
| **После порога** | Гигантский кластер |
| **Связь с 𝒮** | `p_c` ⟹ `𝒮 ≥ 𝒮_c` |
