# Качество тройки (abc)

## Определение

Для взаимно простой тройки $a + b = c$:
$$q(a, b, c) = \frac{\ln c}{\ln \text{rad}(abc)}$$

## Интерпретация

- $q > 1$: $c$ больше радикала — «качественная» тройка
- $q \leq 1$: «обычная» тройка
- Гипотеза abc эквивалентна: $\sup q(a,b,c) \leq 1$ (с поправкой на $\varepsilon$)

## Известные рекорды

Наибольшее известное качество:
$$q \approx 1.6299 \quad \text{(для } a=2, b=3^{10}\cdot 109, c=23^5\text{)}$$

## Связь с гипотезой abc

Эквивалентная формулировка: для любого $\varepsilon > 0$ существует лишь конечное число троек с $q > 1 + \varepsilon$.

## Следствия гипотезы abc

Если $q$ ограничена:
- Теорема Ферма для достаточно больших $n$
- Гипотеза Морделла (теорема Фальтингса)
- Гипотеза Холла
- Конечность числа решений многих диофантовых уравнений
