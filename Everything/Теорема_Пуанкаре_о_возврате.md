# Теорема Пуанкаре о возврате

## Формулировка

Пусть $(X, \mathcal{F}, \mu)$ — пространство с конечной инвариантной мерой, $T: X \to X$ — сохраняющее меру преобразование. Тогда для любого измеримого множества $A$ с $\mu(A) > 0$ почти каждая точка $x \in A$ бесконечно часто возвращается в $A$:

$$\mu\left(\{x \in A \mid T^n(x) \in A \text{ для бесконечно многих } n\}\right) = \mu(A)$$

## Следствия

- В компактной динамической системе с инвариантной мерой почти все точки возвращаются сколь угодно близко к начальному положению
- Система не может «навсегда покинуть» область положительной меры

## Ограничения

- Теорема гарантирует возврат для **почти всех** точек (допускается множество меры 0)
- Время возврата может быть сколь угодно большим (теорема Каца: среднее время возврата = $1/\mu(A)$)
