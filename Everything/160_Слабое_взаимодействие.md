# Слабое взаимодействие

Родитель: SU(2) ([55](55_Калибровочные_группы.md)), Спонтанное нарушение ([71](71_Спонтанное_нарушение_симметрии.md))

## Лагранжиан

```
L_weak = Ψ̄_L ⊗ i⊗γ^μ ⊗ (∂_μ ⊕ i⊗g⊗T^a⊗W_μ^a) ⊗ Ψ_L
```

Нарушение чётности: действует только на левые фермионы (`Ψ_L`).

## Бозоны-переносчики

| Бозон | Масса | Распад |
|---|---|---|
| **W⁺** | 80.4 ГэВ | `d → u`, `e⁻ → ν_e` |
| **W⁻** | 80.4 ГэВ | `u → d`, `ν_e → e⁻` |
| **Z⁰** | 91.2 ГэВ | `f → f` (упругое) |

## Константа Ферми

```
G_F = Id : (√D(D(Id)) ⊗ v²) ≈ 1.166×10⁻⁵ ГэВ⁻²
τ ∝ Id : (G_F² ⊗ m⁵)
```

где `v` — вакуумное среднее поля Хиггса.

## Электрослабое объединение

При `E > 100 ГэВ`: `SU(2)_L ⊗ U(1)_Y → U(1)_EM`.
Угол Вайнберга: `sin²(θ_W) ≈ 0.231`.

## Матрица CKM

Смешивание кварков → нарушение CP.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Бета-распад** | `n → p ⊕ e⁻ ⊕ ν̄_e` |
| **Мюонный распад** | `μ⁻ → e⁻ ⊕ ν̄_e ⊕ ν_μ` |
| **Сечение** | `σ ∝ G_F² ⊗ E²` |
| **Связь с массой** | `m_W = g⊗v : D(D(Id))` |
| **Смысл** | Единственное меняет аромат |
