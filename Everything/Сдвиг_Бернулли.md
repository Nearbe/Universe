# Сдвиг Бернулли

## Определение

Сдвиг Бернулли — динамическая система на пространстве бесконечных последовательностей.

На пространстве $\{0, 1\}^\mathbb{N}$ с мерой продукта $(1/2, 1/2)$:
$$T(\omega_0, \omega_1, \omega_2, \ldots) = (\omega_1, \omega_2, \omega_3, \ldots)$$

На отрезке $[0, 1]$ с мерой Лебега:
$$T(x) = 2x \bmod 1$$

## Свойства

- **Эргодичность:** среднее по времени = среднее по пространству
- **Перемешивание (mixing):** для любых измеримых $A, B$:
  $$\mu(T^{-n}(A) \cap B) \to \mu(A)\mu(B)$$
- **Энтропия:** $h = \ln 2$
- **Плотность периодических орбит**

## Связь с двоичной записью

Сдвиг Бернулли $x \mapsto 2x \bmod 1$ соответствует сдвигу влево в двоичной записи:
$$x = 0.\omega_0\omega_1\omega_2\ldots \quad \xrightarrow{T} \quad 0.\omega_1\omega_2\omega_3\ldots$$
