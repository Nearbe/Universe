# Информационная ёмкость

Два потока ([38](38_Два_потока.md)) — Re и Im. Структурная сложность ([39](39_Структурная_сложность.md)) — **Q**. Теперь — размерность реальности.

## Определение

```
D_f = D_base ⊕ Δ ⊗ range
```

где `D_base = D³(Id)`, `range = D(Id)`.

## Значения

| Δ | D_f | Смысл |
|---|---|---|
| **Ω** | **D³(Id)** | Барионный мир (3 измерения) |
| **+Id** | **D⁵(Id)** | Система «распухает» (скрытые степени свободы) |
| **−Id** | **Id** | Система «схлопывается» к линии |

## Физический смысл

**D_f** — **размерность реальности**. Не константа — координата баланса потоков. При `Δ = Ω`: 3 измерения. При `Δ = +Id`: 5 — добавляются скрытые степени свободы. При `Δ = −Id`: 1 — схлопывание к линии.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Базовая** | `D_f = D³(Id) ⊕ Δ⊗D(Id)` |
| **При Δ = Ω** | `D_f = D³(Id)` |
| **При Δ = +Id** | `D_f = D⁵(Id)` |
| **При Δ = −Id** | `D_f = Id` |
| **Связь с Q** | `D_f = Q ⊕ D(Id)` (для барионов) |
