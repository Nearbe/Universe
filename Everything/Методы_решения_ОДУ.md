# Методы решения ОДУ

## Задача Коши

$$\frac{dy}{dt} = f(t, y), \quad y(t_0) = y_0$$

## Метод Эйлера

$$y_{n+1} = y_n + h f(t_n, y_n)$$

Точность: $O(h)$. Простейший, но неустойчивый.

## Методы Рунге-Кутты

**Классический RK4:**
$$k_1 = f(t_n, y_n)$$
$$k_2 = f(t_n + h/2, y_n + hk_1/2)$$
$$k_3 = f(t_n + h/2, y_n + hk_2/2)$$
$$k_4 = f(t_n + h, y_n + hk_3)$$
$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$

Точность: $O(h^4)$.

## Методы Адамса

Многошаговые методы: используют несколько предыдущих точек.
- **Адамс-Башфорт (явный):** прогнозирует
- **Адамс-Мултон (неявный):** корректирует

## Жёсткие системы

Требуют неявных методов (обратный Эйлер, BDF) из-за сильной разницы временных масштабов.
