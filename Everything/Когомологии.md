# Когомологии

## Определение

Когомологии — алгебраические инварианты топологического пространства, двойственные гомологиям.

## Когомологии Де Рама

Для гладкого многообразия $M$:
$$H^k_{dR}(M) = \frac{\{\text{замкнутые } k\text{-формы}\}}{\{\text{точные } k\text{-формы}\}}$$

## Когомологии сингулярные

Определяются через коцепи — функции на сингулярных симплексах.

## Свойства

- **Гомотопическая инвариантность:** гомотопически эквивалентные пространства имеют изоморфные когомологии
- **Кольцевая структура:** произведение cup делает $H^*(X)$ градуированным кольцом
- **Двойственность Пуанкаре:** для замкнутого ориентируемого $n$-мерного многообразия: $H^k \cong H_{n-k}$

## Примеры

- $H^0(S^n) = \mathbb{R}$ (связность)
- $H^n(S^n) = \mathbb{R}$ (ориентация)
- $H^k(S^n) = 0$ для $0 < k < n$

## Разложение Ходжа

На кэлеровом многообразии:
$$H^k(X, \mathbb{C}) = \bigoplus_{p+q=k} H^{p,q}(X)$$
