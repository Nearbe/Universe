# Топологические ограничения Δ

Информационная ёмкость ([47](47_Информационная_ёмкость.md)) — **D_f**. Онтологические состояния ([130](130_Онтологические_состояния.md)) — диапазоны.

## Ограничения

Для наблюдаемого пространства:

```
Δ ∈ [⊖Id, +Id]
```

| Δ | D_f | Смысл |
|---|---|---|
| **⊖Id** | **Id** | Линейная структура |
| **Ω** | **D(D(Id))** | Плоское пространство |
| **+Id** | **D(D(D(Id)))** | Объёмное пространство |

## Исключения

| Состояние | Δ | Смысл |
|---|---|---|
| Инфляция | `→ +Π` | Супер-фрактальное |
| Тёмная энергия | `→ ⊖Π` | Суб-фрактальное |
| Квантовые флуктуации | временный выход за `[⊖Id, +Id]` | `Δt ∼ ħ:ΔE` |

## Физический смысл

Δ измеряет **отклонение от плоского пространства** (`D_f = D(D(Id))`):
- `Δ = Ω`: равновесие
- `Δ > Ω`: отклонение к объёму
- `Δ < Ω`: отклонение к линии

## Полный набор операций

| Δ | D_f | Геометрия |
|---|---|---|
| `⊖Id` | `Id` | Линия |
| `Ω` | `D(D(Id))` | Плоскость |
| `+Id` | `D(D(D(Id)))` | Объём |
| `→+Π` | `→Π` | Инфляция |
| `→⊖Π` | `→Ω` | Тёмная энергия |
