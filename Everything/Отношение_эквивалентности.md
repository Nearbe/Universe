# Отношение эквивалентности

## Определение

Отношение $\sim$ на множестве $X$ — отношение эквивалентности, если:

1. **Рефлексивность:** $x \sim x$ для всех $x \in X$
2. **Симметричность:** $x \sim y \implies y \sim x$
3. **Транзитивность:** $x \sim y$ и $y \sim z \implies x \sim z$

## Класс эквивалентности

$$[x] = \{y \in X \mid y \sim x\}$$

## Фактормножество

$$X/\!\!\sim = \{[x] \mid x \in X\}$$

Множество всех классов эквивалентности.

## Примеры

- Равенство ($=$) — тривиальное отношение эквивалентности
- Сравнение по модулю $n$: $a \equiv b \pmod{n}$
- Изоморфизм групп, гомеоморфизм пространств
- Подобие треугольников
- Равномощность множеств

## Связь с разбиением

Классы эквивалентности образуют разбиение множества:
- Каждый элемент принадлежит ровно одному классу
- Объединение классов = всё множество

## Применения

Построение новых математических объектов: рациональные числа из целых, вещественные из рациональных, факторпространства.
