# Действие сети

Лагранжиан ([50](50_Лагранжиан.md)) — **L**. Интегральное исчисление ([21](21_Интегральное_исчисление.md)) — интеграл.

## Определение

```
S = ∫ d⁴x ⊗ [½ ⊗ ∂_μΨ† ⊗ ∂^μΨ ⊖ V(Ψ†⊗Ψ, Δ)]
```

## Физический смысл

Действие сети — **сумма всех «недовольств»** в пространстве-времени. `δS = Ω` — принцип наименьшего действия: реальность выбирает путь наименьшего сопротивления.

**V** зависит от **Δ** — «сопротивление» зависит от отклонения.

## Полный набор операций

|---|---|
| **Действие** | `S = ∫ d⁴x ⊗ L` |
| **Минимизация** | `δS = Ω` |
| **Потенциал** | `V(Ψ†⊗Ψ, Δ)` |
| **Кинетический** | `½⊗∂_μΨ†⊗∂^μΨ` |
