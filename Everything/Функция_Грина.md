# Функция Грина

## Определение

Функция Грина $G(x, s)$ линейного дифференциального оператора $L$ — решение уравнения:
$$L G(x, s) = \delta(x - s)$$

где $\delta$ — дельта-функция Дирака.

## Применение к решению уравнений

Решение $Lu = f$ выражается через функцию Грина:
$$u(x) = \int G(x, s) f(s) \, ds$$

## Свойства

- Зависит только от оператора $L$ и граничных условий
- Симметрия: $G(x, s) = G(s, x)$ для самосопряжённых операторов
- Фундаментальное решение: частный случай для $L$ на всём пространстве

## Примеры

**Для оператора Лапласа в $\mathbb{R}^3$:**
$$G(x, s) = -\frac{1}{4\pi |x - s|}$$

**Для одномерного оператора $-d^2/dx^2$:**
$$G(x, s) = \begin{cases} x(1-s), & x \leq s \\ s(1-x), & x \geq s \end{cases}$$

## В физике

Функция Грина = функция отклика системы на точечный источник. В квантовой теории поля — пропагатор.
