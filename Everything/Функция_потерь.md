# Функция потерь

## Определение

Функция $L(y, \hat{y})$, измеряющая ошибку предсказания $\hat{y}$ относительно истинного значения $y$.

## Для классификации

**Кросс-энтропия (Binary):**
$$L = -[y \log \hat{y} + (1-y) \log(1-\hat{y})]$$

**Кросс-энтропия (Categorical):**
$$L = -\sum_{c} y_c \log \hat{y}_c$$

## Для регрессии

**MSE (среднеквадратичная ошибка):**
$$L = \frac{1}{n}\sum_{i=1}^n (y_i - \hat{y}_i)^2$$

**MAE (средняя абсолютная ошибка):**
$$L = \frac{1}{n}\sum_{i=1}^n |y_i - \hat{y}_i|$$

**Huber:** комбинация MSE и MAE, робастна к выбросам.

## Свойства

- **Дифференцируемость:** необходима для backpropagation
- **Выпуклость:** гарантирует глобальный минимум (для линейных моделей)
- **Робастность:** устойчивость к выбросам

## Регуляризация

$$L_{\text{total}} = L_{\text{data}} + \lambda R(\theta)$$

$L1$ (Lasso) и $L2$ (Ridge) регуляризация.
