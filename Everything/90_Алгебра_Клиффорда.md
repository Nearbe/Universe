# Алгебра Клиффорда (Геометрический Калькулятор)

Кватернионы ([87](87_Кватернионы.md)) — для 3D. Алгебра Клиффорда — для **любой размерности**.

## Объединение Всего

В обычной алгебре есть векторы, площади (бивекторы), объёмы. И у каждого свои правила.
Клиффорд говорит: **«Всё есть один тип объекта — Мультивектор»**.

И умножение у них единое.
- Вектор × Вектор = Скаляр (Точка) + Бивектор (Плоскость).
- Это объединяет метрику (длину) и ориентацию (вращение) в одной формуле.

## Физический Смысл

Алгебра Клиффорда — это **Процессор Геометрии**.
Она позволяет считать повороты, отражения и проекции без тригонометрии и матриц.
В системе **𝕌** это естественный способ описания спиноров и полей.
Пространство не «фон». Оно **алгебраическая структура**, с которой можно умножаться.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Геометрическое произведение** | `a ⊗ b = a·b ⊕ a∧b` |
| **Скаляр** | `a·b` — метрика |
| **Бивектор** | `a∧b` — ориентация/вращение |
| **Мультивектор** | `M = s ⊕ v ⊕ B ⊕ T ⊕ ...` |
| **Обратный** | `M⁻¹ = M̃ : (M⊗M̃)` |
