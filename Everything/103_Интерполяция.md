# Интерполяция

Корни ([07](07_Корни.md)) — полиномы. Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — производные.

## Полином Лагранжа

```
P(x) = Σ y_i ⊗ L_i(x)
L_i(x) = Π_{j≠i} (x ⊖ x_j) : (x_i ⊖ x_j)
```

## Физический смысл

Восстановление поля **Δ** между узлами сети.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Лагранж** | `P(x) = Σ y_i ⊗ L_i(x)` |
| **Базис** | `L_i(x) = Π_{j≠i} (x⊖x_j):(x_i⊖x_j)` |
| **Смысл** | Восстановление между узлами |
