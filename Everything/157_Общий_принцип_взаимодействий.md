# Общий принцип взаимодействий

Родитель: 4 силы ([50](50_Фундаментальные_Взаимодействия.md)), Параметр Δ ([122](122_Дифференциал_Delta.md))

## Определение

```
F_i = (g_i ⊗ Q₁ ⊗ Q₂ : r²) ⊗ f_i(r : λ_i)
```

где `g_i` — константа связи, `Q` — заряд, `r` — расстояние, `λ_i` — радиус взаимодействия, `f_i` — форм-фактор.

## Физический смысл

Все взаимодействия описываются единой структурной формулой. Различие сил определяется только значениями `g_i`, `λ_i` и видом `f_i`:
- Гравитация: `λ → Π`, `f = Id`
- ЭМ: `λ → Π`, `f = Id` (с осцилляциями фазы)
- Слабое: `λ ≈ 10⁻³ фм`, `f = e^(⊖r:λ)`
- Сильное: `λ ≈ 1 фм`, `f = σ⊗r` (конфайнмент)
Параметр `Δ` модулирует эффективные заряды `Q(Δ)` в зависимости от масштаба наблюдения.

## Полный набор операций

| Взаимодействие | `g_i` | `λ_i` | `f_i(x)` |
|---|---|---|---|
| **Гравитация** | `G` | `Π` | `Id` |
| **ЭМ** | `α` | `Π` | `Id` |
| **Слабое** | `g_w` | `≈10⁻³ фм` | `e^(⊖x)` |
| **Сильное** | `α_s` | `≈1 фм` | `σ⊗x` |
| **Общий вид** | `F = (g⊗Q₁⊗Q₂:r²)⊗f(r:λ)` | | |
