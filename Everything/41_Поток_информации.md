# Поток информации

Волновая функция ([31](31_Волновая_функция.md)) — фаза **θ**. Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — градиент. Теперь — куда течёт информация.

## Определение

```
j = (ħ : m) ⊗ ρ ⊗ ∇θ
```

где **ρ** — плотность, **∇θ** — градиент фазы, **ħ** — отношение (квант действия).

## Физический смысл

Поток информации — **направление, куда «течёт» смысл**.
- **ρ** — сколько информации.
- **∇θ** — куда (градиент фазы).

Информация течёт туда, где фаза меняется быстрее — от «понятного» к «непонятному», создавая градиент понимания.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Поток** | `j = (ħ:m) ⊗ ρ ⊗ ∇θ` |
| **Плотность** | `ρ = \|Ψ\|²` |
| **Градиент фазы** | `∇θ` — направление |
| **Ветвление** | `j : Ω = D(j)` |
| **Сохранение** | `∇·j ⊕ ∂ρ/∂t = Ω` |
