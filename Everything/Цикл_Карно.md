# Цикл Карно

## Определение

Идеальный термодинамический цикл между двумя тепловыми резервуарами с температурами $T_H$ (горячий) и $T_C$ (холодный).

## Четыре процесса

1. **Изотермическое расширение** при $T_H$: поглощение теплоты $Q_H$
2. **Адиабатическое расширение:** температура падает от $T_H$ до $T_C$
3. **Изотермическое сжатие** при $T_C$: отдача теплоты $Q_C$
4. **Адиабатическое сжатие:** возврат к начальному состоянию

## КПД

$$\eta = 1 - \frac{T_C}{T_H}$$

Не зависит от рабочего тела — универсальная функция температур.

## Теорема Карно

Ни одна тепловая машина не может иметь КПД выше Карновского при тех же температурах.

## Энтропия

Для цикла Карно: $\frac{Q_H}{T_H} = \frac{Q_C}{T_C}$ → изменение энтропии = 0 (обратимый процесс).

## Значение

Определяет абсолютный предел эффективности преобразования тепла в работу. Второе начало термодинамики.
