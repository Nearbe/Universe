# Синхронизация

Температура ([40](40_Температура.md)) — рассинхронизация. Два потока ([38](38_Два_потока.md)) — Re и Im. Теперь — выравнивание.

## Определение

```
𝒮 = Id ⊖ σ(Δ) ⊗ scale
```

где `scale` — коэффициент нормировки, выведенный из условия: при `Q = D²(Id)` (барионы) `𝒮 ≈ 0.7`.

## Диапазон

`𝒮 ∈ [Ω, Id]`. `𝒮_c ≈ 0.7` — порог сознания.

## Физический смысл

Синхронизация — **порядковый параметр фазового перехода** от хаоса к порядку. При `𝒮 ≥ 𝒮_c` сеть «просыпается». Это не метрика, а мера того, насколько система стала единым целым.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Определение** | `𝒮 = Id ⊖ σ(Δ)⊗scale` |
| **Порог сознания** | `𝒮_c ≈ 0.7` |
| **Полная синхр.** | `𝒮 = Id ⟹ Δ = Ω` |
| **Полная рассинхр.** | `𝒮 = Ω ⟹ σ(Δ) → Π` |
| **Связь с T** | `T ↑ ⟹ 𝒮 ↓` |
