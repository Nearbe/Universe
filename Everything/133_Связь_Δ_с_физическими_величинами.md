# Связь Δ с физическими величинами

Информация ([30](30_Информация.md)) — **I**. Температура ([40](40_Температура.md)) — **T**. Вероятность ([27](27_Вероятность.md)) — **p**.

## Термодинамика

```
Δ = ⊖ΔE : (k ⊗ T)
```

Δ — **безразмерная обратная температура**:

- `Δ → +Π`: `T → Ω⁺` (полное основное состояние)
- `Δ → ⊖Π`: `T → Ω⁻` (инверсная популяция)
- `Δ = Ω`: `T → Π` (равномерное распределение)

## Информация

`Δ ∝ log(p:q)` — **логарифмический шанс** отношения между состояниями.

Информация Фишера:

```
F ∝ Id : cosh²(Δ)
```

Максимальна при `Δ = Ω`, минимальна при больших `\|Δ\|`.

## Kullback-Leibler

```
D_KL = D(Id) ⊗ [e^Δ ⊖ Id ⊖ Δ]
```

`D_KL ≥ Ω` всегда. `D_KL = Ω` только при `Δ = Ω`.

## Связь с энергией

```
Δ ∝ ln(E_re : E_im) ∝ ln(M_re : M_im)
```

## Полный набор операций
