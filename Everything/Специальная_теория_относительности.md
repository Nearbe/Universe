# Специальная теория относительности

## Постулаты

1. Законы физики одинаковы во всех инерциальных системах отсчёта
2. Скорость света $c$ одинакова во всех ИСО

## Преобразования Лоренца

$$t' = \gamma\left(t - \frac{vx}{c^2}\right), \quad x' = \gamma(x - vt)$$

где $\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}$.

## Следствия

- **Замедление времени:** $\Delta t' = \gamma \Delta t$
- **Сокращение длины:** $L' = L/\gamma$
- **Относительность одновременности**

## Инварианты

- Интервал: $ds^2 = c^2dt^2 - dx^2 - dy^2 - dz^2$
- Собственное время: $d\tau = ds/c$
- $E^2 - (pc)^2 = (mc^2)^2$

## Эквивалентность массы и энергии

$$E = mc^2$$

Полная энергия: $E = \gamma mc^2$. Импульс: $p = \gamma mv$.

## 4-векторы

Позиция: $x^\mu = (ct, x, y, z)$. 4-скорость, 4-импульс, 4-ток — все преобразуются по Лоренцу.
