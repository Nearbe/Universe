# Случайная величина

## Определение

Случайная величина $X$ — измеримая функция из пространства элементарных исходов $\Omega$ в $\mathbb{R}$:

$$X: \Omega \to \mathbb{R}$$

## Типы

**Дискретная:** принимает конечное или счётное число значений.

Описывается функцией вероятности: $P(X = x_i) = p_i$.

**Непрерывная:** описывается плотностью $f(x)$:
$$P(a \leq X \leq b) = \int_a^b f(x)\,dx$$

## Функция распределения

$$F(x) = P(X \leq x)$$

## Числовые характеристики

- **Математическое ожидание:** $\mathbb{E}[X]$
- **Дисперсия:** $\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2]$
- **Медиана:** $F(m) = 0.5$
- **Мода:** наиболее вероятное значение
- **Квантили:** $x_p = F^{-1}(p)$

## Совместное распределение

Для двух величин $(X, Y)$:
$$F(x, y) = P(X \leq x, Y \leq y)$$

Независимость: $F(x, y) = F_X(x) F_Y(y)$.
