# Градиентный спуск

## Определение

Итеративный метод минимизации функции:

$$\theta_{t+1} = \theta_t - \eta \nabla f(\theta_t)$$

где $\eta$ — скорость обучения (learning rate).

## Варианты

**Полный (Batch):** градиент по всем данным. Точный, но медленный.

**Стохастический (SGD):** градиент по одному примеру. Быстрый, но шумный.

**Мини-батч:** градиент по подвыборке. Компромисс.

## Оптимизации

**Momentum:**
$$v_t = \beta v_{t-1} + (1-\beta)\nabla f(\theta_t), \quad \theta_{t+1} = \theta_t - \eta v_t$$

**Adam:**
Адаптивные learning rate для каждого параметра + momentum.

**RMSProp, AdaGrad:** адаптивные learning rate.

## Проблемы

- **Локальные минимумы:** особенно в глубоких сетях
- **Седловые точки:** градиент ≈ 0, но не минимум
- **Взрыв/затухание градиента:** в глубоких сетях

## Сходимость

Для выпуклых функций: $O(1/T)$ для выпуклых, $O(1/\sqrt{T})$ для невыпуклых.
