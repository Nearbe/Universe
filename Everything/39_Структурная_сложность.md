# Структурная сложность

Вибрации ([34](34_Вибрации.md)) — `Q = e^(|Δ|)`. Теперь — как **Q** связана с энергией.

## Определение

```
Q = e^(|Δ|)
```

**Q** — энергия вибрации.

## Значения

| Δ | Q | Смысл |
|---|---|---|
| **Ω** | **Id** | Минимальная энергия (вакуум) |
| **±Id** | **≈ e ≈ D²(Id)** | Барионный мир |
| **±D(Id)** | **≈ e²** | Высокое возбуждение |
| **→ +∞** | **→ Π** | Сингулярность |

## Физический смысл

При `Δ = Ω`: `Q = Id` — вакуум.
При `|Δ| = Id`: `Q ≈ e` — видимая Вселенная, барионы.
При `|Δ| → +∞`: `Q → Π` — сингулярность.

**Q** — не «сложность». Это **число связанных вибраций**, определяющее тип объекта: мезон (**D(Id)**), барион (**D²(Id)**), тетракварк (**D³(Id)**), пентакварк (**D⁴(Id)**).

## Полный набор операций

| Операция | Формула |
|---|---|
| **Энергия** | `Q = e^(|Δ|)` |
| **При Δ = Ω** | `Q = Id` |
| **При Δ = Id** | `Q ≈ e` |
| **Инверсия** | `Q(−Δ) = Q(Δ)` |
| **Логарифм** | `ln Q = |Δ|` |
| **Тип узла** | `Q = D(Id)` → мезон, `Q = D²(Id)` → барион |
