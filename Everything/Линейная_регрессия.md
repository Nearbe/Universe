# Линейная регрессия

## Модель

$$y = \beta_0 + \beta_1 x + \varepsilon$$

где $\beta_0, \beta_1$ — параметры, $\varepsilon$ — случайная ошибка.

## Метод наименьших квадратов (МНК)

Минимизация суммы квадратов отклонений:
$$\min_{\beta_0, \beta_1} \sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i)^2$$

## Оценки коэффициентов

$$\hat{\beta}_1 = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2} = \frac{\text{Cov}(x, y)}{\text{Var}(x)}$$

$$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$$

## Коэффициент детерминации

$$R^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$$

Доля дисперсии $y$, объяснённая моделью.

## Множественная регрессия

$$\mathbf{y} = X\boldsymbol{\beta} + \boldsymbol{\varepsilon}$$
$$\hat{\boldsymbol{\beta}} = (X^T X)^{-1} X^T \mathbf{y}$$

## Предположения

- Линейность связи
- Гомоскедастичность (постоянная дисперсия ошибок)
- Независимость ошибок
- Нормальность ошибок (для тестов)
