# Фазовое пространство

## Определение

Фазовое пространство — пространство всех возможных состояний динамической системы.

Для механической системы с $n$ степенями свободы — пространство координат $(q_1, \ldots, q_n, p_1, \ldots, p_n)$, где $q_i$ — обобщённые координаты, $p_i$ — обобщённые импульсы.

## Свойства

- Каждая точка = полное состояние системы
- Траектория = эволюция системы во времени
- Объём фазового пространства сохраняется (теорема Лиувилля для гамильтоновых систем)

## В теории динамических систем

Фазовое пространство $(X, \mathcal{F}, \mu)$ — измеримое пространство с сохраняющейся мерой $\mu$ и преобразованием $T: X \to X$.

## Размерность

Размерность фазового пространства = число степеней свободы $\times$ 2 (координата + импульс).
