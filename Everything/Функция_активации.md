# Функция активации

## Определение

Неотрицательная функция, применяемая к выходу нейрона. Вводит нелинейность в сеть.

## Основные

**Sigmoid:**
$$\sigma(x) = \frac{1}{1 + e^{-x}}$$
Диапазон: $(0, 1)$. Проблема: затухание градиента.

**Tanh:**
$$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$
Диапазон: $(-1, 1)$. Центрирована, но градиент затухает.

**ReLU:**
$$\text{ReLU}(x) = \max(0, x)$$
Проста, не затухает для $x > 0$. Проблема: «умирающие» нейроны.

**Leaky ReLU:**
$$\text{LeakyReLU}(x) = \max(\alpha x, x), \quad \alpha \ll 1$$

**GELU:**
$$\text{GELU}(x) = x \cdot \Phi(x)$$
Используется в Transformers.

**Softmax:**
$$\text{softmax}(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$$
Для многоклассовой классификации.

## Свойства

- Нелинейность (иначе сеть = линейная модель)
- Дифференцируемость (для backpropagation)
- Ограниченность (не всегда)
