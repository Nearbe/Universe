# Функция Мёбиуса

## Определение

Функция Мёбиуса $\mu(n)$ определяется для натуральных $n$:

$$\mu(n) = \begin{cases} 1, & n = 1 \\ 0, & n \text{ делится на квадрат простого} \\ (-1)^k, & n = p_1 p_2 \cdots p_k \text{ (произведение } k \text{ различных простых)} \end{cases}$$

## Свойства

- Мультипликативная: $\mu(mn) = \mu(m)\mu(n)$ при $\gcd(m,n)=1$
- $\sum_{d|n} \mu(d) = 0$ при $n > 1$
- $\sum_{d|n} \mu(d) = 1$ при $n = 1$

## Формула обращения Мёбиуса

Если $g(n) = \sum_{d|n} f(d)$, то:
$$f(n) = \sum_{d|n} \mu(d) g(n/d)$$

## Ряд Дирихле

$$\sum_{n=1}^{\infty} \frac{\mu(n)}{n^s} = \frac{1}{\zeta(s)} \quad (\Re(s) > 1)$$

## Связь с гипотезой Римана

Гипотеза Римана эквивалентна оценке:
$$\left|\sum_{n \leq x} \mu(n)\right| = O(x^{1/2 + \varepsilon})$$
