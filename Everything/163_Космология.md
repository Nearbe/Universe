## Поле Δ как скалярное поле

```
ρ_Δ = (∂_tΔ)² : D(D(Id)) ⊕ V(Δ)
p_Δ = (∂_tΔ)² : D(D(Id)) ⊖ V(Δ)
w_Δ = p_Δ : ρ_Δ
```

| Режим | Условие | `w_Δ` |
|---|---|---|
| Кинетический | `(∂_tΔ)² ≫ V` | `+Id` |
| Потенциальный | `V ≫ (∂_tΔ)²` | `⊖Id` |
| Равновесие | `(∂_tΔ)² = V` | `Ω` |

## Параметр Хаббла

```
H(a) = H₀ ⊗ √[Ω_r⊗a⁻⁴ ⊕ Ω_m⊗a⁻³ ⊕ Ω_k⊗a⁻² ⊕ Ω_Λ]
H₀ ≈ 67-74 км/с/Мпк
```

Напряжённость Хаббла отражает эволюцию `Δ`.
