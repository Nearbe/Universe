# Жадные алгоритмы

## Определение

Алгоритм, делающий на каждом шаге локально оптимальный выбор в надежде получить глобальный оптимум.

## Когда работает

Жадный алгоритм даёт оптимальное решение, если задача обладает:
- **Свойством жадного выбора:** локально оптимальный выбор ведёт к глобально оптимальному решению
- **Оптимальной подструктурой:** остаток задачи после жадного выбора — та же задача меньшего размера

## Примеры

**Алгоритм Дейкстры:** кратчайший путь в графе с неотрицательными весами.

**Алгоритм Крускала:** минимальное остовное дерево — на каждом шаге добавляем самое лёгкое ребро, не создающее цикл.

**Задача о выборе заявок:** сортировка по времени окончания, жадный выбор непересекающихся.

**Код Хаффмана:** оптимальное префиксное кодирование — жадное объединение двух наименьших частот.

## Когда НЕ работает

**Задача о рюкзаке (0/1):** жадный выбор по удельной стоимости не даёт оптимума.

**Задача коммивояжёра:** жадный (ближайший сосед) может дать сколь угодно плохое решение.

## Сложность

Обычно $O(n \log n)$ из-за сортировки, затем $O(n)$ жадный проход.
