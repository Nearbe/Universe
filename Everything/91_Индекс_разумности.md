# Индекс разумности

Информационная ёмкость ([47](47_Информационная_ёмкость.md)) — **D_f**. Глобальный параметр ([52](52_Глобальный_параметр.md)) — `⟨Δ⟩`.

## Определение

```
C_reason = (Id ⊕ Δ) : D(Id)
```

При `Δ ∈ [⊖Id, +Id]`: `C_reason ∈ [Ω, Id]`.

Альтернативная форма:

```
C_reason = (D_f ⊖ Id) : D(D(D(D(Id))))
```

## Физический смысл

`C_reason` — **мера осмысленности** системы.

- `Δ = ⊖Id`: `C_reason = Ω` — система «слепа» к себе
- `Δ = Ω`: `C_reason = Id : D(Id) = 0.5` — нейтральное сознание
- `Δ = +Id`: `C_reason = Id` — максимальная самоосознанность

## Полный набор операций

| Операция | Формула |
|---|---|
| **Основная** | `C_reason = (Id⊕Δ):D(Id)` |
| **Через D_f** | `C_reason = (D_f⊖Id):D⁴(Id)` |
| **Минимум** | `Δ=⊖Id ⟹ C_reason=Ω` |
| **Нейтральное** | `Δ=Ω ⟹ C_reason=Id:D(Id)` |
| **Максимум** | `Δ=+Id ⟹ C_reason=Id` |
