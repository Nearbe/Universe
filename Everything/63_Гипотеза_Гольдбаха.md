# Гипотеза Гольдбаха (Симметрия Весов)

Любое чётное число `> D(Id)` есть сумма двух простых.

## Баланс Зарядов

В системе **𝕌** чётное число — это **Нейтральное Состояние** (`Δ ≈ Ω`).
Простое число — это **Заряд** (Атом, [25](25_Строение_чисел.md)).

Чтобы система пришла в равновесие (стала чётной), она должна скомпенсировать заряды.
Один атом не даст нейтральности. Два атома — да.

```
Even = Prime₁ ⊕ Prime₂
```

Это не магия чисел. Это **Закон Сохранения Симметрии**.
Любая возмущённая система (чётное число) состоит из двух устойчивых возмущений (простых чисел), которые гасят друг друга.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Чётное** | `Δ ≈ Ω` (нейтральное) |
| **Простое** | Атом = **D(Id)** |
| **Баланс** | `Even = P₁ ⊕ P₂` |
| **Нейтральность** | Заряды компенсируют друг друга |

## Физический Смысл

Гольдбах прав, потому что **природа не терпит одиноких зарядов** в равновесии.
Они всегда спариваются.
Чётность — это и есть факт спаривания.
