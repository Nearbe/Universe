# Круговой метод Харди-Литтлвуда

## Идея

Представить число решений диофантова уравнения как интеграл от произведения экспоненциальных сумм:

$$R(N) = \int_0^1 S(\alpha)^k e^{-2\pi i N \alpha} d\alpha$$

где $S(\alpha) = \sum_{p \leq x} e^{2\pi i \alpha p}$ — сумма по простым.

## Дуги Фарея

Разбиение отрезка $[0, 1]$:
- **Основные дуги:** окрестности рациональных $a/q$ с малым $q$ — дают основной вклад
- **Добавочные дуги:** остальное — оцениваются сверху

## Асимптотическая формула

$$R(N) = \mathfrak{S}(N) \cdot J(N) + \text{ошибка}$$

- $\mathfrak{S}(N)$ — сингулярный ряд (арифметический множитель, произведение по простым)
- $J(N)$ — сингулярный интеграл (аналитический множитель)

## Применения

- Проблема Варинга: $x_1^k + \cdots + x_s^k = N$
- Теорема Винградова: $N = p_1 + p_2 + p_3$
- Гипотеза Гольдбаха: $2n = p + q$ (асимптотика известна, но не для всех $n$)
- Проблема близнецов

## Трудности

Оценка сумм по добавочным дугам — ключевая аналитическая задача.
