# Глобальная частота

Частота и период ([42](42_Частота_и_период.md)) — `f = e^(−Δ)`. Глобальный параметр ([52](52_Глобальный_параметр.md)) — `⟨Δ⟩`.

## Определение

```
f_glob = e^(⊖Δ_global)
```

где `Δ_global = (Id : N) ⊗ Σᵢ Δᵢ`.

## Физический смысл

Глобальная частота — **средний темп вычисления всей сети**. Когда `Δ_global ≈ Ω`: `f_glob ≈ Id` — равновесие. Когда отклоняется — сеть «ускоряется» или «замедляется» как целое.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Глобальная** | `f_glob = e^(⊖Δ_global)` |
| **Среднее Δ** | `Δ_global = (Id:N) ⊗ Σᵢ Δᵢ` |
| **Равновесие** | `Δ_global = Ω ⟹ f_glob = Id` |
| **Ускорение** | `Δ_global < Ω ⟹ f_glob > Id` |
