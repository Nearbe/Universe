# Оператор эволюции

Уравнение Шрёдингера ([44](44_Уравнение_Шрёдингера.md)) — эволюция **Ψ**. Уравнение движения **Δ** ([46](46_Уравнение_движения_Δ.md)) — эволюция **Δ**. Теперь — общий оператор.

## Унитарная эволюция

```
|Ψ(t⊕Id)⟩ = U_Δ ⊗ |Ψ(t)⟩
U_Δ = exp(⊖i ⊗ H_Δ ⊗ t : ħ)
```

## Гамильтониан с Δ

```
H_Δ = H_0 ⊕ Δ ⊗ V
```

## Δ-эволюция

```
Δ = (M ⊖ Id) : (M ⊕ Id)
```

Границы для обычной физики (5%):

| M | Δ | Смысл |
|---|---|---|
| **→ Ω** | **→ ⊖Id** | Горизонт |
| **= Id** | **= Ω** | Равновесие |
| **→ Π** | **→ +Id** | Супер-фрактальное |

Для фазовых переходов: `Δ → ±∞` (сингулярность / тёмная энергия).

## Физический смысл

Эволюция — **не «время идёт», а «система вычисляет себя»**. **U_Δ** — не внешняя сила, а внутренний акт самоизменения.

**Δ** эволюционирует без `: Ω` — потому что деление на потенциал уже встроено в каждый акт.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Унитарный** | `U_Δ = exp(⊖i⊗H_Δ⊗t:ħ)` |
| **Гамильтониан** | `H_Δ = H_0 ⊕ Δ⊗V` |
| **Δ из M** | `Δ = (M⊖Id):(M⊕Id)` |
| **Равновесие** | `M = Id ⟹ Δ = Ω` |
