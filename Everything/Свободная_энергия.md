# Свободная энергия

## Свободная энергия Гельмгольца

$$F = U - TS$$

Для систем при постоянных $T, V$. Минимизируется в равновесии.

Дифференциал: $dF = -S\,dT - P\,dV + \mu\,dN$.

## Свободная энергия Гиббса

$$G = U - TS + PV = H - TS$$

Для систем при постоянных $T, P$. Минимизируется в равновесии.

Дифференциал: $dG = -S\,dT + V\,dP + \mu\,dN$.

## Связь со статистической суммой

$$F = -k_B T \ln Z$$

где $Z$ — статистическая сумма канонического ансамбля.

## Химический потенциал

$$\mu = \left(\frac{\partial G}{\partial N}\right)_{T,P}$$

Условие равновесия: $\mu_1 = \mu_2$ (равенство химических потенциалов фаз).

## Применения

- Фазовые переходы: сравнение $G$ разных фаз
- Химическое равновесие: $\Delta G < 0$ → реакция идёт
- Электрохимия: $\Delta G = -nFE$
- Биология: АТФ-гидролиз ($\Delta G < 0$)
