# Градиент и лапласиан Δ

Дифференциальные операторы ([97](97_Дифференциальные_операторы.md)) — **∇**, **∇²**. Уравнение Клейна-Гордона ([64](64_Уравнение_Клейна_Гордона.md)) — **□**.

## Определение

```
∇Δ = (∂Δ:∂x, ∂Δ:∂y, ∂Δ:∂z)
∇²Δ = ∂²Δ:∂x² ⊕ ∂²Δ:∂y² ⊕ ∂²Δ:∂z²
```

## Физический смысл

**∇Δ** — градиент отклонения. Показывает, где система более «реальна» (Re доминирует), а где более «виртуальна» (Im).

**∇²Δ** — лапласиан отклонения. Описывает распространение возмущений **Δ** в сети.

## Полный набор операций

| Оператор | Формула | Смысл |
|---|---|---|
| **∇Δ** | `(∂Δ:∂x, ∂Δ:∂y, ∂Δ:∂z)` | Направление роста Δ |
| **∇²Δ** | `Σ ∂²Δ:∂x_i²` | Распространение возмущений |
| **∇Δ > Ω** | Re доминирует | «Реальность» |
| **∇Δ < Ω** | Im доминирует | «Виртуальность» |
