# LU-разложение

## Определение

Представление квадратной матрицы $A$ в виде произведения:

$$PA = LU$$

где:
- $L$ — нижняя треугольная матрица
- $U$ — верхняя треугольная матрица
- $P$ — матрица перестановок (для устойчивости)

## Алгоритм

Метод Гаусса с выбором ведущего элемента:
1. Для каждого столбца $k$: найти строку с максимальным элементом
2. Поменять строки местами (обновить $P$)
3. Исключить элементы ниже диагонали

## Применение к системам уравнений

Решение $Ax = b$:
1. $PAx = Pb \implies LUx = Pb$
2. Прямая подстановка: $Ly = Pb$
3. Обратная подстановка: $Ux = y$

## Сложность

$O(n^3/3)$ операций — вдвое быстрее метода Гаусса для множественных правых частей.

## Условия существования

LU-разложение существует, если все ведущие главные миноры ненулевые. С перестановками — для любой невырожденной матрицы.
