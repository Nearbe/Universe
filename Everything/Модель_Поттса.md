# Модель Поттса

## Определение

Обобщение модели Изинга на $q \geq 2$ состояний. На каждом узле решётки спин $s_i \in \{1, 2, \ldots, q\}$.

## Гамильтониан

$$H = -J \sum_{\langle i,j \rangle} \delta_{s_i, s_j}$$

где $\delta$ — символ Кронекера.

## Частные случаи

- **$q = 2$:** модель Изинга
- **$q \to 1$:** перколяция (связь через случайные кластеры)
- **$q \to 0$:** остовные деревья (spanning trees)

## Фазовый переход

- **2D:** при $q \leq 4$ — непрерывный переход, при $q > 4$ — первого рода
- **Критическая температура:** $\frac{k_B T_c}{J} = \frac{1}{\ln(1 + \sqrt{q})}$

## Точное решение

Точное решение для 2D (Baxter, 1973). Критические индексы зависят от $q$.

## Применения

- Физика: полиморфные фазовые переходы, пены
- Материаловедение: рост зёрен
- Теория графов: раскраска (хроматический многочлен)
