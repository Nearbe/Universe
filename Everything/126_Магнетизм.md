# Магнетизм

Уравнения Максвелла ([53](53_Уравнения_Максвелла.md)) — электромагнетизм. Калибровочные группы ([55](55_Калибровочные_группы.md)) — **U(1)**.

## Определение

```
B = ∇ × A
```

**A** — векторный потенциал (фаза калибровки **U(1)**).

## Физический смысл

Магнетизм — **закрученность фаз**. Когда фаза **θ** сети имеет вихрь, возникает магнитное поле. **B** — мера «скрученности» пространства.

Электричество + магнетизм = электромагнетизм. Но на уровне сети — это один процесс: фаза течёт (электричество) и фаза закручена (магнетизм).

## Полный набор операций

| Операция | Формула |
|---|---|
| **Поле** | `B = ∇×A` |
| **Потенциал** | **A** — фаза калибровки |
| **Вихрь** | `∇×A ≠ Ω` ⟹ магнетизм |
| **Связь с E** | `E = ⊖∇φ ⊖ ∂A:∂t` |
