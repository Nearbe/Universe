# QR-разложение

## Определение

Представление матрицы $A$ размера $m \times n$ ($m \geq n$) в виде:

$$A = QR$$

где:
- $Q$ — ортогональная (или унитарная) матрица $m \times m$: $Q^T Q = I$
- $R$ — верхняя треугольная матрица $m \times n$

## Методы вычисления

- **Процесс Грама-Шмидта:** ортогонализация столбцов
- **Отражения Хаусхолдера:** численно более устойчиво
- **Вращения Гивенса:** эффективно для разреженных матриц

## Применение к методу наименьших квадратов

Решение $\min \|Ax - b\|_2$:
$$\|QRx - b\|_2 = \|Rx - Q^Tb\|_2$$

Решение обратной подстановкой из $Rx = Q^Tb$.

## Связь с собственными значениями

QR-алгоритм — основной метод нахождения собственных значений:
$$A_k = Q_k R_k, \quad A_{k+1} = R_k Q_k$$

Последовательность $A_k$ сходится к верхней треугольной форме Шура.

## Применения

- Метод наименьших квадратов
- Нахождение собственных значений
- Решение переопределённых систем
