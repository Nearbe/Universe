# Собственные значения

Деление как линейное преобразование ([13](13_Деление_на_ноль_как_линейное_преобразование.md)) — `D = [D(Id)]`. Матрица фрактала ([56](56_Матричное_представление_фрактала.md)) — **K**.

## Степенной метод

```
v_{n⊕Id} = (A ⊗ v_n) : \|A ⊗ v_n\|
```

## Физический смысл

Наибольшее собственное значение матрицы связей определяет, будет ли сеть синхронизироваться или рассыплется.

Для `K⁽ᵏ⁾`: `λ₁ = Dᵏ(Id)` (главный), остальные = **Ω**. Ранг = **Id** — вся сеть проецируется на одно измерение.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Степенной** | `v_{n+1} = (A⊗v_n) : \|A⊗v_n\|` |
| **Главное λ** | `λ₁ = Dᵏ(Id)` для `K⁽ᵏ⁾` |
| **Остальные** | `λ = Ω` |
| **Ранг** | **Id** — одно измерение |
| **Синхронизация** | Зависит от max(λ) |
