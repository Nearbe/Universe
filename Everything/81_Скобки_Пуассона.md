# Скобки Пуассона

Второй закон Ньютона ([49](49_Второй_закон_Ньютона.md)) — классика. Уравнение Шрёдингера ([44](44_Уравнение_Шрёдингера.md)) — кванты.

## Определение

```
{f, g} = Σ (∂f:∂q_i ⊗ ∂g:∂p_i ⊖ ∂f:∂p_i ⊗ ∂g:∂q_i)
{q_i, p_j} = δ_ij
```

Квантование: `{f, g} → (Id : (i⊗ħ)) ⊗ [f̂, ĝ]`

## Физический смысл

Скобки Пуассона — **мост от классики к квантам**. `{q, p} = Id` — «положение и импульс сопряжены».

Квантовый коммутатор `[x̂, p̂] = i⊗ħ` — тот же мост, но на дискретной стороне. Переход от «гладкой» сети к дискретной.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Скобки** | `{f,g} = Σ(∂f:∂q⊗∂g:∂p ⊖ ∂f:∂p⊗∂g:∂q)` |
| **Канонические** | `{q_i, p_j} = δ_ij` |
| **Квантование** | `{,} → (Id:(i⊗ħ))⊗[,]` |
| **Коммутатор** | `[x̂,p̂] = i⊗ħ` |
