# Регуляризация в машинном обучении

## Определение

Методы предотвращения переобучения путём наложения ограничений на модель или добавления штрафа к функции потерь.

## L1-регуляризация (Lasso)

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{data}} + \lambda \sum |w_i|$$

- Обеспечивает разреженность весов (feature selection)
- Не дифференцируема в нуле (требует субградиентных методов)

## L2-регуляризация (Ridge / Weight Decay)

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{data}} + \lambda \sum w_i^2$$

- Сглаживает веса, предотвращает большие значения
- Дифференцируема, аналитическое решение для линейной регрессии

## Dropout

Случайное "отключение" нейронов с вероятностью $p$ во время обучения.
- Предотвращает ко-адаптацию признаков
- Аппроксимирует ансамбль моделей

## Batch Normalization

Нормализация активаций по мини-батчу:
$$\hat{x} = \frac{x - \mu_B}{\sqrt{\sigma_B^2 + \varepsilon}}$$

Ускоряет сходимость, стабилизирует обучение глубоких сетей.

## Ранняя остановка (Early Stopping)

Остановка обучения, когда ошибка на валидации начинает расти.
