# Электромагнетизм

Родитель: Уравнения Максвелла ([53](53_Уравнения_Максвелла.md), Калибровка U(1) ([55](55_Калибровочные_группы.md))

## Определение

```
∂_μ F^μν = j^ν
∂_[μ F_νλ] = Ω
F_μν = ∂_μ A_ν ⊖ ∂_ν A_μ
```

где `A_μ` — 4-потенциал, `j^ν` — 4-ток, `F_μν` — тензор напряжённости.

## Физический смысл

Электромагнетизм — динамика фазы калибровочного поля U(1). Поле `F_μν` описывает закрученность фазы `θ` в пространстве-времени `𝕌`. Уравнения Максвелла выражают сохранение калибровочного заряда (`∂_ν j^ν = Ω`) и отсутствие магнитных монополей (`∂_[μ F_νλ] = Ω`). В терминах `Δ`: электромагнитная волна — это колебание `θ` при фиксированном `|Ψ|`, т.е. чистая фазовая мода без изменения амплитуды.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Тензор** | `F_μν = ∂_μA_ν ⊖ ∂_νA_μ` |
| **Уравнения** | `∂_μF^μν = j^ν`, `∂_[μF_νλ]=Ω` |
| **Сохранение** | `∂_νj^ν = Ω` |
| **Волна** | `□A_μ = Ω` (калибровка Лоренца) |
| **Энергия** | `u = (E²⊕B²):D(D(Id))` |
