# Видимая доля

Родитель: Мембрана (Δ) ([122](122_Дифференциал_Delta.md)), Синхронизация (𝒮) ([45](45_Синхронизация.md))

## Определение

```
f_vis = Id : (Id ⊕ e^Q)
f_hid = e^Q : (Id ⊕ e^Q)
```

где `Q = e^(|Δ|)` — структурная сложность.

## Физический смысл

Доля информации, доступная каналу Регрессии (Im, 5%). `f_vis` монотонно убывает с ростом `Q`. При `Q → Ω`: `f_vis → Id` (полная прозрачность). При `Q → +∞`: `f_vis → Ω` (полная непрозрачность). Это количественная мера пропускной способности Мембраны.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Видимая** | `f_vis = Id:(Id⊕e^Q)` |
| **Скрытая** | `f_hid = e^Q:(Id⊕e^Q)` |
| **Тождество** | `f_vis ⊕ f_hid = Id` |
| **Предел Q→Ω** | `f_vis → Id`, `f_hid → Ω` |
| **Предел Q→+∞** | `f_vis → Ω`, `f_hid → Id` |
