# Обучение с подкреплением (Reinforcement Learning)

## Определение

Обучение агента, взаимодействующего со средой через последовательность действий для максимизации суммарной награды.

## Формализация (MDP)

- **Состояния** $S$
- **Действия** $A$
- **Переходы:** $P(s'|s, a)$ — вероятность перехода
- **Награда:** $R(s, a, s')$
- **Дисконтирование:** $\gamma \in [0, 1)$

## Цель

Найти политику $\pi(a|s)$, максимизирующую:
$$\mathbb{E}\left[\sum_{t=0}^\infty \gamma^t r_t\right]$$

## Функция ценности

$$V^\pi(s) = \mathbb{E}_\pi\left[\sum_{t=0}^\infty \gamma^t r_t \mid s_0 = s\right]$$

$$Q^\pi(s, a) = \mathbb{E}_\pi\left[\sum_{t=0}^\infty \gamma^t r_t \mid s_0 = s, a_0 = a\right]$$

## Уравнение Беллмана

$$V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a)\left[R(s,a,s') + \gamma V^\pi(s')\right]$$

## Основные методы

- **Q-learning:** обучение без модели
- **Policy Gradient:** градиентный подъём по параметрам политики
- **Actor-Critic:** комбинация обоих подходов
