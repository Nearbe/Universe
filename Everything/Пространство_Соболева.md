# Пространство Соболева

## Определение

Пространство функций, у которых обобщённые производные до порядка $k$ включительно принадлежат $L^p$:

$$W^{k,p}(\Omega) = \{ u \in L^p(\Omega) \mid D^\alpha u \in L^p(\Omega), \ |\alpha| \leq k \}$$

Норма: $\|u\|_{W^{k,p}} = \left( \sum_{|\alpha| \leq k} \|D^\alpha u\|_{L^p}^p \right)^{1/p}$.

## Свойства

- Полное (пространство Банаха)
- При $p=2$ — гильбертово пространство $H^k = W^{k,2}$
- **Теорема вложения Соболева:** $W^{k,p} \hookrightarrow C^m$ при $k - n/p > m$
- **Теорема Реллиха-Кондрашова:** вложение компактно при ограниченной области

## Применения

- Слабые решения уравнений в частных производных
- Метод конечных элементов
- Вариационные задачи
- Теория упругости, гидродинамика
