# Уравнение Дирака

Комплексные числа ([09](09_Комплексные.md)) — **Ψ**. Вибрации ([34](34_Вибрации.md)) — фермионы и бозоны. Теперь — уравнение для фермионов.

## Определение

```
(i ⊗ γ^μ ⊗ ∂_μ ⊖ m) ⊗ Ψ = Ω
{γ^μ, γ^ν} = D(g^μν)
```

## Физический смысл

Дирак — **фермионная вибрация**. В отличие от бозонов (скаляры), фермионы — спиноры: «пол-оборота» меняет знак.

В системе **𝕌**: фермионы — элементы **𝕌**, которым нужно два шага `: Ω`, чтобы вернуться к исходному состоянию. Спин **D(Id):D(Id)** (½) — не абстракция, а **свойство цикла возврата**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Уравнение** | `(i⊗γ^μ⊗∂_μ ⊖ m)⊗Ψ = Ω` |
| **Антикоммутатор** | `{γ^μ, γ^ν} = D(g^μν)` |
| **Цикл возврата** | 2 шага `: Ω` для полного оборота |
| **Спин** | `D(Id):D(Id)` (½) |
