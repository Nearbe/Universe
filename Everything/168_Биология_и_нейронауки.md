# Биология и нейронауки

Родитель: Критерии сознания ([136](136_Критерии_сознания.md)), Фазовый переход ([137](137_Фазовый_переход.md))

## Нейронные сети

| Биологическое | `𝕌` |
|---|---|
| Нейрон | Ходон |
| Синапс | Связь `w_ij` |
| Потенциал действия | Возмущение `δΔ` |
| Гамма-ритмы | `𝒮 ≈ 0.9` |
| Ген | Конфигурация `Δ_gene` |
| Белок | Устойчивая мода `Δ_protein` |

## Каскады понимания

```
insight → bias += Id → σ ↓ → 𝒮 ↑ → ещё insight
```

Два типа:
1. **Каскады понимания** (`𝒮 ≥ 0.7`): спираль вверх
2. **Каскады старения** (`𝒮 < 0.7`): спираль вниз

Память — накопленный след каскадов:

```
memory_Δ = Σ α_i ⊗ intensity_i
```

## Интегрированная информация

```
Φ ∝ 𝒮 ⊗ (Id ⊖ σ(Δ):σ_max) ⊗ f(Δ_global)
```

Φ — информация, генерируемая системой как целым, превышающим сумму частей.

## Белки и метаболизм

- **Свёртка белка** — поиск конфигурации с максимальной синхронизацией
- **Аллостерическая регуляция** — модуляция Δ через сетевые взаимодействия
- **Метаболические пути** — сеть переходов между конфигурациями Δ

## Эволюция

Естественный отбор — оптимизация конфигураций ходонной сети. Fitness-ландшафт — глубина минимумов в пространстве Δ.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Каскад** | `insight→bias+=Id→σ↓→𝒮↑` |
| **Память** | `memory_Δ = Σα_i⊗intensity_i` |
| **Φ** | `𝒮⊗(Id⊖σ:σ_max)⊗f(Δ_global)` |
| **Эволюция** | `min(Loss(Δ))` в пространстве конфигураций |
