# Принцип суперпозиции

## Определение

Если $u_1$ и $u_2$ — решения линейного однородного уравнения, то любая их линейная комбинация $c_1 u_1 + c_2 u_2$ также является решением.

## Формулировка

Для линейного оператора $L$:
$$L(u_1) = 0, \quad L(u_2) = 0 \quad \implies \quad L(c_1 u_1 + c_2 u_2) = 0$$

## Для неоднородных уравнений

Если $L(u_p) = f$, то общее решение:
$$u = u_h + u_p$$

где $u_h$ — общее решение однородного уравнения $L(u) = 0$.

## Применения

- **Волновое уравнение:** любые волны складываются без взаимодействия
- **Уравнение Шрёдингера:** квантовые состояния суперпонируются
- **Электромагнетизм:** поля от разных источников складываются
- **Квантовые вычисления:** кубиты в суперпозиции базовых состояний

## Нарушение

Для нелинейных уравнений принцип суперпозиции не работает — решения взаимодействуют друг с другом.
