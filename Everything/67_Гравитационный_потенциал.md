# Гравитационный потенциал

Гравитация ([43](43_Гравитация.md)) — бегущая геометрия. Масса ([32](32_Масса.md)) — **ρ**. Теперь — статический потенциал.

## Определение

```
∇²Φ = K ⊗ ρ
```

где **K** — константа связи.

## Физический смысл

Гравитационный потенциал — **форма информационной ямы**. Масса создаёт «вмятину» в сети, и другие элементы «скатываются» в неё не из-за силы, а из-за геометрии.

Гравитация — не сила, а кривизна связей.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Потенциал** | `∇²Φ = K⊗ρ` |
| **Сила** | `F = ⊖∇Φ` |
| **Энергия** | `U = m⊗Φ` |
| **Без массы** | `ρ = Ω ⟹ ∇²Φ = Ω` |
