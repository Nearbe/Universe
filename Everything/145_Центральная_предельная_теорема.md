# Центральная предельная теорема

Родитель: Вероятность ([27](27_Вероятность.md)), Распределения ([100](100_Распределения_вероятностей.md))

## Определение

Для независимых одинаково распределённых величин `X_i` с `⟨X⟩ = μ`, `Var(X) = σ²`:

```
S_n = Σ_{i=Id}^{n} X_i
Z_n = (S_n ⊖ n⊗μ) : (σ ⊗ √n)
lim_{n→∞} P(Z_n ≤ z) = Φ(z)
```

где `Φ(z)` — функция распределения стандартного нормального закона.

## Физический смысл

Сумма независимых флуктуаций `Δ` в любой точке сети `𝕌` стремится к гауссовскому профилю при `n → ∞`. Это обеспечивает устойчивость макроскопических параметров сети (среднее `⟨Δ⟩`, `σ(Δ)`) относительно микроскопических шумов. Теорема объясняет, почему локальный хаос (`λ > Ω`) не разрушает глобальную структуру `𝕌`.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Нормированная сумма** | `Z_n = (S_n⊖n⊗μ):(σ⊗√n)` |
| **Предел** | `lim P(Z_n≤z) = Φ(z)` |
| **Φ(z)** | `(Id:√(D(D(Id))⊗π)) ⊗ ∫ e^(⊖t²:D(D(Id))) dt` |
| **Скорость сходимости** | `O(Id:√n)` |
