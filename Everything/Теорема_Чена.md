# Теорема Чена

## Формулировка

Для любого достаточно большого чётного $N$:
$$N = p + P_2$$

где $p$ — простое, а $P_2$ — число не более чем с двумя простыми множителями.

## Метод

Усиленное решето Бруна-Титчмарша + оценки сумм по простым.

Чен Цзинжунь (1973): доказал, комбинируя решето с оценками сумм характеров.

## Связь с гипотезой Гольдбаха

Гольдбах: $2n = p + p$ (два простых).
Чен: $2n = p + P_2$ (простое + «почти простое»).

Это ближайший результат к Гольдбаху аналитическими методами.

## Обобщения

- **Чен для близнецов:** бесконечно много простых $p$, таких что $p+2 = P_2$
- **Чен для полиномов:** аналогичные результаты для значений полиномов

## Значение

Показывает, что «почти Гольдбах» достижим решето-методами. Полный Гольдбах требует принципиально новых идей.
