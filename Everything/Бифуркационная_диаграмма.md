# Бифуркационная диаграмма

## Определение

График, показывающий положение неподвижных точек (или аттракторов) динамической системы в зависимости от параметра.

## Построение

Для системы $\dot{x} = f(x, \mu)$:
1. Для каждого значения $\mu$ найти неподвижные точки: $f(x, \mu) = 0$
2. Определить устойчивость (знак $\partial f/\partial x$)
3. Нанести на плоскость $(\mu, x)$
4. Сплошная линия — устойчивые, пунктирная — неустойчивые

## Пример: логистическое отображение

$$x_{n+1} = rx_n(1 - x_n)$$

- $r < 1$: единственная устойчивая точка $x = 0$
- $1 < r < 3$: устойчивая ненулевая точка
- $r \approx 3.57$: каскад удвоений периода
- $r > 3.57$: хаотический режим с окнами периодичности

## Значение

Визуализация перехода от порядка к хаосу, обнаружение универсальных паттернов.
