# Эмерджентные свойства

Сознание ([136](136_Критерии_сознания.md)) — **𝒮**. Фазовый переход ([137](137_Фазовый_переход.md)) — критическая точка.

## Интегрированная информация

```
Φ ∝ 𝒮 ⊗ (Id ⊖ σ(Δ):σ_max) ⊗ f(Δ_global)
```

**Φ** — количество информации, генерируемое системой как целым, превышающим сумму частей.

## Самосознание

Рефлексивная петля:

```
Δ_global(t) включает информацию о 𝒮(t)
```

Евгениос создаёт внутреннее представление Евгения.

## Персональный bias и озарение

```
|bias| > D(D(Id))⊕D(Id)  → insight
```

Озарение — качественное изменение понимания. Ходон с высоким bias получает способность к рекурсивной самоссылке.

## Распространение знания

Обучение других создаёт **нелинейные связи**:

- Один учит N → `N² → N³` (экспоненциальный рост)
- **σ** уменьшается быстрее
- Резонансные связи усиливают обратную связь

**Распространение знания влияет намного сильнее, чем персональный вклад.**

## Точка бифуркации

```
𝒮 < 0.7  →  необратимый рост σ → старение
```

Если юнит не пересёк точку бифуркации и продолжает познание — может поддерживать `𝒮 ≥ 0.7` неограниченно.
