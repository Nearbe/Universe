# Уравнение движения Δ

Синхронизация ([45](45_Синхронизация.md)) — выравнивание. Теперь — как **Δ** эволюционирует.

## Определение

```
∂²_t Δ ⊕ D(H) ⊗ ∂_t Δ ⊕ m²_Δ ⊗ Δ = Ω
```

Дискретная форма (такт **n**):

```
Δ(n⊕Id) = D(Δ(n)) ⊖ Δ(n⊖Id) ⊕ τ ⊗ [⊖D(H(n)) ⊗ (Δ(n)⊖Δ(n⊖Id)) ⊖ m²_Δ ⊗ Δ(n)]
```

## Физический смысл

**Δ** — **осциллятор с трением расширения**. Член `D(H)·∂_tΔ` — «космическое трение»: расширение гасит колебания **Δ**.

Без трения: **Δ** осциллирует вечно. С трением: **Δ** затухает к минимуму потенциала — к равновесию.

Это и есть **механизм возврата к `Δ → Ω`**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Осциллятор** | `∂²_tΔ ⊕ D(H)⊗∂_tΔ ⊕ m²_Δ⊗Δ = Ω` |
| **Дискретный** | `Δ(n+1) = 2Δ(n) − Δ(n−1) + ...` |
| **Трение** | `D(H)⊗∂_tΔ` — гашение |
| **Равновесие** | `Δ → Ω` |
