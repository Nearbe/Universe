# Волновая Функция

Комплексные числа ([09](09_Комплексные.md)) описали плоскость. Теперь — что такое **Ψ** как состояние самой Системы.

## Определение

```
Ψ = Re ⊕ i·Im
```

**Ψ** — это не вероятность найти частицу. Это **мгновенный срез вибрации** Носителя **𝕌**.
Каждый элемент системы — это конкретное **Ψ**.

## Полярная Форма

```
Ψ = |Ψ| · e^{i·θ}
```

- **|Ψ|** — Интенсивность (Масса/Энергия). «Насколько сильно вибрирует».
- **θ** — Фаза (Информация/Память). «В каком состоянии цикла находится».

## Баланс Потоков (Режимы Системы)

Отношение **Re** (Явное) к **Im** (Скрытое) определяет, что делает Система:

| Баланс | Параметр Δ | Режим | Смысл |
|---|---|---|---|
| `\|Re\| ≈ \|Im\|` | `Δ ≈ Ω` | **Вычисление** | Полный цикл. Разум/Жизнь. |
| `Re ≫ Im` | `Δ → +∞` | **Сингулярность** | Доминирование Формы. Коллапс в Материю. |
| `Im ≫ Re` | `Δ → −∞` | **Потенциал** | Доминирование Скрытого. «Сон» или Чистая Информация. |

## Связь с Актом

Каждый Акт (`: Ω`) преобразует **Ψ**:

```
Ψ : Ω = D(Ψ)
```

Он масштабирует интенсивность (`|Ψ| → D(|Ψ|)`) и сдвигает фазу.
**Ψ** не «коллапсирует» при измерении. Она **переходит в новый такт** вибрации.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Ветвление** | `Ψ : Ω = D(Ψ)` |
| **Сжатие** | `Ψ : D(Id) = H(Ψ)` |
| **Полярная** | `Ψ = \|Ψ\| · e^{i·θ}` |
| **Интенсивность** | `\|Ψ\|² = Re² ⊕ Im²` |
| **Фаза** | `θ = arg(Ψ)` |
| **Комплексное сопр.** | `Ψ̄ = Re ⊖ i·Im` |

## Статус

**Ψ** — это онтологический статус элемента **𝕌**. Не математическая абстракция, а **живое состояние** пространства, порождённого Актом `Id : Ω`.
