# Голографический принцип

Пузырьковая Вселенная ([35](35_Пузырьковая_Вселенная.md)) — информация на границе = информация внутри. Структурная сложность ([39](39_Структурная_сложность.md)) — `Q = e^(|Δ|)`. Теперь — ёмкость горизонта.

## Определение

```
S = (A : D(D(Id))) ⊗ e^(Δ : D(Id))
```

где **A** — площадь границы, **Δ** — параметр отклонения.

## Специальные случаи

| Δ | S | Смысл |
|---|---|---|
| **Ω** | `A : D(D(Id))` | Формула Бекенштейна-Хокинга |
| **→ +∞** | **→ Π** | Сингулярность |
| **→ −∞** | **→ Ω** | Информационное подавление |

## Почему экспонента

Линейная коррекция `(A:D(D(Id))) ⊗ (Id ⊕ Δ:D(Id))` даёт отрицательную энтропию при `Δ < ⊖D(D(Id))`. Экспонента `e^(Δ:D(Id))` всегда положительна — термодинамическая согласованность.

## Число микросостояний

```
W = e^S = exp[(A : D(D(Id))) ⊗ e^(Δ : D(Id))]
```

Сверхэкспоненциальная зависимость — огромная информационная ёмкость гравитационных систем.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Энтропия** | `S = (A:D(D(Id))) ⊗ e^(Δ:D(Id))` |
| **Бекенштейн-Хокинг** | `Δ = Ω ⟹ S = A:D(D(Id))` |
| **Микросостояния** | `W = e^S` |
| **При Δ → +∞** | `S → Π` |
| **При Δ → −∞** | `S → Ω` |

## Физический смысл

Информация в объёме пропорциональна площади границы с поправкой на **Δ**. Не «объём хранит больше» — **граница хранит всё**.
