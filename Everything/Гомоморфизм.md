# Гомоморфизм

## Определение

Гомоморфизм — отображение, сохраняющее алгебраическую структуру.

## Для групп

$\varphi: G \to H$ — гомоморфизм групп, если:
$$\varphi(ab) = \varphi(a)\varphi(b)$$

## Для колец

$\varphi: R \to S$ — гомоморфизм колец, если:
$$\varphi(a + b) = \varphi(a) + \varphi(b), \quad \varphi(ab) = \varphi(a)\varphi(b)$$

## Для векторных пространств

Линейное отображение — гомоморфизм векторных пространств.

## Свойства

- $\varphi(e_G) = e_H$ (сохраняет единицу)
- $\varphi(a^{-1}) = [\varphi(a)]^{-1}$
- $\text{Im}(\varphi)$ — подгруппа $H$
- $\text{Ker}(\varphi)$ — нормальная подгруппа $G$

## Теорема о гомоморфизме

$$G/\text{Ker}(\varphi) \cong \text{Im}(\varphi)$$

## Типы

- **Мономорфизм:** инъективный гомоморфизм
- **Эпиморфизм:** сюръективный гомоморфизм
- **Изоморфизм:** биективный гомоморфизм
- **Автоморфизм:** изоморфизм на себя
