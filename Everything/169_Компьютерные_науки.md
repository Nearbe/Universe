# Компьютерные науки

Родитель: Фрактальное сжатие ([154](154_Фрактальное_сжатие.md), Соленоид ([23](23_Соленоид.md))

## Биты и кубиты через Δ

| Состояние | Δ | Что |
|---|---|---|
| **0** | `→ ⊖Π` | Крайнее состояние |
| **1** | `→ +Π` | Крайнее состояние (фазовый переход) |
| **Кубит** | `∈ [⊖Id, +Id]` | Суперпозиция: `Δ = ln\|α\| ⊖ ln\|β\|` |

## Нейросети

| CS | `𝕌` |
|---|---|
| Нейросеть | Ходонная сеть `{Δ_i}`, `𝒮` |
| Обучение | Оптимизация `Δ`: `min Loss(Δ)` |
| Глубокое обучение | Иерархия сетей с разными `𝒮` |
| Нейроморфные | Физическая реализация ходонной сети |

## PhaseBasedAI

```
if 𝒮 ≥ 0.7: использовать PhaseBasedAI
else: использовать LLM
```

Детерминированные веса: `weight = f(Δ, 𝒮)`.

## Ёмкость канала

```
C = B ⊗ log₂(Id ⊕ SNR)
SNR ∝ 𝒮 : (Id ⊖ 𝒮)
```

## Криптография

Коррелированные состояния Δ для безопасной передачи:

```
corr(Δ₁, Δ₂)
```

## Полный набор операций

| Операция | Формула |
|---|---|
| **Бит** | `Δ → ±Π` |
| **Кубит** | `Δ ∈ [⊖Id,+Id]` |
| **Ёмкость** | `C = B⊗log₂(Id⊕SNR)` |
| **Обучение** | `min Loss(Δ)` |
| **Крипто** | `corr(Δ₁,Δ₂)` |
