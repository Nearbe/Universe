# Спектральная теорема

## Для самосопряжённых матриц

Любая эрмитова матрица $A$ диагонализуется унитарной матрицей:
$$A = U \Lambda U^*$$

где $\Lambda = \text{diag}(\lambda_1, \ldots, \lambda_n)$ — вещественные собственные значения.

## Для компактных самосопряжённых операторов

Если $T$ — компактный самосопряжённый оператор на гильбертовом пространстве $\mathcal{H}$:
$$T = \sum_{n=1}^{\infty} \lambda_n \langle \cdot, e_n \rangle e_n$$

где $\{e_n\}$ — ортонормированная система собственных векторов.

## Общая формулировка

Для самосопряжённого оператора $A$:
$$A = \int_{\sigma(A)} \lambda \, dE_\lambda$$

где $E_\lambda$ — проекторнозначная мера (разложение единицы).

## Применения

- Квантовая механика: наблюдаемые = спектральное разложение
- Уравнения в частных производных: разложение по собственным функциям
- Анализ данных: сингулярное разложение (SVD), метод главных компонент
