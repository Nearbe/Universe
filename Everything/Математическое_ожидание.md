# Математическое ожидание

## Определение

Для дискретной случайной величины:
$$\mathbb{E}[X] = \sum_i x_i P(X = x_i)$$

Для непрерывной:
$$\mathbb{E}[X] = \int_{-\infty}^{\infty} x f(x) \, dx$$

## Свойства

- **Линейность:** $\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y]$
- **Монотонность:** если $X \geq Y$, то $\mathbb{E}[X] \geq \mathbb{E}[Y]$
- $\mathbb{E}[c] = c$ для константы
- Для независимых: $\mathbb{E}[XY] = \mathbb{E}[X]\mathbb{E}[Y]$

## Примеры

| Распределение | $\mathbb{E}[X]$ |
|---|---|
| Бернулли($p$) | $p$ |
| Биномиальное($n, p$) | $np$ |
| Пуассона($\lambda$) | $\lambda$ |
| Равномерное($a, b$) | $\frac{a+b}{2}$ |
| Нормальное($\mu, \sigma^2$) | $\mu$ |
| Экспоненциальное($\lambda$) | $1/\lambda$ |

## Условное матожидание

$$\mathbb{E}[X|Y] = g(Y), \quad \text{где } g(y) = \mathbb{E}[X|Y=y]$$

Свойство: $\mathbb{E}[\mathbb{E}[X|Y]] = \mathbb{E}[X]$.
