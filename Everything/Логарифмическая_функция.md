# Логарифмическая функция

## Определение

Натуральный логарифм — функция, обратная к экспоненте:
$$\ln: \mathbb{R}^+ \to \mathbb{R}, \quad \ln(e^x) = x$$

Интегральное определение:
$$\ln x = \int_1^x \frac{dt}{t}$$

## Свойства

- $\ln(xy) = \ln x + \ln y$
- $\ln(x^a) = a \ln x$
- $\frac{d}{dx} \ln x = \frac{1}{x}$
- $\ln 1 = 0,\ \ln e = 1$

## Логарифм по основанию $b$

$$\log_b x = \frac{\ln x}{\ln b}$$

## Комплексный логарифм

Многозначная функция на $\mathbb{C} \setminus \{0\}$:
$$\operatorname{Log} z = \ln|z| + i \arg z$$

Главная ветвь: $-\pi < \arg z \leq \pi$.

## Асимптотика

$$\ln n! \sim n \ln n - n \quad \text{(формула Стирлинга)}$$
