# Гомологии

## Определение

Гомологии — последовательность абелевых групп $H_n(X)$, сопоставляемых топологическому пространству $X$.

## Симплициальные гомологии

Для симплициального комплекса $K$:
- $C_n(K)$ — группа $n$-цепей (формальные суммы симплексов)
- $\partial_n: C_n \to C_{n-1}$ — граничный оператор, $\partial^2 = 0$
- $H_n(K) = \ker \partial_n / \text{im } \partial_{n+1}$

## Сингулярные гомологии

Определяются через непрерывные отображения симплексов в пространство.

## Свойства

- **Гомотопическая инвариантность:** $X \simeq Y \implies H_n(X) \cong H_n(Y)$
- **Точная последовательность пары:** $\cdots \to H_n(A) \to H_n(X) \to H_n(X, A) \to H_{n-1}(A) \to \cdots$
- **Аксиома вырезания:** $H_n(X, A) \cong H_n(X \setminus U, A \setminus U)$ при $\overline{U} \subset \text{int}(A)$

## Примеры

- $H_0(S^n) = \mathbb{Z}$ (связность)
- $H_n(S^n) = \mathbb{Z}$
- $H_k(S^n) = 0$ для $0 < k < n$
