# Нормальное распределение

## Определение

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

где $\mu$ — среднее, $\sigma > 0$ — стандартное отклонение.

## Обозначение

$$X \sim \mathcal{N}(\mu, \sigma^2)$$

## Характеристики

- **Математическое ожидание:** $\mathbb{E}[X] = \mu$
- **Дисперсия:** $\text{Var}(X) = \sigma^2$
- **Медиана = мода = $\mu$**
- **Коэффициент асимметрии:** 0 (симметрично)
- **Эксцесс:** 3

## Стандартное нормальное

$$Z \sim \mathcal{N}(0, 1), \quad \phi(z) = \frac{1}{\sqrt{2\pi}} e^{-z^2/2}$$

## Правило трёх сигм

- $P(\mu - \sigma < X < \mu + \sigma) \approx 68.3\%$
- $P(\mu - 2\sigma < X < \mu + 2\sigma) \approx 95.4\%$
- $P(\mu - 3\sigma < X < \mu + 3\sigma) \approx 99.7\%$

## Центральная предельная теорема

Сумма независимых одинаково распределённых величин стремится к нормальному распределению.
