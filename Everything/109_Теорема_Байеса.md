# Теорема Байеса

Вероятность ([27](27_Вероятность.md)) — мера неопределённости.

## Определение

```
P(A|B) = P(B|A) ⊗ P(A) : P(B)
```

## Физический смысл

Байес — **обучение на опыте**.

- `P(A)` — априорная (до данных)
- `P(B|A)` — правдоподобие
- `P(A|B)` — апостериорная (после данных)

Каждый элемент **𝕌** обновляет своё **Δ** по Байесу при получении информации от соседей.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Байес** | `P(A\|B) = P(B\|A)⊗P(A):P(B)` |
| **Полная вероятность** | `P(B) = Σ P(B\|A_i)⊗P(A_i)` |
| **Обновление** | `P_новая = P_старая ⊗ P(данные\|A) : P(данные)` |
