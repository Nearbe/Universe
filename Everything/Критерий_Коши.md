# Критерий Коши

## Для последовательностей

Последовательность $(a_n)$ сходится тогда и только тогда, когда она фундаментальна:
$$\forall \varepsilon > 0 \ \exists N: \ \forall n, m > N \quad |a_n - a_m| < \varepsilon$$

## Для рядов

Ряд $\sum a_n$ сходится тогда и только тогда, когда:
$$\forall \varepsilon > 0 \ \exists N: \ \forall n > m > N \quad \left|\sum_{k=m+1}^n a_k\right| < \varepsilon$$

## Для равномерной сходимости

Последовательность функций $f_n$ сходится равномерно тогда и только тогда, когда:
$$\forall \varepsilon > 0 \ \exists N: \ \forall n, m > N, \ \forall x \quad |f_n(x) - f_m(x)| < \varepsilon$$

## Для интегралов

Несобственный интеграл $\int_a^\infty f(x)\,dx$ сходится тогда и только тогда, когда:
$$\forall \varepsilon > 0 \ \exists M: \ \forall B > A > M \quad \left|\int_A^B f(x)\,dx\right| < \varepsilon$$

## Значение

Позволяет проверять сходимость без знания предела.
