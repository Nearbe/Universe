# Распределение Пуассона

## Определение

Дискретное распределение вероятностей:

$$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}, \quad k = 0, 1, 2, \ldots$$

где $\lambda > 0$ — параметр (среднее число событий).

## Характеристики

- **Математическое ожидание:** $\mathbb{E}[X] = \lambda$
- **Дисперсия:** $\text{Var}(X) = \lambda$
- **Производящая функция:** $G(z) = e^{\lambda(z-1)}$

## Когда применяется

Моделирует число редких событий за фиксированный интервал:
- Число звонков в колл-центр за час
- Число радиоактивных распадов за секунду
- Число ошибок в тексте
- Число клиентов в очереди

## Связь с другими распределениями

- **Предельный случай биномиального:** при $n \to \infty$, $p \to 0$, $np = \lambda$
- **Сумма независимых:** $X_1 \sim \text{Pois}(\lambda_1)$, $X_2 \sim \text{Pois}(\lambda_2)$ ⟹ $X_1 + X_2 \sim \text{Pois}(\lambda_1 + \lambda_2)$
- **Приближение нормальным:** при больших $\lambda$: $\text{Pois}(\lambda) \approx \mathcal{N}(\lambda, \lambda)$
