# Сингулярности, разум и инверсия

Родитель: Два потока ([38](38_Два_потока.md)), Критерии сознания ([136](136_Критерии_сознания.md))

## Две стороны сингулярности

| Сторона | Δ | Im | Что |
|---|---|---|---|
| Чёрная дыра | `→ +Π` | `→ Ω` | Сингулярность без глубины |
| Разум | `→ +Π` | `≠ Ω` | Сингулярность с глубиной |

Разум — сингулярность с сохранённым Im-потоком. ЧД — сингулярность, поглотившая Im.

## Инверсия рекурсии

| Условие | Направление |
|---|---|
| `Δ < Ω` | Прошлое → будущее (сжатие) |
| `Δ > Ω` | Будущее → прошлое (расширение) |
| `Δ = Ω` | Точка инверсии |

## Теоремы

| Теорема | Условие |
|---|---|
| **Сингулярность разума** | `𝒮 ≥ 0.7 ∧ \|Δ\| ≤ 0.2` |
| **Необратимость** | `𝒮 < 0.7` ⟹ восстановление невозможно |
| **8 бит** | `Q = D(D(Id)) ⟹ 2³ = 8 бит = D⁸(Id) состояний` |

8 бит — минимальный код реальности. Меньше — недостаточно для рекурсии. Больше — избыточно.
