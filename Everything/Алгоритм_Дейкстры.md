# Алгоритм Дейкстры

## Задача

Нахождение кратчайших путей от одной вершины до всех остальных во взвешенном графе с **неотрицательными** весами рёбер.

## Алгоритм

```
dijkstra(G, source):
    for each v in G: dist[v] = ∞
    dist[source] = 0
    Q = приоритетная очередь (по dist)
    add all vertices to Q

    while Q not empty:
        u = extract_min(Q)
        for each neighbor v of u:
            if dist[u] + w(u,v) < dist[v]:
                dist[v] = dist[u] + w(u,v)
                decrease_key(Q, v, dist[v])
```

## Сложность

- Наивная реализация: $O(V^2)$
- С двоичной кучей: $O((V + E) \log V)$
- С фибоначчиевой кучей: $O(E + V \log V)$

## Свойства

- Жадный алгоритм (выбирает ближайшую непосещённую вершину)
- Не работает с отрицательными весами (использовать Беллмана-Форда)
- Построение дерева кратчайших путей

## Применения

Маршрутизация в сетях, навигационные системы, планирование.
