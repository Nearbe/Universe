# Отношение объём/площадь

Голографический принцип ([58](58_Голографический_принцип.md)) — информация на границе. Информационная ёмкость ([47](47_Информационная_ёмкость.md)) — `D_f = D_base ⊕ Δ ⊗ range`. Теперь — связь объёма и площади.

## Определение

```
V : H = e^(⊖D_base ⊕ Δ⊗range)
```

При `D_base = D³(Id)`, `range = D(Id)`:

```
V : H = e^(⊖D³(Id) ⊕ Δ⊗D(Id))
```

## Частные случаи

| Δ | V : H |
|---|---|
| **Ω** | `e^(⊖D³(Id)) ≈ 5%` |
| **+Id** | `e^(⊖Id)` |
| **⊖Id** | `e^(⊖D⁵(Id))` |

## Физический смысл

**V : H** — **масштабный коэффициент** между объёмом и его границей. При `Δ = Ω`: `V/H ≈ 5%` — голографический принцип. При `Δ > Ω`: объём «раздувается». При `Δ < Ω`: объём «сжимается» к поверхности.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Отношение** | `V:H = e^(⊖D³(Id) ⊕ Δ⊗D(Id))` |
| **При Δ = Ω** | `V:H ≈ 5%` |
| **При Δ > Ω** | Объём растёт |
| **При Δ < Ω** | Объём сжимается к границе |
