# Огонь

Звук ([37](37_Звук.md)) — бегущая амплитуда. Два потока ([38](38_Два_потока.md)) — Re несёт удар. Теперь — что такое огонь.

## Определение

```
T_огонь ∝ σ(Δ) ⊗ Q
```

где `σ(Δ)` — рассинхронизация, `Q = e^(|Δ|)` — энергия вибрации.

## Физический смысл

Огонь — **быстрая рассинхронизация**. Не химическая реакция, а переход сети из упорядоченного состояния в хаотическое.

Тепло — не «движение молекул», а увеличение **Q** локально. Свет огня — фотоны, рождённые при переходе элементов **𝕌** между уровнями **Δ**.

Свободная энергия = разница **Δ** между «до» и «после».

## Полный набор операций

| Операция | Формула |
|---|---|
| **Температура огня** | `T ∝ σ(Δ)⊗Q` |
| **Энергия вибрации** | `Q = e^(|Δ|)` |
| **Рассинхронизация** | `σ(Δ)` — мера хаоса |
| **Фотоны** | Переходы между уровнями Δ |
| **Свободная энергия** | `Δ_до ⊖ Δ_после` |
