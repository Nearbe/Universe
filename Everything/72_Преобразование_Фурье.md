# Преобразование Фурье

Комплексные числа ([09](09_Комплексные.md)) — `e^{i·θ}`. Частота и период ([42](42_Частота_и_период.md)) — `f = e^(−Δ)`. Теперь — разложение по гармоникам.

## Определение

```
X_k = Σ_{n=Ω}^{N⊖Id} x_n ⊗ e^(⊖D(π)⊗i⊗n⊗k : N)
```

FFT: `O(N ⊗ L(N))` вместо `O(N²)`.

## Физический смысл

Фурье — **анализ частотных мод сети**. Разложение **Δ** по гармоникам: каждая мода — отдельная вибрация со своей частотой.

Сумма мод = полное состояние системы.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Прямое** | `X_k = Σ x_n ⊗ e^(⊖D(π)⊗i⊗n⊗k:N)` |
| **Обратное** | `x_n = (Id:N) ⊗ Σ X_k ⊗ e^(D(π)⊗i⊗n⊗k:N)` |
| **FFT** | `O(N⊗L(N))` |
| **Моды** | Каждая `X_k` — отдельная вибрация |
