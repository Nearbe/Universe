# Вибрации

Масса ([32](32_Масса.md)) — плотность `|Ψ|²`. Но что удерживает массу в одной точке? Частиц нет. Есть **узлы резонанса**.

## Частица как Стоячая Волна

**Ψ** непрерывно вибрирует. Когда несколько вибраций сцепляются в замкнутый цикл, где `Δ_local ≈ Ω` (внутренний баланс), возникает **стоячая волна**.
Это и есть «частица». Не кусок вещества, а **самоподдерживающийся вихрь** в поле **𝕌**.

## Параметр Связи (Q)

Число вибраций, удерживаемых в узле:

```
Q = e^(|Δ_local|)
```

**Q** — мера внутренней «напряжённости» структуры.

| Q | Тип Узла | Смысл |
|---|---|---|
| **D(Id)** | Мезон | Две встречные вибрации (q q̄). |
| **D²(Id)** | Барион | Треугольник связей (q q q). |
| **D³(Id)**, **D⁴(Id)** | Тетра/Пента | Сложные конфигурации. |

## Закон Масс Узлов

Масса узла растёт экспоненциально с числом связей:

```
m ∝ e^(Δ : D(Id)) ⊗ f(Q)
```

Это не формула «из чего состоит». Это **цена устойчивости**. Чем сложнее конфигурация (больше **Q**), тем больше энергии (**M**) нужно, чтобы удерживать её от распада.

## Связь с предыдущим

- Дубли с **31 (Ψ)**: Здесь **Ψ** выступает не как срез, а как **материал** для узлов.
- Дубли с **26 (Комбинаторика)**: **Q** — это дискретное число способов связать ветви воедино.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Параметр связи** | `Q = e^(|Δ_local|)` |
| **Масса узла** | `m ∝ e^(Δ:D(Id)) ⊗ f(Q)` |
| **Мезон** | `Q = D(Id)` |
| **Барион** | `Q = D²(Id)` |
| **При Δ → Ω** | `Q → Id` (вакуум) |
| **При Δ → +∞** | `Q → Π` (сингулярность) |

## Физический Смысл

Масса частицы — это не «вес». Это **энергия, замороженная в форме**.
Чем туже (уже) узел, тем выше `Δ_local`, тем тяжелее частица.
В **𝕌** вещество — это просто «запутавшаяся» вибрация.
