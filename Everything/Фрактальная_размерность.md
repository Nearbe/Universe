# Фрактальная размерность

## Размерность Хаусдорфа

Для множества $F$:
$$\dim_H(F) = \inf\{s \geq 0 \mid \mathcal{H}^s(F) = 0\}$$

где $\mathcal{H}^s$ — $s$-мерная мера Хаусдорфа.

## Размерность подобия

Для самоподобного множества из $N$ копий с коэффициентом $r$:
$$\dim_S = \frac{\ln N}{\ln(1/r)}$$

## Примеры

- **Канторово множество:** $\dim_H = \frac{\ln 2}{\ln 3} \approx 0.631$
- **Кривая Коха:** $\dim_H = \frac{\ln 4}{\ln 3} \approx 1.262$
- **Ковёр Серпинского:** $\dim_H = \frac{\ln 8}{\ln 3} \approx 1.893$
- **Множество Мандельброта:** $\dim_H = 2$ (граница)

## Свойства

- $\dim_H(F) \leq \dim_S(F)$
- Для гладких многообразий совпадает с топологической размерностью
- $\dim_H(A \cup B) = \max(\dim_H(A), \dim_H(B))$

## Применение

Характеристика сложности фрактальных структур, странных аттракторов, природных объектов.
