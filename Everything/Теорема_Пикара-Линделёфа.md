# Теорема Пикара-Линделёфа

## Формулировка

Для задачи Коши:
$$\frac{dy}{dx} = f(x, y), \quad y(x_0) = y_0$$

Если $f(x, y)$ непрерывна в окрестности $(x_0, y_0)$ и удовлетворяет условию Липшица по $y$:
$$|f(x, y_1) - f(x, y_2)| \leq L |y_1 - y_2|$$

то решение существует и единственно в некоторой окрестности $x_0$.

## Доказательство

Метод последовательных приближений Пикара:
$$y_{n+1}(x) = y_0 + \int_{x_0}^x f(t, y_n(t)) \, dt$$

Последовательность сходится к решению по теореме Банаха о неподвижной точке.

## Значение

Гарантирует корректность начальных задач для ОДУ. Без условия Липшица решение может быть неединственным.

## Контрпример без условия Липшица

$$\frac{dy}{dx} = \sqrt{|y|}, \quad y(0) = 0$$

Имеет два решения: $y(x) = 0$ и $y(x) = \frac{x^2}{4}$.
