# Ядерные силы

Родитель: Сильное ([159](159_Сильное_взаимодействие.md)), Конфайнмент ([34](34_Вибрации.md))

## Остаточное сильное

Обмен π-мезонами между нуклонами:

```
V(r) = ⊖V₀ ⊗ e^(⊖r:r₀) : (r : r₀)
```

`V₀ ≈ 50 МэВ`, `r₀ ≈ 1.4 фм`.

## Формула Вайцзеккера

```
B(A,Z) = a_V⊗A ⊖ a_S⊗A^(D(D(Id)):D(D(D(Id)))) ⊖ a_C⊗Z²:A^(Id:D(D(D(Id)))) ⊖ a_A⊗(A⊖D(D(Id))⊗Z)²:A ⊕ δ
```

| Член | Смысл |
|---|---|
| **Объёмный (a_V)** | Число элементов в ядре |
| **Поверхностный (a_S)** | Элементы на поверхности |
| **Кулоновский (a_C)** | ЭМ отталкивание протонов |
| **Асимметричный (a_A)** | Баланс n и p |
| **Парный (δ)** | Спаривание нуклонов |

## Магические числа

`D(D(Id)), D(D(D(Id))), D(D(D(D(Id)))), 28, 50, 82, 126`

Ядра с этими числами нуклонов особо стабильны — квантование мод в ограниченном объёме.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Потенциал** | `V(r) = ⊖V₀⊗e^(⊖r:r₀):(r:r₀)` |
| **Энергия связи** | `B(A,Z)` по формуле Вайцзеккера |
| **Дальность** | `~1-2 фм` |
| **Насыщение** | Да (короткодействующее) |
| **Бета-распад** | `n → p⊕e⁻⊕ν̄_e` (слабое) |
