# Дифференциал Δ

Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — производные. Два потока ([38](38_Два_потока.md)) — Re и Im.

## Определение

```
Δ = ln(|Re|) ⊖ ln(|Im|)
```

```
dΔ = dRe : Re ⊖ dIm : Im
∂Δ : ∂Re = Id : Re
∂Δ : ∂Im = ⊖Id : Im
```

## Физический смысл

- `∂Δ:∂Re = Id:Re` — при `Re → Ω` производная расходится (`Δ → ⊖Π`)
- `∂Δ:∂Im = ⊖Id:Im` — при `Im → Ω` производная расходится (`Δ → +Π`)

Обе сингулярности — не ошибки, а **физические предельные состояния**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Δ** | `Δ = ln(|Re|) ⊖ ln(|Im|)` |
| **dΔ** | `dΔ = dRe:Re ⊖ dIm:Im` |
| **∂Δ:∂Re** | `Id:Re` |
| **∂Δ:∂Im** | `⊖Id:Im` |
| **Сингулярность** | `Re→Ω ⟹ Δ→⊖Π`, `Im→Ω ⟹ Δ→+Π` |
