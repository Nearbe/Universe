# Формула Стирлинга

## Определение

Асимптотическая формула для факториала и гамма-функции:

$$n! \sim \sqrt{2\pi n} \left(\frac{n}{e}\right)^n \quad (n \to \infty)$$

## Обобщение

Для гамма-функции:
$$\Gamma(z) \sim \sqrt{\frac{2\pi}{z}} \left(\frac{z}{e}\right)^z \quad (|z| \to \infty, \ |\arg z| < \pi)$$

## Ряд Стирлинга

Более точное разложение:
$$\ln \Gamma(z) \sim \left(z - \frac{1}{2}\right)\ln z - z + \frac{1}{2}\ln(2\pi) + \sum_{k=1}^{\infty} \frac{B_{2k}}{2k(2k-1)z^{2k-1}}$$

где $B_{2k}$ — числа Бернулли.

## Применение

- Оценки роста $|\Gamma(s)|$ на вертикальных прямых
- Вывод $|\chi(1/2 + it)| = 1$ для дзета-функции Римана
- Теория вероятностей (приближение биномиального распределения)
- Статистическая физика
