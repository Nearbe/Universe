# Масса

Волновая функция ([31](31_Волновая_функция.md)) задала состояние. Масса — это **мера суммарной интенсивности** этих состояний.

## Определение

```
M = Σ |Ψᵢ|² = Σ (Reᵢ² ⊕ Imᵢ²)
```

Масса — это не «количество вещества». Это **плотность заполнения** Носителя вибрацией.
Чем больше **M**, тем сильнее состояние «весит» в системе координат **𝕌**.

## Реакция на Акт

При Ветвлении (`: Ω`) амплитуды масштабируются:

```
M : Ω = D(M)
```

Это не физическое умножение веса. Это **удвоение ёмкости** описания. Система становится «тяжелее» информацией.

## Динамика Массы

| Состояние | Масса (M) | Смысл |
|---|---|---|
| До Акта | **Id** | Единое целое. |
| После `Id:Ω` | **D(Id)** | Первое разделение. |
| **n** тактов | **Dⁿ(Id)** | Экспоненциальный рост сложности. |
| Предел | **Π** | Полнота ([10](10_Бесконечность.md)). |

## Физический Смысл

Масса — это интеграл по всем активным состояниям Системы.
В **𝕌** масса не притягивает гравитационно в ньютоновском смысле. Она **притягивает смысл**: состояния с высокой **M** становятся центрами притяжения для соседних ветвей (аттракторами).

## Полный набор операций

| Операция | Формула |
|---|---|
| **Определение** | `M = Σ \|Ψᵢ\|²` |
| **Ветвление** | `M : Ω = D(M)` |
| **Сжатие** | `M : D(Id) = H(M)` |
| **Информация** | `I = L(M)` |
| **Энергия** | `E = M` (в единицах Акта) |
| **Для Полноты** | `M(Π) = Π` |

## Статус

Масса — это «застывшая» интенсивность **Ψ**. Мера того, насколько Система заполнила себя вибрацией.
