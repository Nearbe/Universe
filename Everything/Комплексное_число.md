# Комплексное число

## Определение

Комплексное число — элемент множества $\mathbb{C} = \{a + bi \mid a, b \in \mathbb{R},\ i^2 = -1\}$.

- $a = \Re(z)$ — действительная часть
- $b = \Im(z)$ — мнимая часть
- $i$ — мнимая единица, $i^2 = -1$

## Основные операции

**Сложение:**
$$(a + bi) + (c + di) = (a + c) + (b + d)i$$

**Умножение:**
$$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$$

**Сопряжение:**
$$\overline{a + bi} = a - bi$$

**Модуль:**
$$|z| = \sqrt{a^2 + b^2} = \sqrt{z \overline{z}}$$

## Полярная форма

$$z = r e^{i\theta} = r(\cos\theta + i\sin\theta)$$

где $r = |z|$, $\theta = \arg(z)$.

## Свойства

- $\overline{z_1 z_2} = \overline{z_1} \cdot \overline{z_2}$
- $|z_1 z_2| = |z_1| \cdot |z_2|$
- $\mathbb{C}$ — алгебраически замкнутое поле (основная теорема алгебры)
