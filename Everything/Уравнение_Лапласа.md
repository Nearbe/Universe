# Уравнение Лапласа

## Определение

$$\Delta u = 0$$

где $\Delta = \sum_{i=1}^n \frac{\partial^2}{\partial x_i^2}$ — оператор Лапласа.

## Гармонические функции

Решения уравнения Лапласа называются гармоническими функциями.

## Свойства

- **Принцип максимума:** гармоническая функция не имеет локальных максимумов внутри области
- **Среднее значение:** $u(x_0) = \frac{1}{|S_r|} \int_{|x-x_0|=r} u(x) \, dS$
- **Бесконечная гладкость:** гармонические функции аналитичны
- **Принцип Дирихле:** решение определяется граничными условиями единственным образом

## Решение в разных областях

**Круг (ряд Пуассона):**
$$u(r, \theta) = \frac{a_0}{2} + \sum_{n=1}^\infty r^n(a_n \cos n\theta + b_n \sin n\theta)$$

**Шар (формула Пуассона):**
$$u(x) = \frac{R^2 - |x|^2}{n \omega_n R} \int_{\partial B_R} \frac{f(y)}{|x-y|^n} \, dS_y$$

## Применения

Электростатика, гидродинамика, теория потенциала, теория вероятностей.
