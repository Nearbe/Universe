# Мера Лебега

## Определение

Мера Лебега $\lambda$ — стандартная мера длины/площади/объёма на $\mathbb{R}^n$.

На $\mathbb{R}$: $\lambda([a, b]) = b - a$.

## Построение

1. Определяется на полукольце интервалов: $\lambda((a, b]) = b - a$
2. Продолжается до внешней меры: $\lambda^*(E) = \inf \sum \lambda(I_k)$
3. Ограничивается на $\sigma$-алгебру измеримых по Лебегу множеств

## Свойства

- Счётная аддитивность: $\lambda(\bigcup E_i) = \sum \lambda(E_i)$ для попарно непересекающихся
- Инвариантность относительно сдвигов: $\lambda(E + a) = \lambda(E)$
- Множества меры 0: счётные множества имеют меру 0
- Полнота: подмножества множеств меры 0 измеримы

## Интеграл Лебега

$$\int f \, d\lambda = \lim_{n \to \infty} \int s_n \, d\lambda$$

где $s_n$ — простые функции, аппроксимирующие $f$.
