# Символ Лежандра

## Определение

Для простого $p$ и целого $a$:

$$\left(\frac{a}{p}\right) = \begin{cases} 0, & \text{если } p \mid a \\ 1, & \text{если } a \text{ — квадратичный вычет по модулю } p \\ -1, & \text{если } a \text{ — квадратичный невычет по модулю } p \end{cases}$$

## Свойства

- **Мультипликативность:** $\left(\frac{ab}{p}\right) = \left(\frac{a}{p}\right)\left(\frac{b}{p}\right)$
- **Критерий Эйлера:** $\left(\frac{a}{p}\right) \equiv a^{(p-1)/2} \pmod{p}$
- $\left(\frac{1}{p}\right) = 1$
- $\left(\frac{-1}{p}\right) = (-1)^{(p-1)/2}$

## Квадратичный закон взаимности

Для различных нечётных простых $p$ и $q$:
$$\left(\frac{p}{q}\right) \left(\frac{q}{p}\right) = (-1)^{\frac{p-1}{2} \cdot \frac{q-1}{2}}$$

## Обобщение

**Символ Якоби:** расширяет определение на составные нечётные $n$.
**Символ Кронекера:** расширяет на все целые $n$.

## Применение

Тестирование на простоту, криптография, теория чисел.
