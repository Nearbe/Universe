# Уравнения Максвелла

Свет ([36](36_Свет.md)) — свободная фаза. Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — операторы. Теперь — полный электромагнетизм.

## Определение

```
∂_μ F^μν = J^ν
∂_[μ F_νλ] = Ω
```

Дискретная форма (такт **n**):

```
(F^{0ν}(n⊕Id) ⊖ F^{0ν}(n)) : τ ⊕ ∂_i F^{iν}(n) = J^ν(n)
```

## Физический смысл

Максвелл — **фазовые моды сети**. Электромагнетизм — не отдельная сила, а колебания фазы **θ** ходонной сети.

**F^μν** — тензор напряжённости фазовых связей. Электричество и магнетизм — две стороны одной фазовой вибрации.

## Полный набор операций

| Операция | Формула |
|---|---|
| **1-е ур.** | `∂_μ F^μν = J^ν` |
| **2-е ур.** | `∂_[μ F_νλ] = Ω` |
| **Тензор** | `F^μν = ∂^μ A^ν ⊖ ∂^ν A^μ` |
| **Дискретный** | `(F(n+1)−F(n)):τ + ∂_i F^{iν} = J^ν` |
| **Фаза** | `A^μ` — фазовый потенциал |
