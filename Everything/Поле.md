# Поле

## Определение

Поле $F$ — коммутативное кольцо с единицей, в котором каждый ненулевой элемент имеет обратный:

$$\forall a \neq 0\ \exists a^{-1}: a \cdot a^{-1} = 1$$

## Аксиомы

$(F, +, \cdot)$ satisfies:
1. $(F, +)$ — абелева группа (нейтраль: 0)
2. $(F\setminus\{0\}, \cdot)$ — абелева группа (нейтраль: 1)
3. Дистрибутивность: $a(b+c) = ab + ac$

## Примеры

- $\mathbb{Q}$ — рациональные числа
- $\mathbb{R}$ — вещественные числа
- $\mathbb{C}$ — комплексные числа
- $\mathbb{F}_p = \mathbb{Z}/p\mathbb{Z}$ — конечное поле из $p$ элементов

## Характеристика

Наименьшее $n$: $1+1+\cdots+1 = 0$ ($n$ раз).
- $\mathbb{Q}, \mathbb{R}, \mathbb{C}$: характеристика 0
- $\mathbb{F}_p$: характеристика $p$

## Расширения полей

$K \supset F$ — расширение, если $K$ — поле, содержащее $F$.

Степень расширения: $[K:F] = \dim_F K$.

## Конечные поля

Любое конечное поле имеет $p^n$ элементов ($p$ — простое). Обозначение: $\mathbb{F}_{p^n}$.
