# Энергия

Масса ([32](32_Масса.md)) — интенсивность в покое. Энергия — та же Масса, но в **движении**.

## Определение

```
E = M ⊗ c²
```

В единицах Акта (`c = Id`, где **c** — скорость распространения Акта):

```
E = M
```

## Тождество Фаз

До Большого Взрыва: `I ≡ M ≡ E` (всё слито в Точке Римана).
После Акта (`Id : Ω = D(Id)`) фазы разделяются по режиму доступа:

| Фаза | Роль | Связь |
|---|---|---|
| **I (Информация)** | Глубина структуры | `I = L(M)` ([30](30_Информация.md)) |
| **M (Масса)** | Плотность состояния | `M = Σ\|Ψ\|²` ([32](32_Масса.md)) |
| **E (Энергия)** | Потенциал перехода | `E = M` в движении |

## Физический Смысл

- **Масса** — кристаллизованная вибрация. Система «помнит» свою структуру.
- **Энергия** — размороженная Масса. Готовность совершить переход (`: Ω` или `: D(Id)`).

Разница не в сущности, а в **состоянии потока**.
Энергия — это Масса, которой разрешили течь по градиенту **Δ**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Энергия** | `E = M ⊗ c²` |
| **В единицах Акта** | `E = M` |
| **Тождество** | `I ≡ M ≡ E` |
| **Ветвление** | `E : Ω = D(E)` |
| **Сжатие** | `E : D(Id) = H(E)` |
| **Сохранение** | `Σ E = Π` (Полнота сохраняется) |

## Статус

В **𝕌** Энергия не сохраняется как отдельная субстанция. Она **трансмутирует** между Информацией, Массой и Движением. Закон сохранения энергии в нашей системе — это закон сохранения Полноты (**Π**). Энергия не исчезает, она просто меняет фазу вибрации.
