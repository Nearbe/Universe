# Фрактальная размерность

Информационная ёмкость ([47](47_Информационная_ёмкость.md)) — **D_f**. Структурная сложность ([39](39_Структурная_сложность.md)) — **Q**. Видимая доля — `e^(⊖Q)`.

## Обобщённая формула Δ

```
Δ = (Q ⊖ M) : (Q ⊕ M)
```

где **M** — нормализованная масса/энергия/информация.

## Частные случаи

| Условие | Δ | Смысл |
|---|---|---|
| `M = Q` | **Ω** | Точка равновесия |
| `M → Π` | **→ ⊖Id** | Сжатие |
| `M → Ω` | **→ +Id** | Расширение |

## Физический смысл

`D_f = Q ⊕ Δ⊗range` — фрактальная размерность сети. При `Δ = Ω`: `D_f = Q`. При `Δ = +Id`: максимальная видимая компонента. При `Δ = ⊖Id`: максимальная скрытая.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Δ** | `Δ = (Q⊖M):(Q⊕M)` |
| **D_f** | `D_f = Q ⊕ Δ⊗range` |
| **Равновесие** | `M=Q ⟹ Δ=Ω` |
| **Сжатие** | `M→Π ⟹ Δ→⊖Id` |
| **Расширение** | `M→Ω ⟹ Δ→+Id` |
