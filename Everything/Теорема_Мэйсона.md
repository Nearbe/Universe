# Теорема Мэйсона

## Формулировка (ABC для многочленов)

Для взаимно простых многочленов $a(t) + b(t) = c(t)$:
$$\max(\deg a, \deg b, \deg c) \leq N_0(abc) - 1$$

где $N_0(abc)$ — число различных корней произведения $abc$.

## Доказательство

Из тождества:
$$\frac{a'}{a} - \frac{b'}{b} = \frac{a'b - ab'}{ab}$$

Числитель имеет степень $\leq \deg a + \deg b - 1$, знаменатель — степень $N_0(ab)$. Оценка числа полюсов даёт неравенство.

## Значение

- Аналог гипотезы abc для многочленов — **полностью доказан**
- Показывает, что abc-феномен имеет алгебраическую природу
- Мотивирует abc-гипотезу для целых через аналогии с теорией функций

## Применения

- Доказательство теоремы Фальтингса для функциональных полей
- Оценка степени решений полиномиальных уравнений
- Теорема Хурвица для накрытий Римановой сферы
