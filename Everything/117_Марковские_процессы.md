# Марковские процессы

Вероятность ([27](27_Вероятность.md)) — мера. Уравнение движения **Δ** ([46](46_Уравнение_движения_Δ.md)) — эволюция.

## Определение

```
P(X_{n⊕Id} = j | X_n = i) = P_ij
```

## Физический смысл

Эволюция **Δ** — марковский процесс. Следующее состояние зависит только от текущего, не от истории.

Элемент «помнит» только себя сейчас.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Переход** | `P(X_{n+1}=j \| X_n=i) = P_ij` |
| **Матрица переходов** | `Σ_j P_ij = Id` |
| **Стационарное** | `π_j = Σ_i π_i⊗P_ij` |
| **Смысл** | Будущее зависит только от настоящего |
