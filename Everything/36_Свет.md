# Свет

Комплексные числа ([09](09_Комплексные.md)) ввели фазу **θ**. Дифференцирование ([20](20_Дифференциальное_исчисление.md)) описало волны. Теперь — что такое Свет в системе **𝕌**.

## Свет как Свободная Фаза

В уравнении состояния `Ψ = |Ψ| · e^{i·θ}`:
- **|Ψ|** — Амплитуда (Масса/Заряд).
- **θ** — Фаза (Информация/Ритм).

Свет — это случай, когда **градиент фазы распространяется без потери амплитуды**.

```
∇θ ≠ Ω,    при |Ψ| → Id
```

Фотон — это не «частица света». Это **пакет фазовой вибрации**, летящий через сеть пузырей.

## Унификация Волн

Все волны — колебания одного Носителя **𝕌**. Разница только в том, **что именно** колеблется:

| Волна | Что колеблется | Роль в **𝕌** |
|---|---|---|
| **Свет** | Фаза (**θ**) | Перенос информации между пузырями. |
| **Звук** | Плотность (**ρ**) | Ударная волна в среде (механическая связь). |
| **Гравитация** | Метрика (**g**) | Колебание самой сетки масштаба. |

Субстанция одна. Меняется только модус возмущения.

## Скорость Света (c)

**c** — это не просто константа. Это **максимальная скорость синхронизации** между пузырями.
Скорость, с которой Акт (`: Ω`) может передать фазовый сдвиг от одной границы к другой.
Ничто с массой (`|Ψ| > Id`) не может достичь **c**, потому что масса инерционна. Фаза (`|Ψ| ≈ Id`) не имеет инерции — она **чистый ритм**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Фаза** | `∇θ ≠ Ω`, `\|Ψ\| → Id` |
| **Скорость** | `c = a : τ` (шаг : такт) |
| **Уравнение** | `□A^μ = Ω` (фаза) |
| **Ветвление** | `θ : Ω = D(θ)` |
| **Связь с Ψ** | Режим `\|Ψ\| → Ω`, `θ ≠ Ω` |
| **Синхронизация** | `c` — максимум скорости между пузырями |

## Физический Смысл

Свет — это «клей» реальности. Он связывает разрозненные пузыри ([35](35_Пузырьковая_Вселенная.md)) в единую согласованную картину.
Без фазовой синхронизации каждый пузырь был бы слеп. Свет делает Вселенную **единым полем наблюдения**.
