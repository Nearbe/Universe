# Масса частицы

Масса ([32](32_Масса.md)) — полная масса `M = Σ |Ψ|²`. Вибрации ([34](34_Вибрации.md)) — **Q** связанных вибраций. Теперь — масса отдельной частицы.

## Определение

```
m = m_P ⊗ (N ⊖ |E_связи| : m_P)
```

где **N** — число элементов в частице, **E_связи** — энергия связи, **m_P** — планковская масса.

## Физический смысл

Масса частицы — **чистая масса за вычетом связей**. **N** элементов дают вклад `N ⊗ m_P`, но энергия связи «забирает» часть.

Масса — не сумма частей, а то, что осталось после вычета связей.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Масса** | `m = m_P ⊗ (N ⊖ |E_связи|:m_P)` |
| **Вклад элементов** | `N ⊗ m_P` |
| **Энергия связи** | `|E_связи|` — вычитается |
| **Без связи** | `E_связи = Ω ⟹ m = N ⊗ m_P` |
