# Масштабирование ёмкости

Родитель: Ёмкость состояний ([156](156_Ёмкость_состояний.md)), Ренормализация ([28](28_Масштабная_инвариантность.md))

## Определение

```
C(Λ) = C₀ ⊗ (Λ : Λ₀)^{D_f ⊖ D(D(Id))}
```

где `Λ` — масштаб наблюдения (глубина Хребта), `C₀` — ёмкость на опорном масштабе `Λ₀`.

## Физический смысл

Ёмкость сети растёт степенным образом с увеличением глубины рекурсии. Показатель степени `D_f ⊖ D(D(Id))` определяет, насколько быстро система накапливает информационный объём при переходе на более мелкие масштабы. При `D_f > D(D(Id))` ёмкость супер-линейна (фрактальное накопление). При `D_f < D(D(Id))` — сублинейна (ограниченная разрешающая способность).

## Полный набор операций

| Операция | Формула |
|---|---|
| **Закон** | `C(Λ) = C₀ ⊗ (Λ:Λ₀)^{D_f⊖D(D(Id))}` |
| **D_f > D(D(Id))** | Супер-линейный рост |
| **D_f = D(D(Id))** | Линейный рост |
| **D_f < D(D(Id))** | Сублинейный рост |
| **Предел Λ→Π** | `C(Λ) → Π` (полная ёмкость) |
