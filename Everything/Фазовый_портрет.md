# Фазовый портрет

## Определение

Фазовый портрет — геометрическое изображение траекторий динамической системы в фазовом пространстве.

## Построение

Для системы $\dot{x} = f(x, y)$, $\dot{y} = g(x, y)$:
1. Найти неподвижные точки: $f = 0$, $g = 0$
2. Линеаризовать в окрестности каждой точки
3. Определить тип точки (узел, седло, фокус, центр)
4. Нарисовать типичные траектории

## Типы неподвижных точек

Для линеаризации $\dot{\mathbf{x}} = A\mathbf{x}$:

| Собственные значения | Тип |
|---|---|
| Оба отрицательные | Устойчивый узел |
| Оба положительные | Неустойчивый узел |
| Разных знаков | Седло |
| Комплексные, Re < 0 | Устойчивый фокус |
| Комплексные, Re > 0 | Неустойчивый фокус |
| Чисто мнимые | Центр |

## Примеры

- **Маятник:** центры (устойчивое равновесие) и седла (неустойчивое)
- **Хищник-жертва:** предельный цикл
- **Лоренц:** странный аттрактор
