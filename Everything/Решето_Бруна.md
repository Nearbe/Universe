# Решето Бруна

## Идея

Модификация решета Эратосфена с использованием принципа включения-исключения до ограниченной глубины.

## Формула

Для множества $\mathcal{A}$ и множества простых $\mathcal{P}$:
$$S(\mathcal{A}, \mathcal{P}, z) = \sum_{d \mid P(z)} \mu(d) |\mathcal{A}_d|$$

где $P(z) = \prod_{p < z} p$, $\mathcal{A}_d = \{a \in \mathcal{A} : d \mid a\}$.

## Метод Бруна

Обрезать суммирование по $d$ на уровне $D$:
$$S \approx \sum_{d \mid P(z),\, d < D} \mu(d) |\mathcal{A}_d|$$

Ошибка контролируется через комбинаторные тождества.

## Результаты

- **Теорема Бруна:** $\sum_{p \text{ близнецы}} \frac{1}{p}$ сходится (ряд Бруна $\approx 1.902$)
- **Близнецы:** $\pi_2(x) \ll \frac{x}{(\ln x)^2}$
- **Гольдбах:** $r_2(N) \ll \mathfrak{S}(N) \frac{N}{(\ln N)^2}$ (верхняя оценка)

## Значение

Первое решето, давшее нетривиальные результаты для аддитивных проблем. Основа для теоремы Чена.
