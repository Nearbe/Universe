# Алгоритм Гровера

## Задача

Поиск в неструктурированной базе данных из $N$ элементов. Классически: $O(N)$ проверок.

## Алгоритм

Квантовый алгоритм находит отмеченный элемент за $O(\sqrt{N})$ операций.

**Шаги:**
1. Инициализация: суперпозиция всех состояний $H^{\otimes n}|0\rangle$
2. **Оракул:** инвертирует фазу отмеченного состояния: $|x\rangle \to -|x\rangle$
3. **Диффузия (отражение относительно среднего):** усиливает амплитуду отмеченного
4. Повторить шаги 2–3 примерно $\frac{\pi}{4}\sqrt{N}$ раз
5. Измерение

## Амплитудное усиление

Каждая итерация поворачивает состояние к отмеченному. Амплитуда отмеченного растёт, остальных — убывает.

## Сложность

- **Классическая:** $O(N)$
- **Квантовая:** $O(\sqrt{N})$
- Ускорение квадратичное, но не экспоненциальное

## Применения

- Поиск в базе данных
- Решение задачи выполнимости (SAT)
- Криптоанализ (ускорение перебора ключей)
- Оптимизация
