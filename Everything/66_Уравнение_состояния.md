# Уравнение состояния

Два потока ([38](38_Два_потока.md)) — Re и Im. Структурная сложность ([39](39_Структурная_сложность.md)) — **Q**. Теперь — давление реальности.

## Определение

```
w = ⊖Id ⊕ Δ : D(D(D(Id)))
```

## Частные случаи

| Δ | w | Физика |
|---|---|---|
| **Ω** | **⊖Id** | Космологическая постоянная |
| **+D(D(D(Id)))** | **Ω** | Нерелятивистское вещество |
| **→ +∞** | **→ +∞** | Жёсткая среда |

## Физический смысл

**w** — **характер давления реальности**. При `Δ = Ω`: `w = ⊖Id` — отрицательное давление, расширение ускоряется. При `Δ > Ω`: давление менее отрицательное — расширение замедляется.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Уравнение** | `w = ⊖Id ⊕ Δ:D³(Id)` |
| **При Δ = Ω** | `w = ⊖Id` (тёмная энергия) |
| **При Δ = D³(Id)** | `w = Ω` (пыль) |
| **При Δ → +∞** | `w → +∞` (жёсткость) |
