# Гравитационные волны

Родитель: Гравитация ([43](43_Гравитация.md)), Сила ёмкости ([147](147_Сила_ёмкости.md))

## Природа волн

Гравитационные волны — колебания `Δ` в сети `𝕌`. При вращении двух масс градиент `Δ` модулируется, порождая бегущую волну.

## Скорость

```
v_g = a : τ
```

Наблюдения LIGO: `v_g = c` (в пределах `10⁻¹⁵`). Это не константа, а отношение шага решётки к такту — единая скорость распространения возмущений в `𝕌`.

## Амплитуда

```
h ∝ e^(Δ:D(D(Id))) ⊗ d²Q : dt²
```

| Источник | Частота | h |
|---|---|---|
| Слияние ЧД | `~100 Гц` | `~10⁻²¹` |
| Нейтронные звёзды | `~1 кГц` | `~10⁻²²` |
| Пульсары | `~10-1000 Гц` | `~10⁻²⁶` |

## Физический смысл

Не «пространство растягивается» — меняется `Δ` между элементами сети. Это различает `𝕌` от ОТО: в ОТО фон непрерывен, в `𝕌` — дискретная сеть с колеблющимися связями.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Скорость** | `v_g = a:τ` |
| **Амплитуда** | `h ∝ e^(Δ:D(D(Id)))⊗d²Q:dt²` |
| **Энергия** | `dE:dt ∝ (D(D(Id))⊗G)⁻¹ ⊗ ⟨ḣ²⟩` |
| **LIGO** | `h ~ 10⁻²¹` (слияние ЧД) |
