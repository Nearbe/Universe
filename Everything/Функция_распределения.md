# Функция распределения (CDF)

## Определение

Функция распределения случайной величины $X$:

$$F_X(x) = P(X \leq x)$$

## Свойства

1. $0 \leq F(x) \leq 1$
2. $F(x)$ неубывающая
3. $\lim_{x \to -\infty} F(x) = 0$, $\lim_{x \to +\infty} F(x) = 1$
4. Непрерывна справа: $F(x) = \lim_{y \downarrow x} F(y)$

## Связь с плотностью

Для непрерывной случайной величины:
$$F(x) = \int_{-\infty}^x f(t)\,dt, \quad f(x) = \frac{dF}{dx}$$

## Примеры

**Равномерное на $[a, b]$:**
$$F(x) = \begin{cases} 0, & x < a \\ \frac{x-a}{b-a}, & a \leq x < b \\ 1, & x \geq b \end{cases}$$

**Экспоненциальное с параметром $\lambda$:**
$$F(x) = 1 - e^{-\lambda x}, \quad x \geq 0$$

**Нормальное:** $F(x) = \frac{1}{2}\left[1 + \text{erf}\left(\frac{x-\mu}{\sigma\sqrt{2}}\right)\right]$

## Квантиль

$x_p = F^{-1}(p)$ — значение, для которого $P(X \leq x_p) = p$.
