# Функция распределения простых чисел

## Определение

$$\pi(x) = \#\{p \text{ prime} \mid p \leq x\}$$

## Теорема о распределении простых чисел

$$\pi(x) \sim \frac{x}{\ln x} \quad \text{при } x \to \infty$$

Эквивалентная формулировка:
$$\pi(x) \sim \text{Li}(x) = \int_2^x \frac{dt}{\ln t}$$

## Явная формула Римана

$$\pi(x) = \text{Li}(x) - \sum_{\rho} \text{Li}(x^\rho) - \ln 2 + \int_x^{\infty} \frac{dt}{t(t^2-1)\ln t}$$

где суммирование ведётся по всем нетривиальным нулям $\zeta(s)$.

## Связь с гипотезой Римана

Гипотеза Римана эквивалентна оценке:
$$\pi(x) = \text{Li}(x) + O(\sqrt{x} \ln x)$$

Это наилучшая возможная оценка ошибки.

## Функция Чебышёва

$$\psi(x) = \sum_{p^k \leq x} \ln p$$

$\psi(x) \sim x$ эквивалентно теореме о распределении простых чисел.
