# Стационарное уравнение Шрёдингера

## Определение

$$\hat{H}\psi = E\psi$$

где $\hat{H} = -\frac{\hbar^2}{2m}\nabla^2 + V(x)$ — оператор Гамильтона, $E$ — энергия.

## Одномерный случай

$$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} + V(x)\psi = E\psi$$

## Примеры решений

**Свободная частица ($V = 0$):**
$$\psi(x) = A e^{ikx} + B e^{-ikx}, \quad E = \frac{\hbar^2 k^2}{2m}$$

**Гармонический осциллятор ($V = \frac{1}{2}m\omega^2 x^2$):**
$$E_n = \hbar\omega\left(n + \frac{1}{2}\right), \quad n = 0, 1, 2, \ldots$$

**Частица в ящике ($V = 0$ при $0 < x < L$, $V = \infty$ иначе):**
$$E_n = \frac{\hbar^2\pi^2 n^2}{2mL^2}, \quad \psi_n(x) = \sqrt{\frac{2}{L}}\sin\left(\frac{n\pi x}{L}\right)$$

## Свойства

- Спектр дискретен для связанных состояний
- Волновые функции ортогональны: $\int \psi_m^* \psi_n \, dx = \delta_{mn}$
- Полнота: любая функция разлагается по собственным функциям
