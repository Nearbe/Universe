# Распределения вероятностей

Вероятность ([27](27_Вероятность.md)) — мера неопределённости. Биномиальное ([99](99_Биномиальное_распределение.md)) — **P(k)**.

## Ключевые распределения

**Гауссовское:**

```
P(Δ) = (Id : √(D(D(Id))⊗π⊗σ²)) ⊗ exp[⊖(Δ⊖⟨Δ⟩)² : (D(D(Id))⊗σ²)]
```

**Пуассона:**

```
P(k) = (λ^k : k!) ⊗ e^(⊖λ)
```

## Физический смысл

Распределение **Δ** — **статистика отклонений**. Гауссовское: большинство элементов близки к среднему, редкие отклонения экспоненциально маловероятны.

Если распределение не гауссовское — система в фазовом переходе.

## Полный набор операций

| Распределение | Формула | Когда |
|---|---|---|
| **Гауссовское** | `exp[⊖(Δ−⟨Δ⟩)²:(D(D(Id))⊗σ²)]` | Равновесие |
| **Пуассона** | `(λ^k:k!)⊗e^(⊖λ)` | Редкие события |
| **Биномиальное** | `C(n,k)⊗p^k⊗(Id⊖p)^(n⊖k)` | Двоичный выбор |
