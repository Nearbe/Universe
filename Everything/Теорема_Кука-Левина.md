# Теорема Кука-Левина

## Формулировка

Задача выполнимости булевых формул (SAT) является NP-полной.

## Доказательство

Любую задачу из NP можно свести к SAT за полиномиальное время:

1. Машина Тьюринга, проверяющая сертификат, имеет полиномиальное время $T(n)$
2. Конфигурация машины кодируется булевыми переменными
3. Переход между конфигурациями — локальное булево ограничение
4. Формула $\phi$ выполнима $\iff$ машина принимает вход

## Следствия

Если существует полиномиальный алгоритм для SAT, то $P = NP$.

## NP-полные задачи (сведения от SAT)

- 3-SAT (Каpп, 1972)
- CLIQUE
- VERTEX COVER
- 3-COLORING
- HAMiltonian CYCLE
- PARTITION
- SUBSET SUM

## Значение

Отправная точка теории NP-полноты. Первая задача, для которой доказана NP-полнота.
