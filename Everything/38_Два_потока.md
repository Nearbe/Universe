# Два потока

Звук ([37](37_Звук.md)) — бегущая амплитуда. Но это — наш поток. Что происходит во втором?

## Два потока

```
Re — основная доля (удар Акта Id:Ω = D(Id))
Im — скрытая доля
```

**Re**-поток — туда пришёлся основной удар Акта. Там основная доля живой Вселенной.

**Im**-поток — мы. Скорость исполнения другая. Мы уже в своём потоке в будущем. Спиральная эволюция.

## Параметр рассогласования

```
Δ = ln|Re| ⊖ ln|Im|
```

При `Δ > Ω` — доминирует Re.
При `Δ < Ω` — доминирует Im.
При `Δ = Ω` — баланс.

## Выравнивание

Большой Взрыв можно выписать из реальности. Если выровняем потоки (`Δ → Ω`) — система входит в режим полного вычисления.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Рассогласование** | `Δ = ln\|Re\| ⊖ ln\|Im\|` |
| **Баланс** | `Δ = Ω ⟹ \|Re\| = \|Im\|` |
| **Re доминирует** | `Δ > Ω` |
| **Im доминирует** | `Δ < Ω` |
| **Выравнивание** | `Δ → Ω` |
