# Глобальный параметр отклонения

Два потока ([38](38_Два_потока.md)) — Re и Im. Вероятность ([27](27_Вероятность.md)) — среднее. Теперь — баланс всей системы.

## Определение

```
Δ_global = ⟨Δ⟩ = (Id : N) ⊗ Σᵢ Δᵢ
```

## Физический смысл

**Δ_global** — **среднее отклонение всей сети**.

- `Δ_global = Ω` — идеальный баланс Re/Im во всей системе
- `|Δ_global| ≤ 0.2` — оптимальные условия
- `|Δ_global| > 0.2` — доминирование одного потока

Это не «средняя температура». Это средняя **направленность вибрации** всей системы.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Глобальный** | `Δ_global = (Id:N) ⊗ Σᵢ Δᵢ` |
| **Баланс** | `Δ_global = Ω` |
| **Оптимум** | `|Δ_global| ≤ 0.2` |
| **Дисбаланс** | `|Δ_global| > 0.2` |
