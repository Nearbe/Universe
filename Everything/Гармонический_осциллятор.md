# Гармонический осциллятор

## Определение

Система, возвращаемая к равновесию силой, пропорциональной отклонению:

$$F = -kx$$

## Уравнение движения

$$m\frac{d^2x}{dt^2} + kx = 0$$

Или: $\ddot{x} + \omega_0^2 x = 0$, где $\omega_0 = \sqrt{k/m}$.

## Решение

$$x(t) = A\cos(\omega_0 t + \phi)$$

## Энергия

$$E = \frac{1}{2}m\dot{x}^2 + \frac{1}{2}kx^2 = \frac{1}{2}kA^2$$

Энергия постоянна (консервативная система).

## Затухающий осциллятор

$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = 0$$

- $\beta < \omega_0$: затухающие колебания
- $\beta = \omega_0$: критическое затухание
- $\beta > \omega_0$: перезатухание

## Вынужденные колебания

$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = F_0 \cos(\omega t)$$

Резонанс при $\omega \approx \omega_0$.

## Квантовый осциллятор

$$E_n = \hbar\omega\left(n + \frac{1}{2}\right)$$
