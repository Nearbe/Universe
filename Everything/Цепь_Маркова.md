# Цепь Маркова

## Определение

Последовательность случайных величин $X_0, X_1, X_2, \ldots$ удовлетворяет свойству Маркова, если:

$$P(X_{n+1} = x | X_0, \ldots, X_n) = P(X_{n+1} = x | X_n)$$

Будущее зависит только от настоящего, не от прошлого.

## Матрица перехода

Для конечного числа состояний:
$$P_{ij} = P(X_{n+1} = j | X_n = i)$$

Матрица $P = (P_{ij})$ — стохастическая: $\sum_j P_{ij} = 1$.

## Классификация состояний

- **Возвратное:** система возвращается с вероятностью 1
- **Транзитивное:** вероятность возврата < 1
- **Периодическое:** возврат возможен только через кратные $d$ шагов
- **Эргодическое:** возвратное, апериодическое, с конечным средним временем возврата

## Стационарное распределение

Вектор $\pi$: $\pi P = \pi$, $\sum \pi_i = 1$.

Для эргодической цепи существует единственное $\pi$, и $P^n_{ij} \to \pi_j$ при $n \to \infty$.

## Применения

Моделирование очередей, финансы, обработка естественного языка, MCMC-методы.
