# Теория Ландау фазовых переходов

## Феноменологический подход

Свободная энергия как функционал параметра порядка $\eta$:

$$F(\eta) = F_0 + a(T - T_c)\eta^2 + \frac{b}{2}\eta^4 + \cdots$$

## Минимизация

- **$T > T_c$:** минимум при $\eta = 0$ (симметричная фаза)
- **$T < T_c$:** минимум при $\eta = \pm\sqrt{-a(T-T_c)/b}$ (нарушенная симметрия)

## Критические индексы (среднее поле)

- $\eta \sim (T_c - T)^\beta$, $\beta = 1/2$
- Теплоёмкость: скачок ($\alpha = 0$)
- Восприимчивость: $\chi \sim |T - T_c|^{-\gamma}$, $\gamma = 1$

## Классификация

- **Первого рода:** $b < 0$, нужен член $\eta^6$, есть гистерезис
- **Второго рода:** $b > 0$, непрерывный переход

## Область применимости

Работает выше верхней критической размерности ($d_c = 4$ для $\phi^4$). Ниже — не учитывает флуктуации (ренормализационная группа исправляет).

## Значение

Универсальность: разные системы с одинаковой симметрией имеют одинаковые критические индексы.
