# Производящая функция

## Для последовательности вероятностей

Для дискретной случайной величины $X$ с $P(X = k) = p_k$:

$$G(z) = \mathbb{E}[z^X] = \sum_{k=0}^{\infty} p_k z^k$$

## Свойства

- $G(1) = 1$ (нормировка)
- $\mathbb{E}[X] = G'(1)$
- $\mathbb{E}[X(X-1)] = G''(1)$
- $\text{Var}(X) = G''(1) + G'(1) - [G'(1)]^2$

## Для суммы независимых

Если $X$ и $Y$ независимы:
$$G_{X+Y}(z) = G_X(z) \cdot G_Y(z)$$

## Примеры

| Распределение | $G(z)$ |
|---|---|
| Бернулли($p$) | $q + pz$ |
| Биномиальное($n, p$) | $(q + pz)^n$ |
| Пуассона($\lambda$) | $e^{\lambda(z-1)}$ |
| Геометрическое($p$) | $\frac{p}{1-qz}$ |

## Производящая функция моментов

$$M(t) = \mathbb{E}[e^{tX}] = G(e^t)$$

## Применения

- Комбинаторика (подсчёт конфигураций)
- Теория вероятностей (распределение сумм)
- Теория очередей
- Ветвящиеся процессы
