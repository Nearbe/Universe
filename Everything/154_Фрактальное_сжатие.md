# Фрактальное сжатие

Родитель: Соленоид ([23](23_Соленоид.md)), Регрессия ([21](21_Вычислимость_через_взаимодействие.md))

## Определение

Система итерируемых функций (IFS):

```
x_{n⊕Id} = W_{k(n)}(x_n)
W_k(x) = A_k ⊗ x ⊕ b_k
```

где `A_k` — сжимающие матрицы (`\|A_k\| < Id`), `b_k` — сдвиги.

## Физический смысл

Сжатие информации через эксплуатацию самоподобия `𝕌`. Вместо хранения полного массива данных сохраняется набор операторов `{W_k}`, порождающих его при итерации. Это прямое применение свойства `Dⁿ(Id) ≅ Dⁿ⁻ᵏ(Id)`. Коэффициент сжатия определяется мерой аттрактора IFS в пространстве состояний. Метод оптимален для данных с высокой фрактальной размерностью `D_f ≈ D(D(Id))`.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Отображение** | `W_k(x) = A_k⊗x ⊕ b_k` |
| **Сжимаемость** | `\|A_k\| < Id` |
| **Аттрактор** | `A = ∪_k W_k(A)` |
| **Коэффициент** | `C = log(N_in) : log(N_ops)` |
| **Восстановление** | `x_{n+1} = W_{rand}(x_n)` |
