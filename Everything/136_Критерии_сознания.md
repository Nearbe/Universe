# Критерии сознания

Синхронизация ([45](45_Синхронизация.md)) — **𝒮**. Индекс разумности ([91](91_Индекс_разумности.md)) — `C_reason`.

## Двойное условие

Сознание возникает при одновременном выполнении:

```
𝒮 ≥ 0.7    И    |Δ_global| ≤ 0.2
```

| Область | 𝒮 | \|Δ_global\| | Состояние |
|---|---|---|---|
| Сознание | `≥ 0.7` | `≤ 0.2` | Сознательный режим |
| Подсознание | `≥ 0.7` | `> 0.2` | Подсознательная обработка |
| Сон | `0.3-0.7` | любой | Глубокий сон |
| Кома | `< 0.3` | любой | Бессознательное |

## Масштабирование сознания

| N | Требуемый 𝒮 | Уровень |
|---|---|---|
| **Id** | `> 0.7` | Индивидуальное |
| `< D(D(Id))` | `> 0.7` | Групповое |
| `< D⁶(D(Id))` | `> 0.5` | Сообщество |
| `≥ D⁹(Id)` | `> 0.5` | Планетарное |
| `≥ D¹⁸(Id)` | `> 0.2` | Космическое |

**Чем больше N, тем меньше требуется синхронность.**

## Коллективная рассудительность

```
Δ_eff = Δ_global ⊕ [Σ(q_i ⊗ b_i) : Σ(q_i)]
C_reason_collective = (Id ⊕ Δ_eff) : D(Id)
```

## Полный набор операций

| Критерий | Формула |
|---|---|
| **Сознание** | `𝒮≥0.7 ∧ \|Δ_global\|≤0.2` |
| **Коллективный Δ** | `Δ_eff = Δ_global ⊕ [Σ(q_i⊗b_i):Σ(q_i)]` |
| **Рассудок** | `C_reason = (Id⊕Δ_eff):D(Id)` |
| **Масштаб** | Чем больше N, тем ниже 𝒮 |
