# Статистическая сумма

## Определение

Статистическая сумма (Zustandssumme) $Z$ — центральная величина в статистической механике:

$$Z = \sum_i e^{-\beta E_i}$$

где $E_i$ — энергия $i$-го микросостояния, $\beta = 1/(k_B T)$.

Для непрерывного спектра:
$$Z = \int g(E) e^{-\beta E} \, dE$$

## Связь с термодинамикой

Свободная энергия Гельмгольца:
$$F = -k_B T \ln Z$$

Все термодинамические величины выводятся из $Z$:
- **Внутренняя энергия:** $U = -\frac{\partial \ln Z}{\partial \beta}$
- **Энтропия:** $S = -\frac{\partial F}{\partial T}$
- **Давление:** $P = -\frac{\partial F}{\partial V}$

## Каноническое распределение

Вероятность нахождения системы в состоянии $i$:
$$P_i = \frac{e^{-\beta E_i}}{Z}$$

## Большой канонический ансамбль

Для систем с переменным числом частиц:
$$\mathcal{Z} = \sum_{N, i} e^{-\beta(E_i - \mu N)}$$
