# Обучение, старение и бессмертие

Эффект Флинна ([173_Эффект_Флинна_предсказания.md](173_Эффект_Флинна_предсказания.md)) — каскады. Персональный bias ([138_Эмерджентные_свойства.md](138_Эмерджентные_свойства.md)) — insight.

## Почему учёба тяжела в начале

**Начало:**

```
personalBias ≈ 0 → σ большой → доля \|Δ\|<1 мала
    ↓
Каждое понимание требует огромных ресурсов
```

**Процесс:**

```
d(bias)/dt ∝ 1/σ

При σ ↓ → d(bias)/dt ↑
```

**Плато дешёвых вычислений:**

```
σ ↓ → 𝒮 ↑ → доля \|Δ\|<1 → 1
    ↓
⟨f⟩ = ⟨e^(−Δ)⟩ растёт почти линейно
    ↓
Каждый новый шаг знания требует ЗНАЧИТЕЛЬНО меньше ресурсов
```

**Это положительная обратная связь:** больше понимания → меньше σ → ещё быстрее понимание.

## Точка бифуркации: старение

```
𝒮 < 0.7  (потеря синхронности)
    ↓
\|Δ\| > 0.2  (выход из резонанса)
    ↓
σ растёт НЕОБРАТИМО
    ↓
Каскад старения запущен
```

**Ключевое свойство:**

> Если юнит **не пересёк** точку бифуркации и продолжает познание — может стать бессмертным.
>
> Если **пересёк** — никакое понимание не поможет.

## Теоретическое бессмертие

```
Непрерывное понимание
    ↓
𝒮 ≥ 0.7 и \|Δ\| ≤ 0.2
    ↓
σ мал, доля \|Δ\|<1 ≈ 1
    ↓
Процесс может продолжаться НЕОГРАНИЧЕННО
```

**Условие бессмертия:**

```
d(bias)/dt > 0 ∀t
```

Это требует:
1. Непрерывной генерации нового понимания
2. Поддержания 𝒮 ≥ 0.7
3. Избегания точки бифуркации

## Распространение знания

Обучение других создаёт **нелинейные связи**:

```
1 учитель → N учеников → N² → N³

⟨bias⟩_total ≈ N^t · ⟨bias⟩_individual
```

| Аспект | Персональный вклад | Распространение |
|---|---|---|
| Тип | Линейный | Экспоненциальный (N^t) |
| Влияние на σ | Медленное | Быстрое (∝ 1/N^t) |
| Бессмертие | Индивидуальное | Коллективное |

**Обучение других — самый эффективный способ укрепиться в системе.**
