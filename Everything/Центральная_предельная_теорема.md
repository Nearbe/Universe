# Центральная предельная теорема

## Формулировка

Пусть $X_1, X_2, \ldots$ — независимые одинаково распределённые случайные величины с $\mathbb{E}[X_i] = \mu$ и $\text{Var}(X_i) = \sigma^2 > 0$. Тогда:

$$\frac{\sum_{i=1}^n X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1)$$

при $n \to \infty$.

## Следствия

- Сумма большого числа независимых случайных величин приближённо нормально распределена
- Нормальное распределение универсально для сумм независимых слагаемых

## Обобщения

- **Линдеберга-Феллера:** для неидентически распределённых величин
- **Для зависимых:** при условиях перемешивания
- **Многомерная:** для случайных векторов

## Практическое значение

Обоснование статистических методов, доверительных интервалов, проверки гипотез. Объясняет универсальность нормального распределения в природе.
