# Критическая прямая

## Определение

Критическая прямая — вертикальная прямая $\Re(s) = 1/2$ в комплексной плоскости.

## Контекст

Возникает в теории дзета-функции Римана. Нетривиальные нули $\zeta(s)$ лежат в критической полосе $0 < \Re(s) < 1$.

## Гипотеза Римана

Утверждает, что все нетривиальные нули $\zeta(s)$ лежат на критической прямой.

## Значение

Расположение нулей относительно критической прямой определяет точность формулы Римана для $\pi(x)$ — количества простых чисел, не превосходящих $x$.

Если $\Re(\rho) = 1/2$ для всех нетривиальных нулей:
$$\pi(x) = \text{Li}(x) + O(\sqrt{x} \log x)$$
