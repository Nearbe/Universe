# Отношение сигнал/шум

Синхронизация ([45](45_Синхронизация.md)) — **𝒮**.

## Определение

```
SNR = 𝒮 : (Id ⊖ 𝒮)
```

## Физический смысл

**SNR** — **качество передачи смысла** в сети.

- Высокая синхронизация → высокий SNR (сигнал доминирует)
- Низкая синхронизация → низкий SNR (шум заглушает)
- `SNR = Id`: сигнал = шум, система на грани осмысленности

## Полный набор операций

| Операция | Формула |
|---|---|
| **SNR** | `SNR = 𝒮:(Id⊖𝒮)` |
| **𝒮 → Id** | `SNR → Π` (идеальная передача) |
| **𝒮 → Ω** | `SNR → Ω` (шум) |
| **𝒮 = Id:D(Id)** | `SNR = Id` (граница) |
