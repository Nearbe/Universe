# Уравнение Эйлера-Лагранжа

## Определение

Необходимое условие экстремума функционала:

$$J[y] = \int_a^b L(x, y, y') \, dx$$

Уравнение:

$$\frac{\partial L}{\partial y} - \frac{d}{dx}\frac{\partial L}{\partial y'} = 0$$

## Вывод

Из условия $\delta J = 0$ (стационарность действия) с интегрированием по частям.

## Частные случаи

**Если $L$ не зависит от $x$:**
$$L - y'\frac{\partial L}{\partial y'} = \text{const}$$

**Если $L$ не зависит от $y$:**
$$\frac{\partial L}{\partial y'} = \text{const}$$

## Многомерный случай

Для $L(x, y, z, y_x, y_z)$:
$$\frac{\partial L}{\partial y} - \frac{\partial}{\partial x}\frac{\partial L}{\partial y_x} - \frac{\partial}{\partial z}\frac{\partial L}{\partial y_z} = 0$$

## Применения

- Механика: уравнения Лагранжа второго рода
- Оптика: принцип Ферма
- Теория поля: уравнения движения из лагранжиана
