# Метод Монте-Карло

Вероятность ([27](27_Вероятность.md)) — мера. Интегральное исчисление ([21](21_Интегральное_исчисление.md)) — интегралы.

## Интегрирование

```
∫ f(x) dx ≈ (V : N) ⊗ Σ f(x_i)
```

## MCMC

Принятие: `min(Id, P(x') : P(x))`

## Физический смысл

Оценка параметров сети из экспериментальных данных. Случайная выборка вместо точного вычисления.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Интеграл** | `∫f(x)dx ≈ (V:N)⊗Σf(x_i)` |
| **MCMC принятие** | `min(Id, P(x'):P(x))` |
| **Выборка** | Случайные точки → оценка |
| **Точность** | `O(√N)` |
