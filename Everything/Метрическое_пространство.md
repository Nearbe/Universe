# Метрическое пространство

## Определение

Метрическое пространство — пара $(X, d)$, где $X$ — множество, $d: X \times X \to \mathbb{R}$ — функция расстояния, удовлетворяющая:

1. $d(x, y) \geq 0$, причём $d(x, y) = 0 \iff x = y$
2. $d(x, y) = d(y, x)$ (симметрия)
3. $d(x, z) \leq d(x, y) + d(y, z)$ (неравенство треугольника)

## Примеры

- **Евклидово:** $d(x, y) = \sqrt{\sum (x_i - y_i)^2}$
- **Манхэттенское:** $d(x, y) = \sum |x_i - y_i|$
- **Дискретное:** $d(x, y) = 0$ если $x = y$, иначе $1$
- **Пространство непрерывных функций:** $d(f, g) = \max |f - g|$

## Открытые и замкнутые множества

- **Открытый шар:** $B_r(x) = \{y \mid d(x, y) < r\}$
- **Открытое множество:** объединение открытых шаров
- **Замкнутое:** дополнение открытого

## Полнота

Пространство полно, если каждая фундаментальная последовательность сходится к элементу этого пространства.
