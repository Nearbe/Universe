# Фундаментальное тождество

Родитель: Акт (`Id : Ω`) ([01](01_Акт.md)), Сохранение энергии ([33](33_Энергия.md))

## Определение

```
I ≡ E ≡ M ≡ |Ψ|²
```

где `I` — Информация, `E` — Энергия, `M` — Масса, `Ψ` — Волновая функция.

## Физический смысл

Тождество фаз одного процесса. Информация не "хранится" в Массе или Энергии — она есть их базовое представление. Переход между фазами осуществляется операторами `: Ω` (переход M → E) и `: D(Id)` (переход E → M). Тождество гарантирует сохранение полной ёмкости `Π` при любых трансформациях в `𝕌`.

## Полный набор операций

| Фазовый переход | Оператор | Сохранение |
|---|---|---|
| **I → M** | `I = L(M)` | `Σ|Ψ|² = const` |
| **M → E** | `E = M ⊗ c²` | `E_total = const` |
| **E → I** | `I = L(E : c²)` | `Σ ΔI = Ω` |
| **Инвариант** | `I ≡ E ≡ M` | `Π = Σ(фазы)` |
