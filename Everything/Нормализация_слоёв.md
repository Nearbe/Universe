# Нормализация слоёв (Layer Normalization)

## Определение

Нормализация активаций внутри каждого примера (в отличие от Batch Norm, нормализующего по батчу).

## Формула

$$\text{LayerNorm}(x) = \frac{x - \mu}{\sqrt{\sigma^2 + \varepsilon}} \cdot \gamma + \beta$$

где $\mu = \frac{1}{d}\sum x_i$, $\sigma^2 = \frac{1}{d}\sum (x_i - \mu)^2$.

$\gamma, \beta$ — обучаемые параметры масштаба и сдвига.

## Отличие от Batch Normalization

| | Batch Norm | Layer Norm |
|---|---|---|
| Нормализация | по батчу | по признакам |
| Зависимость от batch size | Да | Нет |
| RNN/Transformers | Плохо | Хорошо |

## Применения

- **Transformers:** стандартный компонент (Pre-LN или Post-LN)
- **RNN:** стабилизирует обучение
- **BERT, GPT:** LayerNorm после каждого подслоя

## Варианты

- **RMSNorm:** только нормировка корнем из среднего квадрата (без вычитания среднего)
- **GroupNorm:** нормализация по группам каналов
- **InstanceNorm:** нормализация по каждому каналу отдельно
