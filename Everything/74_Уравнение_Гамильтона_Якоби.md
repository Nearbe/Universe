# Уравнение Гамильтона-Якоби

Уравнение Шрёдингера ([44](44_Уравнение_Шрёдингера.md)) — действительная часть. Квантовый потенциал ([68](68_Квантовый_потенциал.md)) — **Q**.

## Определение

```
∂S/∂t ⊕ (∇S)² : D(m) ⊕ Q ⊕ V = Ω
```

## Физический смысл

Гамильтон-Якоби — **классический предел квантового намерения**. **S** — действие, «путь наименьшего сопротивления».

**Q** — квантовая поправка: когда `√ρ` неоднороден, возникает «квантовая сила», отклоняющая от классического пути. Без **Q** — Ньютон ([49](49_Второй_закон_Ньютона.md)).

## Полный набор операций

| Операция | Формула |
|---|---|
| **Полное** | `∂S/∂t ⊕ (∇S)²:D(m) ⊕ Q ⊕ V = Ω` |
| **Без Q** | `∂S/∂t ⊕ (∇S)²:D(m) ⊕ V = Ω` → Ньютон |
| **Действие** | `S` — «путь наименьшего сопротивления» |
| **Квант. поправка** | `Q` отклоняет от классики |
