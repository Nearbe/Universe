# Оценка ошибок

Метод конечных разностей ([104](104_Метод_конечных_разностей.md)) — дискретизация.

## Погрешности

| Метод | Порядок |
|---|---|
| Эйлер | `O(h)` |
| RK4 | `O(h⁴)` |
| Конечные разности | `O(h²)` |

## Физический смысл

Накопление ошибок при моделировании **Δ**. Экспоненциальный рост для неустойчивых задач (`Δ → ±∞`).

## Полный набор операций

| Метод | Погрешность |
|---|---|
| **Эйлер** | `O(h)` — первый порядок |
| **RK4** | `O(h⁴)` — четвёртый порядок |
| **Конечные разности** | `O(h²)` — второй порядок |
