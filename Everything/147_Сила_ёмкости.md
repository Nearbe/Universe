# Сила ёмкости

Родитель: Голографический принцип ([58](58_Голографический_принцип.md)), Энтропия ([166](166_Термодинамика.md))

## Определение

```
F = T ⊗ ∇C
```

где `T` — температура, `C` — информационная ёмкость границы, `∇` — градиент по координатам сети.

## Физический смысл

Сила возникает как отклик системы на градиент информационной ёмкости. При наличии разницы `C` между соседними узлами сети, возникает поток, выравнивающий ёмкость. Это термодинамическая интерпретация гравитации и других взаимодействий: сила есть не первичная сущность, а производная энтропийного градиента `∇C`. Формула воспроизводит закон Ньютона и уравнения поля Эйнштейна в пределе `Δ → Ω`.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Сила** | `F = T⊗∇C` |
| **Ёмкость** | `C = (A:D³(Id)) ⊗ e^(Δ:D(Id))` |
| **Градиент** | `∇C = (∂C:∂x_i)` |
| **Ньютоновский предел** | `F = m⊗a` при `Δ→Ω, C∝r` |
| **Гравитационный предел** | `F = G⊗m₁⊗m₂:r²` |
