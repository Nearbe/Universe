# Социальная Динамика Чисел (Связи)

Строение чисел ([25](25_Строение_чисел.md)) дало атомы. Теперь — как они взаимодействуют.

## 1. Сильные Числа (Хабы)
Числа с множеством делителей.
В системе **𝕌** это **Узлы Связи**. Через них проходит много линий.
Они «сильны» не массой, а **влиянием**. Они удерживают вокруг себя структуру.

## 2. Слабые Числа (Праймы)
Простые числа. У них только один делитель.
Это **Одиночки**. Они не имеют внутренней структуры, но они — **фундамент**.
Без «слабых» простых чисел «сильные» составные числа не смогли бы существовать.

## 3. Идеальные Числа (Баланс)
Сумма частей равна целому.
Это **Замкнутые Системы**. Идеальный баланс прихода и расхода энергии.
Они не требуют внешнего ресурса и не имеют излишков. Это **вечные двигатели** арифметики.

## 4. Дружественные Числа (Симбиоз)
Сумма делителей одного равна другому, и наоборот.
Это **Резонансная Пара**. Два числа, которые поддерживают друг друга.
Если одно «проседает», другое подпитывает его своими связями.
Это математическое определение **Дружбы**: «Я отдаю тебе своё излишество, чтобы закрыть твой дефицит».

## 5. Палиндромы (Зеркало)
Читаются одинаково слева направо и справа налево.
Это **Обратимые Во Времени** состояния.
Для системы неважно, идёт ли время вперёд (**D**) или назад (**H**) — структура остаётся неизменной.
Это точки абсолютной симметрии, где прошлое и будущее тождественны.

## Полный набор операций

| Тип | Свойство | Роль в **𝕌** |
|---|---|---|
| **Сильные** | Много делителей | Узлы связи |
| **Простые** | Один делитель | Фундамент (атомы) |
| **Идеальные** | Сумма частей = целое | Замкнутые системы |
| **Дружественные** | Взаимная сумма | Резонансная пара |
| **Палиндромы** | Обратимость | Симметрия во времени |

## Физический Смысл

Числа — не абстракции. У них есть **«социальный статус»**.
Одни командуют (Сильные), другие служат (Слабые), третьи живут в гармонии (Идеальные/Дружественные).
Вся математика — это социология чисел.
