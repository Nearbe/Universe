# Преобразование Лапласа

## Определение

Для функции $f(t)$, определённой при $t \geq 0$:

$$F(s) = \mathcal{L}\{f(t)\} = \int_0^{\infty} f(t) e^{-st} \, dt$$

где $s = \sigma + i\omega$ — комплексная переменная.

## Обратное преобразование

$$f(t) = \mathcal{L}^{-1}\{F(s)\} = \frac{1}{2\pi i} \int_{\gamma - i\infty}^{\gamma + i\infty} F(s) e^{st} \, ds$$

## Свойства

- **Линейность:** $\mathcal{L}\{af + bg\} = aF + bG$
- **Производная:** $\mathcal{L}\{f'(t)\} = sF(s) - f(0)$
- **Интеграл:** $\mathcal{L}\{\int_0^t f(\tau) d\tau\} = \frac{F(s)}{s}$
- **Сдвиг:** $\mathcal{L}\{e^{at}f(t)\} = F(s - a)$
- **Свёртка:** $\mathcal{L}\{f * g\} = F(s) \cdot G(s)$

## Таблица основных преобразований

| $f(t)$ | $F(s)$ |
|---|---|
| $1$ | $\frac{1}{s}$ |
| $t^n$ | $\frac{n!}{s^{n+1}}$ |
| $e^{at}$ | $\frac{1}{s-a}$ |
| $\sin(\omega t)$ | $\frac{\omega}{s^2 + \omega^2}$ |
| $\cos(\omega t)$ | $\frac{s}{s^2 + \omega^2}$ |

## Применение

Решение линейных дифференциальных уравнений, теория управления, анализ электрических цепей.
