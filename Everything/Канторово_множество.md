# Канторово множество

## Определение (средняя треть)

Канторово множество $\mathcal{C}$ — множество, получаемое из отрезка $[0, 1]$ последовательным удалением средних третей:

$$\mathcal{C} = [0,1] \setminus \bigcup_{n=0}^{\infty} \bigcup_{k=0}^{3^n-1} \left(\frac{3k+1}{3^{n+1}}, \frac{3k+2}{3^{n+1}}\right)$$

## Эквивалентное определение

$\mathcal{C}$ — множество всех чисел из $[0, 1]$, у которых троичная запись содержит только цифры 0 и 2:

$$\mathcal{C} = \left\{ \sum_{n=1}^{\infty} \frac{a_n}{3^n} \mid a_n \in \{0, 2\} \right\}$$

## Свойства

- Несчётно, но имеет меру Лебега 0
- Совершенно (замкнуто, без изолированных точек)
- Нульмерно, компактно, метризуемо
- Гомеоморфно $\{0, 1\}^\mathbb{N}$ с топологией произведения
- Фрактальная размерность: $\dim_H(\mathcal{C}) = \frac{\ln 2}{\ln 3}$

## Универсальность

Любое компактное метризуемое нульмерное совершенное пространство гомеоморфно $\mathcal{C}$.
