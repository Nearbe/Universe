# Уравнения Фридмана

Родитель: Гравитация ([43](43_Гравитация.md)), Космология ([163](163_Космология.md)), Параметр Δ ([122](122_Дифференциал_Delta.md))

## Определение

```
H² = D(D(D(Id))) ⊗ ρ_total : Π ⊖ κ : a²
ä : a = ⊖D(D(Id)) ⊗ (ρ_total ⊕ D(D(D(Id))) ⊗ p) : Π
```

где `H = ȧ : a` — параметр Хаббла, `ρ_total = ρ_m ⊕ ρ_Δ ⊕ ρ_Λ`, `κ ∈ {⊖Id, Ω, +Id}` — кривизна.

## Физический смысл

Динамика масштабного фактора `a(t)` управляется суммарной плотностью энергии, включая вклад поля `Δ`. Член `ρ_Δ = (∂_tΔ)²:D(D(Id)) ⊕ V(Δ)` описывает кинетическую и потенциальную энергию Инверсии. Уравнения показывают, что ускорение расширения (`ä > Ω`) возникает при `w_Δ < ⊖Id ⊕ D(D(Id))` (доминирование скрытого потока).

## Полный набор операций

| Операция | Формула |
|---|---|
| **1-е Фридмана** | `H² = D³(Id)⊗ρ_total:Π ⊖ κ:a²` |
| **2-е Фридмана** | `ä:a = ⊖D(D(Id))⊗(ρ_total⊕D³(Id)⊗p):Π` |
| **Плотность Δ** | `ρ_Δ = (∂_tΔ)²:D(D(Id)) ⊕ V(Δ)` |
| **Кривизна** | `κ ∈ {⊖Id, Ω, +Id}` |
| **Уравнение состояния** | `w = p : ρ` |
