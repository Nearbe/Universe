# Принцип Мопертюи

Интегральное исчисление ([21](21_Интегральное_исчисление.md)) — интегралы. Лагранжиан ([50](50_Лагранжиан.md)) — действие. Теперь — принцип при фиксированной энергии.

## Определение

```
S₀ = ∫ p ⊗ dq
δS₀ = Ω    (при фиксированной энергии E)
```

## Физический смысл

Мопертюи — **кратчайший путь при фиксированной энергии**. Не «наименьшее действие», а «наименьший импульс на путь».

Элементы **𝕌** с фиксированной энергией выбирают путь минимального «пробега» по фазовому пространству.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Действие** | `S₀ = ∫ p⊗dq` |
| **Минимизация** | `δS₀ = Ω` |
| **Энергия** | `E = const` |
| **Связь с Лагранжианом** | `S = ∫ L dt`, `S₀` при фиксированной E |
