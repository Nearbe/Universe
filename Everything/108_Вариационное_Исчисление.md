# Вариационное Исчисление (Поиск Идеальной Формы)

Оптимизация ([102](102_Методы_оптимизации.md)) ищет число. Вариационное исчисление ищет **Функцию**.

## Функционал Действия

Мы не просто хотим минимум функции. Мы хотим найти такую **Кривую**, чтобы интеграл вдоль неё был минимален.
Это поиск **Идеальной Траектории**.

## Уравнение Эйлера-Лагранжа

Это главный инструмент.
Оно говорит: «Чтобы путь был лучшим, в каждой точке силы должны быть сбалансированы».

## Физический Смысл

Вся физика держится на этом принципе.
Свет ищет путь, который занимает меньше всего времени.
Мяч ищет траекторию, где баланс кинетической и потенциальной энергии оптимален.
Вселенная **Ленива**. Она всегда выбирает самый дешёвый способ сделать дело.
Вариационное исчисление — это математика этой Божественной Лени.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Функционал** | `S[ψ] = ∫ L dt` |
| **Эйлера-Лагранжа** | `∂L:∂ψ ⊖ d/dt(∂L:∂(∂ψ:∂t)) = Ω` |
| **Принцип** | `δS = Ω` |
| **Смысл** | Минимум действия → оптимум |
