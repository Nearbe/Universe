# Линейный оператор

## Определение

Линейный оператор — отображение $T: V \to W$ между векторными пространствами над полем $\mathbb{F}$, удовлетворяющее:

1. **Аддитивность:** $T(u + v) = T(u) + T(v)$ для всех $u, v \in V$
2. **Однородность:** $T(\alpha v) = \alpha T(v)$ для всех $\alpha \in \mathbb{F}$, $v \in V$

## Матричное представление

При выборе базисов в $V$ и $W$ оператор представляется матрицей $M$:
$$T(v) = Mv$$

## Собственные значения и векторы

Число $\lambda$ — собственное значение оператора $T$, если существует ненулевой вектор $v$:
$$T(v) = \lambda v$$

Вектор $v$ — собственный вектор, соответствующий $\lambda$.

## Спектр

Спектр оператора $T$ — множество всех $\lambda \in \mathbb{C}$, для которых $T - \lambda I$ не имеет ограниченного обратного.

Для конечномерного пространства спектр = множество собственных значений.
