# Частота и период

Степени ветвления ([06](06_Степени_двойки.md)) — хребет. Корни ([07](07_Корни.md)) — дробные шаги. Теперь — время вычисления.

## Определение

```
f = e^(−Δ)     — частота вибрации
T = e^Δ = Id : f  — период
```

## Физический смысл

Частота — **темп вычисления**. Период — **длительность одного шага**.

| Δ | f | T | Состояние |
|---|---|---|---|
| **Ω** | **Id** | **Id** | Базовый такт |
| **→ +∞** | **→ Ω** | **→ Π** | Остановка у горизонта |
| **→ −∞** | **→ Π** | **→ Ω** | Мгновенное вычисление |

**f** — не «колебания в секунду». Это **актов `:` за такт** хода.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Частота** | `f = e^(−Δ)` |
| **Период** | `T = e^Δ = Id : f` |
| **При Δ = Ω** | `f = Id, T = Id` |
| **При Δ → +∞** | `f → Ω, T → Π` |
| **При Δ → −∞** | `f → Π, T → Ω` |
| **Связь с T** | `f ⊗ T = Id` |
