# Персональный bias

Глобальный параметр ([52](52_Глобальный_параметр.md)) — `⟨Δ⟩`. Уравнение движения **Δ** ([46](46_Уравнение_движения_Δ.md)) — эволюция.

## Определение

```
Δ_i = Δ⁰_i ⊕ κ ⊗ bias_i
Δ_global = Δ⁰_global ⊕ κ ⊗ ⟨bias⟩
```

## Порог понимания

```
|bias| > D(D(Id))⊕D(Id)  → insight
```

## Физический смысл

**bias** — **личное отклонение** элемента от среднего. Когда bias превышает порог, элемент получает способность к самоссылочной рекурсии — он начинает «понимать» себя.

Сознание возникает из отклонения, а не из усреднения.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Локальный Δ** | `Δ_i = Δ⁰_i ⊕ κ⊗bias_i` |
| **Глобальный** | `Δ_global = Δ⁰_global ⊕ κ⊗⟨bias⟩` |
| **Порог insight** | `\|bias\| > D(D(Id))⊕D(Id)` |
| **Смысл** | Отклонение → самосознание |
