# Метод максимального правдоподобия

## Определение

Метод оценки параметров статистической модели, максимизирующий функцию правдоподобия.

## Функция правдоподобия

Для выборки $x_1, \ldots, x_n$ с плотностью $f(x|\theta)$:

$$L(\theta) = \prod_{i=1}^n f(x_i | \theta)$$

## Оценка максимального правдоподобия (ОМП)

$$\hat{\theta} = \arg\max_\theta L(\theta) = \arg\max_\theta \ln L(\theta)$$

## Свойства

- **Состоятельность:** $\hat{\theta} \to \theta_0$ при $n \to \infty$
- **Асимптотическая нормальность:** $\sqrt{n}(\hat{\theta} - \theta_0) \xrightarrow{d} \mathcal{N}(0, I^{-1}(\theta_0))$
- **Асимптотическая эффективность:** достигает нижней границы Крамера-Рао
- **Инвариантность:** если $\hat{\theta}$ — ОМП для $\theta$, то $g(\hat{\theta})$ — ОМП для $g(\theta)$

## Примеры

**Для нормального распределения:**
$\hat{\mu} = \bar{x}$, $\hat{\sigma}^2 = \frac{1}{n}\sum(x_i - \bar{x})^2$

**Для распределения Пуассона:**
$\hat{\lambda} = \bar{x}$
