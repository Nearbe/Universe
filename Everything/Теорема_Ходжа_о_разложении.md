# Теорема Ходжа о разложении

## Формулировка

На компактном кэлеровом многообразии $X$ комплексной размерности $n$:

$$H^k(X, \mathbb{C}) = \bigoplus_{p+q=k} H^{p,q}(X)$$

где $H^{p,q}(X)$ — когомологии, представимые формами типа $(p,q)$.

## Свойства разложения

- **Сопряжение:** $\overline{H^{p,q}} = H^{q,p}$
- **Симметрия Ходжа:** $h^{p,q} = h^{q,p} = h^{n-p,n-q}$
- **Числа Ходжа:** $h^{p,q} = \dim H^{p,q}$

## Операторы Лefschetz

Для класса Кэлера $\omega$:
$$L: H^{p,q} \to H^{p+1,q+1}, \quad L(\alpha) = \omega \wedge \alpha$$

**Теорема о жёстком разложении:** примитивные классы разлагаются по степеням $L$.

## Гипотеза Ходжа

Каждый класс Ходжа $H^{p,p} \cap H^{2p}(X, \mathbb{Q})$ — рациональная линейная комбинация классов алгебраических циклов.

Одна из 7 задач тысячелетия. Доказана для $p=1$ (Лефшец), открыта для $p \geq 2$.

## Применения

Алгебраическая геометрия, теория струн (модули Калаби-Яу), теория мотивов.
