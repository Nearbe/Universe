# Химия

Родитель: Строение чисел ([25](25_Строение_чисел.md)), Фрактальное пространство ([155](155_Фрактальное_пространство.md))

## Химическая связь

| Тип | Механизм | Энергия (эВ) |
|---|---|---|
| Ковалентная | Общие электроны | `D(D(Id))-Id` |
| Ионная | Электростатика | `Id-5` |
| Металлическая | Делокализованные e⁻ | `Id-3` |
| Водородная | Диполь-диполь | `0.Id-0.5` |
| Ван-дер-Ваальса | Дисперсия | `0.0Id-0.Id` |

## Равновесие

```
K_eq = ([C]^c ⊗ [D]^d) : ([A]^a ⊗ [B]^b)
ΔG° = ⊖R ⊗ T ⊗ ln(K_eq)
```

В терминах `𝕌`: `ΔG° ∝ ⊖ln(K_eq) ∝ ΔI`.

## Скорость реакций

```
k = A ⊗ exp(⊖E_a : (R⊗T))
```

В терминах `𝕌`: `k ∝ exp(⊖I_barrier)`.
Катализатор снижает информационный барьер.

## Спектроскопия

| Тип | Что измеряет | Связь с Δ |
|---|---|---|
| UV-Vis | Электронные переходы | `\|Ψ\|²` |
| IR, Раман | Колебания ядер | Модуляция Δ |
| ЯМР | Спины в поле | Поляризация сети |
| ЭПР | Неспаренные e⁻ | Флуктуации Δ |

## Полный набор операций

| Операция | Формула |
|---|---|
| **K_eq** | `([C]^c⊗[D]^d):([A]^a⊗[B]^b)` |
| **ΔG°** | `⊖R⊗T⊗ln(K_eq)` |
| **Аррениус** | `k = A⊗exp(⊖E_a:(R⊗T))` |
| **Катализ** | `↓I_barrier ⟹ ↑k` |
