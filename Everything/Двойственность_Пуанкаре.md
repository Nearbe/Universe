# Двойственность Пуанкаре

## Формулировка

Для замкнутого ориентируемого $n$-мерного многообразия $M$ существует изоморфизм:

$$H^k(M) \cong H_{n-k}(M)$$

где $H^k$ — когомологии, $H_{n-k}$ — гомологии.

## Следствия

- **Бетти-числа:** $b_k = b_{n-k}$ (симметрия)
- **Эйлерова характеристика:** $\chi(M) = \sum_{k=0}^n (-1)^k b_k$
- Для нечётного $n$: $\chi(M) = 0$

## Обобщения

**Двойственность Лефшеца:** для многообразия с краем
**Двойственность Александера:** для подмногообразий

## Примеры

**Сфера $S^n$:**
$b_0 = b_n = 1$, все остальные $b_k = 0$.

**Тор $T^2$:**
$b_0 = b_2 = 1$, $b_1 = 2$.

## Значение

Фундаментальная связь между гомологиями и когомологиями, ключевой инструмент алгебраической топологии.
