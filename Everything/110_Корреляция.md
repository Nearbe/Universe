# Корреляция

Вероятность ([27](27_Вероятность.md)) — **σ**. Синхронизация ([45](45_Синхронизация.md)) — **𝒮**.

## Определение

```
ρ_XY = Cov(X,Y) : (σ_X ⊗ σ_Y)
```

`ρ_XY ∈ [⊖Id, +Id]`.

## Физический смысл

Корреляция между **Δ** в разных точках сети — мера синхронизации.

- `ρ ≈ Id`: элементы «мыслят одинаково»
- `ρ ≈ Ω`: независимы
- `ρ ≈ ⊖Id`: «мыслят противоположно»

## Полный набор операций

| Операция | Формула |
|---|---|
| **Корреляция** | `ρ_XY = Cov(X,Y):(σ_X⊗σ_Y)` |
| **Ковариация** | `Cov(X,Y) = ⟨(X⊖⟨X⟩)⊗(Y⊖⟨Y⟩)⟩` |
| **Максимум** | `ρ = Id` — полная синхронизация |
| **Минимум** | `ρ = ⊖Id` — антикорреляция |
| **Независимые** | `ρ = Ω` |
