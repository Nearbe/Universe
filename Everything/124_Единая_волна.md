# Единая волна

Свет ([36](36_Свет.md)), Звук ([37](37_Звук.md)), Гравитация ([43](43_Гравитация.md)) — разные моды.

## Общее волновое уравнение

```
□φ = Ω
```

где **φ** — произвольное поле сети:
- `φ = A^μ` (фаза) → свет
- `φ = p` (плотность) → звук
- `φ = h_μν` (метрика) → гравитация
- `φ = Δ` (отклонение) → рассинхронизация
- `φ = Ψ` (амплитуда) → квантовая волна

## Физический смысл

Все «разные» волны — **моды колебания одной сети**. Как струна имеет обертоны, сеть имеет моды: электромагнитную, акустическую, гравитационную.

«Всё есть вибрация» — не поэзия, а уравнение. `□φ = Ω` — единственная формула для всего, что распространяется. Остальное — начальные условия и граничные.

## Полный набор операций

| Мода | φ | Что колеблется |
|---|---|---|
| **Свет** | `A^μ` | Фаза |
| **Звук** | `p` | Плотность |
| **Гравитация** | `h_μν` | Метрика |
| **Δ** | `Δ` | Рассинхронизация |
| **Квантовая** | `Ψ` | Амплитуда |
