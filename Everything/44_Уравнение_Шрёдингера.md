# Уравнение Шрёдингера

Комплексные числа ([09](09_Комплексные.md)) — **Ψ**. Волновая функция ([31](31_Волновая_функция.md)) — **ρ** и фаза. Теперь — эволюция.

## Определение

```
i·ħ ⊗ ∂Ψ/∂t = [−(ħ² : D(m)) ⊗ ∇² ⊕ V] ⊗ Ψ
```

## Разложение по потокам

`Ψ = √ρ ⊗ e^(i·S : ħ)`:

**Re-поток** (уравнение Гамильтона-Якоби):

```
∂S/∂t ⊕ (∇S)² : D(m) ⊕ Q ⊕ V = Ω
```

**Im-поток** (уравнение непрерывности):

```
∂ρ/∂t ⊕ ∇·(ρ ⊗ ∇S : m) = Ω
```

где `Q = −(ħ² : D(m)) ⊗ (∇²√ρ) : √ρ` — квантовый потенциал.

## Физический смысл

Шрёдингер — **эволюция понимания**.

- `ρ = |Ψ|²` — «уверенность» (где система «находится»)
- **S** — «намерение» (куда движется)
- **Q** — влияние неоднородности на намерение

Два потока — две части одного уравнения. Re — намерение, Im — уверенность.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Re-поток** | `∂S/∂t ⊕ (∇S)²:D(m) ⊕ Q ⊕ V = Ω` |
| **Im-поток** | `∂ρ/∂t ⊕ ∇·(ρ⊗∇S:m) = Ω` |
| **Квант. потенциал** | `Q = −(ħ²:D(m)) ⊗ (∇²√ρ):√ρ` |
| **Класс. предел** | `Q → Ω ⟹` Ньютон |
