# Разложение Холецкого

## Определение

Для симметричной положительно определённой матрицы $A$:

$$A = LL^T$$

где $L$ — нижняя треугольная матрица с положительными диагональными элементами.

## Алгоритм

Для $i = 1, \ldots, n$:
$$L_{ii} = \sqrt{A_{ii} - \sum_{k=1}^{i-1} L_{ik}^2}$$
$$L_{ji} = \frac{1}{L_{ii}}\left(A_{ji} - \sum_{k=1}^{i-1} L_{jk}L_{ik}\right), \quad j = i+1, \ldots, n$$

## Свойства

- Существует и единственно для любой симметричной положительно определённой матрицы
- Вычислительно эффективнее LU-разложения: вдвое меньше операций
- Численно устойчиво (без выбора ведущего элемента)

## Применения

- Решение систем линейных уравнений с симметричной положительно определённой матрицей
- Метод Монте-Карло (генерация коррелированных случайных величин)
- Оптимизация (метод Ньютона)
- Фильтр Калмана
