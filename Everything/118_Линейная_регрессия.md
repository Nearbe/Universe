# Линейная регрессия

Корреляция ([110](110_Корреляция.md)) — **Cov**. Вероятность ([27](27_Вероятность.md)) — **Var**.

## Определение

```
y = a ⊗ x ⊕ b ⊕ ε
â = Cov(X,Y) : Var(X)
```

## Физический смысл

Анализ зависимости между **Δ** в разных частях сети: «как **Δ** в одном кластере предсказывает **Δ** в другом».

## Полный набор операций

| Операция | Формула |
|---|---|
| **Модель** | `y = a⊗x ⊕ b ⊕ ε` |
| **Наклон** | `â = Cov(X,Y):Var(X)` |
| **Сдвиг** | `b̂ = ⟨Y⟩ ⊖ â⊗⟨X⟩` |
| **Смысл** | Предсказание Δ одного кластера из другого |
