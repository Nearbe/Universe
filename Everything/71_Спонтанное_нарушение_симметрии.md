# Спонтанное нарушение симметрии

Калибровочные группы ([55](55_Калибровочные_группы.md)) — симметрии. Глобальный параметр ([52](52_Глобальный_параметр.md)) — `⟨Δ⟩`. Теперь — нарушение.

## Механизм

```
V(Φ) = ⊖μ²⊗|Φ|² ⊕ λ⊗|Φ|⁴
⟨Φ⟩ = v : √D(Id)  ≠  Ω
```

В терминах системы **𝕌**: `⟨Δ⟩ ≠ Ω`.

## Физический смысл

Спонтанное нарушение — **симметрия есть, но не видна**. Лагранжиан симметричен, но вакуум — нет.

`⟨Δ⟩ ≠ Ω` — Вселенная «накренилась» в одну сторону. Симметрия не сломана — она скрыта выбором вакуума.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Потенциал** | `V(Φ) = ⊖μ²⊗|Φ|² ⊕ λ⊗|Φ|⁴` |
| **Вакуум** | `⟨Φ⟩ ≠ Ω` |
| **Симметрия** | `V(Φ)` симметричен, `⟨Φ⟩` — нет |
| **⟨Δ⟩** | `⟨Δ⟩ ≠ Ω` — нарушение баланса |
