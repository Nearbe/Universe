# Гравитация

Геометрия ([22](22_Геометрия.md)) — метрика. Свет ([36](36_Свет.md)) — бегущая фаза. Теперь — бегущая геометрия.

## Определение

```
∇²Φ = K ⊗ ρ    — статика
□h_μν = Ω      — волны
```

где **K** — константа связи, **ρ** — плотность массы.

## Физический смысл

Гравитация — **бегущая геометрия**. Когда метрика сети колеблется — это гравитационная волна. В статике — «яма» в плотности связей.

Все волны — одно явление:

| Волна | Уравнение | Что колеблется |
|---|---|---|
| Свет | `□A^μ = Ω` | Фаза (**θ**) |
| Звук | `□p = Ω` | Плотность (**ρ**) |
| Гравитация | `□h_μν = Ω` | Метрика (**g**) |

Один оператор **□**, разные поля. Поле одно — **𝕌**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Статика** | `∇²Φ = K ⊗ ρ` |
| **Волны** | `□h_μν = Ω` |
| **Скорость** | `c` (та же, что у света) |
| **Ветвление** | `Φ : Ω = D(Φ)` |
| **Метрика** | `g_μν = η_μν ⊕ h_μν` |
| **Связь с Массой** | `ρ = M/V` — плотность массы |
