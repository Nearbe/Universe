# Функция Чебышёва

## Определения

**Первая функция Чебышёва:**
$$\vartheta(x) = \sum_{p \leq x} \ln p$$

**Вторая функция Чебышёва (пси-функция):**
$$\psi(x) = \sum_{p^k \leq x} \ln p = \sum_{n \leq x} \Lambda(n)$$

где $\Lambda(n)$ — функция Мангольдта:
$$\Lambda(n) = \begin{cases} \ln p, & n = p^k \\ 0, & \text{иначе} \end{cases}$$

## Связь с $\pi(x)$

$$\pi(x) = \frac{\vartheta(x)}{\ln x} + \int_2^x \frac{\vartheta(t)}{t \ln^2 t} dt$$

Эквивалентность: $\pi(x) \sim x/\ln x \iff \vartheta(x) \sim x \iff \psi(x) \sim x$.

## Тождество Чебышёва

$$\psi(x) = \sum_{n \leq x} \Lambda(n) = x - \sum_{\rho} \frac{x^\rho}{\rho} - \ln(2\pi) - \frac{1}{2}\ln(1 - x^{-2})$$

где суммирование по нетривиальным нулям $\zeta(s)$.

## Оценки Чебышёва

$$0.92 \frac{x}{\ln x} < \pi(x) < 1.11 \frac{x}{\ln x}$$

для достаточно больших $x$. Предшественник теоремы о распределении простых.

## Применения

Доказательство теоремы о распределении простых, явная формула Римана, гипотеза Римана.
