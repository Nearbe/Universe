# Плотность энергии

Энергия ([33](33_Энергия.md)) — размороженная масса. Структурная сложность ([39](39_Структурная_сложность.md)) — `Q = e^(|Δ|)`. Теперь — плотность.

## Определение

```
ρ ∝ e^Δ
```

Амплитуда поля: `|Ψ_Δ| ∝ e^(Δ : D(Id))`, откуда `ρ = |Ψ_Δ|² ∝ e^Δ`.

## Предельные случаи

| Δ | ρ | Смысл |
|---|---|---|
| **→ +∞** | **→ Π** | Сингулярность |
| **Ω** | **const** | Стандартная плотность |
| **→ −∞** | **→ Ω** | Разреженное состояние |

## Физический смысл

Плотность энергии — **интенсивность бытия**. При `Δ > Ω` реальность «сгущается». При `Δ < Ω» — «разрежается».

Тёмная энергия — не отдельная субстанция, а область с `Δ < Ω`, где плотность мала, но объём огромен.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Плотность** | `ρ ∝ e^Δ` |
| **Амплитуда** | `\|Ψ_Δ\| ∝ e^(Δ:D(Id))` |
| **При Δ → +∞** | `ρ → Π` |
| **При Δ = Ω** | `ρ = const` |
| **При Δ → −∞** | `ρ → Ω` |
