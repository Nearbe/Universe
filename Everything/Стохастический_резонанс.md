# Стохастический резонанс

## Определение

Явление, при котором добавление шума в нелинейную систему усиливает отклик на слабый периодический сигнал.

## Механизм

Система с двумя устойчивыми состояниями (бистабильная):
- Слабый сигнал недостаточен для переключения между состояниями
- Шум помогает системе преодолевать барьер
- При оптимальном уровне шума переключения синхронизуются с сигналом

## Количественная мера

Отношение сигнал/шум (SNR) на выходе достигает максимума при определённой интенсивности шума.

## Применения

- **Нейронаука:** усиление слабых сенсорных сигналов
- **Климатология:** объяснение ледниковых циклов
- **Электроника:** обнаружение субпороговых сигналов
- **Биология:** механорецепторы раков (экспериментальное подтверждение)

## Модель

Ланжевеновская частица в двухъямном потенциале:
$$\dot{x} = -V'(x) + A\cos(\omega t) + \xi(t)$$

где $\xi(t)$ — белый шум.
