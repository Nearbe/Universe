# Быстрая сортировка

## Определение

Алгоритм сортировки, основанный на принципе «разделяй и властвуй». Среднее время $O(n \log n)$.

## Алгоритм

```
quicksort(A, lo, hi):
    if lo < hi:
        p = partition(A, lo, hi)
        quicksort(A, lo, p - 1)
        quicksort(A, p + 1, hi)

partition(A, lo, hi):
    pivot = A[hi]
    i = lo
    for j = lo to hi - 1:
        if A[j] < pivot:
            swap(A[i], A[j])
            i++
    swap(A[i], A[hi])
    return i
```

## Сложность

- **Лучший случай:** $O(n \log n)$ — опорный элемент делит пополам
- **Средний случай:** $O(n \log n)$
- **Худший случай:** $O(n^2)$ — массив уже отсортирован, опорный = последний

## Оптимизации

- Случайный выбор опорного элемента (избегает худшего случая)
- Медиана трёх (выбор из первого, среднего и последнего)
- Переключение на сортировку вставкой для малых подмассивов

## Свойства

- Неустойчивая (порядок равных элементов может измениться)
- На месте (in-place), не требует дополнительной памяти
- На практике быстрее большинства $O(n \log n)$ алгоритмов
