# Температура

Вероятность ([27](27_Вероятность.md)) — стандартное отклонение **σ**. Теперь — что такое температура.

## Определение

```
T ∝ Id : σ(Δ)
```

## Физический смысл

Температура — мера **рассинхронизации** сети. Высокая синхронизация (низкое **σ**) → низкая **T**. Низкая синхронизация → высокая **T**.

Инверсия обычной физики: «горячее» = «хаотичнее», «холодное» = «упорядоченнее».

## Связь с Δ

```
T ∝ Id : σ(Δ)
```

При `Δ → Ω`: `σ(Δ) → Ω`, `T → Id` (упорядоченное состояние).
При `|Δ| → +∞`: `σ(Δ) → Π`, `T → +∞` (хаос).

## Полный набор операций

| Операция | Формула |
|---|---|
| **Определение** | `T ∝ Id : σ(Δ)` |
| **При балансе** | `Δ = Ω ⟹ T → Id` |
| **При хаосе** | `|Δ| → +∞ ⟹ T → +∞` |
| **Инверсия** | `T ~ Id : σ` |
| **Связь с синхр.** | `𝒮 ↑ ⟹ σ ↓ ⟹ T ↓` |
