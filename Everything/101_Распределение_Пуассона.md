# Распределение Пуассона

Распределения вероятностей ([100](100_Распределения_вероятностей.md)) — обзор. Вероятность ([27](27_Вероятность.md)) — мера.

## Определение

```
P(k) = (λ^k : k!) ⊗ e^(⊖λ)
```

```
⟨k⟩ = λ,    σ² = λ
```

## Физический смысл

Редкие события в сети: туннелирование, спонтанные флуктуации **Δ**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Вероятность** | `P(k) = (λ^k:k!) ⊗ e^(⊖λ)` |
| **Среднее** | `⟨k⟩ = λ` |
| **Дисперсия** | `σ² = λ` |
| **Редкие события** | `λ ≪ n` |
