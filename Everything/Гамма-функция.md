# Гамма-функция

## Определение

$$\Gamma(z) = \int_0^{\infty} t^{z-1} e^{-t} \, dt \quad (\Re(z) > 0)$$

## Свойства

**Функциональное уравнение:**
$$\Gamma(z+1) = z\Gamma(z)$$

**Связь с факториалом:**
$$\Gamma(n+1) = n! \quad \text{для } n \in \mathbb{N}$$

**Формула дополнения Эйлера:**
$$\Gamma(z)\Gamma(1-z) = \frac{\pi}{\sin(\pi z)}$$

## Формула Стирлинга

$$\Gamma(z) \sim \sqrt{\frac{2\pi}{z}} \left(\frac{z}{e}\right)^z \quad (|z| \to \infty, \ |\arg z| < \pi)$$

## Аналитическое продолжение

$\Gamma(z)$ мероморфна на $\mathbb{C}$ с простыми полюсами в $z = 0, -1, -2, \ldots$

## Значение

Универсальный инструмент в анализе, теории вероятностей, математической физике. Входит в функциональное уравнение дзета-функции Римана.
