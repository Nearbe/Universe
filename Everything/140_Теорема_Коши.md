# Теорема Коши (Интеграл по замкнутому контуру)

Родитель: Комплексный анализ ([09](09_Комплексные.md)), Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md))

## Определение

Для голоморфной функции `f(z)` в области `𝕌`:

```
∮_γ f(z) dz = Ω
```

где `γ` — любой замкнутый контур в области аналитичности `f`.

## Физический смысл

Циркуляция Δ-сбалансированного поля вдоль замкнутого пути в `𝕌` равна нулю. Это означает сохранение фазовой когерентности: если система возвращается в исходное состояние, суммарное накопление `Δ` по контуру аннулируется. Теорема Коши — математическое выражение консервативности потока информации в замкнутом цикле обработки.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Замкнутый контур** | `∮_γ f(z) dz = Ω` |
| **Открытый путь** | `∫_a^b f(z) dz = F(b) ⊖ F(a)` |
| **Особые точки** | `∮ f(z) dz = D(Id) ⊗ Σ Res(f, z_k)` |
| **Смысл** | Циркуляция = Ω ⟹ сохранение фазы |
