# Группа

## Определение

Группа $(G, \cdot)$ — множество с бинарной операцией, удовлетворяющее:

1. **Ассоциативность:** $(ab)c = a(bc)$
2. **Единица:** существует $e$: $ea = ae = a$
3. **Обратный элемент:** для каждого $a$ существует $a^{-1}$: $aa^{-1} = e$

## Примеры

- $(\mathbb{Z}, +)$ — целые числа со сложением
- $(\mathbb{R}\setminus\{0\}, \cdot)$ — ненулевые вещественные с умножением
- $S_n$ — группа перестановок $n$ элементов
- $GL(n, \mathbb{R})$ — обратимые матрицы
- $\mathbb{Z}_n$ — целые по модулю $n$

## Свойства

- Единица единственна
- Обратный элемент единствен
- $(ab)^{-1} = b^{-1}a^{-1}$

## Порядок

- **Порядок группы:** $|G|$ — число элементов
- **Порядок элемента:** наименьшее $n$: $a^n = e$

## Теорема Лагранжа

Для конечной группы: порядок подгруппы делит порядок группы.

## Абелева группа

Коммутативна: $ab = ba$ для всех $a, b \in G$.
