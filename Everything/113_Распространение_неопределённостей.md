# Распространение неопределённостей

Дифференциальное исчисление ([20](20_Дифференциальное_исчисление.md)) — производные. Масса ([32](32_Масса.md)), Энергия ([33](33_Энергия.md)), Информация ([30](30_Информация.md)).

## Определение

```
δf = √[(∂f:∂x ⊗ δx)² ⊕ (∂f:∂y ⊗ δy)² ⊕ ...]
```

## Физический смысл

Неопределённость информации = неопределённость массы = неопределённость энергии:

```
ΔI = ΔE = ΔM
```

Они не различимы на уровне ошибки.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Общая** | `δf = √[Σ(∂f:∂x_i ⊗ δx_i)²]` |
| **Тождество** | `ΔI = ΔE = ΔM` |
| **Сумма** | `δ(a⊕b) = √(δa² ⊕ δb²)` |
| **Произведение** | `δ(a⊗b) = \|a⊗b\| ⊗ √[(δa:a)² ⊕ (δb:b)²]` |
