# Теорема Рисса

## Представление линейных функционалов

Каждый непрерывный линейный функционал $f$ на гильбертовом пространстве $\mathcal{H}$ имеет вид:
$$f(x) = \langle x, z \rangle$$

для единственного $z \in \mathcal{H}$. При этом $\|f\| = \|z\|$.

## Для $L^p$ пространств

Для $1 < p < \infty$, $(L^p)^* \cong L^q$, где $\frac{1}{p} + \frac{1}{q} = 1$:
$$f(g) = \int fg \, d\mu$$

## Для мер

Теорема Рисса-Маркова: каждый положительный линейдный функционал на $C_c(X)$ имеет вид:
$$f(\phi) = \int_X \phi \, d\mu$$

для единственной меры Радона $\mu$.

## Применения

- Квантовая механика: бра-векторы = кет-векторы
- Теория вероятностей: представление математических ожиданий
- Функциональный анализ: двойственность пространств
