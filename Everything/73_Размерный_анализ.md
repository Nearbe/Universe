# Размерный анализ

Масса ([32](32_Масса.md)) — **M**. Структурная сложность ([39](39_Структурная_сложность.md)) — **Q**. Теперь — размерности величин.

## Размерности

| Величина | Размерность |
|---|---|
| **Ψ** | **M** (масса) |
| **Δ** | **Id** (безразмерная) |
| **g** | **Id** (безразмерная) |
| **ħ** | **M ⊗ L² : T** |
| **c** | **L : T** |

## Физический смысл

**Δ** безразмерна — это **отношение**, а не величина с единицами. **Ψ** имеет размерность массы — амплитуда вибрации есть масса.

Безразмерные константы связи = перенормируемость = система не «разваливается» на малых масштабах.

## Полный набор операций

| Операция | Формула |
|---|---|
| **[Ψ]** | **M** |
| **[Δ]** | **Id** |
| **[ħ]** | **M⊗L²:T** |
| **[c]** | **L:T** |
| **Перенормировка** | Безразмерные константы → стабильность |
