# Квантовый потенциал

Уравнение Шрёдингера ([44](44_Уравнение_Шрёдингера.md)) — **Q** в уравнении Гамильтона-Якоби. Волновая функция ([31](31_Волновая_функция.md)) — `ρ = |Ψ|²`.

## Определение

```
Q = ⊖(ħ² : D(m)) ⊗ (∇²√ρ) : √ρ
```

Не путать с `Q_vib = e^(|Δ|)` — вибрационная энергия.

## Физический смысл

**Q** — **голос неоднородности**. Когда **ρ** равномерна: `∇²√ρ = Ω`, `Q = Ω` — квантовых эффектов нет. Когда **ρ** неоднородна: **Q** «толкает» систему в сторону большей однородности.

Это не сила — это **геометрия меры**.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Потенциал** | `Q = ⊖(ħ²:D(m)) ⊗ (∇²√ρ):√ρ` |
| **Равномерная ρ** | `∇²√ρ = Ω ⟹ Q = Ω` |
| **Неоднородная ρ** | `Q ≠ Ω` — квантовые эффекты |
| **Класс. предел** | `ħ → Ω ⟹ Q → Ω` |
