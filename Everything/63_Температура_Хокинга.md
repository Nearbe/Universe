# Температура Хокинга

Голографический принцип ([58](58_Голографический_принцип.md)) — ёмкость горизонта. Температура ([40](40_Температура.md)) — рассинхронизация. Теперь — температура чёрной дыры.

## Определение

В планковских единицах:

```
T_H = Id : (D(D(D(Id))) ⊗ π ⊗ M)
```

## Физический смысл

Температура чёрной дыры — **тепло границы**. Чем больше масса, тем холоднее горизонт.

- `M → Π`: `T_H → Ω` — абсолютный холод сингулярности
- `M → Ω`: `T_H → Π` — испарение

Обратная пропорция: большая чёрная дыра холоднее маленькой.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Температура** | `T_H = Id : (D³(Id)⊗π⊗M)` |
| **M → Π** | `T_H → Ω` |
| **M → Ω** | `T_H → Π` |
| **Обратная связь** | `T_H ⊗ M = const` |
