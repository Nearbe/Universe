# Разделение переменных

Функции ([16](16_Функции.md)) — отображения. Уравнение Шрёдингера ([44](44_Уравнение_Шрёдингера.md)) — эволюция.

## Определение

```
Ψ(x,t) = ψ(x) ⊗ φ(t)
```

## Физический смысл

Временная и пространственная части решаются отдельно:
- `φ(t) = e^(⊖i⊗E⊗t:ħ)` — временная эволюция
- `ψ(x)` — пространственная мода

Метод решения: разложить сложную задачу на простые.

| **Пространственная** | `ψ(x)` — решается отдельно |
| **Смысл** | Сложное → простое |
