# Стандартное отклонение

Вероятность ([27](27_Вероятность.md)) — дисперсия. Синхронизация ([45](45_Синхронизация.md)) — **𝒮**.

## Определение

Дискретная:

```
σ(Δ) = √[(Id : N) ⊗ Σᵢ (Δᵢ ⊖ ⟨Δ⟩)²]
```

## Физический смысл

`σ(Δ)` — мера **рассогласования** сети.

- Малое `σ` = все элементы «поют в унисон»
- Большое `σ` = «каждый в своём ритме»

`𝒮 = Id ⊖ σ⊗scale` — насколько сеть едина.

## Полный набор операций

| Операция | Формула |
|---|---|
| **σ(Δ)** | `√[(Id:N) ⊗ Σ(Δ_i⊖⟨Δ⟩)²]` |
| **Дисперсия** | `σ² = (Id:N) ⊗ Σ(Δ_i⊖⟨Δ⟩)²` |
| **Связь с 𝒮** | `𝒮 = Id ⊖ σ⊗scale` |
| **Малая σ** | `σ → Ω` ⟹ унисон |
