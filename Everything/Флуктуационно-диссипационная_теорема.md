# Теорема Флуктуационно-диссипационная

## Формулировка

Отклик системы на внешнее возмущение связан с флуктуациями в равновесии:

$$\chi''(\omega) = \frac{\omega}{2k_B T} S(\omega)$$

где $\chi''$ — мнимая часть восприимчивости, $S(\omega)$ — спектральная плотность флуктуаций.

## Частный случай

Для броуновской частицы:
$$\langle \eta(t)\eta(t') \rangle = 2\gamma k_B T \delta(t-t')$$

Интенсивность шума пропорциональна трению и температуре.

## Обобщённая форма

$$\langle A(t)B(0) \rangle = \frac{k_B T}{\pi} \int_{-\infty}^{\infty} \frac{\chi''_{AB}(\omega)}{\omega} e^{-i\omega t} d\omega$$

## Значение

Связывает равновесные флуктуации (пассивные) с диссипацией при внешнем воздействии (активные). Универсальна для любых линейных систем вблизи равновесия.

## Применения

Термодинамика необратимых процессов, спектроскопия, молекулярная динамика.
