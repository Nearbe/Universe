# Классы PSPACE и EXPTIME

## Определения

- **PSPACE:** задачи, решаемые на машине Тьюринга с полиномиальной памятью: $O(n^k)$ ячеек.
- **EXPTIME:** задачи, решаемые за экспоненциальное время: $O(2^{p(n)})$.
- **PSPACE-complete:** задачи из PSPACE, к которым сводится любая задача из PSPACE.

## Соотношения классов

$$P \subseteq NP \subseteq PSPACE \subseteq EXPTIME$$

Известно строго: $P \subsetneq EXPTIME$. Остальные включения, вероятно, строгие, но не доказаны.

## PSPACE-complete примеры

- Истинность формул логики высказываний с кванторами (TQBF)
- Обобщённые шахматы, го, сакки (на доске $n \times n$)
- Планирование с ограниченными ресурсами

## EXPTIME-complete примеры

- Проверка эквивалентности регулярных выражений с пересечением
- Некоторые задачи верификации программ
- Обобщённые игры с экспоненциальной длиной партии

## Значение

Определяют границы feasible вычислений. PSPACE тесно связано с интерактивными доказательствами и криптографией.
