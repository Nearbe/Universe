# Собственное значение

## Определение

Число $\lambda$ — собственное значение линейного оператора $A: V \to V$, если существует ненулевой вектор $v \in V$:
$$Av = \lambda v$$

Вектор $v$ — собственный вектор, соответствующий $\lambda$.

## Характеристический многочлен

Собственные значения — корни характеристического многочлена:
$$\det(A - \lambda I) = 0$$

## Спектр

Спектр $\sigma(A)$ — обобщение понятия собственных значений на бесконечномерные пространства:
$$\sigma(A) = \{\lambda \in \mathbb{C} \mid A - \lambda I \text{ не имеет ограниченного обратного}\}$$

## Свойства

- Сумма собственных значений = след матрицы
- Произведение собственных значений = определитель
- Для самосопряжённого оператора все собственные значения вещественны
- Для унитарного оператора все собственные значения лежат на единичной окружности
