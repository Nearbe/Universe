# Энтропия Шеннона

## Определение

Для дискретного распределения вероятностей $P = \{p_1, \ldots, p_n\}$:

$$H(P) = -\sum_{i=1}^{n} p_i \log_2 p_i$$

## Свойства

- $H \geq 0$, равенство только для вырожденного распределения
- Максимум при равномерном распределении: $H_{\max} = \log_2 n$
- Аддитивность: $H(X, Y) = H(X) + H(Y|X)$
- Непрерывность по $p_i$

## Условная энтропия

$$H(X|Y) = H(X, Y) - H(Y)$$

## Взаимная информация

$$I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)$$

## Энтропия источника

Для стационарного источника:
$$H = \lim_{n \to \infty} \frac{1}{n} H(X_1, \ldots, X_n)$$

## Связь с физикой

Энтропия Шеннона совпадает с энтропией Больцмана-Гиббса с точностью до множителя $k_B$.
