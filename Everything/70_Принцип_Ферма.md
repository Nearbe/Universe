# Принцип Ферма

Принцип Мопертюи ([69](69_Принцип_Мопертюи.md)) — кратчайший путь по импульсу. Свет ([36](36_Свет.md)) — свободная фаза.

## Определение

```
δ ∫ n ⊗ ds = Ω
```

где **n** — показатель преломления, **ds** — элемент пути.

## Связь с Мопертюи

| Принцип | Формулировка | Область |
|---|---|---|
| Ферма | `δ ∫ n⊗ds = Ω` | Оптика |
| Мопертюи | `δ ∫ p⊗dq = Ω` | Механика |

## Физический смысл

Ферма — **свет ленив**. Луч выбирает путь наименьшей оптической длины.

Фазовые моды сети подчиняются Ферма — фаза **θ** распространяется по пути наименьшего «фазового сопротивления».

## Полный набор операций

| Операция | Формула |
|---|---|
| **Оптическая длина** | `∫ n⊗ds` |
| **Минимизация** | `δ ∫ n⊗ds = Ω` |
| **Связь с Мопертюи** | Оба — принцип наименьшего «пробега» |
