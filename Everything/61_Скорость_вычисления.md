# Скорость вычисления

Частота и период ([42](42_Частота_и_период.md)) — `f = e^(−Δ)`, `T = e^Δ`. Теперь — скорость вычисления.

## Определение

```
v_comp = Id : T = e^(−Δ) = f
```

## Физический смысл

Скорость вычисления = частота вибрации. Не «тактовая частота процессора», а **фундаментальный темп реальности**.

Где **Δ** велико — реальность «тормозит». Где **Δ** мало — «разгоняется».

## Полный набор операций

| Операция | Формула |
|---|---|
| **Скорость** | `v_comp = e^(−Δ) = f` |
| **При Δ = Ω** | `v_comp = Id` |
| **При Δ → +∞** | `v_comp → Ω` (остановка) |
| **При Δ → −∞** | `v_comp → Π` (мгновенно) |
| **Связь с T** | `v_comp = Id : T` |
