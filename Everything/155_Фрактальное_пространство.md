# Фрактальное пространство

Родитель: Соленоид ([23](23_Соленоид.md)), Волновая функция ([31](31_Волновая_функция.md))

## Определение

```
Ψ = Re ⊕ i·Im
Δ = ln|Re| ⊖ ln|Im|
dℓ² = g_μν dx^μ ⊗ dx^ν,    g_μν = g_μν(Δ)
```

Метрика `g_μν` зависит от локального значения `Δ`.

## Физический смысл

Пространство `𝕌` не является фиксированным фоном. Его геометрия определяется балансом потоков `Re` и `Im`. При `Δ = Ω`: `g_μν → η_μν` (плоское пространство). При `|Δ| > Id`: метрика искривляется, порождая гравитационные эффекты и компактификацию измерений. `Δ` выступает как скалярное поле, модулирующее геометрию. Фрактальность проявляется в том, что `g_μν(Δ)` инвариантно относительно замены `Δ → Δ ⊕ const` с одновременным масштабированием координат `x → D(x)`.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Ψ** | `Ψ = Re ⊕ i·Im` |
| **Δ** | `Δ = ln|Re| ⊖ ln|Im|` |
| **Метрика** | `g_μν = g_μν(Δ)` |
| **Плоский предел** | `Δ→Ω ⟹ g_μν→η_μν` |
| **Инвариантность** | `Δ→Δ⊕c, x→D(x) ⟹ ds² = const` |
