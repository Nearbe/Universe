# Матрица плотности

## Определение

Оператор $\rho$, описывающий статистическое состояние квантовой системы (в том числе смешанное):

$$\rho = \sum_i p_i |\psi_i\rangle\langle\psi_i|$$

где $p_i$ — вероятности, $\sum p_i = 1$.

## Свойства

- Эрмитова: $\rho^\dagger = \rho$
- Нормировка: $\text{Tr}(\rho) = 1$
- Положительно определена: $\langle \phi | \rho | \phi \rangle \geq 0$
- Для чистого состояния: $\rho^2 = \rho$, $\text{Tr}(\rho^2) = 1$
- Для смешанного: $\text{Tr}(\rho^2) < 1$

## Уравнение движения

Уравнение фон Неймана:
$$i\hbar \frac{\partial \rho}{\partial t} = [H, \rho]$$

## Приведённая матрица плотности

Для подсистемы $A$ составной системы $AB$:
$$\rho_A = \text{Tr}_B(\rho_{AB})$$

Позволяет вычислять средние по подсистеме, не зная состояния всей системы.

## Энтропия фон Неймана

$$S = -k_B \text{Tr}(\rho \ln \rho)$$
