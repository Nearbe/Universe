# Даламбертиан

Дифференциальные операторы ([97](97_Дифференциальные_операторы.md)) — **□**. Уравнение Клейна-Гордона ([64](64_Уравнение_Клейна_Гордона.md)) — `(□ ⊕ m²) ⊗ φ = Ω`.

## Определение

```
□ = (Id : c²) ⊗ ∂² : ∂t² ⊖ ∇²
```

Дискретная форма (такт **n**):

```
□_d φ(n) = (φ(n⊕Id) ⊖ D(φ(n)) ⊕ φ(n⊖Id)) : (c²⊗τ²) ⊖ ∇²φ(n)
```

## Физический смысл

**□** — волновой оператор. `□φ = Ω`: возмущение распространяется без потерь. Вибрация элемента **𝕌**, не затухающая в сети.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Непрерывный** | `□ = (Id:c²)⊗∂²:∂t² ⊖ ∇²` |
| **Дискретный** | `□_d φ(n) = (φ(n+1)−D(φ(n))+φ(n−1)):(c²τ²) − ∇²φ(n)` |
| **Свободная волна** | `□φ = Ω` |
| **С массой** | `(□ ⊕ m²)⊗φ = Ω` |
