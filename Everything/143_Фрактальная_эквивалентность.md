# Фрактальная эквивалентность

Родитель: Хребет (`Dⁿ(Id)`) ([06](06_Степени_двойки.md)), Соленоид ([23](23_Соленоид.md))

## Определение

Для любого масштаба `k ∈ ℤ`:

```
F(x) ≅ F(Dᵏ(x))
```

где `≅` — структурное подобие с точностью до оператора масштабирования `Dᵏ`.

## Физический смысл

Законы `𝕌` инвариантны относительно смены глубины рекурсии. Структура на уровне `n` изоморфна структуре на уровне `n+k` с точностью до метрики `Dᵏ(Id)`. Это свойство обеспечивает универсальность физических констант и математических структур на всех наблюдаемых масштабах. Эквивалентность нарушается только при `|Δ| > 1` (фазовые переходы).

## Полный набор операций

| Операция | Формула |
|---|---|
| **Самоподобие** | `F(x) ≅ F(Dᵏ(x))` |
| **Масштабирование** | `x → Dᵏ(x) ⟹ метрика → Dᵏ(Id)` |
| **Инвариант** | `Δ(x) = Δ(Dᵏ(x))` |
| **Нарушение** | `|Δ| > 1` ⟹ эквивалентность рушится |
