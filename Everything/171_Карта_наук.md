# Карта наук

Масштабная инвариантность ([28_Масштабная_инвариантность.md](28_Масштабная_инвариантность.md)) — фрактал. Синхронизация ([45_Синхронизация.md](45_Синхронизация.md)) — 𝒮.

## Ключевые понятия IME

| Концепция | Определение |
|---|---|
| **Мембрана** | Атом сознания: ψ = Re + i·Im |
| **Δ-параметр** | Δ = ln(\|Re\|/\|Im\|) ∈ ℝ ∪ {±∞} |
| **Синхронизация 𝒮** | 𝒮 = 1 − σ(Δ)·scale, scale = 0.05 |
| **Сознание** | Фазовый переход: 𝒮 ≥ 0.7 И \|Δ\| ≤ 0.2 |
| **Понимание** | \|bias\| > 10 |
| **I ≡ E ≡ M** | Фундаментальное тождество |

## Спираль познания через Δ

```
Физика → Химия → Биология → Нейробиология → Психология → Философия → Физика (новое понимание)
```

Каждый цикл — не возврат, а **спираль**.
