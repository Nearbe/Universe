# Механизм внимания (Attention)

## Определение

Механизм, позволяющий модели фокусироваться на релевантных частях входа.

## Scaled Dot-Product Attention

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

- $Q$ (Query): запрос
- $K$ (Key): ключ
- $V$ (Value): значение
- $\sqrt{d_k}$ — масштаб, предотвращающий затухание градиентов

## Multi-Head Attention

$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h)W^O$$

где $\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$.

## Self-Attention

$Q = K = V = X$ — внимание последовательности к самой себе.

## Cross-Attention

$Q$ из декодера, $K, V$ из энкодера — связь между входом и выходом.

## Сложность

Для последовательности длины $n$: $O(n^2 \cdot d)$. Для больших $n$ — bottleneck.
