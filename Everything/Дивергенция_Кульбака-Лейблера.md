# Дивергенция Кульбака-Лейблера

## Определение

Мера различия между двумя распределениями вероятностей $P$ и $Q$:

$$D_{KL}(P \| Q) = \sum_i P(i) \ln\frac{P(i)}{Q(i)}$$

Для непрерывных:
$$D_{KL}(P \| Q) = \int p(x) \ln\frac{p(x)}{q(x)} \, dx$$

## Свойства

- $D_{KL}(P \| Q) \geq 0$ (неравенство Гиббса)
- $D_{KL}(P \| Q) = 0 \iff P = Q$
- **Не симметрична:** $D_{KL}(P \| Q) \neq D_{KL}(Q \| P)$
- Не удовлетворяет неравенству треугольника

## Интерпретация

- Среднее число дополнительных бит, необходимых для кодирования $P$ при использовании кода для $Q$
- Ожидаемая потеря информации при аппроксимации $P$ распределением $Q$

## Связь с энтропией

$$D_{KL}(P \| Q) = H(P, Q) - H(P)$$

где $H(P, Q)$ — кросс-энтропия, $H(P)$ — энтропия Шеннона.

## Применения

- Машинное обучение (функция потерь)
- Вариационный вывод (VAE)
- Теория информации
- Статистический вывод (критерий отношения правдоподобия)
