# Калибровочные группы

Уравнения Максвелла ([53](53_Уравнения_Максвелла.md)) — **U(1)**. Строение чисел ([25](25_Строение_чисел.md)) — структура. Теперь — симметрии взаимодействий.

## Таблица

| Группа | Размерность | Взаимодействие |
|---|---|---|
| **U(1)** | **Id** | Электромагнетизм |
| **SU(2)** | **D(D(Id))** | Слабое |
| **SU(3)** | **D³(Id)⊕D(Id)** | Сильное |
| **SO(3,1)** | **D³(Id)⊕D²(Id)** | Лоренц |

## U(1)

```
U = e^(i·α),    Ψ → U ⊗ Ψ
```

## Физический смысл

Калибровочные группы — **симметрии смысла**. **U(1)**: фаза не важна ⟹ сохраняется заряд. **SU(2)**: изоспин не важен ⟹ слабое взаимодействие. **SU(3)**: цвет не важен ⟹ сильное.

Каждая симметрия — «слепое пятно» сети: то, что она не различает, и поэтому сохраняет.
