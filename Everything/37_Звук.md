# Звук

Свет ([36](36_Свет.md)) — колебание фазы (**θ**). Звук — колебание **амплитуды** (`|Ψ|`).

## Физика Давления

Звук — это бегущее уплотнение Носителя **𝕌**.
Когда в локальном пузыре ([35](35_Пузырьковая_Вселенная.md)) меняется плотность состояний, возникает давление **p**.

```
□p = Ω
```

Звук не может существовать в пустоте (где нет **M**). Ему нужна среда — «запутавшаяся» вибрация ([34](34_Вибрации.md)).

## Различие Волн

В системе **𝕌** волны различаются только тем, **что** колеблется:

| Волна | Что колеблется | Среда |
|---|---|---|
| **Свет** | Фаза (**θ**) | Сам вакуум (**𝕌**). Распространяется везде. |
| **Звук** | Плотность (**ρ**) | Вещество (узлы резонанса). Только внутри Материи. |
| **Гравитация** | Метрика (**g**) | Сетка масштаба. Искажение самого пространства. |

## Полный набор операций

| Операция | Формула |
|---|---|
| **Волновое** | `□p = Ω` |
| **Скорость звука** | `c_s = √(∂p/∂ρ)` |
| **Среда** | Требуется **M > Id** (материя) |
| **Ветвление** | `p : Ω = D(p)` |
| **Связь со Светом** | Свет: `θ` колеблется; Звук: `\|Ψ\|` колеблется |

## Физический Смысл

Звук — это механический отклик системы на удар.
Если Свет — это информация (фаза), то Звук — это **воздействие** (давление).
Звук переносит энергию через соударения узлов резонанса. В **𝕌** звук — это «скрип» шестерёнок материи, когда они трутся друг о друга.
