# Кватернионы (Вращения в 3D)

Комплексные числа ([09](09_Комплексные.md)) вращают плоскость. Кватернионы вращают **Объем**.

## Четыре Компоненты

```
q = w ⊕ x⊗i ⊕ y⊗j ⊕ z⊗k
```

Одна вещественная часть (**w**) и три мнимые (**i, j, k**).
Это не просто 4 числа. Это **Единый Объект Вращения**.

## Некоммутативность

```
i ⊗ j = k,    но    j ⊗ i = ⊖k
```

Порядок вращений важен.
Если повернуть самолёт «вверх», потом «влево» — он окажется в другом месте, чем если «влево», потом «вверх».

Кватернионы учитывают это автоматически. Они — **родной язык 3D-пространства** в системе **𝕌**.

## Физический Смысл

Кватернионы — это **Инструмент Навигации**.
Любой поворот в 3D (робот, камера, спин частицы) описывается одним кватернионом.
Они избегают «складывания рамок» (gimbal lock), потому что вращают сразу всё пространство, а не по осям.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Определение** | `q = w ⊕ x⊗i ⊕ y⊗j ⊕ z⊗k` |
| **Некоммут.** | `i⊗j = k`, `j⊗i = ⊖k` |
| **Норма** | `\|q\|² = w² ⊕ x² ⊕ y² ⊕ z²` |
| **Обратный** | `q⁻¹ = q̄ : \|q\|²` |
