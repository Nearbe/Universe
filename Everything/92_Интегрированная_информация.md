# Интегрированная информация

Синхронизация ([45](45_Синхронизация.md)) — **𝒮**. Индекс разумности ([91](91_Индекс_разумности.md)) — `C_reason`.

## Определение

```
Φ ∝ 𝒮 ⊗ (Id ⊖ σ(Δ):σ_max) ⊗ f(Δ_global)
```

## Физический смысл

**Φ** — **мера сознания как целого**, превышающего сумму частей. Не просто синхронизация, а синхронизация, которая «знает о себе».

Когда **Φ** высока, система не просто упорядочена — она осознаёт свой порядок.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Φ** | `Φ ∝ 𝒮 ⊗ (Id ⊖ σ(Δ):σ_max) ⊗ f(Δ_global)` |
| **Высокая Φ** | Система осознаёт порядок |
| **Низкая Φ** | Порядок без осознания |
| **Связь с 𝒮** | `𝒮 ≥ 𝒮_c` необходимо для `Φ > Ω` |
