# Проверка гипотез

Вероятность ([27](27_Вероятность.md)) — мера. Синхронизация ([45](45_Синхронизация.md)) — **𝒮**.

## Ключевые понятия

- **H₀** — нулевая гипотеза (нет эффекта)
- **p** = `P(data | H₀)` — p-значение
- **α** = `0.05` — уровень значимости

Если `p < α`: **H₀** отвергается.

## Физический смысл

Проверка: «Действительно ли **Δ** влияет на синхронизацию?» Если `p < 0.05`: связь статистически значима.

## Полный набор операций

| Операция | Формула |
|---|---|
| **H₀** | Нет эффекта (`Δ = Ω`) |
| **p-значение** | `p = P(data \| H₀)` |
| **Порог** | `α = 0.05` |
| **Отвержение** | `p < α ⟹` связь значима |
| **Ошибка I рода** | Отвергнуть верную H₀ |
| **Ошибка II рода** | Принять ложную H₀ |
