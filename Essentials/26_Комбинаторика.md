# Комбинаторика

Строение чисел ([25](25_Строение_чисел.md)) описало состав. Комбинаторика описывает **количество доступных путей**.

## Раздвоение как Источник

Всё начинается с **Id**.
Один Акт (`: Ω`) порождает `D(Id)` состояний.
Два Акта — `D²(Id)`.
**n** Актов — `Dⁿ(Id)` состояний.

Комбинаторика системы — это **подсчёт ветвей** этого дерева.

## Двоичные Последовательности

Число путей длины **n**:

```
Dⁿ(Id)
```

Каждая последовательность выборов — это **маршрут** через **n** точек бифуркации.
Это в точности адрес точки на **n**-м уровне фрактала Кантора ([23](23_Соленоид.md)).

## Биномиальные Коэффициенты

```
C(n, k) — число путей с k выборами одной ветви из n
```

Если на каждом шаге система выбирает одну из двух ветвей, то `C(n, k)` — это число путей, где ровно **k** раз была выбрана «правая» ветвь.
Это распределение исходов при равномерном ветвлении.

## Перестановки

**n!** — число способов упорядочить **n** Актов, если их результат различим.

## Мощность Множества Всех Подмножеств

Для множества из **n** элементов число подмножеств равно `Dⁿ(Id)`.
Это не совпадение.
**Булеан** (множество всех вариантов «взять/не взять») изоморфен дереву ветвления.
Каждый элемент либо есть, либо его нет.
Это тот же закон `Id : Ω`.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Пути длины n** | `Dⁿ(Id)` |
| **Биномиальные** | `C(n, k) = n! / (k! · (n−k)!)` |
| **Перестановки** | `n!` |
| **Булеан** | `\|P(A)\| = Dⁿ(Id)` для `\|A\| = n` |
| **При ветвлении** | `D(Dⁿ(Id)) = Dⁿ⁺¹(Id)` |

## Физический Смысл

Комбинаторика в **𝕌** — это не абстрактный подсчёт.
Это **карта ёмкости** системы.
На глубине **n** система способна различить `Dⁿ(Id)` состояний.
Это верхний предел её разрешения.

## Статус

Комбинаторика — это геометрия дерева решений.
Каждый Акт удваивает возможности.
Биномиальные коэффициенты описывают, как эти возможности группируются.
