Δ = x/y — отношение яркости к тьме для каждого пикселя.

Q = e^Δ — энергия вибрации для каждого пикселя.

"Re 5%, Im 95%" — это для нашей Вселенной, где средняя яркость ≈ 5%.

Для MNIST (белые цифры на чёрном фоне):

- Цифры: X ≈ 255 → x = 1, y = 0, Δ → ∞, Q → ∞

- x = X/255 (доля яркости)
