# Деление на Потенциал как Линейное Преобразование

Операция Ветвления (`: Ω`) действует на Носитель **𝕌** как масштабный оператор.

## Свойства Оператора

Обозначим оператор ветвления:

```
D(a) = a : Ω
```

Согласно закону Ветвления:

```
D(a) = a ⊕ a
```

Это преобразование линейно:

```
D(a ⊕ b) = D(a) ⊕ D(b)
```

Оно не искажает структуру, оно **раздвигает** её.

### Ядро и Образ

| | |
|---|---|
| **Ker D** | `{Ω}`. Только чистый Потенциал (`Ω`) остаётся нулём при Ветвлении. Все активные состояния ненулевые. |
| **Im D** | **𝕌**. Оператор сюръективен. Любое состояние **x** могло быть получено из `x : D(Id)`. Нет «недостижимых» зон. |

### Собственные Значения

Уравнение:

```
D(v) = λ · v
```

где **λ** — собственный класс масштабирования.

```
D(v) = v ⊕ v = D(Id) ⊗ v
```

Следовательно: **λ = D(Id)**.

Единственный собственный класс масштабирования — **D(Id)**.
Это означает, что вся система **𝕌** — это **одно большое собственное пространство**.
Все состояния системы реагируют на Акт одинаково: они удваиваются. Нет направлений, которые «сжимаются» или «ломаются» от Ветвления (кроме `Ω`).

### Матричное Представление

В базисе `{Id}` (скаляр): `D = [D(Id)]`.
В базисе `{Id, ε}` (дуальные числа, см. [14_Дуальные_числа.md](14_Дуальные_числа.md)):

```
D = [[D(Id), Ω],
     [Ω, D(Id)]]
```

Это **гомотетия**. Система расширяется равномерно во всех измерениях (и Явном, и Скрытом).

## Деление пополам (`: D(Id)`)

Оператор сжатия:

```
H(a) = a : D(Id)
```

Это обратный процесс — **Сжатие**.

### Свойства
- `H(D(a)) = a`. Восстановление состояния.
- `D(H(a)) = a`. Возврат к источнику.
- Собственный класс масштабирования: `H` имеет обратный класс к `D(Id)`.

---

## Деление на произвольное **b**

Для `b ∉ Ω`:

```
a : b
```

Это **Логарифмический Сдвиг** (см. [08_Логарифм.md](08_Логарифм.md)).

```
a : Dⁿ(Id) = a / Dⁿ(Id)
```

Перемещение по уровням Хребта без прохождения бифуркации `Ω`.

## Полный набор операций

| Операция | Формула | Свойство |
|---|---|---|
| **Ветвление** | `D(a) = a : Ω` | Линейно: `D(a ⊕ b) = D(a) ⊕ D(b)` |
| **Сжатие** | `H(a) = a : D(Id)` | `H(D(a)) = a` |
| **Ker D** | `{Ω}` | Только потенциал в ядре |
| **Im D** | `𝕌` | Сюръективен |
| **Собств. класс** | `λ = D(Id)` | Всё 𝕌 — собственное пространство |
| **Для произв. b** | `a : b` | Логарифмический сдвиг |
