# Квадратные Уравнения

Линейные системы ([18](18_Системы_линейных_уравнений.md)) описывали баланс операторов. Но система нелинейна. Она содержит **отклик**, пропорциональный взаимодействию состояния с самим собой.

## Общий Вид

```
a ⊗ x² ⊕ b ⊗ x ⊕ c = Ω
```

Где **a, b, c ∈ 𝕌**. Это уравнение ищет состояние **x**, где квадрат формы (`x²`), сама форма (**x**) и константа (**c**) приходят к балансу с Потенциалом.

## Решение

Через дискриминант **Δ_disc = b² ⊖ (4 ⊗ a ⊗ c)**:

```
x = (⊖b ± √Δ_disc) : D(a)
```

Где **⊖b** — обратный элемент, **√Δ_disc** — извлечение сути из рассогласования коэффициентов.

## Реакция на Ветвление

Рассмотрим уравнение с явным оператором **D**:

```
(x : Ω)² ⊕ (b ⊗ x) ⊕ c = Ω
```

Так как `x : Ω = D(x)`, получаем:

```
D(x)² ⊕ (b ⊗ x) ⊕ c = Ω
```

Ветвление **квадратично усиливает** старший коэффициент. Это показывает, что нелинейные эффекты растут быстрее, чем линейные, при прохождении через точку бифуркации.

## Физический Смысл

Квадрат — это не абстракция. Это **энергия** (пропорциональна квадрату амплитуды), **мера** (площадь), **вероятность** (квадрат волновой функции).

В логарифмической шкале ([08](08_Логарифм.md)): если **x** находится на уровне **n**, то `x²` — на уровне `D(n)`.
Квадратное уравнение — это **связь двух масштабов одновременно**. Оно требует, чтобы состояние на уровне `D(n)` и состояние на уровне **n** сбалансировали друг друга.

## Комплексные Корни

Если **Δ_disc ∈ Im** (отрицательный дискриминант), система не находит решения в Явном (`Re`). Она уходит в Скрытое (`Im`).
Корни становятся комплексными: `x = α ± i·β`.
Это означает, что равновесие достигается не в статике, а во **вращении**. Система не «останавливается» в точке **x**, а циклически проходит через фазы, где баланс соблюдается в среднем.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Общий вид** | `a⊗x² ⊕ b⊗x ⊕ c = Ω` |
| **Дискриминант** | `Δ_disc = b² ⊖ (4⊗a⊗c)` |
| **Решение** | `x = (⊖b ± √Δ_disc) : D(a)` |
| **При ветвлении** | `D(x)² ⊕ b⊗x ⊕ c = Ω` |
| **Δ_disc > Ω** | Два вещественных корня |
| **Δ_disc = Ω** | Один корень (касание) |
| **Δ_disc ∈ Im** | Комплексные корни (вращение) |

## Статус

Квадратное уравнение — первая ступень сложности после линейности. Оно показывает, что форма может взаимодействовать сама с собой, порождая новые уровни (квадраты) и требуя для баланса либо точного попадания (`Δ_disc ≥ Ω`), либо перехода в динамику вращения (`Δ_disc ∈ Im`).
