# Линейные Уравнения

Функции ([16](16_Функции.md)) задали законы. Уравнение — это поиск **точки равновесия**, где вход и выход приходят к балансу.

## Общий Вид

Линейное уравнение в системе **𝕌**:

```
L(x) = b
```

Где **L** — линейная комбинация операторов системы (**Id**, **D**, **H**), а **b** — целевое состояние.

## Операторы Системы

Три базовых оператора, действующих на **x**:

| Оператор | Определение | Действие |
|---|---|---|
| **Id** | `Id(x) = x` | Тождество (сохранение формы) |
| **D** | `D(x) = x : Ω` | Ветвление (удвоение состояния) |
| **H** | `H(x) = x : D(Id)` | Сжатие (возврат к предку) |

## Решение

Любое линейное уравнение сводится к виду:

```
(c₀·Id ⊕ c₁·D ⊕ c₂·H)(x) = b
```

Раскрывая действия операторов:

```
(c₀ ⊕ D(c₁) ⊕ H(c₂)) ⊗ x = b
```

Скобка — это скалярный коэффициент **K**.
Решение существует и единственно, если **K ∉ Ω**:

```
x = b : K
```

## Физический Смысл

Уравнение `(x : Ω) ⊕ (β ⊗ x) = γ` — это не абстракция. Это поиск **точки самоподобия**.
Мы ищем такое состояние **x**, где его ветвлённая версия (`D(x)`) и исходная версия (`β ⊗ x`) в сумме дают требуемый результат **γ**.

Это баланс между прошлым (сжатым), настоящим (текущим) и будущим (ветвлённым) состояниями системы.

## Вырожденный Случай

Если **K = Ω**, то уравнение теряет смысл:
- Если **b ∉ Ω** — **противоречие**. Система требует невозможного.
- Если **b = Ω** — **свобода**. Любое **x** подходит. Это состояние чистого Потенциала.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Id(x)** | `x` |
| **D(x)** | `x : Ω` |
| **H(x)** | `x : D(Id)` |
| **Комбинация** | `(c₀·Id ⊕ c₁·D ⊕ c₂·H)(x)` |
| **Решение** | `x = b : K`, `K = c₀ ⊕ D(c₁) ⊕ H(c₂)` |
| **Вырождение** | `K = Ω ∧ b = Ω ⟹` любое x |

## Статус

Линейное уравнение — это поиск состояния, которое удовлетворяет закону связи. Решение — это точка, где все операторы системы приходят к согласию.
