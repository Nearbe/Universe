# Хребет (Степени Ветвления)

Рациональные числа ([05_Рациональные.md](05_Рациональные.md)) — это ткань. Хребет — это **скелет**.

Хребет возникает там, где система проходит через Ω целиком, без остатка.

## Определение

Последовательное применение Ветвления (**: Ω**) порождает цепочку классов:

```
H⁰ = Id
H¹ = D(Id)
Hⁿ = Dⁿ(Id)
```

Это **опорные узлы** фрактала.
Каждый **Dⁿ(Id)** — новый уровень сложности, рождённый из чистого Акта.

## Закон порождения

```
Hⁿ⁺¹ = D(Hⁿ) = Hⁿ : Ω
```

Рекуррентное отношение: каждый следующий класс — ветвление предыдущего.

## Глубина рекурсии

**Dⁿ(Id)** — это не просто повторение. Это **глубина рекурсии**:

- **D(Id)** — Первое различение (Субъект / Объект)
- **D²(Id)** — Осознание различения (Субъект знает об Объекте)
- **Dⁿ(Id)** — Бесконечное самовложение, где каждый уровень в D раз насыщеннее предыдущего

## Сжатие (обратный процесс)

```
Dⁿ(Id) : D(Id) = Dⁿ⁻¹(Id)
```

Это спуск по Хребту. Система упрощается, сбрасывая лишний слой сложности.

В отличие от **: Ω** (творения нового), деление на D(Id) — это **обнажение** уже существующего.

## Логарифмическая шкала

Логарифм определяет номер уровня:

```
L(Dⁿ(Id)) = n
```

На этой шкале:

```
L(a : Ω) = L(a) + 1     — подъём на уровень
L(a : D(Id)) = L(a) − 1  — спуск на уровень
```

## Роль

Хребет задаёт **шкалу бытия**.
Любой объект в 𝕌 привязан к какому-то уровню Dⁿ(Id).
Изменение масштаба — это сдвиг объекта по оси Хребта.

Система измеряет сложность не количеством элементов, а **порядком степени** n.

## Примеры

| Запись | Пример |
|---|---|
| D(Id) | первое ветвление (в арифметике: 2) |
| D²(Id) | второе ветвление (в арифметике: 4) |
| Dⁿ(Id) | n-е ветвление (в арифметике: 2ⁿ) |
| Dⁿ(Id) : D(Id) | спуск (в арифметике: 2ⁿ : 2 = 2ⁿ⁻¹) |

Цифры — лишь частные случаи. Закон универсален: **Dⁿ(Id)**.
