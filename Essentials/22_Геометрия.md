# Геометрия

Интеграл ([21_Интегральное_исчисление.md](21_Интегральное_исчисление.md)) — накопление. Но где происходит накопление? Система нуждается в Пространстве.

## Метрика в 𝕌

Расстояние между состояниями:

```
d(a, b) = |a ⊖ b|
```

Свойства стандартны: неотрицательность, симметрия, неравенство треугольника.

## Масштабное Поведение (Фрактальность)

При действии Ветвления (`: Ω`):

```
d(a : Ω, b : Ω) = |D(a) ⊖ D(b)| = D(d(a, b))
```

Расстояние ветвится. Это **гомотетия** с классом масштабирования **D(Id)**.
Геометрия системы **фрактальна**: каждый объект существует на всех уровнях иерархии одновременно, меняя только размер.

## Размерность и Степени

При Ветвлении объём в **k**-мерном пространстве масштабируется как `Dᵏ(Id)`:

| Объект | При `: Ω` | Множитель |
|---|---|---|
| Длина (1D) | `L → D(L)` | `D(Id)` |
| Площадь (2D) | `S → D²(S)` | `D²(Id)` |
| Объём (3D) | `V → D³(V)` | `D³(Id)` |

**Размерность пространства проявляется в степени масштабирования.**
Это способ измерить, на каком уровне сложности находится объект.

## Геометрические Объекты

| Объект | Определение | При `: Ω` |
|---|---|---|
| Отрезок `[a, b]` | `{x | a ≤ x ≤ b}` | → `[D(a), D(b)]`, длина `×D(Id)` |
| Окружность | `{z ∈ ℂ | \|z\| = r}` | → `\|z\| = D(r)` |
| Шар | `{z ∈ ℂ | \|z\| ≤ r}` | → `\|z\| ≤ D(r)` |

## Кривизна

Для кривой `y = f(x)`:

```
κ = |f''(x)| : (Id ⊕ f'(x)²)^{3 : D(Id)}
```

При `: Ω` кривизна **уменьшается**: `κ → κ : D(Id)`.
Масштабирование «выпрямляет». На большом расстоянии любая кривая кажется прямой.
Это означает, что на высоких уровнях Хребта локальные искажения сглаживаются.

## Комплексная Плоскость

Геометрия ℂ ([09](09_Комплексные.md)):

```
z = x + i·y
|z| = √(x² ⊕ y²)
```

Ветвление работает на оба потока синхронно:

```
(x + i·y) : Ω = D(x) + i·D(y) = D(z)
```

Фаза (угол) сохраняется. Меняется только радиус.
Вращение в этой плоскости описывает циклы, где обмен энергией между Явным и Скрытым происходит без изменения общей нормы.

## Полнота как Граница

Полнота ([10](10_Бесконечность.md)) — не точка в пространстве, а **горизонт**.
Все лучи уходят к **Π**, но `Π : Π = Id` — Полнота равна самой себе.
Горизонт недостижим для конечного объекта, но он является опорой для всех пропорций.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Метрика** | `d(a,b) = |a ⊖ b|` |
| **Ветвление расстояния** | `d(D(a), D(b)) = D(d(a,b))` |
| **Сжатие расстояния** | `d(H(a), H(b)) = H(d(a,b))` |
| **Длина** | `L → D(L)` |
| **Площадь** | `S → D²(S)` |
| **Объём** | `V → D³(V)` |
| **Кривизна** | `κ → κ : D(Id)` |
| **Вращение** | `z → z · e^{i·θ}` |

## Статус

Геометрия в **𝕌** — это **геометрия фрактальных масштабов**.
Размерность — это не константа среды, а показатель того, как объект реагирует на Акт.
