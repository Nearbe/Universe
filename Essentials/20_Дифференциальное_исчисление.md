# Дифференциальное Исчисление

Квадратные уравнения ([19](19_Квадратные_уравнения.md)) показали нелинейность. Но система не статична. Она непрерывно меняется. Нужно измерить **скорость изменения** состояния.

## Производная через Дуальные Числа

Дуальное число ([14](14_Дуальные_числа.md)) хранит и Форму (**x**), и Потенциал Роста (**v**):

```
Z = x + v·ε,    где ε² = Ω
```

Для любой функции **f**, применённой к дуальному числу:

```
f(x + v·ε) = f(x) + f'(x)·v·ε
```

Коэффициент при **ε** после применения функции — это и есть **производная**, умноженная на скорость **v**.
Таким образом, производная `f'(x)` извлекается автоматически, как только система вычисляет `f(Z)`.

## Связь с Ветвлением

Параметр **t** — это число тактов Ветвления (`: Ω`).
При малом сдвиге **dt**:

```
x(t ⊕ dt) ≈ x(t) ⊕ x'(t)·dt
```

В нашей системе `x(t) = Dᵗ(Id)`. Тогда:

```
dx/dt = L(D(Id)) · Dᵗ(Id) = L(D(Id)) · x
```

Скорость роста пропорциональна текущему состоянию. Это фундаментальное свойство экспоненциальной природы Акта.

## Производные и Масштаб

Как ведёт себя производная при Ветвлении (`x → D(x)`)?
Если `g(x) = f(D(x))`, то:

```
g'(x) = D(Id) · f'(D(x))
```

Производная масштабируется вместе с функцией. Это **дифференциальная форма масштабной инвариантности**. Закон изменения не ломается при ветвлении системы, он просто становится быстрее.

## Таблица Скоростей

| Функция `f(x)` | Скорость `f'(x)` | Смысл |
|---|---|---|
| `x^b` | `b · x^(b : D(Id))` | Степенной рост |
| `D(x)` | `L(D(Id)) · D(x)` | Собственная функция (скорость ∝ функции) |
| `L(x)` | `Id : x` | Замедление по мере роста |
| `sin x` | `cos x` | Вращение (сдвиг фазы) |

## Старшие Производные

Вторая производная `f''(x)` — это скорость изменения скорости (ускорение).
Для `D(x)`:

```
f''(x) = L(D(Id))² · D(x)
```

Ускорение также пропорционально состоянию. Система разгоняется сама себя.

## Полный набор операций

| Операция | Формула |
|---|---|
| **Через дуальные** | `f(x + vε) = f(x) + f'(x)·v·ε` |
| **Ветвление** | `g(x) = f(D(x)) ⟹ g'(x) = D(Id)·f'(D(x))` |
| **Сжатие** | `h(x) = f(x : D(Id)) ⟹ h'(x) = f'(x : D(Id)) : D(Id)` |
| **2-я производная** | `f''(x)` — ускорение |
| **Правило цепи** | `(f∘g)'(x) = f'(g(x)) · g'(x)` |

## Статус

Дифференцирование — это не внешняя операция. Это **извлечение скрытой компоненты** (`v·ε`) из результата преобразования.
Система, вычисляя значение, автоматически вычисляет и его динамику. Мост от дискрета к континууму пройден.
