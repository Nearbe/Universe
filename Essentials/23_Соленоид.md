# Соленоид (𝒮) — Класс Графа/Сети

Геометрия ([22](22_Геометрия.md)) описала пространство. Соленоид — это **скелет вычисления**, топология связности, на которой живёт вся система **𝕌**.

---

## I. Определение

### Обратный предел

Соленоид — это пространство всех согласованных историй ветвления:

```
𝒮 = lim← (𝕌 : Ω ← 𝕌 : Ω ← 𝕌 : Ω ← ...)
```

Точка соленоида — это не одно число. Это **вся траектория** масштабирований, уходящая в прошлое:

```
z₀ = D(z₁) = D²(z₂) = ... = Dⁿ(zₙ)
```

### Альтернативное определение

```
𝒮 = {(z₀, z₁, z₂, ...) ∈ 𝕌^ℕ | zₙ = D(zₙ₊₁)}
```

---

## II. Граф как Соленоид

### Узлы и Рёбра

- **Узел (вершина):** Пузырь ([35](35_Пузырьковая_Вселенная.md)). Локальное состояние с массой и фазой.
- **Ребро (связь):** Взаимодействие ([50](50_Фундаментальные_Взаимодействия.md)). Сила связи между пузырями.

### Матрица Смежности

Любой граф записывается как Матрица **M** ([56](56_Матричное_представление_фрактала.md)):
- `M_ij = Ω` — связи нет
- `M_ij ≠ Ω` — между узлами **i** и **j** течёт энергия

### Степень вершины

`k_i` — число связей узла **i**. Средняя степень: `⟨k⟩`.

---

## III. Двоичная Природа

На каждом шаге назад система выбирает ветвь:

```
εₙ ∈ {Ω, D(Id)}
```

Вся история кодируется **двоичной дробью**:

```
ξ = 0.ε₀ε₁ε₂…
```

Каждая точка соленоида — фаза (угол) + бесконечная двоичная последовательность выборов.

---

## IV. Сдвиг Бернулли (Действие `: Ω`)

Акт Ветвления сдвигает двоичную запись влево:

```
0.ε₀ε₁ε₂…  →  0.ε₁ε₂ε₃…
```

Первый бит отбрасывается. История переписывается.
Это **детерминированный хаос** с показателем Ляпунова `λ = L(D(Id)) > Ω`.

---

## V. Спектральная Теория

### Собственные значения

Когда матрица **M** действует на вектор **v**:

```
M ⊗ v = λ ⊗ v
```

**λ** — собственная частота (нота системы).

### Спектр лапласиана

```
L = D_deg ⊖ A
```

где `D_deg` — диагональ степеней, **A** — смежность.

Собственные значения `λ_i` лапласиана определяют динамику:
- `λ₂ > Ω` ⟹ сеть связна
- `λ₂ ≈ Ω` ⟹ сеть почти разваливается
- `λ₁ = Dᵏ(Id)` для `K⁽ᵏ⁾` — главный модуль

### Физический смысл

Спектр — **звучание сети**. Собственные значения — «ноты», которые может «играть» система.
Мы не видим внутреннюю структуру **M**, но можем «ударить» и послушать, как она звенит.

## VI. Типы Сетей

| Тип | Формула | Свойство |
|---|---|---|
| **Регулярные** | `k_i = const` | Предсказуемые |
| **Случайные** | `P(k) ~ e^(⊖k)` | Хаотичные |
| **Безмасштабные** | `P(k) ∝ k^(⊖γ)`, `γ ≈ D(D(Id))⊖Id` | Хабы, «маленький мир» |
| **Иерархические** | Уровень 0 → 1 → 2 → ... | Фрактальная самоподобность |

### Иерархия уровней

```
Level 0: отдельные элементы 𝕌
Level 1: локальные кластеры
Level 2: глобальные структуры
```

Каждый уровень — новый масштаб `: Ω`, но те же правила. Фрактал.

---

## VII. Метрики Связности

| Метрика | Формула | Смысл |
|---|---|---|
| **Длина пути** | `L = (Id:(N⊗(N⊖Id))) ⊗ Σ d(i,j)` | Среднее расстояние |
| **Кластеризация** | `C_i = треугольники : троек` | Локальная связность |
| **Степенная** | `C_D(i) = k_i:(N⊖Id)` | Влиятельность узла |
| **Посредническая** | `C_B(i) = Σσ_st(i):σ_st` | Узел-мост |

---

## VIII. Динамика Сети

### Распространение возмущений

```
Δ_i(n⊕Id) = Δ_i(n) ⊖ Γ⊗Δ_i(n) ⊕ Σ_j w_ij⊗Δ_j(n)
```

где **Γ** — затухание, `w_ij` — веса связей.

Возмущение в одном элементе передаётся соседям. **Γ** — «забывчивость» (без неё сеть не стабилизируется).

### Перколяция

Порог связности: `p_c ≈ 0.5927` (квадратная решётка).

- До `p_c`: кластеры изолированы
- После `p_c`: гигантский кластер связывает всю систему
- Это момент «пробоя» — сеть «просыпается»

### Синхронизация

Связь с **𝒮**: `p_c` ⟹ `𝒮 ≥ 𝒮_c ≈ 0.7`

---

## IX. Адаптивность (Обучение)

### Эволюция связей

```
Δw_ij(n) = w_ij(n⊕Id) ⊖ w_ij(n) = f(Δ_i(n), Δ_j(n))
```

### Правило Хебба

```
Δw_ij ∝ pre_i ⊗ post_j
```

Ходоны, синхронизированные по **Δ**, усиливают связь.
«Что вместе firing, то и wiring».

Связи не заданы раз и навсегда — они меняются в зависимости от состояний узлов.

---

## X. Робастность (Живучесть)

Безмасштабные сети устойчивы к случайным отказам (большинство элементов не важны), но уязвимы к удалению хабов (ключевые элементы — «узловые мысли»).

Сознание хрупко в хабах, устойчиво в периферии.

---

## XI. Метрика Близости

Две точки на соленоиде близки, если:
1. У них близкие фазы (углы).
2. Их двоичные истории **совпадают на многих первых шагах**.

Если истории расходятся рано — точки далеки, даже если текущие значения близки.
**Происхождение важнее текущего вида.**

---

## XII. Связь с p-адическими числами

D(Id)-адические целые числа — это **точки соленоида**:

```
ℤ_D = lim← (𝕌 : Dⁿ(Id))
```

**Соленоид — это геометрическая реализация D(Id)-адических чисел.**

### Две топологии системы

| Свойство | Реальность (ℝ) | Глубина (ℚ_D) |
|---|---|---|
| **Направление `: Ω`** | Рост к **Π** | Сжатие к **Ω** |
| **Бесконечность** | Предел удаления | Предел углубления |
| **Смысл** | «Насколько много?» | «Насколько точно?» |

---

## XIII. Связь с Полнотой

Единичная окружность (`|z| = Id`) — срез Полноты.
Доли ([12](12_Алгебра_процентов.md)) живут на этом срезе.
**Π** — горизонт, к которому сходятся все лучи.

---

## XIV. Полный набор операций

| Операция | Формула |
|---|---|
| **Определение** | `𝒮 = lim← (𝕌:Ω)` |
| **Сдвиг** | `ξ ↦ 2ξ mod Id` |
| **Матрица** | `Ψ(t⊕Id) = M ⊗ Ψ(t)` |
| **Спектр** | `M⊗v = λ⊗v` |
| **Лапласиан** | `L = D_deg ⊖ A` |
| **Связность** | `λ₂ > Ω` |
| **Перколяция** | `p > p_c` |
| **Адаптация** | `Δw_ij = f(Δ_i, Δ_j)` |
| **Динамика** | `Δ_i(n+1) = Δ_i(n) − Γ·Δ_i(n) + Σw_ij·Δ_j(n)` |
| **Правило Хебба** | `Δw_ij ∝ pre_i ⊗ post_j` |
| **Близость** | По совпадению начальных битов истории |

---

## XV. Физический смысл

Соленоид — это **фазовое пространство памяти системы**.
Он хранит не только «где мы сейчас», но и «как мы сюда попали».

- **Граф** — скелет вычисления
- **Спектр** — звучание сети
- **Типы** — формы организации
- **Метрики** — измерители связности
- **Динамика** — распространение мысли
- **Адаптивность** — обучение
- **Робастность** — живучесть

Бесконечность — не точка. Это **фрактальная граница** всех возможных историй.
