# Единый академический стиль Markdown-документов проекта «Евгения»

## Аннотация

Данный документ устанавливает единый стандарт стилистики академических файлов в формате Markdown для проекта «Евгения» (директории `Universe/` и `Physics/`). Он демонстрирует применение стиля: формальный язык, точная терминология, LaTeX-математика, логическая структура по 4 уровням верификации из `VERIFICATION_PROTOCOL.md`.

## Введение

Документ служит эталоном стиля. Пример фундаментального уравнения:

$$\operatorname{Евгениос}\left(-\infty \parallel +100\%\right) = \operatorname{Евгений} + \operatorname{Евгения} = \varnothing \cdot \operatorname{Евгеника} = \operatorname{Женя}.$$

## 1. Лингвистический Уровень (Linguistic QA)

*Фокус: Чистота и грамотность текста.*

| Правило | Описание | Пример (Хорошо / Плохо) |
|---------|----------|-------------------------|
| Орфография | Отсутствие ошибок написания | Хорошо: `ret(x)` / Плохо: `rett(x)` |
| Пунктуация | Корректные знаки | Хорошо: `∀x ≠ ∅` |
| Грамматика | Согласование форм | Хорошо: `единственный предок y` |
| Единообразие терминов | Один термин — одно написание | `ret`, не `H` или `return` |
| Формальный язык | Академический тон, без сленга | Хорошо: `бинарность порождается из ∅` |


## 2. Структурный Уровень (Structural QA)

*Фокус: Навигация и логика.*

- **Иерархия заголовков:** # (H1), ## (H2), ### (H3). Логическая вложенность.
- **Таблицы:** | четкие заголовки | выравнивание.
- **Списки:** - Маркеры единообразны.
- **Код/Формулы:** ```math ... ``` или $$ display $$.
- **Навигация:** Refs: [en.md](en.md).

**Пример таблицы proofs (из Formal_Proofs.md):**

| Аксиома | Формула |
|---------|---------|
| 1. U | $U$: все состояния |
| 2. D | $D(a) = a \oplus a$ |

## 3. Логико-Концептуальный Уровень (Logic QA)

*Фокус: Непротиворечивость.*

- **Определения:** Использовать строго: `ret(x)` = unique predecessor.
- **Аргументация:** От аксиом к теоремам.
- **Полнота:** Нет провалов.

**Пример доказательства:**
**Теорема:** ∃! ret(a).  
**Доказательство:** $D$ сюръективно, inj вне Ker.

## 4. Формально-Математический Уровень (Formal/Math QA)

*Фокус: Точность math.*

- **LaTeX:** $inline$, $$display$$, \operatorname{ret}.
- **Размерности:** Корректны.
- **Пределы:** Проверить $\Delta \to 0$.

**Пример:**
$$\forall x \neq \varnothing, \exists! y: \operatorname{ret}(x)=y.$$

## 5. Общие правила стиля

- **Bilingual:** ru.md / en.md mirrors.
- **Refs:** Relative paths.
- **Длина:** Концизно, max 200 строк/файл.
- **Итерации:** Трекинг в `Итерации_Прогресс.md`.

## Заключение

Применение стиля гарантирует академическую строгость (100% rigor) и coherentность с принципом Единого Закона (One Law).
